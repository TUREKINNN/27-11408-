# 📏 11408 知识库笔记风格规范（v1.0）

> **本规范是所有笔记、题库的"宪法"**，所有生成内容必须严格遵守。

---

## 一、YAML Frontmatter（强制）

每个 `.md` 文件**首部必须**包含以下元数据：

```yaml
---
title: "文件标题"
date: 2026-06-21
tags: [标签1, 标签2, ...]
科目: [数学一/408/英语一]
章节: [章节名]
来源: [讲义/教材/题库]
难度: [基础/中等/拔高]
---
```

---

## 二、标题层级规范

- `# ` 一级：文件主标题（仅 1 个）
- `## ` 二级：节（如：一、选择题）
- `### ` 三级：题号或子节（如：### 第1题）
- `#### ` 四级：极少使用

---

## 三、数学公式铁律

> 1. **行内公式**：`$...$`（如：`$f(x) = x^2$`）
> 2. **块级公式**：`$$...$$`（独占段落，前后空行）
> 3. **中文与公式之间必须留空格**（如：`当 $x \to 0$ 时`）
> 4. 禁止使用 Word/HTML 公式；禁止使用无法渲染的 LaTeX 符号
> 5. 分数用 `\frac{a}{b}`；开方用 `\sqrt{a}`；希腊字母 `\alpha, \beta, \ldots`

---

## 四、特殊语法块

| 语法 | 用途 | 示例 |
|---|---|---|
| `> [!note]` | 重要结论、关键知识点 | `> [!note] 重要结论` |
| `> [!warning]` | 易错点、坑点 | `> [!warning] 易错点` |
| `> [!success]` | 题目详细解答 | `> [!success]- 解答步骤` |
| `> [!info]` | 考点分析、知识拓展 | `> [!info] 考点分析` |
| `> [!question]` | 题目（折叠） | `> [!question] 题号` |
| `> [!example]` | 例题 | `> [!example] 例：...` |
| `

## 五、双链（Wiki Link）规范

### 必须双链的位置
1. **每个知识点笔记末尾**的"相关知识点"区
2. **每道题的"考点分析"**中提到的所有知识点
3. **每章笔记的"跨章联系"**区
4. **错题本**中每道题对应原题位置

### 双链格式
- 基础：`文件名` 或 `文件名`
- 章节定位：`01_函数极限连续#极限的定义`
- 多链并列：`[[02_408/03_计算机组成原理/01_笔记/03_综合笔记/3.2.1 SRAM和DRAM]]`、`[[02_408/01_数据结构/01_笔记/02_综合笔记/7.4.1-B树]]`、`LICENSE`

### 命名约定
- 笔记：`科目_章节名.md`，如 `数学一_函数极限连续.md`
- 题库：`科目_章节_题型.md`，如 `数学一_函数极限连续_填空题.md`
- 单词：`词性_单词.md`，如 `verb_abandon.md`
- 题目：`Q_科目_章节_序号.md`（如可拆分为独立文件时）

## 六、标签（Tag）体系

### 一级标签（必选）
- `#数学一` 或 `#408` 或 `#英语一`

### 二级标签（必选）
- `#高等数学` / `#线性代数` / `#概率论`
- `#数据结构` / `#计组` / `#操作系统` / `#计网`
- `#阅读` / `#写作` / `#完形` / `#翻译` / `#新题型` / `#英语单词`

### 三级标签（可选）
- 考点：`#极限` `#导数` `#积分` ...
- 难度：`#基础` `#拔高` `#真题`
- 题型：`#填空题` `#选择题` `#解答题`
- 特殊：`#易错点` `#核心结论` `#反例`

---

## 七、图片处理

1. **本地图片**：用 `!图片名.png` 引用，图片存于同目录 `assets/` 子文件夹
2. **外部图片**：尽量转文字描述，几何图形用文字 + ASCII 图示意
3. **图表文字化**：所有图必须在笔记正文里有对应文字描述
4. **公式截图**：必须转 LaTeX 公式
