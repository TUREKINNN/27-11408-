## 🚀 使用方式

### Obsidian 用户（推荐）

1. 下载本仓库
2. 在 Obsidian 中 **「打开本地文件夹」** → 选择 `27考研笔记/`
3. 笔记内置了完整的双向链接，可直接跳转

### 普通浏览

直接在 GitHub 上浏览 `.md` 文件即可，所有笔记均为 Markdown 格式。
