# 写作高频词

> 本分类包含考研英语作文中常用的高级词汇，替换基础词汇提升作文档次。

---

## ✨ 高级形容词

### significant
- **音标**：/sɪɡˈnɪfɪkənt/
- **词性**：adj. 重要的；显著的
- **中文释义**：有重大意义的
- **词根词缀**：sign-（标记）+ -ific（使）+ -ant（形容词后缀）
- **替换词**：important → **significant**
- **例句**：This is a significant improvement.（这是一个显著的进步。）
- **真题例句**：Significant changes have taken place.（已经发生了重大变化。）
- **考研频率**：★★★★★

### essential
- **音标**：/ɪˈsenʃl/
- **词性**：adj. 必要的；本质的
- **中文释义**：不可或缺的
- **词根词缀**：ess-（存在）+ -ent（形容词后缀）+ -ial（形容词后缀）
- **替换词**：necessary → **essential**
- **例句**：Water is essential for life.（水对生命是必要的。）
- **考研频率**：★★★★★

### crucial
- **音标**：/ˈkruːʃl/
- **词性**：adj. 关键的；决定性的
- **中文释义**：极其重要的
- **词根词缀**：cruc-（十字）+ -ial（形容词后缀）
- **替换词**：important → **crucial**
- **例句**：This is a crucial moment.（这是一个关键时刻。）
- **考研频率**：★★★★★

### profound
- **音标**：/prəˈfaʊnd/
- **词性**：adj. 深刻的；深远的
- **中文释义**：意义深远的
- **词根词缀**：pro-（向前）+ found（基础）
- **替换词**：deep → **profound**
- **例句**：The book had a profound impact on me.（这本书对我产生了深远影响。）
- **考研频率**：★★★★

### inevitable
- **音标**：/ɪnˈevɪtəbl/
- **词性**：adj. 不可避免的
- **中文释义**：必然发生的
- **词根词缀**：in-（不）+ evit-（避免）+ -able（能够）
- **替换词**：unavoidable → **inevitable**
- **例句**：Change is inevitable.（变化是不可避免的。）
- **考研频率**：★★★★

## 🔧 高级动词

### demonstrate
- **音标**：/ˈdemənstreɪt/
- **词性**：v. 证明；展示
- **中文释义**：清楚地表明
- **词根词缀**：de-（完全）+ monstr（展示）+ -ate（动词后缀）
- **替换词**：show → **demonstrate**
- **例句**：The results demonstrate the effectiveness.（结果证明了有效性。）
- **考研频率**：★★★★★

### illustrate
- **音标**：/ˈɪləstreɪt/
- **词性**：v. 说明；阐明
- **中文释义**：用例子说明
- **词根词缀**：il-（使）+ lustr（光）+ -ate（动词后缀）
- **替换词**：explain → **illustrate**
- **例句**：This example illustrates the point.（这个例子说明了这一点。）
- **考研频率**：★★★★★

### emphasize
- **音标**：/ˈemfəsaɪz/
- **词性**：v. 强调；着重
- **中文释义**：特别指出
- **词根词缀**：em-（使）+ phas（显示）+ -ize（动词后缀）
- **替换词**：stress → **emphasize**
- **例句**：I want to emphasize the importance.（我想强调重要性。）
- **考研频率**：★★★★★

### advocate
- **音标**：/ˈædvəkeɪt/
- **词性**：v. 提倡；主张
- **中文释义**：公开支持
- **词根词缀**：ad-（向）+ voc（声音）+ -ate（动词后缀）
- **替换词**：support → **advocate**
- **例句**：He advocates for environmental protection.（他提倡环境保护。）
- **考研频率**：★★★★

### implement
- **音标**：/ˈɪmplɪment/
- **词性**：v. 实施；执行
- **中文释义**：使生效
- **词根词缀**：im-（使）+ ple（满）+ -ment（名词后缀）
- **替换词**：carry out → **implement**
- **例句**：We will implement the plan.（我们将实施这个计划。）
- **考研频率**：★★★★★

## 📝 高级名词

### phenomenon
- **音标**：/fəˈnɒmɪnən/
- **词性**：n. 现象
- **中文释义**：可观测的事实或事件
- **词根词缀**：phen-（显示）+ -omenon（名词后缀）
- **替换词**：occurrence → **phenomenon**
- **例句**：This is a common phenomenon.（这是一个普遍现象。）
- **考研频率**：★★★★★

### perspective
- **音标**：/pəˈspektɪv/
- **词性**：n. 观点；视角
- **中文释义**：看待事物的方式
- **词根词缀**：per-（通过）+ spect（看）+ -ive（名词后缀）
- **替换词**：viewpoint → **perspective**
- **例句**：From my perspective, this is correct.（从我的角度看，这是正确的。）
- **考研频率**：★★★★★

### consequence
- **音标**：/ˈkɒnsɪkwəns/
- **词性**：n. 结果；后果
- **中文释义**：某事导致的结果
- **词根词缀**：con-（共同）+ sequ（跟随）+ -ence（名词后缀）
- **替换词**：result → **consequence**
- **例句**：We must face the consequences.（我们必须面对后果。）
- **考研频率**：★★★★★
