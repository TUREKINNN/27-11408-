# 工程与制造

> 本分类包含与工程、制造、机械、建筑、生产相关的词汇。

---

## 🏗️ 工程相关

### engineering
- **音标**：/ˌendʒɪˈnɪərɪŋ/
- **词性**：n. 工程学
- **中文释义**：应用科学原理的学科
- **词根词缀**：engineer-（工程师）+ -ing（名词后缀）
- **例句**：Engineering is a practical field.（工程学是一个实用领域。）
- **近义词**：technology（技术）
- **反义词**：无
- **考研频率**：★★★★

### engineer
- **音标**：/ˌendʒɪˈnɪər/
- **词性**：n. 工程师
- **中文释义**：从事工程工作的人
- **词根词缀**：engineer-（工程师）
- **例句**：He is a software engineer.（他是一名软件工程师。）
- **近义词**：designer（设计师）
- **反义词**：无
- **考研频率**：★★★★

### design
- **音标**：/dɪˈzaɪn/
- **词性**：v./n. 设计
- **中文释义**：规划和创造
- **词根词缀**：de-（出）+ sign（标记）
- **例句**：The bridge was well designed.（这座桥设计得很好。）
- **近义词**：plan（规划）
- **反义词**：无
- **考研频率**：★★★★★

### project
- **音标**：/ˈprɒdʒekt/
- **词性**：n. 项目
- **中文释义**：有特定目标的工作
- **词根词缀**：pro-（向前）+ ject（扔）
- **例句**：The project is on schedule.（项目按计划进行。）
- **近义词**：venture（项目）
- **反义词**：无
- **考研频率**：★★★★★

## 🏭 制造相关

### manufacture
- **音标**：/ˌmænjuˈfæktʃər/
- **词性**：v./n. 制造
- **中文释义**：大规模生产
- **词根词缀**：manu-（手）+ fact（做）+ -ure（名词后缀）
- **例句**：The factory manufactures cars.（这家工厂制造汽车。）
- **近义词**：produce（生产）
- **反义词**：consume（消费）
- **考研频率**：★★★★

### factory
- **音标**：/ˈfæktri/
- **词性**：n. 工厂
- **中文释义**：制造产品的地方
- **词根词缀**：fact-（做）+ -ory（名词后缀）
- **例句**：The factory is large.（工厂很大。）
- **近义词**：plant（工厂）
- **反义词**：无
- **考研频率**：★★★★★

### machine
- **音标**：/məˈʃiːn/
- **词性**：n. 机器
- **中文释义**：执行工作的设备
- **词根词缀**：machine-（机器）
- **例句**：The machine is efficient.（机器效率很高。）
- **近义词**：device（设备）
- **反义词**：无
- **考研频率**：★★★★★

### equipment
- **音标**：/ɪˈkwɪpmənt/
- **词性**：n. 设备
- **中文释义**：用于特定目的的工具
- **词根词缀**：equip-（装备）+ -ment（名词后缀）
- **例句**：The equipment is modern.（设备很现代。）
- **近义词**：apparatus（设备）
- **反义词**：无
- **考研频率**：★★★★

## 🏠 建筑相关

### build
- **音标**：/bɪld/
- **词性**：v. 建造
- **中文释义**：建造建筑物
- **词根词缀**：build-（建造）
- **例句**：They built a new school.（他们建造了一所新学校。）
- **近义词**：construct（建造）
- **反义词**：destroy（破坏）
- **考研频率**：★★★★★

### construction
- **音标**：/kənˈstrʌkʃn/
- **词性**：n. 建造
- **中文释义**：建造的过程
- **词根词缀**：con-（共同）+ struct（建造）+ -ion（名词后缀）
- **例句**：Construction is underway.（建造正在进行中。）
- **近义词**：building（建筑）
- **反义词**：destruction（破坏）
- **考研频率**：★★★★

### architecture
- **音标**：/ˈɑːkɪtektʃər/
- **词性**：n. 建筑学
- **中文释义**：设计建筑物的艺术
- **词根词缀**：archi-（主要）+ tect（建造）+ -ure（名词后缀）
- **例句**：The architecture is beautiful.（建筑很美。）
- **近义词**：design（设计）
- **反义词**：无
- **考研频率**：★★★

### material
- **音标**：/məˈtɪəriəl/
- **词性**：n. 材料
- **中文释义**：用于制造的物质
- **词根词缀**：mater-（物质）+ -ial（名词后缀）
- **例句**：The material is strong.（材料很坚固。）
- **近义词**：substance（物质）
- **反义词**：无
- **考研频率**：★★★★★

## ⚙️ 生产相关

### produce
- **音标**：/prəˈdjuːs/
- **词性**：v. 生产
- **中文释义**：制造产品
- **词根词缀**：pro-（向前）+ duce（引导）
- **例句**：The company produces goods.（公司生产商品。）
- **近义词**：manufacture（制造）
- **反义词**：consume（消费）
- **考研频率**：★★★★★

### production
- **音标**：/prəˈdʌkʃn/
- **词性**：n. 生产
- **中文释义**：制造的过程
- **词根词缀**：pro-（向前）+ duct（引导）+ -ion（名词后缀）
- **例句**：Production has increased.（生产增加了。）
- **近义词**：manufacturing（制造）
- **反义词**：consumption（消费）

- **反义词**：无
- **考研频率**：★★★★★
