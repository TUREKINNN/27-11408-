# 信息技术

> 本分类包含与计算机、网络、数据、信息技术相关的词汇。

---

## 💻 计算机相关

### computer
- **音标**：/kəmˈpjuːtər/
- **词性**：n. 计算机
- **中文释义**：电子计算设备
- **词根词缀**：comput-（计算）+ -er（名词后缀）
- **例句**：Computers have changed our lives.（计算机改变了我们的生活。）
- **近义词**：PC（个人电脑）
- **反义词**：无
- **考研频率**：★★★★★

### software
- **音标**：/ˈsɒftweər/
- **词性**：n. 软件
- **中文释义**：计算机程序
- **词根词缀**：soft-（软）+ ware（物品）
- **例句**：The software is easy to use.（这个软件很容易使用。）
- **近义词**：application（应用程序）
- **反义词**：hardware（硬件）
- **考研频率**：★★★★★

### hardware
- **音标**：/ˈhɑːdweər/
- **词性**：n. 硬件
- **中文释义**：计算机物理设备
- **词根词缀**：hard-（硬）+ ware（物品）
- **例句**：The hardware needs to be upgraded.（硬件需要升级。）
- **近义词**：equipment（设备）
- **反义词**：software（软件）
- **考研频率**：★★★★

### algorithm
- **音标**：/ˈælɡərɪðəm/
- **词性**：n. 算法
- **中文释义**：解决问题的步骤
- **词根词缀**：algorithm-（算法）
- **例句**：The algorithm is very efficient.（这个算法非常高效。）
- **近义词**：procedure（程序）
- **反义词**：无
- **考研频率**：★★★★

## 🌐 网络相关

### internet
- **音标**：/ˈɪntənet/
- **词性**：n. 互联网
- **中文释义**：全球计算机网络
- **词根词缀**：inter-（相互）+ net（网络）
- **例句**：The internet has connected the world.（互联网连接了世界。）
- **近义词**：web（网络）
- **反义词**：无
- **考研频率**：★★★★★

### network
- **音标**：/ˈnetwɜːk/
- **词性**：n. 网络
- **中文释义**：相互连接的系统
- **词根词缀**：net-（网）+ work（工作）
- **例句**：A network of computers shares data.（计算机网络共享数据。）
- **近义词**：system（系统）
- **反义词**：无
- **考研频率**：★★★★★

### website
- **音标**：/ˈwebsaɪt/
- **词性**：n. 网站
- **中文释义**：互联网上的页面集合
- **词根词缀**：web-（网）+ site（地点）
- **例句**：The website provides useful information.（这个网站提供有用信息。）
- **近义词**：site（站点）
- **反义词**：无
- **考研频率**：★★★★

### digital
- **音标**：/ˈdɪdʒɪtl/
- **词性**：adj. 数字的；数码的
- **中文释义**：使用数字技术的
- **词根词缀**：digit-（数字）+ -al（形容词后缀）
- **例句**：We live in a digital age.（我们生活在数字时代。）
- **近义词**：electronic（电子的）
- **反义词**：analog（模拟的）
- **考研频率**：★★★★★

## 📊 数据相关

### data
- **音标**：/ˈdeɪtə/
- **词性**：n. 数据
- **中文释义**：信息的集合
- **词根词缀**：data-（数据）
- **例句**：The data shows a clear trend.（数据显示了明确的趋势。）
- **近义词**：information（信息）
- **反义词**：无
- **考研频率**：★★★★★

### database
- **音标**：/ˈdeɪtəbeɪs/
- **词性**：n. 数据库
- **中文释义**：有组织的数据集合
- **词根词缀**：data-（数据）+ base（基础）
- **例句**：The database stores all the information.（数据库存储所有信息。）
- **近义词**：repository（存储库）
- **反义词**：无
- **考研频率**：★★★★

### information
- **音标**：/ˌɪnfəˈmeɪʃn/
- **词性**：n. 信息
- **中文释义**：有用的数据
- **词根词缀**：in-（进入）+ form（形式）+ -ation（名词后缀）
- **例句**：Information is power.（信息就是力量。）
- **近义词**：data（数据）、knowledge（知识）
- **反义词**：无
- **考研频率**：★★★★★
