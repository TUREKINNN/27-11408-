# 工作与创造

> 本分类包含与工作、创造、生产、设计相关的词汇。

---

## 💼 工作相关

### work
- **音标**：/wɜːk/
- **词性**：n./v. 工作
- **中文释义**：从事劳动
- **词根词缀**：work-（工作）
- **例句**：He works hard every day.（他每天努力工作。）
- **近义词**：labor（劳动）、toil（辛苦工作）
- **反义词**：rest（休息）
- **考研频率**：★★★★★

### job
- **音标**：/dʒɒb/
- **词性**：n. 工作；职业
- **中文释义**：有报酬的工作
- **词根词缀**：job-（工作）
- **例句**：She found a new job.（她找到了一份新工作。）
- **近义词**：position（职位）、occupation（职业）
- **反义词**：unemployment（失业）
- **考研频率**：★★★★★

### career
- **音标**：/kəˈrɪər/
- **词性**：n. 职业；事业
- **中文释义**：长期从事的工作
- **词根词缀**：career-（职业）
- **例句**：She has a successful career.（她事业有成。）
- **近义词**：profession（职业）、occupation（职业）
- **反义词**：hobby（爱好）
- **考研频率**：★★★★★

### employ
- **音标**：/ɪmˈplɔɪ/
- **词性**：v. 雇用
- **中文释义**：给某人工作
- **词根词缀**：em-（使）+ ploy（折叠）
- **例句**：The company employs many people.（这家公司雇用了许多人。）
- **近义词**：hire（雇佣）
- **反义词**：fire（解雇）
- **考研频率**：★★★★

## 🎨 创造相关

### create
- **音标**：/kriˈeɪt/
- **词性**：v. 创造；创建
- **中文释义**：使存在
- **词根词缀**：creat-（创造）+ -e（动词后缀）
- **例句**：Artists create beautiful works.（艺术家创造美丽的作品。）
- **近义词**：produce（生产）、make（制作）
- **反义词**：destroy（破坏）
- **考研频率**：★★★★★

### design
- **音标**：/dɪˈzaɪn/
- **词性**：v./n. 设计
- **中文释义**：规划和创造
- **词根词缀**：de-（出）+ sign（标记）
- **例句**：She designed the building.（她设计了这座建筑。）
- **近义词**：plan（规划）、devise（设计）
- **反义词**：无
- **考研频率**：★★★★★

### invent
- **音标**：/ɪnˈvent/
- **词性**：v. 发明；创造
- **中文释义**：首次创造
- **词根词缀**：in-（进入）+ vent（来）
- **例句**：Edison invented the light bulb.（爱迪生发明了电灯。）
- **近义词**：create（创造）、devise（设计）
- **反义词**：copy（复制）
- **考研频率**：★★★★

### produce
- **音标**：/prəˈdjuːs/
- **词性**：v. 生产；制造
- **中文释义**：制造产品
- **词根词缀**：pro-（向前）+ duce（引导）
- **例句**：The factory produces cars.（这家工厂生产汽车。）
- **近义词**：manufacture（制造）、make（制作）
- **反义词**：consume（消费）
- **考研频率**：★★★★★

## 🔧 生产相关

### manufacture
- **音标**：/ˌmænjuˈfæktʃər/
- **词性**：v./n. 制造；生产
- **中文释义**：大规模生产
- **词根词缀**：manu-（手）+ fact（做）+ -ure（名词后缀）
- **例句**：The company manufactures electronics.（这家公司制造电子产品。）
- **近义词**：produce（生产）、make（制作）
- **反义词**：consume（消费）
- **考研频率**：★★★★

### build
- **音标**：/bɪld/
- **词性**：v. 建造；建立
- **中文释义**：建造建筑物
- **词根词缀**：build-（建造）
- **例句**：They built a new house.（他们建造了一座新房子。）
- **近义词**：construct（建造）、erect（竖立）
- **反义词**：destroy（破坏）
- **考研频率**：★★★★★

### construct
- **音标**：/kənˈstrʌkt/
- **词性**：v. 建造；构造
- **中文释义**：建造建筑物
- **词根词缀**：con-（共同）+ struct（建造）
- **例句**：The bridge was constructed last year.（这座桥是去年建造的。）
- **近义词**：build（建造）、erect（竖立）
- **反义词**：destroy（破坏）
- **考研频率**：★★★★
