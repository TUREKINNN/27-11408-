# 交流与沟通

> 本分类包含与交流、沟通、演讲、写作、讨论相关的词汇。

---

## 🗣️ 口头交流

### speak
- **音标**：/spiːk/
- **词性**：v. 说话；演讲
- **中文释义**：用语言表达
- **词根词缀**：speak-（说话）
- **例句**：She speaks English fluently.（她英语说得很流利。）
- **近义词**：talk（谈话）、communicate（交流）
- **反义词**：listen（听）
- **考研频率**：★★★★★

### talk
- **音标**：/tɔːk/
- **词性**：v./n. 谈话；讨论
- **中文释义**：交谈
- **词根词缀**：talk-（谈话）
- **例句**：Let's talk about the problem.（让我们讨论这个问题。）
- **近义词**：speak（说话）、discuss（讨论）
- **反义词**：listen（听）
- **考研频率**：★★★★★

### discuss
- **音标**：/dɪˈskʌs/
- **词性**：v. 讨论；商讨
- **中文释义**：详细讨论
- **词根词缀**：dis-（分开）+ cuss（敲打）
- **例句**：We need to discuss this issue.（我们需要讨论这个问题。）
- **近义词**：talk about（谈论）、debate（辩论）
- **反义词**：ignore（忽视）
- **考研频率**：★★★★★

### communicate
- **音标**：/kəˈmjuːnɪkeɪt/
- **词性**：v. 交流；沟通
- **中文释义**：传递信息
- **词根词缀**：com-（共同）+ muni（服务）+ -cate（动词后缀）
- **例句**：We need to communicate effectively.（我们需要有效沟通。）
- **近义词**：convey（传达）、express（表达）
- **反义词**：withhold（保留）
- **考研频率**：★★★★★

## ✍️ 书面交流

### write
- **音标**：/raɪt/
- **词性**：v. 写；写作
- **中文释义**：用文字表达
- **词根词缀**：write-（写）
- **例句**：She writes beautifully.（她写得很优美。）
- **近义词**：compose（创作）、author（著述）
- **反义词**：read（读）
- **考研频率**：★★★★★

### read
- **音标**：/riːd/
- **词性**：v. 阅读
- **中文释义**：理解文字
- **词根词缀**：read-（读）
- **例句**：I like to read books.（我喜欢读书。）
- **近义词**：study（学习）、peruse（细读）
- **反义词**：write（写）
- **考研频率**：★★★★★

### publish
- **音标**：/ˈpʌblɪʃ/
- **词性**：v. 出版；发表
- **中文释义**：使公开
- **词根词缀**：publ-（人民）+ -ish（动词后缀）
- **例句**：The book was published last year.（这本书是去年出版的。）
- **近义词**：release（发布）、issue（发行）
- **反义词**：suppress（压制）
- **考研频率**：★★★★

### edit
- **音标**：/ˈedɪt/
- **词性**：v. 编辑；校订
- **中文释义**：修改文字
- **词根词缀**：edit-（编辑）
- **例句**：She edited the article.（她编辑了这篇文章。）
- **近义词**：revise（修订）、modify（修改）
- **反义词**：无
- **考研频率**：★★★

## 🎤 演讲与表达

### speech
- **音标**：/spiːtʃ/
- **词性**：n. 演讲；讲话
- **中文释义**：公开讲话
- **词根词缀**：speech-（演讲）
- **例句**：He gave a great speech.（他做了一场精彩的演讲。）
- **近义词：address（演说）、lecture（讲座）
- **反义词**：silence（沉默）
- **考研频率**：★★★★

### express
- **音标**：/ɪkˈspres/
- **词性**：v. 表达；表示
- **中文释义**：传达思想或感情
- **词根词缀**：ex-（出）+ press（压）
- **例句**：She expressed her opinion.（她表达了自己的观点。）
- **近义词**：convey（传达）、communicate（交流）
- **反义词**：suppress（压制）
- **考研频率**：★★★★★

### explain
- **音标**：/ɪkˈspleɪn/
- **词性**：v. 解释；说明
- **中文释义**：使理解
- **词根词缀**：ex-（出）+ plain（清楚）
- **例句**：Can you explain this?（你能解释一下吗？）
- **近义词**：clarify（澄清）、describe（描述）
- **反义词**：confuse（混淆）
- **考研频率**：★★★★★

### describe
- **音标**：/dɪˈskraɪb/
- **词性**：v. 描述；描写
- **中文释义**：用文字描绘
- **词根词缀**：de-（向下）+ scribe（写）
- **例句**：Describe what you see.（描述你看到的。）
- **近义词**：portray（描绘）、depict（描绘）
- **反义词**：无
- **考研频率**：★★★★★
