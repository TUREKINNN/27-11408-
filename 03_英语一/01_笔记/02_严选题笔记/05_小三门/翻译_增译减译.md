# 增译减译技巧

> [[03_英语一/01_笔记/02_严选题笔记/05_小三门/翻译_翻译技巧|翻译技巧]] | [[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/知识清单]]

## 核心原则

英译汉时，由于两种语言结构差异，常需要：
- **增译**：补充英文省略但中文需要的内容
- **减译**：省略英文重复但中文冗余的内容

## 增译技巧

### 1. 增补主语

英文常省略主语（特别是被动语态），中文需要明确主语。

**例**：
```
原文：He was attacked and wounded.
译文：他遭到袭击，并受了伤。
```
（增补"他遭到"作为主语）

### 2. 增补谓语

英文中某些结构（介词短语、形容词）作表语时，中文需要增补谓语动词。

**例**：
```
原文：He is at home.
译文：他在家里。（"在"是增补的谓语）
```

### 3. 增补宾语

英文中某些动词（discuss, emphasize, reach, enter）中文需要宾语。

**例**：
```
原文：We should discuss this problem.
译文：我们应该讨论一下这个问题。（增补"一下"）
```

### 4. 增补量词

中文有量词，英文没有。

**例**：
```
a book 一本书
two apples 两个苹果
```

### 5. 增补复数概念

英文复数 -s，中文用"们""些""多种"等表达。

**例**：
```
books 一些书/多本书
teachers 教师们
```

### 6. 增补连词

英文中并列复合句常省连词，中文需要。

**例**：
```
原文：He came, he saw, he conquered.
译文：他来了，他看见了，他征服了。（增补逗号）
或：他来了，看见了，征服了。
```

### 7. 增补逻辑词

英文中某些逻辑关系隐含，中文需要明确。

**例**：
```
原文：Being a student, he studies hard.
译文：因为是学生，所以他学习努力。（增补因果关系）
```

## 减译技巧

### 1. 减译代词

英文代词频繁使用，中文可省略以避免重复。

**例**：
```
原文：He opened the door and he entered the room.
译文：他打开门，进入了房间。（减译第二个 he）
```

### 2. 减译连词

英文中 and, or 频繁使用，中文可用逗号或省略。

**例**：
```
原文：He is tall and thin.
译文：他高而瘦。（and 译为"而"或省略）
```

### 3. 减译冠词

英文 a, an, the 中文通常不译。

**例**：
```
the book 这本书 / 书
a student 一个学生 / 学生
```

### 4. 减译介词

英文介词丰富，中文常省略或转换。

**例**：
```
arrive at/in 到达（at/in 不译）
in the morning 在早上 / 早上（in 不译）
```

### 5. 减译 it 等虚词

**例**：
```
It is important to learn English.
学习英语很重要。（It is...to 句型）
```

### 6. 减译动词

英文某些动词中文用名词或形容词表达。

**例**：
```
have a look 看一看
take a walk 散步
```

## 综合应用

### 例 1

```
原文：It is universally acknowledged that 
       trees are indispensable to us.
       (普遍认为，树对我们是不可或缺的。)

译文：人们普遍认为，树对我们来说是不可或缺的。

增译：人们（主语）
减译：It is...that（形式主语）
增补："来说是"
```

### 例 2

```
原文：The fact that he succeeded 
       made his parents very happy.
       
译文：他的成功使他的父母非常高兴。
     （"他成功了"这个事实使他的父母非常高兴。）

减译：The fact that（可用同位语从句直接翻译）
     或保留为"他成功这件事使..."
```

### 例 3

```
原文：He is a teacher and he teaches English.
       
译文：他是英语老师。
     （或：他是老师，教英语。）

增补：英语（减译 and，he teaches）
     或保留为两个分句
```

## 注意事项

1. **不可过度增译**：保持原意，不要添加原文没有的内容
2. **不可过度减译**：不能删掉关键信息
3. **保持通顺**：中译要符合中文表达习惯
4. **照顾上下文**：具体选择根据上下文调整
5. **翻译不是字对字**：要传达原文的含义和精神
