## 8.3 高频必考词样例（A 组）

### abandon
- **音标**：`[əˈbændən]`
- **释义**：放弃，抛弃
- **助记**：a（加强）+ bandon（禁止）→ 加强禁止 → 放弃
- **搭配**：abandon oneself to 沉溺于；abandon hope 放弃希望
- **例句**：He abandoned all hope of finding a job.

### abolish
- **音标**：`[əˈbɒlɪʃ]`
- **释义**：废除，取消
- **助记**：a（加强）+ bol（抛）+ ish（动词后缀）→ 用力抛弃 → 废除
- **搭配**：abolish slavery 废除奴隶制；abolish a law 废除法律

### abnormal
- **音标**：`[æbˈnɔːml]`
- **释义**：反常的，变态的
- **助记**：ab（相反）+ normal（正常）→ 不正常的
- **搭配**：abnormal behavior 反常行为；abnormal weather 反常天气

### abrupt
- **音标**：`[əˈbrʌpt]`
- **释义**：突然的，粗暴的
- **助记**：ab（离开）+ rupt（断裂）→ 突然断裂 → 突然的
- **搭配**：abrupt change 突变；abrupt departure 突然离开

## 8.4 词根词缀体系（高频）

| 词缀 | 含义 | 例词 |
|:-----|:-----|:-----|
| ab- | 离开、相反 | absent, abnormal, abandon, abolish |
| ad- (ac-/af-/ag-) | 方向、添加 | adapt, affect, aggressive |
| com- (co-/con-) | 共同、加强 | combine, connect, cooperate |
| de- | 向下、离开、否定 | descend, destroy, decline |
| dis- | 否定、分离 | disappear, disagree, dismiss |
| in- (im-/il-/ir-) | 不、无、非 | incredible, impossible, illegal |
| re- | 再次、返回 | return, react, reform |
| sub- | 下面 | subway, subject, substance |
| trans- | 跨越 | transfer, transform, transport |
| -able / -ible | 能...的 | readable, sensible, possible |
| -tion / -sion | 名词后缀 | action, decision, explosion |
| -ment | 名词后缀 | development, government |
| -ness | 形容词→名词 | happiness, darkness |
| -ful | 充满...的 | beautiful, powerful |
| -less | 无...的 | careless, hopeless |

## 8.5 必考词记忆法

> 1. **词根词缀法**：abandon = a + bandon（禁止）→ 加强禁止 → 放弃
> 2. **联想法**：abroad = a + broad（广阔）→ 在广阔的地方 → 在国外
> 3. **对比法**：absence（缺席）/ absent（缺席的）/ absentee（缺席者）

## 8.6 学习策略

> 1. **重复是记忆之母**：每天 30 分钟滚动复习
> 2. **语境记单词**：脱离例句记单词 = 记不住
> 3. **分类整理**：按场景/话题/词根分类
>
> **不推荐**：死记硬背单词表（无语境，易遗忘）
