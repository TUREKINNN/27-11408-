## 🎴 二、30 张速记卡片使用说明

### 📖 卡片结构

每张卡片包含 5 大要素：
1. **🏷️ 类别**：前缀/后缀的功能分类（否定类/方向类/程度类/名词后缀等）
2. **⭐ 出现频率**：考研真题中出现次数（5-10 颗星）
3. **🧠 助记法**：核心口诀 / 故事 / 谐音
4. **🎬 联想场景**：可视化记忆场景
5. **📚 5 大核心派生词**：高频词汇
6. **⚠️ 易错提示**：变形规则 / 易混淆对比

### 🔄 学习循环

```
记忆 (10分钟) → 复习 (5分钟) → 默写 (5分钟) → 应用 (5分钟)
   ↓                ↓                ↓                ↓
 联想场景        口头重复         纸笔测试        造句使用
```

### 🧠 记忆法 3 大武器

1. **形象化**：用图像、场景、故事代替抽象记忆
2. **节奏化**：押韵口诀 + 节拍记忆
3. **关联化**：新旧知识串联，构建网络

## 📇 卡片 01：**un-**（不、非、相反）

> 🏷️ **类别**：否定类
> ⭐ **出现频率**：★★★★★★★★（★★★★★★★★ 颗星 / 10 颗星满）

### 🧠 助记法

👿 **uncle 反派叔叔** → 专做反对的事

### 🎬 联想场景

想象一个反派叔叔 'un' 走进房间，把所有东西都'反'过来：他把灯 unlight(关掉)，把门 unclose(打开)

### 📚 5 大核心派生词

| 单词 | 释义 |
|:-----|:-----|
| **unable** | 不能的 |
| **unaware** | 不知道的 |
| **uncover** | 揭露 |
| **unfold** | 展开 |
| **undo** | 撤销 |

### ⚠️ 易错提示

⚠️ un- + 形容词 = 反义 (happy → unhappy)，un- + 动词 = 相反动作 (do → undo)

## 💡 七、速记小技巧

### ✅ 推荐做法

1. **早晨记忆**：刚起床时大脑清醒，适合记忆新内容
2. **晚上默写**：睡前回忆，加深印象
3. **联想图像**：每个助记法配一个具体场景
4. **口诀押韵**：编顺口溜，例如 "un 反 in 否 dis 离"
5. **错题本**：记录易错点，定期回顾

### ❌ 避免误区

1. ❌ **只看不写**：必须动手写才能记住
2. ❌ **一次学太多**：每天 3 个，贪多嚼不烂
3. ❌ **不复盘**：不复习 = 遗忘
4. ❌ **死记硬背**：不理解构词法，效率低
5. ❌ **跳过易错点**：薄弱环节要重点突破
