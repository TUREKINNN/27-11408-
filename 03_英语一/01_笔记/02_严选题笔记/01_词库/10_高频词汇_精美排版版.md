# 📓 第五部分：记忆法与复习策略

## 🎯 五大黄金记忆法

### 1️⃣ 词根词缀法（最强大）
- **核心**：掌握 18 大前缀 + 12 大后缀 = 推测 80% 词汇
- **步骤**：拆分 → 识根 → 组合 → 推测
- **示例**：un(反) + cover(盖) = uncover(揭露)

### 2️⃣ 词族树法（最系统）
- **核心**：一个词根 → 一棵词汇树
- **步骤**：找根 → 辐射 → 联想 → 串联
- **示例**：act → action, active, activity, react, interact, actor

### 3️⃣ 主题聚类法（最实用）
- **核心**：同主题词汇一起记
- **步骤**：分类 → 建网 → 联想 → 应用
- **示例**：思想主题 = think, believe, consider, realize, judge

### 4️⃣ 视觉符号法（最高效）
- **核心**：用符号标注重点，一眼抓住核心
- **符号**：
  - ⭐ 必考 = 优先记
  - 🔗 同义 = 一起记
  - ⚡ 反义 = 对比记
  - 🧠 助记 = 联想记

### 5️⃣ 艾宾浩斯曲线法（最科学）
- **核心**：科学间隔复习，事半功倍
- **节奏**：1天 → 2天 → 4天 → 7天 → 15天
- **工具**：Anki 闪卡 / 墨墨背单词
