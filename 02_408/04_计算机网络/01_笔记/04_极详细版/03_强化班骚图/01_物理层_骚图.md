## 📡 主题 1：物理层基本概念

### 1.1 物理层 4 大功能

```
1. 机械特性：接口形状、尺寸、引脚数
2. 电气特性：电压范围、阻抗匹配
3. 功能特性：某线上的电平含义
4. 过程特性：各物理线路的工作规程
```

### 1.2 数据通信模型

```
        ┌─────┐     信号     ┌─────┐
源系统─→│发送器├─→─────→────→│接收器├─→目的系统
        └─────┘   传输系统   └─────┘
           │                    │
           ↓                    ↓
        调制                    解调
```

### 1.3 三大术语

| 术语 | 英文 | 含义 |
|------|------|------|
| **码元** | Symbol | 数字通信中数字信号的计量单位 |
| **比特** | Bit | 信息量单位 |
| **波特** | Baud | 码元传输速率单位（1 波特 = 1 码元/秒） |

> - 一个码元可携带 **n 比特**信息
> - 当码元有 4 种状态时，1 码元 = 2 bit
> - 当码元有 8 种状态时，1 码元 = 3 bit

## 🏃 主题 2：信道复用技术

### 2.1 三大信道复用

| 复用技术 | 英文 | 原理 | 特点 |
|---------|------|------|------|
| **频分复用** | FDM | 不同频率同时传 | 模拟 |
| **时分复用** | TDM | 不同时隙交替传 | 数字 |
| **波分复用** | WDM | 不同波长同时传 | 光纤 |
| **码分复用** | CDM | 不同码型区分 | 移动通信 |

### 2.2 时分复用 3 大子类型

```
TDM  (时分复用)
├── 同步时分 TDM：固定时隙，可能浪费
├── 异步时分 TDM / 统计时分：按需分配
└── 交织时分：每帧取 1 字节轮流
```

### 2.3 CDM 码分多址

```
每个用户分配一个唯一的码片序列（m bit）
发 1：发码片序列本身
发 0：发码片序列反码
接收：用发送方的码片序列做内积
  = 1：发送 1
  = -1：发送 0
  = 0：未发送
```

> - 发送 1：传 (1,1,-1,-1)
> - 发送 0：传 (-1,-1,1,1)
> - 接收 (1,1,-1,-1) × (1,1,-1,-1)/4 = 1 → 发 1
> - 接收 (-1,-1,1,1) × (1,1,-1,-1)/4 = -1 → 发 0

## 📊 主题 3：奈氏准则 + 香农定理

### 3.1 奈奎斯特准则（无噪声）

```
极限码元传输速率（波特率）：
  B = 2W (Baud)
  W = 信道带宽 (Hz)

极限数据传输速率（比特率）：
  C = 2W log₂V (bps)
  V = 码元的离散取值个数
```

> - 限制：**码元速率 ≤ 2W**
> - 提高 V（更多状态）可提高 C
> - 但 V 越大，**抗干扰越差**

### 3.2 香农定理（有噪声）

```
极限数据传输速率：
  C = W × log₂(1 + S/N) (bps)
  W = 信道带宽 (Hz)
  S/N = 信噪比（无量纲）
  S = 信号功率，N = 噪声功率

信噪比（dB）= 10 log₁₀(S/N)
```

> - S/N = 10^3 = 1000
> - C = 4k × log₂(1+1000) = 4k × 9.97 ≈ **39.88 kbps**

### 3.3 奈氏 vs 香农

| 比较 | 奈氏准则 | 香农定理 |
|------|---------|---------|
| 条件 | 理想无噪声 | 有噪声 |
| 公式 | C = 2W log₂V | C = W log₂(1+S/N) |
| 限制 | 受 V 影响 | 受 S/N 影响 |
| 适用 | 所有信道 | 所有信道 |
| 关系 | 给出 V 限制 | 给出 S/N 限制 |

> - 实际 C ≤ min(奈氏极限, 香农极限)
> - 两者取较小值

## 🔤 主题 4：编码与调制

### 4.1 5 大编码方式

#### 4.1.1 不归零制（NRZ）

```
0：低电平
1：高电平
缺点：无法区分连续多个 0/1（无时钟同步）
```

#### 4.1.2 归零制（RZ）

```
每位码元中间归零
优点：自带时钟
缺点：带宽大
```

#### 4.1.3 反向不归零（NRZI）

```
0：无跳变
1：跳变
优点：抗干扰
```

#### 4.1.4 曼彻斯特编码（Manchester）⭐⭐⭐

```
1：前高后低（下降沿）
0：前低后高（上升沿）
或反之
特点：自带时钟，带宽翻倍（10Mbps 需 20MHz）
以太网采用
```

```
位流:    0   1   0   1   1   0
编码:  _|‾|_|‾ _|‾|_|‾  _|_|‾|_
       低高低高  高低高低
       0=前低后高  1=前高后低
```

#### 4.1.5 差分曼彻斯特

```
每位中间必跳变
0：位开始跳变
1：位开始不跳变
更可靠
```

### 4.2 5 大编码对比

| 编码 | 自带时钟 | 带宽 | 抗干扰 | 应用 |
|------|---------|------|--------|------|
| NRZ | ❌ | 小 | 差 | 早期 |
| RZ | ✔ | 大 | 中 | - |
| NRZI | ❌ | 小 | 较好 | USB |
| Manchester | ✔ | 大 | 较好 | 以太网 |
| 差分 Manchester | ✔ | 大 | 好 | 令牌环 |

### 4.3 调制方式

```
调幅（AM）：用不同幅值表示码元
调频（FM）：用不同频率表示码元
调相（PM）：用不同相位表示码元
正交调幅（QAM）：幅值 + 相位结合
```

## 📡 主题 6：物理层设备

### 6.1 中继器（Repeater）

```
功能：放大整形信号，延长传输距离
特点：工作在物理层
端口：2 个（输入 + 输出）
规则：中继器两端网段必须为同一协议
"5-4-3 规则"：10BASE5 中，最多 4 个中继器，5 段中 3 段挂设备
```

### 6.2 集线器（Hub）

```
本质：多端口中继器
特点：所有端口共享带宽
缺点：所有端口同一冲突域、同一广播域
```

### 6.3 三大设备对比

| 设备 | 工作层 | 功能 | 冲突域 | 广播域 |
|------|--------|------|--------|--------|
| **中继器** | 物理层 | 放大信号 | 不隔离 | 不隔离 |
| **集线器** | 物理层 | 多端口中继器 | 不隔离 | 不隔离 |
| **网桥** | 数据链路层 | 帧转发 | 隔离 | 不隔离 |
| **交换机** | 数据链路层 | 多端口网桥 | 隔离 | 不隔离（VLAN 隔离广播域） |
| **路由器** | 网络层 | 路由选择 | 隔离 | 隔离 |

## ✅ 考前必背 8 大公式

| # | 公式 | 名称 | 应用 |
|---|------|------|------|
| 1 | B = 2W | 奈氏准则 | 极限波特率 |
| 2 | C = 2W log₂V | 奈氏比特率 | 极限比特率 |
| 3 | C = W log₂(1+S/N) | 香农定理 | 极限比特率 |
| 4 | S/N(dB) = 10 log₁₀(S/N) | 信噪比 | dB 计算 |
| 5 | 1 码元 = log₂V 比特 | 码元比特关系 | 速率换算 |
| 6 | 带宽 = 波特率 / 2 | 曼彻斯特带宽 | 以太网 |
| 7 | 5-4-3 规则 | 中继器规则 | 10BASE5 |
| 8 | V = 2^n → 1 码元 n 比特 | V 与比特 | 码元换算 |

## ⚠️ 8 大易错点

1. ❌ **波特率 = 比特率？** 错误！比特率 = 波特率 × log₂V
2. ❌ **奈氏与信道带宽无关？** 错误！奈氏 B = 2W 与带宽正相关
3. ❌ **香农不考虑带宽？** 错误！香农公式含 W
4. ❌ **曼彻斯特效率高？** 错误！效率仅 50%，需 2 倍带宽
5. ❌ **NRZ 不能区分 0/1？** 错误！NRZ 区分电平，但无时钟
6. ❌ **集线器隔离冲突？** 错误！集线器所有端口同冲突域
7. ❌ **中继器 = 路由器？** 错误！中继器是物理层，路由器是网络层
8. ❌ **码分多址可任意加用户？** 错误！码片正交有上限
