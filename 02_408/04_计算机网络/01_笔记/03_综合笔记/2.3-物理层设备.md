# 2.3 物理层设备

> 物理层设备工作在第1层，主要功能是对信号进行处理。核心设备包括**中继器**和**集线器**。

---

## 一、中继器（Repeater）

### 1.1 功能

当传输距离太长时，数字信号会**失真**。中继器的功能是**信号整形再生**：

1. 接收失真的信号
2. 将信号整形为标准电平
3. 转发整形后的信号

> 物理层电气特性规定：0.5\~1.5V 为低电平，4.5\~5.5V 为高电平。
> 中继器将低电平整为 1V，高电平整为 5V，再输出。

### 1.2 特点

- 工作在**物理层**
- 只对信号进行整形再生，**不进行存储转发**
- 不能连接物理协议不同的网段（大多数教材观点）
- 不能连接速率不同的网段（大多数教材观点）

---

## 二、集线器（Hub）

### 2.1 功能

集线器本质上是一个**多端口的中继器**：

- 从一个端口接收信号，整形再生后**广播**到所有其他端口
- 物理上是**星形**拓扑结构
- 逻辑上是**总线形**拓扑结构（数据广播式传输）

### 2.2 冲突域（碰撞域）

> 如果两台主机同时发送数据会导致"冲突"，则这两台主机处于同一个**冲突域**。处于同一冲突域的主机在发送数据前需要进行**信道争用**。

**关键结论**：集线器**不能隔离冲突域**。

```
集线器连接4台主机 → 1个冲突域
交换机连接4台主机 → 4个冲突域（每个端口一个）
```

### 2.3 带宽共享

> 集线器的总带宽由所有连接的主机共享。
> 例如：10Mbps 集线器连接 8 台主机 → 每台主机平均只有 **1.25 Mbps** 带宽。

### 2.4 5-4-3 原则

使用集线器（或中继器）连接 10Base5 网段时：
- 最多串联 **5** 个网段
- 使用 **4** 台集线器（或中继器）
- 只有 **3** 个网段可以挂接计算机

---

## 三、中继器与集线器对比

| 特性 | 中继器 | 集线器 |
|------|--------|--------|
| 端口数 | 通常2个 | 多个 |
| 功能 | 信号整形再生 | 信号整形再生 + 广播 |
| 冲突域 | 不影响 | 不能隔离冲突域 |
| 拓扑 | - | 物理星形、逻辑总线形 |
| 工作层次 | 物理层 | 物理层 |

---

## 考点总结

> 1. 中继器功能：信号整形再生，工作在物理层
> 2. 集线器 = 多端口中继器，物理星形、逻辑总线形
> 3. **集线器不能隔离冲突域**（以太网交换机可以隔离）
> 4. 集线器带宽由所有端口共享
> 5. 5-4-3 原则
> 6. 中继器/集线器不能连接不同物理协议或不同速率的网段
