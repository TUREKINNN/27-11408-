# 3.7 广域网

> 广域网（WAN）覆盖较大地理范围，通过路由器连接局域网。广域网中点对点链路常用 **PPP 协议**。

---

## 一、广域网基本概念

### 1.1 广域网与局域网的关系

```
局域网A ←→ 路由器 ←→ 广域网 ←→ 路由器 ←→ 局域网B
```

- 局域网通过**路由器**连接广域网
- 广域网中的"节点交换机"功能上类似路由器（不是以太网交换机）
- 为确保健壮性，广域网各节点之间通常呈**网状**连接

### 1.2 链路类型与协议

| 链路类型 | 数据链路层协议 | 特点 |
|---------|--------------|------|
| 总线型链路 | CSMA/CD（以太网） | 广播式，多点接入 |
| **点对点链路** | **PPP 协议** | 一对一，无需介质访问控制 |

---

## 二、PPP 协议（Point-to-Point Protocol）

### 2.1 PPP 协议的特点

> 1. **LCP**（Link Control Protocol，链路控制协议）：建立、配置和测试数据链路连接
> 2. **NCP**（Network Control Protocol，网络控制协议）：建立和配置不同的网络层协议
> 3. 将 IP 数据报封装到串行链路的方法

### 2.2 PPP 协议的工作过程

```
物理层建立连接 → LCP配置协商 → LCP链路鉴别 → NCP配置协商 → 数据传输
```

| 阶段 | 操作 | 说明 |
|------|------|------|
| 物理层建立 | 插入物理线路 | 节点之间建立物理链路 |
| LCP 配置协商 | 发送 LCP 配置请求/应答 | 配置 MTU、认证协议等参数 |
| LCP 链路鉴别 | 发送用户名/密码 | 如：PAP 认证协议 |
| NCP 配置协商 | 发送 NCP 请求/确认 | 如：请求分配 IP 地址 |
| 数据传输 | 用 PPP 帧携带 IP 分组 | 正式上网 |

### 2.3 PPP 帧格式

```
标志(7E) | 地址(FF) | 控制(03) | 协议 | 信息 | FCS | 标志(7E)
 1B         1B        1B       2B    可变   2B     1B
```

| 字段 | 说明 |
|------|------|
| **标志** | 0x7E，帧定界符 |
| **地址** | 固定为 0xFF（点对点无需地址） |
| **控制** | 固定为 0x03 |
| **协议** | 指明信息字段的类型（如 0x0021=IP数据报） |
| **信息** | IP 数据报，长度受 MTU 限制（默认 1500B） |
| **FCS** | 帧校验序列 |

### 2.4 PPP 的字节填充

当使用异步传输（面向字节）时：
- 转义符 = **0x7D**
- 0x7E → 0x7D 0x5E
- 0x7D → 0x7D 0x5D
- 小于 0x20 的控制字符 → 0x7D + (原字符 + 0x20)

### 2.5 PPP 的零比特填充

当使用同步传输（面向比特）时：
- 标志字段 = 01111110
- 发送方：信息部分每出现连续 5 个 1，插入一个 0
- 接收方：每收到连续 5 个 1，去掉后面的 0

### 2.6 PPP 与 PPPoE

- PPP 的黄金时代是 1990s\~2000s，当时的"拨号上网"就是用 PPP 协议
- **PPPoE**（PPP over Ethernet）= 用以太网 MAC 帧封装 PPP 帧
- 现代宽带上网常使用 PPPoE

---

## 三、PPP 协议的特点

| 特性 | 说明 |
|------|------|
| 点对点 | 只支持两台设备之间的通信 |
| 面向字节 | 帧的长度以字节为单位 |
| 不可靠 | 不使用序号和确认机制 |
| 全双工 | 支持同时双向通信 |
| 简单 | 不需要介质访问控制 |

> - PPP 用于点对点链路（如拨号上网）
> - 以太网用于广播链路（如局域网）
> - PPP 不需要 CSMA/CD，因为只有两台设备

---

## 考点总结

> 1. PPP 是点对点协议，用于广域网的点对点链路
> 2. PPP 三部分：LCP（链路控制）、NCP（网络控制）、封装方法
> 3. PPP 工作过程：物理连接→LCP协商→鉴别→NCP协商→数据传输
> 4. PPP 不可靠（无序号和确认），简单高效
> 5. PPP 帧格式：标志+地址(FF)+控制(03)+协议+信息+FCS+标志
> 6. PPPoE = 以太网封装 PPP，用于宽带上网
