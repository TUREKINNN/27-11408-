# 第二章 物理层

## 考情分析

物理层是计算机网络中最底层，考试以选择题为主，偶尔有计算题。核心考点：**奈奎斯特定理和香农定理**的公式应用（几乎每年必考）、编码方式的识别、波特率与比特率的关系。分值约3-5分，但公式题是"必拿分"题。

---

## 基本概念

### 2.1 物理层的功能

物理层解决的是如何在连接各种计算机的传输媒体上传输**比特流**，而不是指具体的传输媒体。

物理层的主要任务：
1. **机械特性**：接口形状、引脚数目
2. **电气特性**：电压范围、阻抗、传输速率
3. **功能特性**：各引脚电压的意义
4. **过程特性**：各事件的顺序

### 2.2 数据通信基础

#### 基本术语

| 术语 | 含义 |
|------|------|
| **数据** | 传送信息的实体 |
| **信号** | 数据的电气/电磁表现（模拟信号/数字信号） |
| **码元** | 一个固定时长的信号波形，表示k进制的一个数值 |
| **信源** | 产生和发送数据的源头 |
| **信宿** | 接收数据的终点 |
| **信道** | 信号的传输通道（逻辑概念） |

#### 信道分类

| 按传输信号 | 按传输方向 |
|------------|------------|
| 模拟信道 | 单工信道（只能单向） |
| 数字信道 | 半双工信道（双向交替） |
| | 全双工信道（双向同时） |

#### 码元、波特率、比特率的关系 ⭐⭐

$$
比特率 = 波特率 \times \log_2 V
$$

- **波特率（Baud）**：每秒传输的码元数，单位 Baud
- **比特率（bps）**：每秒传输的比特数，单位 bps
- **V**：码元的离散状态数（即一个码元携带的比特数为 $\log_2 V$）

> 例：波特率 = 1000 Baud，码元有4个状态（V=4），则比特率 = 1000 × 2 = 2000 bps

## 核心知识

### 2.3 编码方式

#### 2.3.1 NRZ编码（不归零编码）

```
NRZ-L（电平）：
1 → 高电平    0 → 低电平

NRZ-I（反向）：
1 → 电平翻转   0 → 电平不变
```

**特点**：无法自同步（连续的1或0无法判断边界），实际中不单独使用。

#### 2.3.2 曼彻斯特编码 ⭐⭐

```
规则：
- 每个码元中间有一次电平跳变
- "1" → 先高后低（↓下降沿）  或  先低后高（↑上升沿）
- "0" → 与"1"相反的跳变方向

时钟信号：
  ___|‾‾‾|___|‾‾‾|___|‾‾‾|___|‾|
数据信号：
  ‾‾‾‾‾|___|___|‾‾‾‾‾|___|___|
曼彻斯特：
  ‾‾\__/\‾‾‾/‾‾‾\__/‾‾‾/‾‾\__
  
  ↑下降=1    ↑上升=0    ↑下降=1    ↑上升=0
```

> 🔑 **以太网使用曼彻斯特编码**：每个比特周期内有一个跳变，时钟频率是数据速率的2倍

#### 2.3.3 差分曼彻斯特编码

```
规则：
- 每个码元中间有一次电平跳变（仅用于时钟同步）
- "0" → 码元开始处有跳变
- "1" → 码元开始处无跳变
```

**曼彻斯特 vs 差分曼彻斯特**：

| 比较项 | 曼彻斯特 | 差分曼彻斯特 |
|--------|----------|--------------|
| 中间跳变 | 有（确定比特值+同步） | 有（仅同步） |
| 起始跳变 | 不固定 | 0有跳变，1无跳变 |
| 抗干扰性 | 较好 | **更好**（差分编码抗干扰更强） |
| 应用 | 以太网 | 令牌环网 |

### 2.4 调制技术 ⭐⭐

将数字数据转换为模拟信号的调制方式：

| 调制方式 | 英文 | 原理 | 特点 |
|----------|------|------|------|
| **ASK** | 幅移键控 | 改变载波**振幅** | 抗干扰差 |
| **FSK** | 频移键控 | 改变载波**频率** | 抗干扰较好 |
| **PSK** | 相移键控 | 改变载波**相位** | 抗干扰强 |
| **QAM** | 正交振幅调制 | 同时改变振幅和相位 | **效率最高**，4G/5G使用 |

> **QAM = ASK + PSK** 的组合，每个码元可以携带更多比特

### 2.5 奈奎斯特定理（奈氏准则）⭐⭐⭐

**适用条件**：理想低通信道（无噪声、带宽有限）

$$
C = 2W\log_2 V \quad (bps)
$$

- **C**：最大数据传输速率
- **W**：信道带宽（Hz）
- **V**：码元的离散电平数（信号状态数）

**含义**：在带宽为W Hz的信道中，若信号有V个离散电平，则最大传输速率为 $2W\log_2 V$ bps。

**推导过程**：
1. 奈奎斯特证明：带宽为W的理想信道，最高码元速率为 **2W Baud**
2. 每个码元携带 $\log_2 V$ bit
3. 所以最高数据速率 = $2W \times \log_2 V$ bps

> ⚠️ **注意**：这是理论极限，实际不可能达到！

### 2.6 香农定理 ⭐⭐⭐

**适用条件**：有噪声的信道

$$
C = W\log_2(1 + SNR) \quad (bps)
$$

- **C**：信道的极限数据传输速率
- **W**：信道带宽（Hz）
- **SNR**：信噪比（信号功率/噪声功率）

信噪比通常用分贝(dB)表示：

$$
SNR_{dB} = 10\log_{10}\frac{S}{N}
$$

$$
\frac{S}{N} = 10^{SNR_{dB}/10}
$$

**推导思路**：
1. 信道容量取决于带宽和信噪比
2. 信噪比越大，能区分的信号状态越多
3. 香农从信息论角度证明了这个极限

**两个定理的关系**：
```
实际最大速率 = min{奈奎斯特极限, 香农极限}
```

> 🔑 **解题步骤**：
> 1. 先算奈奎斯特极限：$2W\log_2 V$
> 2. 再算香农极限：$W\log_2(1+SNR)$
> 3. 取两者较小值

### 2.7 传输介质

| 类型 | 名称 | 特点 |
|------|------|------|
| **导向性** | 双绞线 | 最便宜，100m以内，易受干扰 |
| | 同轴电缆 | 抗干扰强于双绞线 |
| | **光纤** | 速率最高、距离最远、抗电磁干扰 |
| **非导向性** | 无线电波 | 全向传播，穿透强 |
| | 微波 | 直线传播，距离远 |
| | 红外 | 短距离，不能穿墙 |

| 光纤类型 | 特点 |
|----------|------|
| 单模光纤 | 纤芯细，只传一种光模式，距离远、速率高 |
| 多模光纤 | 纤芯粗，多种光模式，适合短距离 |

### 2.8 物理层设备

| 设备 | 功能 | 特点 |
|------|------|------|
| **中继器（Repeater）** | 放大/整形衰减的信号 | 不能连接不同类型的网段 |
| **集线器（Hub）** | 多端口中继器，广播转发 | 共享带宽，不能隔离冲突域 |

> ⚠️ **重要结论**：
> - 中继器和集线器**不能**隔离冲突域
> - 集线器所有端口属于**同一个冲突域**
> - 交换机**可以**隔离冲突域
> - 路由器**可以**隔离广播域

## 深入理解

### 为什么曼彻斯特编码能自同步？

因为每个比特周期内都**必定有一次电平跳变**，接收端可以从跳变中恢复时钟信号。代价是带宽利用率翻倍（需要2倍的信号频率）。

### 奈奎斯特 vs 香农的本质区别

- **奈奎斯特**：只考虑**带宽**和**码元状态数**（理想信道）
- **香农**：考虑**带宽**和**信噪比**（有噪声信道）
- 两个公式不能混用：有噪声时，即使增加码元状态数，信噪比也会限制实际精度

### 基带传输 vs 宽带传输

| 比较 | 基带传输 | 宽带传输 |
|------|----------|----------|
| 信号 | 数字信号直接传输 | 调制后的模拟信号 |
| 频率 | 低频，占用整个信道 | 高频，可用频分复用 |
| 典型 | 以太网 | 有线电视 |

## 真题考法

1. **计算题**：给定带宽和信噪比，求最大数据传输速率（香农定理）
2. **选择题**：编码方式的识别和比较
3. **选择题**：波特率与比特率的换算
4. **选择题**：物理层设备的功能辨析

## 典型例题

### 例题1：奈奎斯特定理应用

**题目**：一个无噪声信道带宽为4kHz，若采用16种不同的码元状态，求最大数据传输速率。

**解答**：
$$C = 2W\log_2 V = 2 \times 4000 \times \log_2 16 = 2 \times 4000 \times 4 = 32000 \text{bps} = 32\text{kbps}$$

### 例题2：香农定理应用

**题目**：一个信道带宽为3kHz，信噪比为30dB，求最大数据传输速率。

**解答**：
$$SNR = 10^{30/10} = 1000$$
$$C = W\log_2(1+SNR) = 3000 \times \log_2(1000) = 3000 \times \frac{\lg 1000}{\lg 2} = 3000 \times \frac{3}{0.301} \approx 3000 \times 9.97 \approx 29.9 \text{kbps}$$

### 例题3：综合应用（两个定理结合）

**题目**：一个信道带宽为4kHz，信噪比为30dB，码元有8个状态，求最大数据传输速率。

**解答**：
- 奈奎斯特极限：$C_1 = 2 \times 4000 \times \log_2 8 = 24000$ bps = 24 kbps
- 香农极限：$C_2 = 4000 \times \log_2(1000) \approx 39.88$ kbps
- 实际最大速率：$\min(C_1, C_2) = 24$ kbps

### 例题4：波特率与比特率

**题目**：某信道码元速率为2000 Baud，使用QPSK调制（4种相位），求数据传输速率。

**解答**：
- QPSK：V = 4，每码元携带 $\log_2 4 = 2$ bit
- 数据速率 = 2000 × 2 = 4000 bps = 4 kbps

---

## 易错点

1. ⚠️ 奈奎斯特公式的对数底数是2，不是10！
2. ⚠️ 信噪比分贝(dB)值不能直接代入香农公式，需先转换为比值
3. ⚠️ 集线器属于物理层设备，交换机属于数据链路层设备
4. ⚠️ 曼彻斯特编码的带宽是NRZ编码的2倍（每个比特有两个信号状态）
5. ⚠️ 中继器不能连接不同速率的局域网
6. ⚠️ 码元状态数V增加，虽然奈奎斯特速率增加，但实际中受噪声限制

## 补充内容

> 📖 **来源**：王道考研26版计网基础考点讲解.pdf

### 物理层功能的深入理解

物理层为数据链路层屏蔽了各种传输媒体的差异，使数据链路层只需要考虑如何完成本层的协议和服务，而不必考虑网络具体的传输媒体是什么。

**信源与信宿**：
- **信源**：信号的来源（即信号的发送方）
- **信宿**：信号的归宿（即数据的接收方）
- **信号**：数据的载体，可分为**数字信号**（信号值是离散的）和**模拟信号**（信号值是连续的）
- **信道**：信号的通道，一条物理线路通常包含两条信道：**发送信道**和**接收信道**

**码元的深入理解**：
- 每一个信号就是一个码元，信号周期称为**码元宽度**
- 若一个码元可能有4种状态，则称为4进制码元（一个码元携带2bit数据）
- 如果一个周期内可能出现K种信号，则：1码元 = log₂K bit
- **波特率**：每秒传输多少个码元，即**每秒可能发生的信号变化次数**

### 奈氏准则与香农公式的补充

**奈氏准则**：
- 码元传输速率 = 2 × 带宽（理想信道：不考虑噪声影响、带宽有限）
- 单位为**码元**（也称为**采样定理**）
- 如果波特率太高，会导致"**码间串扰**"，即接收方无法识别码元
- 带宽越大，信道传输码元的能力越强
- 奈氏准则并未对一个码元最多携带多少比特做出解释

**信噪比**：
- 信号平均功率与噪声平均功率之比，记作 S/N，无单位
- 信噪比(dB) = 10 × log₁₀(S/N)，单位：分贝(dB)

**香农公式**：
- 信道的**极限**信息传输速率 C = W × log₂(1 + S/N)，单位为bit
- 提高信道带宽、加强信号功率、降低噪声功率，都可以提高信道的极限比特率
- 结合奈氏准则：在带宽、信噪比确定的信道上，一个码元可以携带的比特数是有上限的

**重要结论**：在信道带宽一定的情况下，根据奈氏准则和香农公式，要想**提高信息的传输速率**就必须采用**多元制**（更好的调制方法）和努力提高信道中的**信噪比**。

### 编码与调制的补充

**基本概念**：
- **编码**：将二进制数据转换为数字信号（离散）
- **解码**：将数字信号转换为二进制数据
- **调制**：将二进制数据转换为模拟信号（连续）
- **解调**：将模拟信号转换为二进制数据

**自同步能力**：信源和信宿可以根据信号完成"节奏同步"，无需时钟信号。

**曼彻斯特编码的特殊性**：
- 以太网默认曼彻斯特码方式，因为其**同步能力**
- 对于曼彻斯特编码而言，**2Baud = 1bit**

**QAM调制方式详解**：
- 若设计m种幅值、n种相位，则将AM、PM信号两两"复合"，可调制出 mn 种信号
- 1码元 = log₂(mn) bit
- QAM-16：即调制16种信号，1码元携带log₂16 = 4bit的数据

**重要注意**：频率和相位是相关的，因此一次只能调制频率和相位两个中的一个。**采样速率 ≥ 数据传输率**。

### 传输介质的补充

**导向型传输媒体**（信号朝固定方向传播）：

| 介质 | 构成 | 抗干扰 | 应用 |
|------|------|--------|------|
| **双绞线** | 两根导线相互绞合（STP有屏蔽层，UTP无） | 较好（绞合、屏蔽层提升抗干扰） | 局域网、电话线 |
| **同轴电缆** | 内导体（传信号）+ 外导体屏蔽层 | 好（屏蔽层） | 早期局域网、有线电视 |
| **光纤** | 纤芯（高折射率）+ 包层（低折射率），利用**光的全反射** | 非常好（光信号对电磁干扰不敏感） | 长距离高速传输 |

**光纤分类**：
- **单模光纤**：只有一条光线在一根光纤中传输，信号损耗小，适合**长距离**传输
- **多模光纤**：多条光线在一根光纤中传输，信号损耗高，适合**近距离**传输，远距离传输信号时容易失真

**以太网对有线传输介质的命名规则**：
- 10Base5：10Mbps，同轴电缆，最远传输距离500m
- 10BaseF*：10Mbps，光纤
- 10BaseT*：10Mbps，双绞线

**非导向型传输介质**（无线传输）：

| 介质 | 特点 | 代表 |
|------|------|------|
| **无线电波** | 穿透能力强，传输距离长，信号指向性弱（长波通信） | 手机信号、WiFi |
| **微波通信** | 频率带宽高、信号指向性强、保密性差（短波通信） | 卫星通信 |
| **红外/激光** | 信号指向性强 | 短距离通信 |

**电磁波特性**：
- 电磁波公式：C = λf（C为光速，λ为波长，f为频率）
- 频率越高，数据传输能力越强
- 波长越短，信号指向性越强，信号越趋于直线传播
- 波长越长，绕射性越好，即穿墙性越好
- 长波更适合长距离、非直线通信
- 短波更适合短距离、高速通信、指向性强

### 物理层设备的补充

**中继器的特性**：
- 只有**两个端口**，通过一个端口接收信号，将**失真信号整形再生**，并转发至另一端口
- 仅支持**半双工通信**（两端连接的结点不可同时发送数据，会导致"冲突"）
- 中继器**两个端口对应两个"网段"**
- 中继器可以连接**不同介质的局域网**，但**不能连接不同速率和不同数据链路层协议的**

**集线器的特性**：
- 本质上是**多端口中继器**
- 集线器将其中一个端口接收到的**信号整形再生后，转发到所有其他端口**
- 各端口连接的结点不可同时发送数据，会导致"冲突"
- 集线器的N个端口对应N个"网段"，各网段属于同一个"**冲突域**"

**冲突域/碰撞域**：如果两台主机同时发送数据会导致"冲突"，则这两台主机处于同一个"冲突域"。处于同一冲突域的主机在发送数据前需要进行"**信道争用**"。

**集线器和中继器的一些重要特性**：
- 集线器、中继器**不能"无限串联"**
- 10Base5的**5-4-3原则**：集线器/中继器连接10Base5网段时，最多只能串联5个网段，使用4台集线器（或中继器），只有3个网段可以挂接计算机
- 集线器连接的网络，**物理上是星形拓扑**，**逻辑上是总线型拓扑**（数据"广播式"传输，存在"信道争用"问题）
- 集线器连接的各网段**共享带宽**
- 集线器可以连接不同的传输介质，因此两个网段的物理层接口特性可以不同
- 集线器如果连接了速率不同的网段，会导致所有网段"速率向下兼容"（以最低速率为标准）

**速率匹配问题**：
- 发送方速率 > 接收方速率：接收方来不及接受导致溢出，数据丢失
- 接收方速率 > 发送方速率：不会数据丢失，但是效率低
- 因此，速率必须相同

**物理层互连的限制**：物理层互连成功，只表明这两个网段之间可以互相发送物理层信息，但物理层设备只能转发物理信号，不能识别数据链路层协议，所以不能保证可以互相传送数据链路层的帧。要达到在数据链路层互联互通的目的，就要求**速率和数据链路层协议都相同**。
