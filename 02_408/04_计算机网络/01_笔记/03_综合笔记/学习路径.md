# 计算机网络学习路径

## 学习目标
掌握计算机网络的基本概念、原理和技术，能够分析和解决网络问题，为考研408做好准备。

## 阶段一：基础入门（1-2周）

### 第1周：计算机网络概述
**学习内容：**
- 计算机网络的基本概念
- 计算机网络的分类
- 计算机网络体系结构
  - OSI七层模型
  - TCP/IP四层模型
  - 五层参考模型
- 协议、接口、服务的概念

**学习目标：**
- [ ] 理解计算机网络的基本概念
- [ ] 掌握OSI和TCP/IP模型
- [ ] 理解各层的功能和关系

**练习：**
1. 绘制OSI和TCP/IP模型的对比图
2. 总结各层的主要协议

### 第2周：物理层
**学习内容：**
- 数据通信基础
- 编码与调制技术
- 奈奎斯特定理
- 香农定理
- 传输介质
- 物理层设备

**学习目标：**
- [ ] 掌握数据通信的基本概念
- [ ] 理解各种编码方式
- [ ] 熟练应用奈奎斯特和香农定理
- [ ] 了解传输介质的特点

**练习：**
1. 计算给定条件下的数据传输速率
2. 比较各种编码方式的优缺点

## 阶段二：数据链路层（2-3周）

### 第3周：数据链路层基础
**学习内容：**
- 数据链路层的功能
- 组帧技术
- 差错控制
  - 奇偶校验
  - CRC循环冗余码
  - 海明码

**学习目标：**
- [ ] 理解数据链路层的功能
- [ ] 掌握各种组帧方法
- [ ] 掌握差错检测和纠正方法

**练习：**
1. 计算海明码
2. 进行CRC编码和校验

### 第4周：流量控制与可靠传输
**学习内容：**
- 停止-等待协议
- 后退N帧协议（GBN）
- 选择重传协议（SR）
- 滑动窗口机制

**学习目标：**
- [ ] 理解三种流量控制协议的工作原理
- [ ] 掌握滑动窗口机制
- [ ] 能够进行相关计算

**练习：**
1. 分析GBN和SR协议的性能
2. 计算窗口大小对吞吐量的影响

### 第5周：介质访问控制与局域网
**学习内容：**
- 信道划分协议
- 随机访问协议
  - ALOHA
  - CSMA
  - CSMA/CD
  - CSMA/CA
- 轮询访问协议
- 以太网
- 无线局域网

**学习目标：**
- [ ] 理解各种MAC协议的工作原理
- [ ] 掌握CSMA/CD和CSMA/CA协议
- [ ] 了解以太网的工作原理

**练习：**
1. 比较各种MAC协议的优缺点
2. 分析以太网帧格式

## 阶段三：网络层（3-4周）

### 第6周：网络层基础与IP地址
**学习内容：**
- 网络层的功能
- IP地址分类
- 子网划分
- CIDR

**学习目标：**
- [ ] 理解网络层的功能
- [ ] 掌握IP地址的分类和表示
- [ ] 熟练进行子网划分和CIDR计算

**练习：**
1. 进行子网划分计算
2. 设计IP地址分配方案

### 第7周：IPv4协议
**学习内容：**
- IP数据报格式
- IP数据报分片
- ARP协议
- DHCP协议
- ICMP协议
- NAT

**学习目标：**
- [ ] 掌握IP数据报的格式
- [ ] 理解IP数据报分片过程
- [ ] 掌握ARP、DHCP、ICMP协议的工作原理

**练习：**
1. 分析IP数据报首部
2. 计算IP数据报分片参数

### 第8周：路由算法与协议
**学习内容：**
- 路由算法
  - 距离向量算法
  - 链路状态算法
- 路由协议
  - RIP协议
  - OSPF协议
  - BGP协议

**学习目标：**
- [ ] 理解路由算法的基本原理
- [ ] 掌握RIP、OSPF、BGP协议的工作原理

**练习：**
1. 使用距离向量算法计算路由
2. 分析OSPF协议的工作过程

### 第9周：IPv6与其他网络层协议
**学习内容：**
- IPv6协议
- IP组播
- 移动IP

**学习目标：**
- [ ] 了解IPv6的特点
- [ ] 理解IP组播的概念
- [ ] 了解移动IP的工作原理

**练习：**
1. 比较IPv4和IPv6的区别
2. 分析IP组播的工作过程

  - 快恢复

**学习目标：**
- [ ] 掌握TCP流量控制机制

**练习：**
1. 分析TCP滑动窗口的工作过程

## 学习建议

### 学习方法
1. **理解为主**：理解概念和原理，不要死记硬背
2. **多做练习**：通过练习巩固知识，提高解题能力
3. **画图理解**：通过画图理解协议的工作过程
4. **总结归纳**：定期总结知识点，建立知识体系
5. **联系实际**：将理论知识与实际网络应用联系起来

### 时间安排
- 每天学习2-3小时
- 每周完成一章的学习
- 每两周进行一次复习

### 学习资源
1. **教材**：王道考研408计算机网络
2. **视频**：王道考研计算机网络课程
3. **习题**：历年考研真题、模拟试题
4. **工具**：Wireshark抓包工具、网络模拟器

### 注意事项
1. **循序渐进**：按照学习路径逐步学习，不要跳跃
2. **及时复习**：定期复习已学内容，避免遗忘
3. **做好笔记**：记录重点内容和难点问题
4. **多问问题**：遇到问题及时请教，不要积累问题
5. **保持耐心**：计算机网络内容较多，需要耐心学习
