# 2.2 传输介质

> 传输介质（Transmission Medium）也称传输媒体，是物理层之下的第0层。按信号传播方式分为**导向性传输介质**（信号朝固定方向传播）和**非导向性传输介质**（信号朝四面八方传播）。

---

## 一、导向性传输介质

### 1.1 双绞线（Twisted Pair）

双绞线是最常用的有线传输介质，内部由两根相互绞合的导线组成。

| 类型 | 全称 | 特点 |
|------|------|------|
| **UTP** | 非屏蔽双绞线 | 无屏蔽层，成本低 |
| **STP** | 屏蔽双绞线 | 有金属屏蔽层，抗干扰强 |

**提升抗干扰能力的方法**：
- 提高绞合度
- 增加屏蔽层

> 效果：抗电磁干扰能力增强 → 信道噪声功率降低 → 信道极限速率提高

**常见规格**：
- **六类网线**：支持千兆（1Gbps）
- **八类网线**：支持万兆（10Gbps）

### 1.2 同轴电缆（Coaxial Cable）

结构（从内到外）：
1. **内导体**（铜线）— 传输信号
2. **绝缘层**
3. **外导体屏蔽层** — 抗干扰
4. **绝缘保护壳层**

> 内导体越粗 → 电阻越低 → 信号衰减越少 → 传输距离越长

同轴电缆**只支持半双工**通信。

### 1.3 光纤（Fiber Optic）

利用光的**全反射**原理传输信号。

| 类型 | 全称 | 纤芯特点 | 传输距离 | 应用 |
|------|------|---------|---------|------|
| **多模光纤（MMF）** | Multi-Mode Fiber | 纤芯较粗，可传多条光线 | 较近距离 | 橙色/绿色外皮 |
| **单模光纤（SMF）** | Single-Mode Fiber | 纤芯极细（<一个波长），只传一条光线 | 远距离 | 黄色外皮，家用光纤 |

> - 传输损耗低，适合长距离传输
> - 抗电磁干扰能力极强
> - 带宽极大（约 500MHz）
> - 光纤只支持**全双工**通信

---

## 二、非导向性传输介质

利用**电磁波**在自由空间中传播。电磁波公式：$C = \lambda F$

| 特性 | 关系 |
|------|------|
| 频率越高 | 数据传输能力越强 |
| 波长越短 | 信号指向性越强，趋于直线传播 |
| 波长越长 | 绕射性越好，穿墙能力越强 |

| 介质 | 频率范围 | 特点 |
|------|---------|------|
| 无线电波 | 较低 | 穿透力强，信号可向所有方向传播 |
| 微波 | 较高 | 信号指向性强，需对准方向 |
| 红外线 | 更高 | 短距离通信 |
| 可见光 | 最高 | 如 LiFi |

**卫星通信**：
- 三颗地球同步卫星可覆盖全球
- 特点：传播时延较大
- 近地卫星群可降低时延（如 Starlink）

---

## 三、物理层接口的特性

物理层接口定义了四种特性：

| 特性 | 说明 |
|------|------|
| **机械特性** | 接口的形状、引脚数目、排列方式等 |
| **电气特性** | 电压范围、传输速率、距离限制等 |
| **功能特性** | 各引脚信号的意义 |
| **过程特性** | 信号的时序关系（如插拔网线时的反应） |

---

## 四、以太网传输介质命名规则

格式：**速度 + Base + 介质信息**

其中 Base 表示基带传输（传输数字信号，采用曼彻斯特编码）。

| 名称 | 含义 |
|------|------|
| 10Base5 | 10Mbps，同轴电缆，最远500m |
| 10Base2 | 10Mbps，同轴电缆，最远200m（实际185m） |
| 10BaseT | 10Mbps，双绞线 |
| 10BaseF | 10Mbps，光纤 |
| 1000BaseT1 | 1000Mbps，双绞线 |
| 2.5GBaseT | 2.5Gbps，双绞线 |

---

## 考点总结

> 1. 双绞线分 UTP 和 STP，绞合度越高抗干扰越强
> 2. 同轴电缆只支持半双工，内导体越粗传输距离越长
> 3. 光纤分多模和单模，只支持全双工，带宽约500MHz
> 4. 电磁波频率越高传输能力越强，但指向性越强
> 5. 物理层接口四大特性：机械、电气、功能、过程
> 6. 以太网命名规则：速度+Base+介质
