## UDP 校验和计算

### 概念

**UDP 校验和**：检测 UDP 报文段在传输过程中是否出错。可选字段。

### 计算步骤

```
① 将伪首部、UDP 头、UDP 数据按 16 位分组
② 所有 16 位字按二进制反码求和
③ 对和取反码
④ 若结果为 0，置为全 1
```

### 伪首部

```
 0      7 8     15 16    23 24    31
+--------+--------+--------+--------+
|          源 IP 地址              |
+--------+--------+--------+--------+
|          目的 IP 地址            |
+--------+--------+--------+--------+
|  0  | 协议(17)|   UDP 长度        |
+--------+--------+--------+--------+
```

### 示例

数据：0x45 0x00 0x00 0x3C ...

分组：
- 4500
- 003C
- ...

求和：4500 + 003C + ... = 0xXXXX

校验和：~求和结果

## TCP 序号与确认号

### 序号（Sequence Number）

- 每个字节都有唯一序号
- 初始序号（ISN）随机生成
- 防止旧连接的混淆

### 确认号（Acknowledgment Number）

- 期望收到的下一个字节序号
- 累计确认

### 示例

```
A → B: SEQ=100, DATA="Hello" (5 字节)
B → A: ACK=105 (期望收到 105)

A → B: SEQ=105, DATA="World" (5 字节)
B → A: ACK=110
```

## 路由器结构

### 组成部分

1. **输入端口**
   - 接收物理信号
   - 数据链路层处理
   - 转发查找

2. **交换结构**
   - 内存交换
   - 总线交换
   - 交叉开关

3. **输出端口**
   - 排队
   - 数据链路层封装
   - 物理信号发送

4. **路由选择处理器**
   - 运行路由协议
   - 更新路由表

### 三种交换结构

| 类型 | 速度 | 复杂度 |
|------|------|--------|
| 内存交换 | 慢 | 低 |
| 总线交换 | 中 | 中 |
| 交叉开关 | 快 | 高 |

## CSMA/CA 协议

### 概念

**CSMA/CA**（Carrier Sense Multiple Access with Collision Avoidance）：无线局域网使用的 MAC 协议。

### 工作流程

```
① 监听信道
② 若信道空闲，等待 DIFS
③ 若仍空闲，发送数据
④ 接收方等待 SIFS 后发送 ACK
⑤ 发送方收到 ACK，确认成功
```

### RTS/CTS 机制

解决隐藏站问题：

```
① 发送 RTS（Request To Send）
② 接收方回复 CTS（Clear To Send）
③ 其他站收到 CTS 后退避
④ 发送数据
⑤ 接收方发送 ACK
```

### 帧间间隔

| 名称 | 长度 | 用途 |
|------|------|------|
| SIFS | 最短 | ACK、CTS、分片 |
| PIFS | 中等 | 点协调 |
| DIFS | 最长 | 普通数据帧 |

## CIDR 路由聚合

### 概念

**CIDR**（Classless Inter-Domain Routing）：无类域间路由，使用变长子网掩码（VLSM）。

### 表示法

```
192.168.0.0/22
```

- 22 表示网络前缀位数
- 主机位 = 32 - 22 = 10 位
- 主机数 = 2¹⁰ - 2 = 1022

### 路由聚合

将多个连续的网络块合并为一个更大的聚合块。

### 示例

```
192.168.0.0/24
192.168.1.0/24
192.168.2.0/24
192.168.3.0/24

聚合为 192.168.0.0/22
```

### 最长前缀匹配

路由查找时，选择**前缀最长**的路由条目。

### 路由聚合计算步骤

```
① 找出所有网络的公共前缀
② 公共前缀位数 = 聚合前缀长度
③ 计算聚合后的网络地址
```

### 示例

网络 1: 192.168.0000 0000.0/24
网络 2: 192.168.0000 0001.0/24
网络 3: 192.168.0000 0010.0/24
网络 4: 192.168.0000 0011.0/24

公共前缀：192.168.000000 00（22 位）

聚合：192.168.0.0/22
