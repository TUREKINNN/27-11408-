## 🎯 跨科目常见综合点

> 以下是 408 考试中常见的跨科目综合点（用户补充）

### 数据结构 + 计算机组成原理
- 数组在内存中的存储方式（顺序存储 vs 链式存储）
- 二叉树的顺序存储（与内存编址相关）
- 排序算法的硬件实现（比较器、寄存器）

### 数据结构 + 操作系统
- 进程/线程的栈和队列
- 内存管理中的链表/位图
- 文件系统的树形结构（B 树、B+ 树）
- 死锁的资源分配图

### 数据结构 + 计算机网络
- 路由表的二叉树查找
- B 树/B+ 树在文件系统中的应用
- 哈希表在网络设备（路由缓存）中的应用
- 滑动窗口协议（数据结构 + 协议）

### 计算机组成原理 + 操作系统
- 虚拟存储器（MMU + 页面管理）
- 中断机制（CPU + OS）
- 进程上下文切换（CPU 寄存器 + OS PCB）
- DMA（CPU + 设备管理）

### 计算机组成原理 + 计算机网络
- 网络接口卡（NIC）的硬件实现
- CRC 校验（硬件实现 + 数据链路层）
- 加密算法的硬件加速（AES/SHA）
- 协议栈的硬件卸载

### 操作系统 + 计算机网络
- 进程/线程在网络编程中的应用（socket）
- 拥塞控制（TCP 协议 + OS 缓冲区管理）
- 文件传输协议（FTP）的进程模型
- 网络安全（防火墙 + OS 权限）

### 数据结构 + 操作系统 + 计算机网络
- 进程间通信（管道、消息队列、socket）
- 分布式系统中的数据结构
- DNS 域名解析树
