> 以下是 408 考试中最高频的易错公式

<!--
### 1. 流水线总时间
- **正确公式**：T = kT_clk + (n-1)T_clk = (n + k - 1)T_clk
- **常见错误**：T = n × kT_clk（误以为每条都跑满 k 阶段）
- **记忆口诀**：第 1 条 k 阶段，之后每周期 1 条

### 2. Cache 命中率
- **正确公式**：H = 命中次数 / 总访问次数
- **平均访问时间**：T_a = H × T_c + (1-H) × T_m
- **常见错误**：把 Cache 访问时间和主存访问时间混淆

### 3. 平均周转时间
- **正确公式**：T = (T1 + T2 + ... + Tn) / n
- **带权周转时间**：W = T / 服务时间
- **常见错误**：带权周转忘记除以服务时间

### 4. 页面置换 - 缺页率
- **正确公式**：缺页率 = 缺页次数 / 总访问次数
- **常见错误**：把缺页率当命中率

### 5. 磁盘寻道时间
- **正确公式**：T = 寻道时间 + 旋转延迟 + 传输时间
- **常见错误**：忘记旋转延迟

### 6. 有效存取时间（TLB）
- **正确公式**：T_e = H × (T_TLB + T_m) + (1-H) × (T_TLB + 2T_m)
- **常见错误**：TLB 命中时不需要二次访存

### 7. 哈夫曼编码 - 编码长度
- **正确公式**：WPL = Σ(wi × li)
- **常见错误**：把 wi 写成频次而不是权重

### 8. 海明码校验位数
- **正确公式**：2^r ≥ k + r + 1（k 信息位，r 校验位）
- **常见错误**：忘记 +1

### 9. CRC 校验生成多项式
- **正确公式**：G(x) 的最高次幂 = 校验位位数
- **常见错误**：把生成多项式写反

### 10. 子网划分
- **正确公式**：子网数 = 2^n（n 为借用主机位数）
- **常见错误**：忘记 -2（去掉全 0 和全 1）

### 11. 信道利用率
- **正确公式**：U = 发送时间 / (发送时间 + 2×传播时延)
- **常见错误**：传播时延忘了 ×2

### 12. 滑动窗口 - 吞吐量
- **正确公式**：吞吐量 = 窗口大小 / RTT
- **常见错误**：忘记考虑带宽时延积

### 13. 拥塞窗口变化（TCP 慢启动）
- **正确公式**：每收到一个 ACK，cwnd = cwnd + 1（启动期）
- **拥塞避免**：每 RTT，cwnd = cwnd + 1
- **常见错误**：cwnd 翻倍 vs 加 1 混淆

### 14. 流水线吞吐率
- **正确公式**：TP = n / T_total = n / ((n + k - 1) × T_clk)
- **最大吞吐率**：TP_max = 1 / T_clk
- **常见错误**：没有减 1

### 15. 流水线加速比
- **正确公式**：S = 串行时间 / 流水线时间
- **常见错误**：忘记用串行作分母

### 16. Amdahl 定律
- **正确公式**：S = 1 / ((1-f) + f/n)
- **f**：可改进部分比例，n：改进部分加速比
- **常见错误**：f 和 1-f 弄反

### 17. 浮点数表示
- **正确公式**：N = (-1)^s × 1.M × 2^(E-127)
- **常见错误**：E 忘了减偏移量

### 18. 二叉树节点数
- **正确公式**：n = n_0 + n_1 + n_2 = n_0 + 2n_2 + 1
- **常见错误**：n_0 = n_2 + 1 这个公式只对非空二叉树成立

### 19. B+ 树阶数与节点关键字数
- **正确公式**：m 阶 B+ 树节点最多 m 个关键字，最少 ⌈m/2⌉
- **常见错误**：把 B 树和 B+ 树的约束搞混

### 20. 散列表平均查找长度
- **正确公式**：ASL = (Σ 查找次数) / 关键字总数
- **常见错误**：分母写成表长
