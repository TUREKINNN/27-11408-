## 💡 综合题答题模板

### 数据结构综合题模板

```
1. 看清题目要求（输入/输出/限制）
2. 选择合适的数据结构（数组/链表/树/图）
3. 算法设计：
   - 思路描述（先口头说）
   - 伪代码（用文字描述步骤）
   - 复杂度分析（时间+空间）
4. 代码实现（如有要求）
5. 边界测试（空/单元素/极端情况）
```

### 计算机组成原理综合题模板

```
1. 题目分析（涉及哪些知识点）
2. 公式列出（相关公式先写）
3. 代入计算（逐步求解）
4. 单位统一（注意 B/KB/MB/GB 转换）
5. 结果验证（量级检查）
```

### 操作系统综合题模板

```
1. 题目建模（进程/资源/信号量）
2. 算法选择（FCFS/SJF/RR/...）
3. 执行过程（逐步推演，画甘特图）
4. 计算指标（周转时间/带权周转时间/...）
5. 死锁/同步分析
```

### 计算机网络综合题模板

```
1. 题目分析（OSI/TCP-IP 哪一层）
2. 协议分析（TCP/UDP/IP/HTTP...）
3. 关键字段计算（IP 地址/子网掩码/端口...）
4. 状态转换（TCP 三次握手/四次挥手/...）
5. 性能指标（带宽/时延/利用率/...）
```
