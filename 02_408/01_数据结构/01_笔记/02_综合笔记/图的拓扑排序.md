# 图的拓扑排序

> [[02_408/01_数据结构/01_笔记/02_综合笔记/04-图]] | [[02_408/01_数据结构/01_笔记/02_综合笔记/图的关键路径]]

## 概念

**拓扑排序**：将有向无环图（DAG）的顶点排成线性序列，使得对于任意有向边 (u, v)，u 在 v 之前。

- 仅适用于**有向无环图（DAG）**
- 检测图中是否有环

## 算法步骤

```
① 选择一个入度为 0 的顶点并输出
② 从图中删除该顶点及其所有出边
③ 重复 ①②，直到所有顶点都被输出（成功）或不存在入度为 0 的顶点（失败）
```

## 示例

```
   A → B → D
   ↓       ↑
   C ------+
   
邻接表：
A: [B, C]
B: [D]
C: [D]
D: []

入度：A=0, B=1, C=1, D=2

步骤：
1. A 入度 0，输出 A，删除边 A→B, A→C
   入度：B=0, C=0, D=2
2. B 入度 0，输出 B，删除边 B→D
   入度：C=0, D=1
3. C 入度 0，输出 C，删除边 C→D
   入度：D=0
4. D 入度 0，输出 D

拓扑序列：A → B → C → D（或 A → C → B → D）
```

## 时间复杂度

O(V + E)，使用邻接表 + 栈实现

## 逆拓扑排序

使用**深度优先搜索（DFS）**的后序序列的逆序。

## 应用

- 课程安排（AOV 网）
- 任务调度
- 编译依赖分析
- 死锁检测
