# 二叉树的先中后序遍历

## 📊 考情分析
- **考试频率**：极高频，每年必考，是二叉树最核心的内容
- **分值**：选择题2-4分 + 综合题8-15分
- **题型**：选择题、算法设计题、综合应用题
- **真题考查方式**：手写遍历序列、由遍历序列推树结构、遍历算法的递归/非递归实现

## 一、基本概念

### 遍历的定义
**遍历**：按照某种次序把所有结点都访问一遍。

二叉树的递归特性决定了三种基本遍历方式：
- **先序遍历**（Pre-order）：**根**左右（NLR）
- **中序遍历**（In-order）：左**根**右（LNR）
- **后序遍历**（Post-order）：左右**根**（LRN）

> 三种遍历的区别仅在于**访问根结点的时机**不同

### 遍历规则

| 遍历方式 | 顺序 | 含义 |
|----------|------|------|
| 先序 | NLR | 先访问根，再遍历左子树，最后遍历右子树 |
| 中序 | LNR | 先遍历左子树，再访问根，最后遍历右子树 |
| 后序 | LRN | 先遍历左子树，再遍历右子树，最后访问根 |

## 二、核心知识

### 1. 递归遍历实现

```c
// 先序遍历
void PreOrder(BiTree T) {
    if (T != NULL) {
        visit(T);              // 访问根结点
        PreOrder(T->lchild);   // 递归遍历左子树
        PreOrder(T->rchild);   // 递归遍历右子树
    }
}

// 中序遍历
void InOrder(BiTree T) {
    if (T != NULL) {
        InOrder(T->lchild);    // 递归遍历左子树
        visit(T);              // 访问根结点
        InOrder(T->rchild);    // 递归遍历右子树
    }
}

// 后序遍历
void PostOrder(BiTree T) {
    if (T != NULL) {
        PostOrder(T->lchild);  // 递归遍历左子树
        PostOrder(T->rchild);  // 递归遍历右子树
        visit(T);              // 访问根结点
    }
}
```

### 2. 非递归中序遍历（借助栈）

**算法思想**：
1. 初始化一个栈
2. 沿着根的左链依次将左孩子入栈（一路向左）
3. 栈顶元素出栈并访问
4. 对该元素的右子树重复步骤2-3
5. 直到栈为空且当前指针为空

```c
void InOrder2(BiTree T) {
    InitStack(S);
    BiTree p = T;
    while (p || !IsEmpty(S)) {
        if (p) {               // 一路向左
            Push(S, p);
            p = p->lchild;
        } else {               // 出栈，转向右子树
            Pop(S, p);
            visit(p);
            p = p->rchild;
        }
    }
}
```

### 3. 非递归先序遍历（借助栈）

```c
void PreOrder2(BiTree T) {
    InitStack(S);
    BiTree p = T;
    while (p || !IsEmpty(S)) {
        if (p) {
            visit(p);          // 访问根结点（先序：先访问再入栈）
            Push(S, p);
            p = p->lchild;
        } else {
            Pop(S, p);
            p = p->rchild;
        }
    }
}
```

### 4. 非递归后序遍历（借助栈）

```c
void PostOrder2(BiTree T) {
    InitStack(S);
    BiTree p = T, r = NULL;  // r记录最近访问的结点
    while (p || !IsEmpty(S)) {
        if (p) {
            Push(S, p);
            p = p->lchild;
        } else {
            GetTop(S, p);
            if (p->rchild && p->rchild != r) {  // 右子树存在且未访问
                p = p->rchild;
            } else {                              // 可以访问根
                Pop(S, p);
                visit(p);
                r = p;
                p = NULL;
            }
        }
    }
}
```

## 三、深入理解

### 手算遍历的"分支展开法"

对于先序遍历，逐层展开：
```
      A
     / \
    B   C
   / \   \
  D   E   F
     /
    G
```

**先序（NLR）**：A B D E G C F

展开过程：
- 根 A → 访问A
- A 的左右：(左 B) (右 C)
- B 的左右：(左 D) (右 E)
- E 的左右：(左 G) (右 空)
- C 的左右：(左 空) (右 F)

**中序（LNR）**：D B G E A C F

**后序（LRN）**：D G E B F C A

### 时间复杂度分析
- 三种遍历每个结点恰好访问一次
- 时间复杂度：**$O(n)$**
- 空间复杂度（递归/栈）：**$O(h)$**，$h$ 为树的高度
- 最坏空间 $O(n)$（单支树），最好 $O(\log n)$（完全二叉树）

### 遍历序列的性质
- **中序遍历** + **先序/后序/层序之一** → 唯一确定一棵二叉树
- **仅凭一种遍历序列** → 不能唯一确定二叉树

## 四、真题考法分析

### 常考题型
1. 给定二叉树图形，写出先/中/后序遍历序列
2. 给定先序+中序（或后序+中序），构造二叉树
3. 非递归遍历算法的栈操作过程
4. 遍历算法的变形（如求树的高度、结点个数等）

### 命题规律
- 手写遍历序列是基础题
- 非递归算法实现是难题
- 遍历的递归应用（求高度、复制树等）是综合题常见考点

## 五、典型例题

### 概念题

**例1**：先序遍历和中序遍历相同的二叉树是什么形态？

**解**：所有结点只有右孩子的单支树（斜树）。因为先序=NLR，中序=LNR，只有当L始终为空时两者才相同。

**例2**：后序遍历序列和中序遍历序列正好相反的二叉树是什么形态？

**解**：每个结点最多只有左子树（左斜树）。

### 算法题

**例3**：用递归算法求二叉树的高度。

```c
int Depth(BiTree T) {
    if (T == NULL) return 0;
    int left = Depth(T->lchild);
    int right = Depth(T->rchild);
    return (left > right ? left : right) + 1;
}
```

**例4**：用先序遍历的思想，统计二叉树中叶子结点的个数。

```c
int LeafCount(BiTree T) {
    if (T == NULL) return 0;
    if (T->lchild == NULL && T->rchild == NULL)
        return 1;
    return LeafCount(T->lchild) + LeafCount(T->rchild);
}
```

**例5**：用先序遍历的思想，交换二叉树所有结点的左右子树。

```c
void SwapTree(BiTree T) {
    if (T == NULL) return;
    BiTree temp = T->lchild;
    T->lchild = T->rchild;
    T->rchild = temp;
    SwapTree(T->lchild);
    SwapTree(T->rchild);
}
```

### 真题型

**例6**：已知先序遍历为 ABDECFG，中序遍历为 DBEAFCG，画出二叉树。

**解**：
1. 先序第一个 A 为根 → 在中序中找到 A → 左子树 DBE，右子树 FCG
2. 左子树：先序 BDE，中序 DBE → 根 B，左 D，右 E
3. 右子树：先序 CFG，中序 FCG → 根 C，左 F，右 G
4. 得到完整二叉树

## 六、易错点

1. **先序≠从左到右**：先序是"根左右"，根先于子树被访问
2. **中序遍历用于BST**：二叉排序树的中序遍历是有序序列
3. **后序遍历非递归实现**：需要额外变量 $r$ 记录最近访问的结点
4. **非递归先序和中序的代码差异**：仅在 visit 的位置不同
5. **空树的处理**：递归终止条件是 `T == NULL`
6. **手算时别漏结点**：展开法需逐层展开，确保每个结点都被处理
