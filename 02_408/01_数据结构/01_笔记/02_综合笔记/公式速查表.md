## 一、时间复杂度常见公式

### 基本复杂度排序
$$O(1) < O(\log n) < O(n) < O(n\log n) < O(n^2) < O(n^3) < O(2^n) < O(n!)$$

### 常见算法复杂度

| 操作/算法 | 最好 | 平均 | 最坏 | 备注 |
|-----------|------|------|------|------|
| 顺序查找 | O(1) | O(n) | O(n) | |
| 折半查找 | O(1) | O(log₂n) | O(log₂n) | 要求有序 |
| BST查找 | O(log₂n) | O(log₂n) | O(n) | 最坏退化为链表 |
| AVL查找 | O(log₂n) | O(log₂n) | O(log₂n) | 始终平衡 |

### 嵌套循环复杂度计算
```
// 例1：单层循环
for(i=0; i<n; i++)   // O(n)

// 例2：两层嵌套
for(i=0; i<n; i++)
  for(j=0; j<n; j++)  // O(n²)

// 例3：不等嵌套
for(i=0; i<n; i++)
  for(j=0; j<i; j++)  // O(n²)，等差数列求和 n(n-1)/2

// 例4：倍增循环
for(i=1; i<n; i*=2)   // O(log₂n)

// 例5：嵌套倍增
for(i=1; i<n; i*=2)
  for(j=0; j<i; j++)  // O(n)
```

## 二、线性表相关公式

### 顺序表
- 插入平均移动次数：$E_{insert} = \frac{1}{n+1}\sum_{i=1}^{n+1}(n-i+1) = \frac{n}{2}$
- 删除平均移动次数：$E_{delete} = \frac{1}{n}\sum_{i=1}^{n}(n-i) = \frac{n-1}{2}$
- 按位查找：O(1)
- 按值查找：O(n)

### 链表
- 插入/删除：需先找到前驱，O(n)
- 头插法建表：O(n)，结果与输入顺序相反
- 尾插法建表：O(n)，结果与输入顺序相同

## 三、栈和队列公式

### 卡特兰数
n个不同元素进栈，出栈序列的不同排列数：
$$C_n = \frac{1}{n+1}\binom{2n}{n} = \frac{(2n)!}{(n+1)! \cdot n!}$$

| n | C_n |
|---|-----|
| 1 | 1 |
| 2 | 2 |
| 3 | 5 |
| 4 | 14 |
| 5 | 42 |

### 循环队列
- 判空条件：`front == rear`
- 判满条件（牺牲一个空间）：`(rear + 1) % MaxSize == front`
- 元素个数：`(rear - front + MaxSize) % MaxSize`
- 队满时实际存储元素数：`MaxSize - 1`

## 四、二叉树性质公式

### 五条基本性质

**性质1：** 第 i 层最多有 $2^{i-1}$ 个节点（i ≥ 1）

**性质2：** 深度为 k 的二叉树最多有 $2^k - 1$ 个节点

**性质3：** 对任何二叉树，若叶子节点数为 $n_0$，度为2的节点数为 $n_2$，则：
$$n_0 = n_2 + 1$$

**性质4：** 具有 n 个节点的完全二叉树的深度为：
$$h = \lfloor \log_2 n \rfloor + 1$$

**性质5：** 完全二叉树中，对编号为 i 的节点（n为总节点数，1 ≤ i ≤ n）：
- 父节点：$\lfloor i/2 \rfloor$（i=1时为根，无父节点）
- 左孩子：$2i$（若 $2i > n$ 则无左孩子）
- 右孩子：$2i+1$（若 $2i+1 > n$ 则无右孩子）

### 哈夫曼树

- **带权路径长度（WPL）：** $WPL = \sum_{i=1}^{n} w_i \cdot l_i$
  - $w_i$ 为叶子节点的权值
  - $l_i$ 为叶子节点到根的路径长度
- n 个叶子节点的哈夫曼树共有 $2n-1$ 个节点
- 哈夫曼树中不存在度为1的节点
- 构造过程：每次选取权值最小的两个节点合并

## 五、树与森林公式

### 树的基本公式
- 树中所有节点的度之和 = 边数 = 节点数 - 1
- n 个节点的树有 n-1 条边
- 度为 m 的树中，第 i 层最多有 $m^{i-1}$ 个节点
- 高度为 h 的 m 叉树最多有 $\frac{m^h - 1}{m - 1}$ 个节点

### 树与二叉树转换
- 树转二叉树：左孩子右兄弟
- 森林转二叉树：每棵树转后，根节点依次相连

## 六、图的性质公式

### 基本公式
- 无向图边数：$e = \frac{1}{2}\sum_{i=1}^{n} \deg(v_i)$
- 有向图边数：$e = \sum_{i=1}^{n} in\_deg(v_i) = \sum_{i=1}^{n} out\_deg(v_i)$
- 完全无向图边数：$e = \frac{n(n-1)}{2}$
- 完全有向图边数：$e = n(n-1)$
- 连通图最少边数：$n-1$（生成树）
- 强连通图最少边数：$n$（环）

### 存储空间
- 邻接矩阵：$O(n^2)$，适合稠密图
- 邻接表：$O(n+e)$，适合稀疏图

### 遍历复杂度
| 存储方式 | DFS | BFS |
|---------|-----|-----|
| 邻接矩阵 | O(n²) | O(n²) |
| 邻接表 | O(n+e) | O(n+e) |

### 最小生成树算法

| 算法 | 时间复杂度 | 适用场景 |
|------|-----------|---------|
| Prim | O(n²) | 稠密图 |
| Kruskal | O(e log₂e) | 稀疏图 |

### 最短路径算法

| 算法 | 时间复杂度 | 说明 |
|------|-----------|------|
| Dijkstra | O(n²) | 单源，不含负权边 |
| Floyd | O(n³) | 全源，不含负权回路 |

### 拓扑排序
- 时间复杂度：O(n + e)
- 可判断有向图中是否有环

### 关键路径
- 最早发生时间：$ve(j) = \max\{ve(i) + w_{ij}\}$
- 最迟发生时间：$vl(j) = \min\{vl(k) - w_{jk}\}$
- 最早开始时间：$e(i) = ve(\text{弧尾})$
- 最迟开始时间：$l(i) = vl(\text{弧头}) - w_i$
- 关键活动：$e(i) = l(i)$

## 七、查找算法ASL公式

### 顺序查找
- ASL成功 = $\frac{1}{n}\sum_{i=1}^{n}(n-i+1) = \frac{n+1}{2}$
- ASL失败 = $n+1$
- 设置哨兵后，ASL成功 = $\frac{n+1}{2}$（相同，但无需判断下界）

### 折半查找
- ASL成功 ≈ $\log_2(n+1) - 1$
- 判定树是一棵平衡二叉树
- 每次比较后搜索范围减半
- 时间复杂度：O(log₂n)

### 分块查找
- ASL = 块间查找ASL + 块内查找ASL
- 若分为 b 块，每块 s 个元素：
  - ASL成功 ≈ $\frac{b+1}{2} + \frac{s+1}{2} = \frac{n+s^2+s}{2s}$
  - 最优分块：$s = \sqrt{n}$，ASL ≈ $\sqrt{n} + 1$

### 二叉搜索树（BST）
- 平均ASL = O(log₂n)
- 最坏ASL = O(n)（退化为链表）

### 平衡二叉树（AVL）
- ASL = O(log₂n)
- n 个节点的AVL树的最大深度 ≈ 1.44 log₂(n+2)

### B树
- m阶B树每个节点最多 m 棵子树，m-1 个关键字
- 每个节点最少 ⌈m/2⌉ 棵子树，⌈m/2⌉-1 个关键字（根节点除外）
- 根节点至少2棵子树（除非是叶子节点）
- 所有叶子节点在同一层

### 散列表
- 装填因子：$\alpha = \frac{\text{表中记录数}}{\text{表长}}$
- 线性探测法：
  - ASL成功 ≈ $\frac{1}{2}(1 + \frac{1}{1-\alpha})$
  - ASL失败 ≈ $\frac{1}{2}(1 + \frac{1}{(1-\alpha)^2})$
- 链地址法：
  - ASL成功 ≈ $1 + \frac{\alpha}{2}$
  - ASL失败 ≈ $\alpha + e^{-\alpha}$

### 稳定性速记

**稳定的：**
> 直接插入、折半插入、冒泡排序、归并排序、基数排序
> 口诀：**"插冒归基"稳定**

**不稳定的：**
> 希尔排序、快速排序、简单选择排序、堆排序
> 口诀：**"希快选堆"不稳定**

## 九、其他常用结论

### 完全二叉树节点编号（从1开始）
- 节点 i 的父节点：⌊i/2⌋
- 节点 i 的左孩子：2i
- 节点 i 的右孩子：2i+1
- 节点 i 所在层次：⌊log₂i⌋ + 1

### 完全二叉树节点编号（从0开始）
- 节点 i 的父节点：⌊(i-1)/2⌋
- 节点 i 的左孩子：2i+1
- 节点 i 的右孩子：2i+2

### 数组存储（行优先/列优先）
- 二维数组 A[m][n]，元素 a[i][j] 的地址：
  - 行优先：$LOC(a[i][j]) = LOC(a[0][0]) + (i \times n + j) \times k$
  - 列优先：$LOC(a[i][j]) = LOC(a[0][0]) + (j \times m + i) \times k$
  - k 为每个元素占用的字节数

### 递推关系求解
- 二分递推：$T(n) = T(n/2) + O(1) \Rightarrow T(n) = O(\log n)$
- 分治递推：$T(n) = 2T(n/2) + O(n) \Rightarrow T(n) = O(n\log n)$
