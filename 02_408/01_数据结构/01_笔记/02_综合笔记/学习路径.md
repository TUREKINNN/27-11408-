# 408 数据结构 学习路径规划

## 总览

| 阶段 | 时间 | 内容 | 目标 |
|------|------|------|------|
| 基础阶段 | 第1-2周 | 绪论、线性表 | 打好基础，理解基本概念和操作 |
| 核心阶段 | 第3-5周 | 栈队列、树与二叉树、图 | 掌握核心数据结构，熟练算法 |
| 进阶阶段 | 第6-7周 | 查找、排序 | 掌握各类算法，能独立分析复杂度 |

## 基础阶段（第1-2周）

### 学习目标
- 理解数据结构的基本概念
- 掌握逻辑结构和存储结构的分类
- 熟练掌握线性表的顺序存储和链式存储
- 能独立编写线性表相关的算法题

### 第1周：数据结构绪论 + 线性表（上）

#### 第1-2天：数据结构绪论
**重点内容：**
- 数据、数据元素、数据项、数据对象的区别
- 四种逻辑结构：集合、线性、树形、图状
- 四种存储结构：顺序、链式、索引、散列
- 算法的五个特性：有穷性、确定性、可行性、输入、输出
- **时间复杂度的计算方法**（重中之重）
  - 常见复杂度排序：O(1) < O(logn) < O(n) < O(nlogn) < O(n²) < O(2^n) < O(n!)
  - 如何分析循环嵌套的时间复杂度
  - 递归算法的时间复杂度分析

**练习建议：**
- 做10道时间复杂度分析的选择题
- 手动分析3-5个嵌套循环的复杂度

#### 第3-5天：线性表 — 顺序表
**重点内容：**
- 顺序表的定义（静态/动态分配）
- 插入操作：时间复杂度 O(n)，平均移动 n/2 个元素
- 删除操作：时间复杂度 O(n)，平均移动 (n-1)/2 个元素
- 按位查找 O(1)，按值查找 O(n)
- 顺序表的特点：随机访问、存储密度高

**练习建议：**
- 手写顺序表的插入、删除、查找代码
- 做5道顺序表相关的算法设计题

#### 第6-7天：线性表 — 单链表
**重点内容：**
- 单链表的结构定义
- 头插法建表 vs 尾插法建表（区别、特点）
- 带头结点 vs 不带头结点（操作差异）
- 插入操作（前插法、后插法）
- 删除操作（找到前驱节点）
- 查找操作

**练习建议：**
- 手写单链表的增删查改代码
- 画图理解头插法和尾插法的过程
- 做5道链表操作的算法设计题

### 第2周：线性表（下）+ 巩固

#### 第1-2天：双链表、循环链表、静态链表
**重点内容：**
- 双链表的插入（四步操作）
- 双链表的删除（两步操作）
- 循环单链表的判空条件
- 循环双链表的判空条件
- 静态链表的概念

#### 第3-5天：线性表算法专题
**重点内容：**
- **线性表逆置**（多种方法：头插法、就地逆置）
- **有序表合并**（顺序表合并、链表合并）
- **删除特定元素**（删除值为x的元素、删除重复元素）
- **查找倒数第k个元素**（双指针法）
- **判断链表是否有环**（快慢指针法）

**练习建议：**
- 精做10道历年真题中的线性表大题
- 总结算法设计的常用技巧

#### 第6-7天：阶段复习
- 整理笔记，梳理知识框架
- 做一套线性表相关的模拟题
- 查漏补缺，回顾薄弱知识点

## 核心阶段（第3-5周）

### 学习目标
- 熟练掌握栈和队列的操作及应用
- 深入理解二叉树的遍历与性质
- 掌握图的各种算法
- 能独立完成中等难度的算法设计题

### 第3周：栈和队列

#### 第1-2天：栈的基础
**重点内容：**
- 栈的LIFO特性
- 顺序栈的实现（top指针、上溢下溢判断）
- 共享栈的实现
- 链栈的实现
- n个元素进栈的不同出栈序列数 = 卡特兰数 C(2n,n)/(n+1)

#### 第3-4天：栈的应用
**重点内容：**
- **括号匹配算法**（必须掌握）
- **中缀表达式转后缀表达式**（必须掌握）
- **后缀表达式求值**
- 递归与栈的关系
- 递归转非递归的方法

#### 第5-6天：队列
**重点内容：**
- 队列的FIFO特性
- **循环队列**（重中之重）
  - 判空条件：`front == rear`
  - 判满条件：`(rear+1)%MaxSize == front`
  - 元素个数：`(rear-front+MaxSize)%MaxSize`
- 链式队列
- 双端队列

#### 第7天：阶段练习
- 精做10道栈队列相关真题
- 手动模拟中缀转后缀的过程

### 第4周：树与二叉树

#### 第1-2天：二叉树基础与性质
**重点内容：**
- 二叉树的定义与五条性质
- 满二叉树与完全二叉树
- **完全二叉树的性质**：
  - 若有n个节点，高度 h = ⌊log₂n⌋ + 1
  - 编号i的父节点为 ⌊i/2⌋
  - 左孩子为 2i，右孩子为 2i+1
- 二叉树的顺序存储与链式存储

#### 第3-4天：二叉树遍历（核心重点）
**重点内容：**
- **先序遍历**（递归 + 非递归实现）
- **中序遍历**（递归 + 非递归实现）
- **后序遍历**（递归 + 非递归实现）
- **层次遍历**（队列实现）
- **由先序+中序构造二叉树**（必须掌握）
- **由后序+中序构造二叉树**（必须掌握）
- 遍历的应用：求树的高度、节点个数等

#### 第5天：线索二叉树
**重点内容：**
- 线索化的概念与目的
- 中序线索化过程
- 利用线索遍历二叉树

#### 第6天：树、森林与哈夫曼树
**重点内容：**
- 树的存储结构（双亲表示、孩子表示、孩子兄弟表示）
- 树↔二叉树的转换规则
- 森林↔二叉树的转换规则
- **哈夫曼树的构造过程**
- **WPL的计算**
- **哈夫曼编码**（前缀编码）

#### 第7天：阶段练习
- 精做10道二叉树相关真题
- 手动画出遍历过程

### 第5周：图

#### 第1-2天：图的基础与存储
**重点内容：**
- 图的基本术语（度、连通图、强连通图、完全图）
- **邻接矩阵**的存储与特点
- **邻接表**的存储与特点
- 邻接矩阵 vs 邻接表的对比

#### 第3天：图的遍历
**重点内容：**
- **DFS深度优先搜索**（递归实现，类似树的先序遍历）
- **BFS广度优先搜索**（队列实现，类似树的层次遍历）
- 遍历生成树/森林
- 时间复杂度分析

#### 第4天：最小生成树
**重点内容：**
- MST的性质（切割性质）
- **Prim算法**（从顶点出发，适合稠密图）
- **Kruskal算法**（从边出发，适合稀疏图）
- 两种算法的时间复杂度对比

#### 第5-6天：最短路径、拓扑排序、关键路径
**重点内容：**
- **Dijkstra算法**（单源最短路径，不能处理负权边）
- **Floyd算法**（全源最短路径，邻接矩阵递推）
- **拓扑排序**（AOV网，检测环）
- **关键路径**（AOE网，计算最早/最迟时间）

#### 第7天：阶段练习
- 精做10道图相关真题
- 手动模拟Prim、Dijkstra、拓扑排序的过程

## 进阶阶段（第6-7周）

### 学习目标
- 掌握各类查找算法及其ASL计算
- 熟练掌握各种排序算法的过程与复杂度
- 能够分析比较各种算法的优劣

### 第6周：查找

#### 第1天：基本查找方法
**重点内容：**
- 顺序查找及ASL = (n+1)/2
- **折半查找**及ASL（必须掌握判定树画法）
- 分块查找及ASL

#### 第2-3天：树表查找
**重点内容：**
- **二叉搜索树（BST）**的插入、删除、查找
- BST在最坏情况下退化为O(n)
- **AVL树**的四种旋转：
  - LL（右旋）、RR（左旋）、LR（先左旋再右旋）、RL（先右旋再左旋）
- AVL树的插入与调整
- AVL树的删除与调整
- 红黑树的基本性质（了解即可）

#### 第4-5天：B树、B+树与散列表
**重点内容：**
- **B树**的定义（m阶B树的性质）
- B树的插入（分裂）和删除（合并/借）
- **B+树**的定义及与B树的区别
- **散列表**的构造方法
- **开放定址法**（线性探测、二次探测）—— ASL计算
- **链地址法** —— ASL计算
- 装填因子α = 表中记录数/表长

#### 第6-7天：阶段练习
- 精做10道查找相关真题
- 练习AVL旋转、B树插入删除的手动模拟

### 第7周：排序

#### 第1-2天：插入排序与交换排序
**重点内容：**
- **直接插入排序**：稳定，O(n²)，最好O(n)
- 折半插入排序：稳定，O(n²)
- **希尔排序**：不稳定，O(n^1.3)
- **冒泡排序**：稳定，O(n²)，最好O(n)
- **快速排序**：不稳定，平均O(nlogn)，最坏O(n²)
  - 枢轴选取、划分过程
  - 最坏情况：已有序时退化

#### 第3天：选择排序
**重点内容：**
- 简单选择排序：不稳定，O(n²)
- **堆排序**（重中之重）：
  - 大根堆与小根堆的定义
  - **建堆过程**（从最后一个非叶子节点开始调整）
  - 堆的插入与删除
  - 排序过程：O(nlogn)

#### 第4天：归并排序与基数排序
**重点内容：**
- **归并排序**：稳定，O(nlogn)，空间O(n)
- 基数排序：稳定，O(d(n+r))
- 各排序算法的稳定性总结

#### 第5-6天：排序综合与外部排序
**重点内容：**
- 排序算法的对比（时间、空间、稳定性、适用场景）
- 外部排序：多路归并、败者树
- 置换-选择排序、最佳归并树

#### 第7天：阶段练习
- 精做10道排序相关真题
- 手动模拟快速排序、建堆、归并排序的过程

## 冲刺阶段（第8周）

### 学习目标
- 系统复习所有知识点
- 查漏补缺
- 提升解题速度和准确率
- 适应考试节奏

### 每日安排

#### 第1天：绪论 + 线性表回顾
- 30分钟回顾思维导图
- 做2套真题/模拟题中对应部分
- 整理错题

#### 第2天：栈队列 + 串回顾
- 重点回顾循环队列判空判满、中缀转后缀、KMP
- 做2套真题/模拟题中对应部分

#### 第3天：树与二叉树回顾
- 重点回顾遍历、哈夫曼、线索化
- 做2套真题/模拟题中对应部分

#### 第4天：图回顾
- 重点回顾Dijkstra、Prim、拓扑排序、关键路径
- 做2套真题/模拟题中对应部分

#### 第5天：查找回顾
- 重点回顾AVL旋转、B树、散列表ASL
- 做2套真题/模拟题中对应部分

#### 第6天：排序回顾
- 重点回顾快排、堆排序、稳定性
- 做2套真题/模拟题中对应部分

#### 第7天：全真模拟
- 完整做一套408数据结构真题（限时90分钟）
- 对答案，分析错因
- 最后查漏补缺

### 冲刺阶段注意事项
1. **回归基础**：不要钻偏题怪题，把基础题做到满分
2. **重视代码**：大题中的代码填空和算法设计要多练
3. **时间管理**：选择题控制在30分钟内完成
4. **心态调整**：保持规律作息，不要过度焦虑

## 学习小贴士

1. **画图理解**：数据结构一定要画图，特别是链表、树、图的操作过程
2. **手写代码**：不要只看代码，一定要自己动手写
3. **分析复杂度**：每个算法都要能分析时间复杂度和空间复杂度
4. **对比记忆**：容易混淆的概念要对比记忆（如B树vs B+树）
5. **真题为王**：历年真题是最好的练习材料，反复刷
6. **定期复习**：利用艾宾浩斯遗忘曲线，定期回顾已学内容
