# 6.3.1 图的广度优先遍历（BFS）

## 核心思想

类似于树的**层序遍历**，逐层向外扩展访问。

### 树的层序遍历

1. 若树非空，则根节点**入队**
2. 若队列非空，队头元素**出队并访问**，同时将其孩子依次入队
3. 重复②直到队列为空

### 图的 BFS 与树的区别

图中可能存在**回路**，搜索相邻顶点时可能搜到**已访问过**的顶点，需要额外的 `visited[]` 数组标记。

## 算法实现

### 关键辅助函数

- `FirstNeighbor(G, x)`：求图 G 中顶点 x 的**第一个**邻接点
- `NextNeighbor(G, x, y)`：返回顶点 x 除 y 之外的**下一个**邻接点

### BFS 算法（单连通分量）

```c
bool visited[MAX_VERTEX_NUM];  // 访问标记数组
Queue Q;                        // 辅助队列

void BFS(Graph G, int v) {
    visit(v);                   // 访问初始顶点
    visited[v] = true;
    Enqueue(Q, v);
    while (!isEmpty(Q)) {
        Dequeue(Q, v);
        for (w = FirstNeighbor(G, v); w >= 0; 
             w = NextNeighbor(G, v, w)) {
            if (!visited[w]) {
                visit(w);
                visited[w] = true;
                Enqueue(Q, w);
            }
        }
    }
}
```

### BFS 算法（Final 版 - 处理非连通图）

```c
void BFSTraverse(Graph G) {
    for (int i = 0; i < G.vexnum; i++)
        visited[i] = false;
    InitQueue(Q);
    for (int i = 0; i < G.vexnum; i++)  // 遍历所有顶点
        if (!visited[i])
            BFS(G, i);                   // 对每个未访问的连通分量调用BFS
}
```

> - 对**无向图**，调用 BFS 函数的次数 = **连通分量数**
> - 对**有向图**，调用次数要具体分析；对于**强连通图**只需调用 1 次

## BFS 遍历过程示例

```
图结构（8个顶点）：
1—2—3—4
|  |  |  |
5—6—7—8
```

从顶点 1 出发的 BFS 序列：`1, 2, 5, 6, 3, 7, 4, 8`

## 遍历序列的可变性

| 存储结构 | BFS 序列是否唯一 |
|----------|-----------------|
| **邻接矩阵** | **唯一**（表示方式唯一） |
| **邻接表** | **不唯一**（表示方式不唯一） |

## 复杂度分析

### 时间复杂度

| 存储结构 | 访问顶点 | 查找邻接点 | 总时间复杂度 |
|----------|----------|-----------|-------------|
| **邻接矩阵** | O(|V|) | 每个顶点 O(|V|)，共 |V| 个 | **O(|V|²)** |
| **邻接表** | O(|V|) | 查找各顶点邻接点共 O(|E|) | **O(|V|+|E|)** |

### 空间复杂度

- 最坏情况：辅助队列大小为 **O(|V|)**

## 广度优先生成树（森林）

由 BFS 遍历过程确定的生成树：

- **邻接矩阵**表示：生成树**唯一**
- **邻接表**表示：生成树**不唯一**
- 对**非连通图**的 BFS，得到**广度优先生成森林**

> 以起始顶点为根，是高度**最小**的生成树

## BFS 求无权图单源最短路径

BFS 天然按层访问，可以求**无权图**中从源点到各顶点的最短路径：

- `d[]`：记录从源点到各顶点的最短路径长度
- `path[]`：记录路径上的前驱结点

```c
void BFS_MIN_Distance(Graph G, int u) {
    // d[i] 表示从 u 到 i 的最短路径长度
    // path[i] 表示最短路径中 i 的前驱
    for (int i = 0; i < G.vexnum; i++) {
        d[i] = ∞;      // 初始化为无穷
        path[i] = -1;   // 无前驱
    }
    d[u] = 0;
    visited[u] = true;
    Enqueue(Q, u);
    while (!isEmpty(Q)) {
        Dequeue(Q, u);
        for (w = FirstNeighbor(G, u); w >= 0;
             w = NextNeighbor(G, u, w)) {
            if (!visited[w]) {
                d[w] = d[u] + 1;    // 路径长度+1
                path[w] = u;         // 记录前驱
                visited[w] = true;
                Enqueue(Q, w);
            }
        }
    }
}
```
