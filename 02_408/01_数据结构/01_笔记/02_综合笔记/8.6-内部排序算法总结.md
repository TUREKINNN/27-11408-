# 8.6 内部排序算法总结

> [[02_408/01_数据结构/01_笔记/02_综合笔记/06-排序]] | [[02_408/01_数据结构/01_笔记/02_综合笔记/8.7-外部排序]]

## 排序算法对比

| 算法 | 最好 | 平均 | 最坏 | 空间 | 稳定性 |
|------|------|------|------|------|--------|
| 直接插入 | O(n) | O(n²) | O(n²) | O(1) | ✅ 稳定 |
| 折半插入 | O(n log n) | O(n²) | O(n²) | O(1) | ✅ 稳定 |
| 希尔排序 | O(n log n) | O(n^1.3) | O(n²) | O(1) | ❌ 不稳定 |
| 冒泡排序 | O(n) | O(n²) | O(n²) | O(1) | ✅ 稳定 |
| 快速排序 | O(n log n) | O(n log n) | O(n²) | O(log n) | ❌ 不稳定 |
| 简单选择 | O(n²) | O(n²) | O(n²) | O(1) | ❌ 不稳定 |
| 堆排序 | O(n log n) | O(n log n) | O(n log n) | O(1) | ❌ 不稳定 |
| 归并排序 | O(n log n) | O(n log n) | O(n log n) | O(n) | ✅ 稳定 |
| 基数排序 | O(d(n+r)) | O(d(n+r)) | O(d(n+r)) | O(n+r) | ✅ 稳定 |

## 关键特点

### 基于比较的排序

- **下界**：O(n log n)
- 包括：插入、交换、选择、归并

### 不基于比较的排序

- **基数排序**：按位分配，不受 O(n log n) 下界限制
- **计数排序**、**桶排序**：也是不基于比较

### 稳定性分析

| 稳定排序 | 不稳定排序 |
|----------|-----------|
| 直接插入、折半插入 | 希尔排序 |
| 冒泡排序 | 快速排序 |
| 归并排序 | 简单选择、堆排序 |
| 基数排序 | |

### 适用场景

| 场景 | 推荐算法 |
|------|----------|
| n 较小 | 直接插入 |
| 基本有序 | 直接插入 |
| n 较大 | 快速排序 |
| 要求稳定 | 归并排序 |
| 要求原地 | 快速排序/堆排序 |
