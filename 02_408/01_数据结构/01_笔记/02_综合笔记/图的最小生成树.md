# 图的最小生成树

> [[02_408/01_数据结构/01_笔记/02_综合笔记/04-图]] | [[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/索引]]

## 核心概念

**最小生成树（Minimum Spanning Tree, MST）**：对于带权连通无向图，生成树中边的权值之和最小的树。

- **生成树**：包含图中所有顶点的极小连通子图
- **顶点数 = n，边数 = n-1**
- **不唯一**：可能有多棵最小生成树

## 性质

1. 最小生成树的边权之和唯一，但树形不唯一
2. 任意一棵最小生成树都包含权值最小的边
3. 若图有 n 个顶点，则 MST 有 n-1 条边
4. MST 中不能有回路

## Prim 算法（普里姆）

### 思想

从任一顶点出发，逐步扩展，每次选择**连接已选顶点集与未选顶点集的权值最小**的边。

### 步骤

```
① 初始化：已选集合 U = {v0}，候选边集 = 与 v0 相连的所有边
② 循环 n-1 次：
   - 从候选边中选权值最小的边 (u, v)，u∈U, v∉U
   - 将 v 加入 U
   - 更新候选边集
```

### 时间复杂度

- 邻接矩阵：O(V²)
- 邻接表 + 优先队列：O(E log V)

### 示例

```
       1
   A ----- B
   | \   / |
  3|  \ / 4|2
   |   X   |
   D ----- C
      5

Prim 从 A 开始：
U={A}, 候选边：(A,B)=1, (A,C)=4, (A,D)=3
选 (A,B)=1, U={A,B}
候选边：(A,C)=4, (A,D)=3, (B,C)=2
选 (B,C)=2, U={A,B,C}
候选边：(A,D)=3, (B,D)=?
选 (A,D)=3, U={A,B,C,D}
MST = {(A,B), (B,C), (A,D)} 权值=6
```

## Kruskal 算法（克鲁斯卡尔）

### 思想

按**边权递增**的顺序选择边，每次选权值最小且**不构成回路**的边。

### 步骤

```
① 将所有边按权值从小到大排序
② 初始化：每个顶点是一个独立的集合
③ 遍历排序后的边：
   - 若边的两个顶点不在同一集合，则选择该边
   - 合并两个顶点的集合
④ 直到选了 n-1 条边
```

### 时间复杂度

- 排序：O(E log E)
- 并查集：O(E α(V))
- 总：O(E log E)

### 示例

```
边按权值排序：
(A,B)=1, (B,C)=2, (A,D)=3, (A,C)=4, (B,D)=?, (C,D)=5

选 (A,B)=1：并集 {A,B}
选 (B,C)=2：并集 {A,B,C}
选 (A,D)=3：并集 {A,B,C,D}
(A,C)=4：跳过（A、C 已连通）
(C,D)=5：跳过
MST = {(A,B), (B,C), (A,D)} 权值=6
```

## 算法对比

| 项目 | Prim | Kruskal |
|------|------|---------|
| 思想 | 点扩展 | 边选择 |
| 适合 | 稠密图 | 稀疏图 |
| 复杂度 | O(V²) | O(E log E) |
| 数据结构 | 邻接矩阵/优先队列 | 并查集 |
