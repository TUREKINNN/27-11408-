# 由遍历序列构造二叉树

## 📊 考情分析
- **考试频率**：高频必考
- **分值**：选择题2-4分或综合题中的一部分
- **题型**：选择题、手算题
- **真题考查方式**：由前序+中序推导二叉树、由后序+中序推导二叉树、判断遍历序列是否合法

## 一、基本概念

### 核心结论

| 给定的遍历序列 | 能否唯一确定二叉树 |
|---------------|------------------|
| 仅前序 | ❌ 不能 |
| 仅中序 | ❌ 不能 |
| 仅后序 | ❌ 不能 |
| 仅层序 | ❌ 不能 |
| **前序 + 中序** | ✅ **能** |
| **后序 + 中序** | ✅ **能** |
| **层序 + 中序** | ✅ **能** |
| 前序 + 后序 | ❌ 不能（除非只有一个孩子） |

> **关键**：必须包含**中序遍历**才能唯一确定二叉树

### 原因分析
- **前序/后序/层序**：只能确定根的位置，但无法区分左右子树
- **中序**：能明确区分哪些结点在左子树、哪些在右子树

## 二、核心知识

### 方法一：前序 + 中序 构造二叉树

**基本原理**：
- 前序遍历：**根** 左子树 右子树
- 中序遍历：左子树 **根** 右子树

**步骤**：
1. 前序序列的**第一个元素**是根结点
2. 在中序序列中找到根的位置，左侧为左子树的中序，右侧为右子树的中序
3. 根据左子树的长度，在前序序列中划分出左子树的前序和右子树的前序
4. **递归**处理左右子树

**示例**：
```
前序：A B D C E F    中序：D B A E C F
```

步骤：
1. 前序第一个 A → 根为 A
2. 中序中 A 的位置：左侧 DB 为左子树，右侧 ECF 为右子树
3. 左子树有2个结点 → 前序中取 BD 为左子树前序
4. 右子树有3个结点 → 前序中取 CEF 为右子树前序
5. 递归：
   - 左子树：前序 BD，中序 DB → 根 B，左 D，右空
   - 右子树：前序 CEF，中序 ECF → 根 C，左 E，右 F

```
    A
   / \
  B   C
 /   / \
D   E   F
```

### 方法二：后序 + 中序 构造二叉树

**基本原理**：
- 后序遍历：左子树 右子树 **根**
- 中序遍历：左子树 **根** 右子树

**步骤**：
1. 后序序列的**最后一个元素**是根结点
2. 在中序序列中找到根的位置，划分左右子树
3. 根据左子树长度划分后序序列
4. 递归处理

**示例**：
```
后序：D B E F C A    中序：D B A E C F
```

步骤：
1. 后序最后 A → 根为 A
2. 中序中 A 左侧 DB 为左子树，右侧 ECF 为右子树
3. 左子树2个 → 后序前2个 DB 为左子树后序
4. 右子树3个 → 后序中3个 BEFC → BEF 为右子树后序
5. 递归构造

### 方法三：层序 + 中序 构造二叉树

**步骤**：
1. 层序第一个为根
2. 在中序中确定左右子树
3. 根据左右子树的结点，在层序中分离出属于左/右子树的结点
4. 递归处理

## 三、深入理解

### 设计思想
- 构造过程本质上是**分治法**：每次确定根，将问题分解为左右子树的子问题
- 关键在于利用**中序遍历的分割特性**

### 复杂度分析
- 时间复杂度：$O(n^2)$（朴素实现，每次在中序中线性查找根）或 $O(n)$（用哈希表预处理）
- 空间复杂度：$O(n)$（递归栈 + 存储结构）

## 四、真题考法分析

### 常考题型
1. 给定前序+中序，手动画出二叉树
2. 给定后序+中序，写出前序遍历序列
3. 判断给定的遍历序列对能否构造出唯一的二叉树
4. 给定层序+中序，构造二叉树

### 命题规律
- 前序+中序是最常考的组合
- 后序+中序次之
- 前序+后序不能唯一确定是常考的判断题

## 五、典型例题

### 计算题

**例1**：已知前序 ABDECF，中序 DBEACF。画出二叉树，并写出后序遍历序列。

**解**：
1. 前序第一个 A → 根 A
2. 中序 A 左侧 DBE → 左子树，右侧 CF → 右子树
3. 左子树3个结点：前序 BDE，中序 DBE
   - 根 B，左 D，右 E
4. 右子树2个结点：前序 CF，中序 CF
   - 根 C，左 F（在中序中C的左侧），右空
5. 二叉树：
```
    A
   / \
  B   C
 / \ /
D  E F
```
6. 后序遍历：**D E B F C A**

**例2**：已知后序 DEBFCA，中序 DBEACF。写出前序遍历序列。

**解**：同上构造，前序为 **ABDECF**

### 真题型

**例3**：一棵二叉树的中序遍历为 ABCDEFG，后序遍历为 BDCAFGE。求前序遍历。

**解**：
1. 后序最后 E → 根 E
2. 中序 E 左侧 ABCD → 左子树，右侧 FG → 右子树
3. 左子树4个：后序 BDC A，中序 ABCD
   - 根 A，左侧空，右侧 BCD
   - 后序 BDC，中序 BCD → 根 C，左 B，右 D
4. 右子树2个：后序 GE → 去掉E后为 G...，中序 FG
   - 根 G，左 F，右空
5. 前序：**E A C B D G F**

## 六、易错点

1. **前序+后序不能唯一确定二叉树**：除非限制每个结点只有一个孩子
2. **中序是关键**：没有中序就不能区分左右子树
3. **划分长度要准确**：左子树长度 = 中序中根的位置 - 中序起始位置
4. **递归边界**：空子树时递归终止
5. **手算时先确定根再划分**：不要跳步
6. **多解情况**：前序+后序可能对应多种二叉树（当某结点只有一个孩子时）
