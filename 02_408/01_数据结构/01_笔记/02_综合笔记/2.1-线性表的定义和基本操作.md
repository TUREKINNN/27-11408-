## 一、基本概念

### 1.1 线性表的定义

线性表是具有**相同数据类型**的 $n$（$n \geq 0$）个数据元素的**有限序列**，其中 $n$ 为表长，当 $n = 0$ 时线性表是一个空表。

若用 $L$ 命名线性表，则其一般表示为：

$$L = (a_1, a_2, \dots, a_i, a_{i+1}, \dots, a_n)$$

### 1.2 基本术语

| 术语 | 说明 |
|------|------|
| **表长** | $n$，线性表中数据元素的个数 |
| **空表** | $n=0$ 时的线性表 |
| **位序** | $a_i$ 是线性表中的"第 $i$ 个"元素，**位序从1开始** |
| **表头元素** | $a_1$ |
| **表尾元素** | $a_n$ |
| **直接前驱** | 除第一个元素外，每个元素有且仅有一个直接前驱 |
| **直接后继** | 除最后一个元素外，每个元素有且仅有一个直接后继 |

> ⚠️ **易混淆点**：位序从1开始，数组下标从0开始

### 1.3 线性表的关键特性

1. **相同数据类型**：每个数据元素所占空间一样大
2. **有限**：元素个数有限
3. **有序**：元素之间有先后次序
4. **同构**：所有元素类型相同

> 反例：所有整数按递增次序排列——不是线性表（无限）

## 二、核心知识

### 2.1 线性表的基本操作

| 操作 | 功能描述 | 分类 |
|------|----------|------|
| `InitList(&L)` | 初始化表，构造空线性表L，分配内存空间 | 创 |
| `DestroyList(&L)` | 销毁线性表，释放内存空间 | 销 |
| `ListInsert(&L,i,e)` | 在表L第i个位置插入元素e | 增 |
| `ListDelete(&L,i,&e)` | 删除表L第i个元素，用e返回删除值 | 删 |
| `LocateElem(L,e)` | 按值查找，查找关键字为e的元素 | 查 |
| `GetElem(L,i)` | 按位查找，获取第i个位置的元素值 | 查 |
| `Length(L)` | 求表长，返回元素个数 | 辅助 |
| `PrintList(L)` | 输出操作，按顺序输出所有元素 | 辅助 |
| `Empty(L)` | 判空操作，空表返回true | 辅助 |

> 💡 **记忆思路**：创销、增删改查

### 2.2 为什么要实现基本操作？

1. **团队合作编程**：封装数据结构，方便他人使用
2. **避免重复工作**：常用操作封装成函数，降低出错风险

### 2.3 引用参数 "&" 的使用

**核心规则**：对参数的修改结果需要"带回来"时，使用引用 `&`

#### 不使用引用（值传递）
```c
void test(int x) {
    x = 1024;  // 修改的是副本
    printf("test函数内部 x=%d\n", x);
}

int main() {
    int x = 1;
    test(x);
    printf("main函数中 x=%d\n", x);  // 输出1，修改未带回来
}
```

#### 使用引用（引用传递）
```c
void test(int &x) {
    x = 1024;  // 直接修改原变量
    printf("test函数内部 x=%d\n", x);
}

int main() {
    int x = 1;
    test(x);
    printf("main函数中 x=%d\n", x);  // 输出1024，修改带回来了
}
```

### 2.4 函数定义规范

```c
<返回值类型> 函数名(<参数1类型> 参数1, <参数2类型> 参数2, ...)
```

**Tips**：
- 函数名和参数的形式、命名都可改变（参考严蔚敏版《数据结构》）
- 命名要有可读性
- 课本不会说明各个参数的具体类型（抽象描述）

## 三、深入理解

### 3.1 设计思想

- **抽象数据类型（ADT）**：线性表的定义是逻辑层面的，不涉及具体存储方式
- **存储结构不同，运算的实现方式不同**：顺序存储和链式存储的实现差异很大
- **封装思想**：将数据结构和操作封装，对外提供接口

### 3.2 数据结构三要素

| 要素 | 说明 |
|------|------|
| **逻辑结构** | 线性表是线性结构 |
| **数据的运算** | 创销、增删改查 |
| **存储结构（物理结构）** | 顺序存储 / 链式存储 |

## 五、典型例题

### 概念题

**Q1**：下列关于线性表的说法，正确的是（）
- A. 线性表的元素可以是不同数据类型
- B. 线性表的长度可以是无限的
- C. 除第一个和最后一个元素外，每个元素有且仅有一个前驱和一个后继
- D. 线性表必须用数组实现

**答案**：C

**解析**：
- A错：必须是相同数据类型
- B错：线性表是有限序列
- C正确：准确描述了线性表的特性
- D错：可以用顺序表或链表实现

### 算法题

**Q2**：已知线性表L，设计一个操作将L中的元素逆置。

**思路**：利用基本操作 `GetElem` 和 `ListInsert`，或使用两个指针从两端向中间遍历交换。

## 六、易错点

1. **位序从1开始，数组下标从0开始**
2. **引用参数的使用**：`InitList(&L)`、`ListInsert(&L,i,e)` 等需要修改L的操作必须用 `&`
3. **线性表 ≠ 数组**：线性表是逻辑结构，数组是存储结构
4. **空表不是不存在**：空表是存在的，只是长度为0
