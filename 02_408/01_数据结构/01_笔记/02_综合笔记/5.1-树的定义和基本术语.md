# 树的定义和基本术语

## 📊 考情分析
- **考试频率**：基础概念题，几乎每年都会涉及
- **分值**：通常2-4分（选择题）
- **题型**：选择题为主，偶尔出现在综合题的基本概念部分
- **真题考查方式**：判断结点关系、计算度/深度/路径长度等基本概念辨析

## 一、基本概念

### 1. 树的严格定义

树是 $n$（$n \geq 0$）个结点的有限集合，$n = 0$ 时称为**空树**。在任意一棵非空树中应满足：
1. 有且仅有一个特定的称为**根**的结点
2. 当 $n > 1$ 时，其余结点可分为 $m$（$m > 0$）个互不相交的有限集合 $T_1, T_2, \ldots, T_m$，其中每个集合本身又是一棵树，并且称为根结点的**子树**

> **关键**：树是一种**递归定义**的数据结构

### 2. 非空树的特性
- 有且仅有一个根节点
- 没有后继的结点称为"**叶子结点**"（终端结点）
- 有后继的结点称为"**分支结点**"（非终端结点）
- 除了根节点外，任何结点都有且仅有一个**前驱**
- 每个结点可以有0个或多个**后继**

### 3. 基本术语

| 术语 | 定义 |
|------|------|
| **结点的度** | 结点有几个孩子（分支） |
| **树的度** | 各结点的度的最大值 |
| **叶子结点** | 度为0的结点 |
| **分支结点** | 度大于0的结点 |
| **结点的层次（深度）** | 从上往下数，默认从1开始 |
| **结点的高度** | 从下往上数 |
| **树的高度（深度）** | 总共多少层 |
| **路径** | 两个结点之间的路径只能**从上往下** |
| **路径长度** | 经过几条边 |
| **祖先结点** | 从根到该结点路径上的所有结点 |
| **子孙结点** | 以某结点为根的子树中的所有结点 |
| **双亲结点（父节点）** | 直接前驱 |
| **孩子结点** | 直接后继 |
| **兄弟结点** | 同一个父节点的结点 |
| **堂兄弟结点** | 同一层但父节点不同的结点 |

### 4. 有序树 vs 无序树

- **有序树**：逻辑上，树中结点的各子树从左至右是有次序的，不能互换
- **无序树**：逻辑上，树中结点的各子树从左至右是无次序的，可以互换

> 判断有序/无序取决于具体应用场景，看结点的左右位置是否需要反映某些逻辑关系

### 5. 树与森林

- **森林**：$m$（$m \geq 0$）棵互不相交的树的集合
- 空森林：$m = 0$
- 把树的根去掉，剩余子树构成森林
- 给森林中各棵树加上一个共同的根，森林变成树

## 二、核心知识

### 树的逻辑结构特征
- **层次关系**：树体现的是一对多的层次关系
- **递归性**：树的定义是递归的，子树也是树
- **唯一前驱**：除根外每个结点有且仅有一个前驱
- **可多后继**：每个结点可以有0个或多个后继

### 树形结构的应用
- 文件系统的目录结构
- 组织架构图
- 家谱
- HTML DOM树

## 三、深入理解

### 树与其他数据结构的对比

| 特性 | 线性表 | 树 |
|------|--------|-----|
| 逻辑关系 | 一对一 | 一对多 |
| 前驱个数 | 最多1个（除首元素） | 根无前驱，其余恰好1个 |
| 后继个数 | 最多1个 | 可以多个 |
| 层次性 | 无 | 有 |

### 树的基本运算
- 求树的深度
- 求结点的度
- 遍历树的所有结点
- 查找某结点的父节点/孩子

## 四、真题考法分析

### 常考题型
1. 给定树的图形，判断各结点的度、层次、深度
2. 判断结点之间的关系（父子、兄弟、祖先等）
3. 计算树的路径长度
4. 有序树与无序树的辨析

### 命题规律
- 概念辨析题常出现在选择题前几道
- 常与树的性质计算结合考查

## 五、典型例题

### 概念题
**例1**：一棵树中，结点A有3个孩子B、C、D，结点B有2个孩子E、F。则：
- A的度为3
- B的度为2
- 叶子结点有：E、F、C、D
- 树的度为3

### 计算题
**例2**：一棵树有 $n$ 个结点，则树中边的条数为 $n-1$。

**证明**：除根结点外，每个结点恰好有一条边连接其父节点，所以边数 = $n - 1$。

### 易混淆概念
**例3**：区分"结点的层次"和"结点的高度"
- 层次（深度）：从根开始**从上往下**数
- 高度：从叶子开始**从下往上**数

## 六、易错点

1. **路径方向**：树中两个结点之间的路径只能**自上而下**，不能反向
2. **空树**：$n=0$ 的空树也是树的合法情况
3. **子树互不相交**：一棵树的各棵子树之间不能有交集
4. **度为0**：叶子结点的度为0，但它是树的一部分
5. **层次起始**：默认从1开始计数，也有教材从0开始，注意题目要求
6. **有序树 vs 二叉树**：度为2的有序树不等于二叉树（二叉树允许为空，度为2的有序树至少3个结点）
