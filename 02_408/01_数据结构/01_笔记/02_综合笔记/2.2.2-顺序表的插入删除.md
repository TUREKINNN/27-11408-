# 顺序表的插入删除

## 一、基本概念

### 1.1 插入操作

**ListInsert(&L, i, e)**：在表L中的第 $i$ 个位置上插入指定元素 $e$。

**核心思想**：注意位序与数组下标的关系，从后面的元素依次向后移动。

### 1.2 删除操作

**ListDelete(&L, i, &e)**：删除表L中第 $i$ 个位置的元素，并用 $e$ 返回删除元素的值。

**核心思想**：注意位序与数组下标的关系，从前面的元素依次向前移动。

## 二、核心知识

### 2.1 插入操作实现

```c
#define MaxSize 10

typedef struct {
    ElemType data[MaxSize];
    int length;
} SqList;

// 在L的位序i处插入元素e
bool ListInsert(SqList &L, int i, ElemType e) {
    // 判断i的范围是否有效
    if (i < 1 || i > L.length + 1)
        return false;
    // 当前存储空间已满，不能插入
    if (L.length >= MaxSize)
        return false;
    
    // 将第i个元素及之后的元素后移
    for (int j = L.length; j >= i; j--)
        L.data[j] = L.data[j-1];
    
    L.data[i-1] = e;  // 在位置i处放入e
    L.length++;        // 长度加1
    return true;
}
```

**执行过程示例**：
```
初始：[a, b, c, d, e, f]  length=6
在位序3插入c：
  第1步：将f后移 → [a, b, c, d, e, _, f]
  第2步：将e后移 → [a, b, c, d, _, e, f]
  第3步：将d后移 → [a, b, c, _, d, e, f]
  第4步：将c放入data[2] → [a, b, c, c, d, e, f]
结果：[a, b, c, c, d, e, f]  length=7
```

### 2.2 删除操作实现

```c
// 删除表L中第i个位置的元素，并用e返回删除元素的值
bool ListDelete(SqList &L, int i, ElemType &e) {
    // 判断i的范围是否有效
    if (i < 1 || i > L.length)
        return false;
    
    e = L.data[i-1];  // 将被删除的元素赋值给e
    
    // 将第i个位置之后的元素前移
    for (int j = i; j < L.length; j++)
        L.data[j-1] = L.data[j];
    
    L.length--;  // 长度减1
    return true;
}
```

**执行过程示例**：
```
初始：[a, b, c, d, e, f]  length=6
删除位序3的元素c：
  第1步：保存c到e
  第2步：将d前移 → [a, b, d, d, e, f]
  第3步：将e前移 → [a, b, d, e, e, f]
  第4步：将f前移 → [a, b, d, e, f, f]
结果：[a, b, d, e, f]  length=5
```

> ⚠️ 如果参数没有加引用符号 `&`，被调用函数中处理的是参数数据的**复制品**，修改无法带回来。

## 三、深入理解

### 3.1 插入操作时间复杂度分析

**问题规模**：$n = L.length$（表长）

| 情况 | 条件 | 循环次数 | 时间复杂度 |
|------|------|----------|------------|
| **最好** | 新元素插入到表尾（$i = n+1$） | 0次 | $O(1)$ |
| **最坏** | 新元素插入到表头（$i = 1$） | $n$次 | $O(n)$ |
| **平均** | 等概率插入到任意位置 | $\frac{n}{2}$次 | $O(n)$ |

**平均情况推导**：

假设新元素插入到任何一个位置的概率相同，即 $i = 1, 2, 3, \dots, n+1$ 的概率都是 $p = \frac{1}{n+1}$

- $i = 1$，循环 $n$ 次
- $i = 2$，循环 $n-1$ 次
- ...
- $i = n+1$，循环 $0$ 次

$$\text{平均循环次数} = np + (n-1)p + (n-2)p + \dots + 1 \cdot p = \frac{n(n+1)}{2} \cdot \frac{1}{n+1} = \frac{n}{2}$$

$$\text{平均时间复杂度} = O(n)$$

### 3.2 删除操作时间复杂度分析

**问题规模**：$n = L.length$（表长）

| 情况 | 条件 | 循环次数 | 时间复杂度 |
|------|------|----------|------------|
| **最好** | 删除表尾元素（$i = n$） | 0次 | $O(1)$ |
| **最坏** | 删除表头元素（$i = 1$） | $n-1$次 | $O(n)$ |
| **平均** | 等概率删除任意元素 | $\frac{n-1}{2}$次 | $O(n)$ |

**平均情况推导**：

假设删除任何一个元素的概率相同，即 $i = 1, 2, 3, \dots, n$ 的概率都是 $p = \frac{1}{n}$

- $i = 1$，循环 $n-1$ 次
- $i = 2$，循环 $n-2$ 次
- ...
- $i = n$，循环 $0$ 次

$$\text{平均循环次数} = (n-1)p + (n-2)p + \dots + 1 \cdot p = \frac{n(n-1)}{2} \cdot \frac{1}{n} = \frac{n-1}{2}$$

$$\text{平均时间复杂度} = O(n)$$

## 五、典型例题

### 算法题

**Q1**：从顺序表中删除最小值元素（假设唯一），并用最后一个元素填补，返回被删除元素的值。

```c
bool DeleteMin(SqList &L, ElemType &e) {
    if (L.length == 0)
        return false;
    
    e = L.data[0];  // 假设第一个元素最小
    int pos = 0;
    
    for (int i = 1; i < L.length; i++) {
        if (L.data[i] < e) {
            e = L.data[i];
            pos = i;
        }
    }
    
    L.data[pos] = L.data[L.length - 1];  // 用最后一个元素填补
    L.length--;
    return true;
}
```

**Q2**：将顺序表L的所有元素逆置，要求空间复杂度为 $O(1)$。

```c
void Reverse(SqList &L) {
    ElemType temp;
    for (int i = 0; i < L.length / 2; i++) {
        temp = L.data[i];
        L.data[i] = L.data[L.length - 1 - i];
        L.data[L.length - 1 - i] = temp;
    }
}
```

**Q3**：在长度为 $n$ 的顺序表中，删除所有值为 $x$ 的元素，要求时间复杂度 $O(n)$，空间复杂度 $O(1)$。

```c
void DeleteX(SqList &L, ElemType x) {
    int k = 0;  // 记录不等于x的元素个数
    for (int i = 0; i < L.length; i++) {
        if (L.data[i] != x) {
            L.data[k] = L.data[i];
            k++;
        }
    }
    L.length = k;
}
```

## 六、易错点

1. **位序从1开始**：第 $i$ 个元素对应 `data[i-1]`
2. **插入位置范围**：$1 \leq i \leq length+1$
3. **删除位置范围**：$1 \leq i \leq length$
4. **移动方向**：插入从后往前移，删除从前往后移
5. **引用参数**：删除操作的 `e` 需要用 `&` 引用
