# 二叉树的性质

## 📊 考情分析
- **考试频率**：高频必考，每年至少1-2道选择题
- **分值**：2-6分
- **题型**：选择题为主，偶尔填空题
- **真题考查方式**：已知结点数求叶子数/高度、完全二叉树的结点编号推导

## 一、基本概念

### 二叉树的基本记号
- $n_0$：度为0的结点数（叶子结点数）
- $n_1$：度为1的结点数
- $n_2$：度为2的结点数
- $n$：总结点数，$n = n_0 + n_1 + n_2$

## 二、核心知识 — 常考性质

### 性质1：$n_0 = n_2 + 1$

**推导**：
- 结点数：$n = n_0 + n_1 + n_2$ ... ①
- 总度数 = 边数 = $n - 1$（树的性质）
- 总度数 = $0 \cdot n_0 + 1 \cdot n_1 + 2 \cdot n_2 = n_1 + 2n_2$
- 因此：$n = n_1 + 2n_2 + 1$ ... ②
- ② - ①：$n_0 = n_2 + 1$

> **记忆**：叶子结点比二分支结点多一个

### 性质2：二叉树第 $i$ 层至多有 $2^{i-1}$ 个结点（$i \geq 1$）

- 这是 $m$ 叉树性质在 $m=2$ 时的特例
- 第1层：$2^0 = 1$
- 第2层：$2^1 = 2$
- 第 $i$ 层：$2^{i-1}$

### 性质3：高度为 $h$ 的二叉树至多有 $2^h - 1$ 个结点（满二叉树）

- 等比数列求和：$1 + 2 + 4 + \cdots + 2^{h-1} = 2^h - 1$

### 性质4（完全二叉树）：具有 $n$ 个结点的完全二叉树的高度 $h$ 为：

$$h = \lceil \log_2(n+1) \rceil \quad \text{或} \quad h = \lfloor \log_2 n \rfloor + 1$$

**推导**：
- 高为 $h-1$ 的满二叉树共 $2^{h-1} - 1$ 个结点
- 高为 $h$ 的满二叉树共 $2^h - 1$ 个结点
- 因此：$2^{h-1} - 1 < n \leq 2^h - 1$
- 化简：$2^{h-1} < n + 1 \leq 2^h$
- 即：$h-1 < \log_2(n+1) \leq h$
- 所以：$h = \lceil \log_2(n+1) \rceil$

### 性质5（完全二叉树）：由结点数 $n$ 推出 $n_0, n_1, n_2$

完全二叉树最多只有一个度为1的结点，即 $n_1 = 0$ 或 $n_1 = 1$。

**由 $n_0 = n_2 + 1$ 和 $n = n_0 + n_1 + n_2$：**

- 若 $n$ 为**奇数**（$2k-1$ 个结点）：$n_1 = 0, \; n_0 = k, \; n_2 = k-1$
- 若 $n$ 为**偶数**（$2k$ 个结点）：$n_1 = 1, \; n_0 = k, \; n_2 = k-1$

> **快速判断**：$n$ 为偶数 → 一定有 $n_1 = 1$；$n$ 为奇数 → 一定有 $n_1 = 0$

### 性质6（完全二叉树的编号关系）

对 $n$ 个结点的完全二叉树按层序编号（从1开始）：
- 结点 $i$ 的**左孩子**为 $2i$（$2i \leq n$ 时存在）
- 结点 $i$ 的**右孩子**为 $2i+1$（$2i+1 \leq n$ 时存在）
- 结点 $i$ 的**父节点**为 $\lfloor i/2 \rfloor$（$i \geq 2$ 时存在）
- $i \leq \lfloor n/2 \rfloor$ 的结点为**分支结点**
- $i > \lfloor n/2 \rfloor$ 的结点为**叶子结点**

## 三、深入理解

### 设计思想
- 二叉树性质是 $m$ 叉树一般性质在 $m=2$ 时的特例
- 完全二叉树的编号性质使其适合用数组顺序存储
- $n_0 = n_2 + 1$ 是二叉树最基本的计数性质，推导需牢记

### 复杂度相关
- 完全二叉树的高度为 $O(\log n)$，这是堆排序、优先队列等高效算法的基础
- 满二叉树是"最胖"的二叉树，单支树是"最瘦"的二叉树

## 四、真题考法分析

### 常考题型
1. 已知叶子结点数 $n_0$，求最少/最多结点数
2. 已知完全二叉树结点数，求 $n_0, n_1, n_2$
3. 已知完全二叉树某结点编号，求其父/子结点编号
4. 计算完全二叉树的高度

### 命题规律
- $n_0 = n_2 + 1$ 的公式变形是最常考的
- 完全二叉树的奇偶性质是解题关键
- 常设陷阱：问"最多有几个结点"要设 $n_1 = 1$，问"最少"设 $n_1 = 0$

## 五、典型例题

### 计算题

**例1**：一棵完全二叉树有100个结点，求叶子结点数。

**解**：
- $n = 100$（偶数），所以 $k = 50$
- $n_1 = 1, \; n_0 = 50, \; n_2 = 49$
- 叶子结点数 = **50**

**例2**：一棵完全二叉树有99个结点，求叶子结点数。

**解**：
- $n = 99$（奇数），$2k-1 = 99$，$k = 50$
- $n_1 = 0, \; n_0 = 50, \; n_2 = 49$
- 叶子结点数 = **50**

**例3**：一棵二叉树有2000个叶子结点，则至少有多少个结点？

**解**：
- 最少结点 → $n_1 = 0$
- $n_0 = 2000, \; n_2 = 1999$
- $n = 2000 + 0 + 1999 = 3999$

**例4**：一棵二叉树有2000个叶子结点，则至多有多少个结点？

**解**：
- 最多结点 → $n_1$ 尽量大，但无上限（理论上无穷多）
- 实际考试中通常有约束条件，如"高度为 $h$ 的二叉树"

**例5**：完全二叉树的第10层（根为第1层）有5个叶子结点，该二叉树至多有几个结点？

**解**：
- 第10层至多 $2^9 = 512$ 个结点
- 前9层共 $2^9 - 1 = 511$ 个结点
- 第10层有5个结点 → 总结点数 = $511 + 5 = 516$

### 概念题

**例6**：$n$ 个结点的完全二叉树，结点 $n/2$（向下取整）一定是什么结点？

**解**：一定是分支结点（且是最后一个分支结点）。

## 六、易错点

1. **$n_0 = n_2 + 1$ 对所有二叉树成立**，不仅限于完全二叉树
2. **完全二叉树最多一个度为1的结点**：$n_1 = 0$ 或 $1$，不可能更多
3. **奇偶判断**：$n$ 为偶数 → $n_1 = 1$；$n$ 为奇数 → $n_1 = 0$
4. **高度公式两个等价形式**：$\lceil \log_2(n+1) \rceil = \lfloor \log_2 n \rfloor + 1$
5. **编号从1开始**：如果编号从0开始，父子关系公式需调整
6. **最少/最多结点问题**：问"最多"时设 $n_1=1$，问"最少"时设 $n_1=0$
