# 二叉树的层次遍历

## 📊 考情分析
- **考试频率**：高频，常出现在算法题中
- **分值**：选择题2分或算法题8-15分
- **题型**：选择题、算法设计题
- **真题考查方式**：层次遍历的算法实现、结合队列的变形应用

## 一、基本概念

### 层次遍历的定义

按照树的层次，从上到下、从左到右依次访问每个结点。

```
第1层：    A
第2层：   B   C
第3层：  D E F G
第4层： H I J K L
```

层序遍历序列：**A B C D E F G H I J K L**

### 算法思想
1. 初始化一个辅助**队列**
2. 根结点**入队**
3. 若队列非空，则队头结点**出队**，访问该结点，并将其**左、右孩子**插入队尾（如果有的话）
4. 重复步骤3直至队列为空

## 二、核心知识

### 层次遍历的完整代码实现

```c
// 层次遍历
void LevelOrder(BiTree T) {
    InitQueue(Q);
    BiTree p;
    EnQueue(Q, T);           // 根结点入队
    while (!IsEmpty(Q)) {
        DeQueue(Q, p);       // 队头结点出队
        visit(p);            // 访问该结点
        if (p->lchild)
            EnQueue(Q, p->lchild);   // 左孩子入队
        if (p->rchild)
            EnQueue(Q, p->rchild);   // 右孩子入队
    }
}
```

> **注意**：队列中存储的是**指针**（`BiTree`），而不是结点本身，这样更高效。

### 队列的操作过程示例

以二叉树为例：
```
    A
   / \
  B   C
 / \   \
D   E   F
```

| 操作 | 队列状态 | 访问 |
|------|----------|------|
| A入队 | [A] | |
| A出队 | [] | A |
| B入队, C入队 | [B, C] | |
| B出队 | [C] | B |
| D入队, E入队 | [C, D, E] | |
| C出队 | [D, E] | C |
| F入队 | [D, E, F] | |
| D出队 | [E, F] | D |
| E出队 | [F] | E |
| F出队 | [] | F |

遍历序列：A B C D E F

## 三、深入理解

### 时间和空间复杂度
- **时间复杂度**：$O(n)$，每个结点入队出队各一次
- **空间复杂度**：$O(w)$，$w$ 为树的最大宽度
  - 最坏情况（满二叉树最后一层）：$O(n/2) = O(n)$
  - 最好情况（单支树）：$O(1)$

### 层次遍历的应用场景
1. **求树的宽度**（最大层结点数）
2. **求某一层的结点数**
3. **判断完全二叉树**
4. **逐层处理问题**

### 层次遍历求树的宽度

```c
int Width(BiTree T) {
    if (T == NULL) return 0;
    InitQueue(Q);
    EnQueue(Q, T);
    int maxWidth = 0;
    while (!IsEmpty(Q)) {
        int levelSize = QueueSize(Q);  // 当前层结点数
        maxWidth = max(maxWidth, levelSize);
        for (int i = 0; i < levelSize; i++) {
            DeQueue(Q, p);
            if (p->lchild) EnQueue(Q, p->lchild);
            if (p->rchild) EnQueue(Q, p->rchild);
        }
    }
    return maxWidth;
}
```

### 层次遍历判断完全二叉树

```c
bool IsComplete(BiTree T) {
    if (T == NULL) return true;
    InitQueue(Q);
    EnQueue(Q, T);
    bool mustBeLeaf = false;
    while (!IsEmpty(Q)) {
        DeQueue(Q, p);
        if (p == NULL) {
            mustBeLeaf = true;  // 遇到空结点，后面必须全是叶子
        } else {
            if (mustBeLeaf) return false;  // 空后又出现非空
            EnQueue(Q, p->lchild);  // 左孩子（可以为空）
            EnQueue(Q, p->rchild);  // 右孩子（可以为空）
        }
    }
    return true;
}
```

## 四、真题考法分析

### 常考题型
1. 手写层次遍历的输出序列
2. 实现层次遍历算法（队列操作过程）
3. 层次遍历的变形应用（求宽度、判断完全二叉树）
4. 从上到下逐层打印二叉树

### 命题规律
- 层次遍历用**队列**实现，与先/中/后序遍历用**栈/递归**实现形成对比
- 变形应用题常考

## 五、典型例题

### 算法题

**例1**：编写算法，自上而下、从左到右逐层输出二叉树各层结点的值（每层输出一行）。

```c
void LevelPrint(BiTree T) {
    if (T == NULL) return;
    InitQueue(Q);
    EnQueue(Q, T);
    while (!IsEmpty(Q)) {
        int size = QueueSize(Q);
        for (int i = 0; i < size; i++) {
            DeQueue(Q, p);
            printf("%d ", p->data);
            if (p->lchild) EnQueue(Q, p->lchild);
            if (p->rchild) EnQueue(Q, p->rchild);
        }
        printf("\n");  // 换行
    }
}
```

**例2**：求二叉树中第 $k$ 层的结点个数。

```c
int CountKLevel(BiTree T, int k) {
    if (T == NULL || k < 1) return 0;
    if (k == 1) return 1;
    return CountKLevel(T->lchild, k-1) + CountKLevel(T->rchild, k-1);
}
```

## 六、易错点

1. **层次遍历用队列，先/中/后序用栈**：不能混淆
2. **队列存指针不是结点**：节省空间，避免复制
3. **入队前要判断非空**：只有非空的孩子才入队
4. **层次遍历不能唯一确定二叉树**：需要结合中序遍历
5. **求宽度需分层处理**：用当前层大小控制循环
