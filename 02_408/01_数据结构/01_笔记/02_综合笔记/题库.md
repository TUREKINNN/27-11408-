#### 应用题

**5.** 已知顺序表 L，设计算法删除所有值为 x 的元素，要求时间复杂度 O(n)，空间复杂度 O(1)。

**参考答案**：
```c
void delX(SqList *L, ElemType x) {
    int k = 0;
    for (int i = 0; i < L->length; i++) {
        if (L->data[i] != x) {
            L->data[k] = L->data[i];
            k++;
        }
    }
    L->length = k;
}
```

**11.** 给定带权有向图 G，编写 Dijkstra 算法求从源点 v0 到其他各点的最短路径。

**参考答案**：
```c
void Dijkstra(MGraph G, int v0, int dist[], int path[]) {
    int final[MAXV];
    int min, w;
    
    // 初始化
    for (int v = 0; v < G.n; v++) {
        final[v] = 0;
        dist[v] = G.edge[v0][v];
        path[v] = (dist[v] < INF && v != v0) ? v0 : -1;
    }
    dist[v0] = 0;
    final[v0] = 1;
    
    // 主循环
    for (int i = 1; i < G.n; i++) {
        min = INF;
        for (int w = 0; w < G.n; w++) {
            if (!final[w] && dist[w] < min) {
                min = dist[w];
                v = w;
            }
        }
        final[v] = 1;
        
        for (w = 0; w < G.n; w++) {
            if (!final[w] && dist[v] + G.edge[v][w] < dist[w]) {
                dist[w] = dist[v] + G.edge[v][w];
                path[w] = v;
            }
        }
    }
}
```
