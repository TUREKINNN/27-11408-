# 6.3.2 图的深度优先遍历（DFS）

## 核心思想

类似于树的**先根遍历**，尽可能"深"地搜索图的分支。

### 树的先根遍历

先访问根节点，再依次递归访问各子树。先根遍历序列：`1, 2, 5, 6, 3, 4, 7, 8`

### 图的 DFS 与树的区别

图中存在**回路**，需要 `visited[]` 数组标记已访问顶点。DFS 通过**函数调用栈**（递归实现）实现深度优先的探索。

## 算法实现

### DFS 算法（单连通分量）

```c
bool visited[MAX_VERTEX_NUM];

void DFS(Graph G, int v) {
    visit(v);
    visited[v] = true;
    for (w = FirstNeighbor(G, v); w >= 0;
         w = NextNeighbor(G, v, w)) {
        if (!visited[w]) {
            DFS(G, w);    // 递归访问
        }
    }
}
```

### DFS 算法（Final 版 - 处理非连通图）

```c
void DFSTraverse(Graph G) {
    for (int i = 0; i < G.vexnum; i++)
        visited[i] = false;
    for (int i = 0; i < G.vexnum; i++)
        if (!visited[i])
            DFS(G, i);
}
```

> - 对**无向图**，调用 DFS 函数的次数 = **连通分量数**
> - 对**有向图**，调用次数要具体分析；对于**强连通图**只需调用 1 次

## DFS 遍历过程示例

```
图结构（8个顶点，邻接表）：
1: → 2 → 5
2: → 1 → 6
3: → 6 → 7 → 4
4: → 3 → 7 → 8
5: → 1
6: → 2 → 7 → 3
7: → 6 → 8 → 3 → 4
8: → 4 → 7
```

从顶点 2 出发的 DFS 序列：`2, 1, 5, 6, 3, 4, 7, 8`

### 函数调用栈过程

```
DFS(2) → visit 2
  DFS(1) → visit 1 (w=1, 已访问则跳过)
    DFS(5) → visit 5
      回溯到 1
    回溯到 2
  DFS(6) → visit 6 (w=6)
    DFS(3) → visit 3 (w=3)
      DFS(4) → visit 4 (w=4)
        DFS(7) → visit 7 (w=7)
          DFS(8) → visit 8 (w=8)
            回溯...
```

## 遍历序列的可变性

| 存储结构 | DFS 序列是否唯一 | DFS 生成树是否唯一 |
|----------|-----------------|-------------------|
| **邻接矩阵** | **唯一** | **唯一** |
| **邻接表** | **不唯一** | **不唯一** |

## 复杂度分析

### 时间复杂度

| 存储结构 | 访问顶点 | 查找邻接点 | 总时间复杂度 |
|----------|----------|-----------|-------------|
| **邻接矩阵** | O(|V|) | 每个顶点 O(|V|)，共 |V| 个 | **O(|V|²)** |
| **邻接表** | O(|V|) | 查找各顶点邻接点共 O(|E|) | **O(|V|+|E|)** |

### 空间复杂度

- 来自函数调用栈
- 最坏情况：递归深度为 **O(|V|)**（图退化为链状）
- 最好情况：**O(1)**（图呈星形）

## 深度优先生成树（森林）

由 DFS 遍历过程确定的生成树：

- **邻接矩阵**表示：生成树**唯一**
- **邻接表**表示：生成树**不唯一**
- 对**非连通图**的 DFS，得到**深度优先生成森林**

## DFS 实现逆拓扑排序

> 在顶点**退栈前输出**，即可得到逆拓扑排序序列（仅适用于 DAG 图）。
> 若存在回路，DFS 中会检测到**正在访问的顶点被再次访问**。

## 图的遍历与连通性

| 图类型 | BFS/DFS 调用次数 |
|--------|-----------------|
| **无向连通图** | 1 次 |
| **无向非连通图** | = 连通分量数 |
| **有向强连通图** | 1 次 |
| **有向图一般情况** | 需具体分析 |

## BFS vs DFS 对比

| 特性 | BFS | DFS |
|------|-----|-----|
| 数据结构 | **队列** | **栈**（递归） |
| 访问顺序 | 逐层扩展 | 尽可能深 |
| 空间复杂度 | O(|V|)（队列） | O(|V|)（递归栈） |
| 生成树特点 | 高度最小 | 深度优先 |
| 最短路径 | 天然求无权图最短路径 | 不保证最短路径 |
