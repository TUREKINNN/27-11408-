# 二叉树的定义

## 📊 考情分析
- **考试频率**：基础概念，每年必考
- **分值**：选择题2-4分
- **题型**：选择题为主
- **真题考查方式**：满二叉树与完全二叉树的辨析、特殊二叉树的性质判断

## 一、基本概念

### 1. 二叉树的严格定义

二叉树是 $n$（$n \geq 0$）个结点的有限集合：
1. 或者为空二叉树，即 $n = 0$
2. 或者由一个**根结点**和两个互不相交的被称为根的**左子树**和**右子树**组成。左子树和右子树又分别是一棵二叉树。

**特点**：
- 每个结点**至多只有两棵子树**
- 左右子树**不能颠倒**（二叉树是有序树）

> **重要区别**：度为2的有序树 ≠ 二叉树
> - 度为2的有序树：至少3个结点，可以为空子树位置不固定
> - 二叉树：可以为空，左右子树有严格的位置定义

### 2. 二叉树的五种基本形态

| 形态 | 描述 |
|------|------|
| 空二叉树 | $n = 0$ |
| 只有根节点 | $n = 1$ |
| 只有左子树 | 根 + 左子树 |
| 只有右子树 | 根 + 右子树 |
| 左右子树都有 | 根 + 左子树 + 右子树 |

### 3. 特殊的二叉树

#### (1) 满二叉树

高度为 $h$，且含有 $2^h - 1$ 个结点的二叉树。

**特点**：
- 只有最后一层有叶子结点
- 不存在度为1的结点
- 按层序从1开始编号，结点 $i$ 的左孩子为 $2i$，右孩子为 $2i+1$，父节点为 $\lfloor i/2 \rfloor$

#### (2) 完全二叉树

当且仅当其每个结点都与高度为 $h$ 的满二叉树中编号为 $1 \sim n$ 的结点一一对应时，称为完全二叉树。

**特点**：
- 只有最后两层可能有叶子结点
- **最多只有一个度为1的结点**
- 按层序编号，结点 $i$ 的左孩子为 $2i$，右孩子为 $2i+1$，父节点为 $\lfloor i/2 \rfloor$
- $i \leq \lfloor n/2 \rfloor$ 为分支结点，$i > \lfloor n/2 \rfloor$ 为叶子结点

#### (3) 二叉排序树（BST）

左子树上所有结点的关键字均小于根结点的关键字；右子树上所有结点的关键字均大于根结点的关键字；左、右子树又各是一棵二叉排序树。

#### (4) 平衡二叉树（AVL树）

树上任一结点的左子树和右子树的深度之差不超过1。

## 二、核心知识

### 二叉树与度为2的有序树的区别

| | 二叉树 | 度为2的有序树 |
|---|---|---|
| 是否可空 | 可以为空 | 至少1个结点 |
| 子树个数 | 可以0、1、2 | 度为2的结点恰好有2个孩子 |
| 左右定义 | 严格区分左右 | 只是有顺序，无"左右子树"概念 |
| 结点最少 | 0 | 3 |

### 完全二叉树的判断方法

一棵 $n$ 个结点的二叉树是完全二叉树，当且仅当按层序编号后：
- 不存在"空洞"（即编号 $1$ 到 $n$ 每个位置都有结点）

## 三、深入理解

### 满二叉树是特殊的完全二叉树
- 满二叉树一定是完全二叉树
- 完全二叉树不一定是满二叉树

### 二叉树的递归性质
二叉树的定义是递归的，这决定了：
- 二叉树的各种算法大多采用递归实现
- 可以将二叉树的问题分解为左子树和右子树的子问题

## 四、真题考法分析

### 常考题型
1. 判断一棵树是否为完全二叉树
2. 给定结点数，求完全二叉树中叶子结点数
3. 满二叉树与完全二叉树的性质辨析
4. 二叉树与度为2的有序树的区别

### 命题规律
- 满二叉树和完全二叉树的定义辨析是高频考点
- 常与完全二叉树的编号性质结合考查

## 五、典型例题

### 概念题

**例1**：一棵有124个叶子结点的完全二叉树，最多有多少个结点？

**解**：
- 由 $n_0 = n_2 + 1$（二叉树性质）
- $n_2 = 123$
- 最多的情况是 $n_1 = 1$
- 最多结点数 = $124 + 1 + 123 = 248$

**例2**：完全二叉树中，编号为 $i$ 的结点，若 $2i > n$（$n$ 为总结点数），则该结点是？

**解**：没有左孩子，说明是叶子结点。

**例3**：判断正误——"满二叉树就是完全二叉树"。

**解**：✓ 满二叉树是完全二叉树的特例。

## 六、易错点

1. **二叉树是有序树**：左右子树不能互换，互换后是不同的二叉树
2. **空二叉树存在**：$n=0$ 是合法的二叉树，但度为2的树至少3个结点
3. **完全二叉树最多一个度为1的结点**：这是关键性质，度为1的结点只有左孩子
4. **满二叉树编号从1开始**：结点 $i$ 的左孩子为 $2i$，右孩子为 $2i+1$
5. **完全二叉树叶子结点位置**：叶子结点集中在最后两层，且最后一层靠左排列
6. **$i > \lfloor n/2 \rfloor$ 为叶子**：这是完全二叉树的特有性质
