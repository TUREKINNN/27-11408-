# 队列的顺序实现

## 📊 考情分析
- **考试频率**：高，选择题和代码题常考
- **分值**：4~10分
- **题型**：选择题、代码填空题、算法设计题
- **真题考查方式**：循环队列的实现、队满/队空判断、元素个数计算

## 一、基本概念

### 1. 顺序队列的问题
用普通数组实现队列会出现**假溢出**问题：
- 入队：`rear++`
- 出队：`front++`
- 当 `rear == MaxSize` 时，队列已满，但前面可能有空闲空间

### 2. 循环队列
用**模运算**将存储空间在逻辑上变成"环状"，解决假溢出问题。

**核心思想**：`(rear + 1) % MaxSize` 将无限的整数域映射到有限的整数集合 `{0, 1, 2, ..., MaxSize-1}` 上。

## 二、核心知识

### 1. 循环队列的定义
```c
#define MaxSize 10

typedef struct {
    ElemType data[MaxSize];  // 静态数组存放队列元素
    int front;               // 队头指针
    int rear;                // 队尾指针
} SqQueue;
```

### 2. 初始化
```c
void InitQueue(SqQueue *Q) {
    Q->front = 0;  // 队头指针初始化为0
    Q->rear = 0;   // 队尾指针初始化为0
}
```

### 3. 判空/判满方案

#### 方案一：牺牲一个存储单元
```c
// 判空：front == rear
bool QueueEmpty(SqQueue Q) {
    return (Q.front == Q.rear);
}

// 判满：(rear + 1) % MaxSize == front
bool QueueFull(SqQueue Q) {
    return ((Q.rear + 1) % MaxSize == Q.front);
}

// 队列元素个数：(rear + MaxSize - front) % MaxSize
int QueueLength(SqQueue Q) {
    return (Q.rear + MaxSize - Q.front) % MaxSize;
}
```

#### 方案二：增加辅助变量 size
```c
typedef struct {
    ElemType data[MaxSize];
    int front;
    int rear;
    int size;  // 记录队列元素个数
} SqQueue2;

// 入队成功：size++
// 出队成功：size--

// 判空：size == 0
// 判满：size == MaxSize
```

#### 方案三：增加辅助变量 tag
```c
typedef struct {
    ElemType data[MaxSize];
    int front;
    int rear;
    int tag;  // 最近一次操作：0=删除，1=插入
} SqQueue3;

// 判空：front == rear && tag == 0
// 判满：front == rear && tag == 1
```

### 4. 入队操作
```c
bool EnQueue(SqQueue *Q, ElemType x) {
    if ((Q->rear + 1) % MaxSize == Q->front)  // 队满
        return false;
    Q->data[Q->rear] = x;                     // 元素入队
    Q->rear = (Q->rear + 1) % MaxSize;        // 队尾指针后移
    return true;
}
```

### 5. 出队操作
```c
bool DeQueue(SqQueue *Q, ElemType *x) {
    if (Q->front == Q->rear)  // 队空
        return false;
    *x = Q->data[Q->front];                   // 读取队头元素
    Q->front = (Q->front + 1) % MaxSize;      // 队头指针后移
    return true;
}
```

### 6. 读队头元素
```c
bool GetHead(SqQueue Q, ElemType *x) {
    if (Q.front == Q.rear)  // 队空
        return false;
    *x = Q.data[Q.front];
    return true;
}
```

## 三、深入理解

### 1. 设计思想
循环队列通过**模运算**将线性空间变成环形空间，解决了假溢出问题。

**关键公式**：
- 队尾指针后移：`rear = (rear + 1) % MaxSize`
- 队头指针后移：`front = (front + 1) % MaxSize`
- 队列长度：`(rear + MaxSize - front) % MaxSize`

### 2. 三种判满方案的比较
| 方案 | 优点 | 缺点 |
|------|------|------|
| 牺牲一个空间 | 实现简单 | 浪费一个空间 |
| 增加size | 不浪费空间 | 每次操作需要更新size |
| 增加tag | 不浪费空间 | 每次操作需要更新tag |

### 3. 复杂度分析
| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 初始化 | O(1) | 设置front和rear |
| 判空 | O(1) | 比较front和rear |
| 入队 | O(1) | 赋值，rear后移 |
| 出队 | O(1) | 取值，front后移 |
| 读队头 | O(1) | 取值 |

## 五、典型例题

### 概念题
**题目**：循环队列中，front指向队头元素，rear指向队尾元素的下一个位置，则队满的条件是（）
A. front == rear  B. (rear + 1) % MaxSize == front  C. rear == MaxSize  D. front == 0

**答案**：B
**解析**：牺牲一个空间的方案中，队满条件是rear的下一个位置是front。

### 计算题
**题目**：设循环队列的MaxSize=10，front=3，rear=7，则队列中的元素个数是（）
A. 3  B. 4  C. 5  D. 7

**答案**：B
**解析**：元素个数 = (rear + MaxSize - front) % MaxSize = (7 + 10 - 3) % 10 = 4

### 算法题
**题目**：设计一个算法，求循环队列中的元素个数。

```c
int QueueLength(SqQueue Q) {
    return (Q.rear + MaxSize - Q.front) % MaxSize;
}
```

## 六、易错点

1. **队满条件**：
   - 牺牲一个空间：`(rear + 1) % MaxSize == front`
   - ❌ 错误：`rear == MaxSize`

2. **队空条件**：
   - `front == rear`
   - ❌ 错误：`front == 0`

3. **指针后移**：
   - 必须使用模运算：`(rear + 1) % MaxSize`
   - ❌ 错误：`rear = rear + 1`

4. **队列长度计算**：
   - 公式：`(rear + MaxSize - front) % MaxSize`
   - ❌ 错误：`rear - front`
