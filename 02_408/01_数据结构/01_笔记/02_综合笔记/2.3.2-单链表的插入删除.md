# 单链表的插入删除

## 📊 考情分析
- **考试频率**：高频考点，必考
- **分值**：选择题2-4分，算法题10-15分
- **题型**：代码实现、时间复杂度分析、指定结点的前插/后插/删除
- **真题考查方式**：手写插入删除代码、分析带头/不带头结点的区别、封装思想的应用

---

## 一、基本概念

### 1.1 按位序插入

**ListInsert(&L, i, e)**：在表L中的第 $i$ 个位置上插入指定元素 $e$。

**核心思路**：找到第 $i-1$ 个结点，将新结点插入其后。

### 1.2 按位序删除

**ListDelete(&L, i, &e)**：删除表L中第 $i$ 个位置的元素，并用 $e$ 返回删除元素的值。

**核心思路**：找到第 $i-1$ 个结点，将其指针指向第 $i+1$ 个结点，释放第 $i$ 个结点。

## 二、核心知识

### 2.1 按位序插入（带头结点）

```c
// 在第i个位置插入元素e（带头结点）
bool ListInsert(LinkList &L, int i, ElemType e) {
    if (i < 1)
        return false;
    
    LNode *p;  // 指针p指向当前扫描到的结点
    int j = 0;  // 当前p指向的是第几个结点
    p = L;      // p指向头结点，头结点是第0个结点
    
    // 找到第i-1个结点
    while (p != NULL && j < i - 1) {
        p = p->next;
        j++;
    }
    
    if (p == NULL)  // i值不合法
        return false;
    
    // 在p结点后插入新结点s
    LNode *s = (LNode *)malloc(sizeof(LNode));
    s->data = e;
    s->next = p->next;
    p->next = s;
    
    return true;
}
```

**时间复杂度**：
- 最好：$O(1)$（插入到表头）
- 最坏：$O(n)$（插入到表尾）
- 平均：$O(n)$

> ⚠️ **关键**：`s->next = p->next;` 和 `p->next = s;` 顺序不能颠倒！

### 2.2 按位序插入（不带头结点）

```c
// 在第i个位置插入元素e（不带头结点）
bool ListInsert(LinkList &L, int i, ElemType e) {
    if (i < 1)
        return false;
    
    if (i == 1) {  // 插入到表头，需要特殊处理
        LNode *s = (LNode *)malloc(sizeof(LNode));
        s->data = e;
        s->next = L;
        L = s;  // 修改头指针
        return true;
    }
    
    LNode *p = L;
    int j = 1;
    
    while (p != NULL && j < i - 1) {
        p = p->next;
        j++;
    }
    
    if (p == NULL)
        return false;
    
    LNode *s = (LNode *)malloc(sizeof(LNode));
    s->data = e;
    s->next = p->next;
    p->next = s;
    
    return true;
}
```

> 💡 **结论**：不带头结点写代码更不方便，推荐用带头结点。但考试中两种都可能考查，注意审题。

### 2.3 指定结点的后插操作

```c
// 在结点p后插入元素e
bool InsertNextNode(LNode *p, ElemType e) {
    if (p == NULL)
        return false;
    
    LNode *s = (LNode *)malloc(sizeof(LNode));
    if (s == NULL)  // 内存分配失败
        return false;
    
    s->data = e;
    s->next = p->next;
    p->next = s;
    
    return true;
}
```

**时间复杂度**：$O(1)$

### 2.4 指定结点的前插操作

#### 方法1：传入头指针（时间复杂度 $O(n)$）
```c
// 需要传入头指针L，循环查找p的前驱q，再对q后插
bool InsertPriorNode(LinkList L, LNode *p, ElemType e) {
    LNode *q = L;
    while (q->next != p)
        q = q->next;
    return InsertNextNode(q, e);
}
```

#### 方法2：偷天换日（时间复杂度 $O(1)$）
```c
// 在p结点前插入元素e（不改变p的指针）
bool InsertPriorNode(LNode *p, ElemType e) {
    if (p == NULL)
        return false;
    
    LNode *s = (LNode *)malloc(sizeof(LNode));
    if (s == NULL)
        return false;
    
    // 先将p的数据复制到s
    s->next = p->next;
    p->next = s;
    s->data = p->data;  // 将p的数据复制到s
    p->data = e;         // 将e放入p中
    
    return true;
}
```

> 💡 **核心思想**：新结点放在p后面，把p原来的数据挪到新结点，p存放新数据。

### 2.5 按位序删除（带头结点）

```c
// 删除第i个位置的元素，并用e返回删除元素的值
bool ListDelete(LinkList &L, int i, ElemType &e) {
    if (i < 1)
        return false;
    
    LNode *p;
    int j = 0;
    p = L;
    
    // 找到第i-1个结点
    while (p->next != NULL && j < i - 1) {
        p = p->next;
        j++;
    }
    
    if (p->next == NULL)  // i值不合法
        return false;
    
    LNode *q = p->next;   // q指向待删除结点
    e = q->data;           // 用e返回删除元素的值
    p->next = q->next;     // 将q结点从链中断开
    free(q);               // 释放结点的存储空间
    
    return true;
}
```

**时间复杂度**：
- 最好：$O(1)$（删除表头）
- 最坏：$O(n)$（删除表尾）
- 平均：$O(n)$

### 2.6 指定结点的删除

#### 方法1：传入头指针（时间复杂度 $O(n)$）
```c
// 传入头指针，循环寻找p的前驱结点
```

#### 方法2：偷天换日（时间复杂度 $O(1)$）
```c
// 删除指定结点p
bool DeleteNode(LNode *p) {
    if (p == NULL)
        return false;
    
    LNode *q = p->next;  // q指向p的后继
    
    // 如果p是最后一个结点，需要特殊处理
    if (q == NULL) {
        // 只能从头开始找p的前驱，时间复杂度O(n)
        return false;
    }
    
    p->data = q->data;  // 将q的数据复制到p
    p->next = q->next;   // 将q从链中断开
    free(q);             // 释放q
    
    return true;
}
```

> ⚠️ **注意**：如果p是最后一个结点，这种方法会出错！需要特殊处理。

## 三、深入理解

### 3.1 封装的好处

将常用操作封装成基本操作函数：
- 代码逻辑清晰
- 避免重复代码
- 易于维护

```c
// 封装示例：在第i个位置插入元素
bool ListInsert(LinkList &L, int i, ElemType e) {
    LNode *p = GetElem(L, i - 1);  // 找到第i-1个结点
    return InsertNextNode(p, e);   // 在p后插入
}
```

### 3.2 时间复杂度分析

| 操作 | 最好 | 最坏 | 平均 |
|------|------|------|------|
| 按位序插入 | $O(1)$ | $O(n)$ | $O(n)$ |
| 按位序删除 | $O(1)$ | $O(n)$ | $O(n)$ |
| 后插操作 | $O(1)$ | $O(1)$ | $O(1)$ |
| 前插操作（偷天换日） | $O(1)$ | $O(1)$ | $O(1)$ |
| 指定结点删除（偷天换日） | $O(1)$ | $O(1)$ | $O(1)$ |

### 3.3 插入操作的关键顺序

```c
s->next = p->next;  // ① 先让s指向p的后继
p->next = s;        // ② 再让p指向s
```

如果顺序颠倒：
```c
p->next = s;        // ❌ p的后继丢失了
s->next = p->next;  // ❌ s指向了自己
```

## 五、典型例题

### 算法题

**Q1**：删除单链表L中值为x的所有结点。

```c
void DeleteX(LinkList &L, ElemType x) {
    LNode *p = L->next;  // 指向第一个数据结点
    LNode *pre = L;       // p的前驱
    LNode *q;             // 临时指针
    
    while (p != NULL) {
        if (p->data == x) {
            q = p;
            pre->next = p->next;
            p = p->next;
            free(q);
        } else {
            pre = p;
            p = p->next;
        }
    }
}
```

**Q2**：将单链表就地逆置（空间复杂度 $O(1)$）。

```c
void Reverse(LinkList &L) {
    LNode *p = L->next;
    L->next = NULL;
    LNode *q;
    
    while (p != NULL) {
        q = p->next;
        p->next = L->next;  // 头插法
        L->next = p;
        p = q;
    }
}
```

**Q3**：在O(1)时间内删除单链表中给定结点p（p不是最后一个结点）。

```c
bool DeleteNodeO1(LNode *p) {
    if (p == NULL || p->next == NULL)
        return false;
    
    LNode *q = p->next;
    p->data = q->data;
    p->next = q->next;
    free(q);
    return true;
}
```

## 六、易错点

1. **插入顺序**：`s->next = p->next` 必须在 `p->next = s` 之前
2. **带头/不带头审题**：第1个元素的处理方式不同
3. **指定结点删除的特殊情况**：p是最后一个结点时需要特殊处理
4. **释放内存**：删除结点后必须 `free`
5. **引用参数**：修改链表的操作需要传入 `&L`
