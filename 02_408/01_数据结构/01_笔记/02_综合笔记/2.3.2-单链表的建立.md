# 单链表的建立

## 📊 考情分析
- **考试频率**：高频考点
- **分值**：选择题2-4分，算法题10-15分
- **题型**：代码实现、头插法/尾插法的应用、链表逆置
- **真题考查方式**：头插法建立链表、尾插法建立链表、链表逆置算法

---

## 一、基本概念

### 1.1 问题描述

给定很多个数据元素（`ElemType`），如何把它们存到一个单链表里？

**步骤**：
1. **Step 1**：初始化一个单链表
2. **Step 2**：每次取一个数据元素，插入到表尾/表头

### 1.2 两种方法

| 方法 | 插入位置 | 结果顺序 | 时间复杂度 |
|------|----------|----------|------------|
| **尾插法** | 表尾 | 正序 | $O(n)$ |
| **头插法** | 表头 | 逆序 | $O(n)$ |

---

## 二、核心知识

### 2.1 尾插法建立单链表

#### 简单思路
```c
// 伪代码
初始化单链表;
设置变量 length 记录链表长度;
while (还有数据元素) {
    取一个数据元素 e;
    ListInsert(L, length + 1, e);  // 插到尾部
    length++;
}
```

**问题**：每次都从头开始遍历，时间复杂度为 $O(n^2)$。

#### 优化：设置表尾指针
```c
LinkList List_TailInsert(LinkList &L) {
    int x;
    L = (LinkList)malloc(sizeof(LNode));  // 创建头结点
    LNode *s, *r = L;  // r为表尾指针
    
    scanf("%d", &x);
    while (x != 9999) {  // 输入9999表示结束
        s = (LNode *)malloc(sizeof(LNode));
        s->data = x;
        r->next = s;  // 在r结点之后插入元素
        r = s;         // r指向新的表尾结点
        scanf("%d", &x);
    }
    
    r->next = NULL;  // 尾结点指针置空
    return L;
}
```

**执行过程**：
```
输入：10, 16, 27, 9999

初始：L → [头] → NULL
       r

插入10：L → [头] → [10] → NULL
                 r

插入16：L → [头] → [10] → [16] → NULL
                        r

插入27：L → [头] → [10] → [16] → [27] → NULL
                               r

结果：L → [头] → [10] → [16] → [27] → NULL
```

**时间复杂度**：$O(n)$

> 💡 **关键**：永远保持 `r` 指向最后一个结点。

### 2.2 头插法建立单链表

```c
LinkList List_HeadInsert(LinkList &L) {
    LNode *s;
    int x;
    
    L = (LinkList)malloc(sizeof(LNode));  // 创建头结点
    L->next = NULL;  // 初始为空链表
    
    scanf("%d", &x);
    while (x != 9999) {  // 输入9999表示结束
        s = (LNode *)malloc(sizeof(LNode));
        s->data = x;
        s->next = L->next;  // 新结点指向原来的第一个结点
        L->next = s;         // 头结点指向新结点
        scanf("%d", &x);
    }
    
    return L;
}
```

**执行过程**：
```
输入：10, 16, 27, 9999

初始：L → [头] → NULL

插入10：L → [头] → [10] → NULL

插入16：L → [头] → [16] → [10] → NULL

插入27：L → [头] → [27] → [16] → [10] → NULL

结果：L → [头] → [27] → [16] → [10] → NULL（逆序！）
```

**时间复杂度**：$O(n)$

> ⚠️ **注意**：一定要先把 `L->next = NULL`，否则会有脏数据。

### 2.3 头插法的重要应用：链表逆置

**问题**：给定一个链表，如何将其逆置？

```c
void Reverse(LinkList &L) {
    LNode *p = L->next;  // 指向第一个数据结点
    L->next = NULL;       // 断开链表
    LNode *q;
    
    while (p != NULL) {
        q = p->next;       // 保存下一个结点
        p->next = L->next; // 头插法
        L->next = p;
        p = q;
    }
}
```

**执行过程**：
```
初始：L → [头] → [10] → [16] → [27] → NULL

第1步：L → [头] → [10] → NULL
       L → [头] → [16] → [27] → NULL

第2步：L → [头] → [16] → [10] → NULL
       L → [头] → [27] → NULL

第3步：L → [头] → [27] → [16] → [10] → NULL

结果：L → [头] → [27] → [16] → [10] → NULL
```

## 三、深入理解

### 3.1 头插法 vs 尾插法对比

| 对比项 | 头插法 | 尾插法 |
|--------|--------|--------|
| 插入位置 | 表头 | 表尾 |
| 结果顺序 | **逆序** | **正序** |
| 实现复杂度 | 简单 | 需要维护表尾指针 |
| 主要应用 | **链表逆置** | 正向建立链表 |
| 是否需要表尾指针 | 不需要 | 需要 |

### 3.2 头插法核心代码

```c
s->next = L->next;  // 新结点指向原来的第一个结点
L->next = s;         // 头结点指向新结点
```

### 3.3 尾插法核心代码

```c
r->next = s;  // 表尾结点指向新结点
r = s;         // 更新表尾指针
```

### 3.4 初始化注意事项

```c
// 养成好习惯：初始化单链表时，先把头指针指向NULL
L = (LinkList)malloc(sizeof(LNode));
L->next = NULL;  // 这一句不能省略
```

如果不加 `L->next = NULL`，头结点的 next 可能指向随机地址（脏数据），导致程序崩溃。

### 4.1 常考题型

1. **手写代码**：头插法/尾插法建立链表
2. **链表逆置**：利用头插法实现

### 4.2 命题规律

- 头插法的逆序特性是常考点
- 链表逆置是高频算法题
- 注意带头/不带头结点的代码差异

## 五、典型例题

### 算法题

**Q1**：用尾插法建立带头结点的单链表，元素值从数组读取。

```c
LinkList CreateListFromArray(ElemType arr[], int n) {
    LinkList L = (LinkList)malloc(sizeof(LNode));
    LNode *r = L;
    
    for (int i = 0; i < n; i++) {
        LNode *s = (LNode *)malloc(sizeof(LNode));
        s->data = arr[i];
        r->next = s;
        r = s;
    }
    
    r->next = NULL;
    return L;
}
```

**Q2**：用头插法实现链表逆置。

```c
void Reverse(LinkList &L) {
    LNode *p = L->next;
    L->next = NULL;
    LNode *q;
    
    while (p != NULL) {
        q = p->next;
        p->next = L->next;
        L->next = p;
        p = q;
    }
}
```

**Q3**：将一个带头结点的单链表A分解为两个链表A和B，A中存放序号为奇数的元素，B中存放序号为偶数的元素（相对顺序不变）。

```c
void SplitList(LinkList &A, LinkList &B) {
    B = (LinkList)malloc(sizeof(LNode));
    B->next = NULL;
    
    LNode *ra = A;  // A的表尾指针
    LNode *rb = B;  // B的表尾指针
    LNode *p = A->next;
    int i = 1;
    
    while (p != NULL) {
        if (i % 2 == 1) {  // 奇数位序
            ra->next = p;
            ra = p;
        } else {  // 偶数位序
            rb->next = p;
            rb = p;
        }
        p = p->next;
        i++;
    }
    
    ra->next = NULL;
    rb->next = NULL;
}
```

## 六、易错点

1. **初始化必须置空**：`L->next = NULL` 不能省略
2. **头插法结果是逆序**：输入10, 16, 27得到27, 16, 10
3. **尾插法必须维护表尾指针**：否则每次都要遍历到表尾
4. **循环结束条件**：输入9999表示结束（具体值可变）
5. **链表逆置用头插法**：这是头插法的重要应用
