# 双端队列

## 📊 考情分析
- **考试频率**：中等，选择题常考
- **分值**：2~4分
- **题型**：选择题为主
- **真题考查方式**：双端队列的特性、输出序列合法性判断

## 一、基本概念

### 1. 双端队列的定义
**双端队列（Deque）**是只允许从**两端插入、两端删除**的线性表。

**变种**：
- **输入受限的双端队列**：只允许从一端插入、两端删除
- **输出受限的双端队列**：只允许从两端插入、一端删除

### 2. 与栈、队列的关系
| 数据结构 | 插入 | 删除 |
|---------|------|------|
| 栈 | 一端 | 一端（同一端） |
| 队列 | 一端 | 另一端 |
| 双端队列 | 两端 | 两端 |

**注意**：若只使用双端队列一端的插入、删除操作，则效果等同于栈。

## 二、核心知识

### 1. 双端队列的基本操作
```
插入 ← [1, 2, 3, 4, 5] → 插入
删除 ←                → 删除
```

**四种操作**：
- 前端插入：`push_front(x)`
- 后端插入：`push_back(x)`
- 前端删除：`pop_front()`
- 后端删除：`pop_back()`

### 2. 输出序列合法性判断

**输入序列**：1, 2, 3, 4

**栈的合法输出序列（14种）**：
```
1,2,3,4  1,2,4,3  1,3,2,4  1,3,4,2  1,4,2,3  1,4,3,2
2,1,3,4  2,1,4,3  2,3,1,4  2,3,4,1  2,4,1,3  2,4,3,1
3,1,2,4  3,2,1,4
```

**重要结论**：
- 在栈中合法的输出序列，在双端队列中**必定合法**
- 在双端队列中合法的输出序列，在栈中**不一定合法**

### 3. 输入受限的双端队列
只允许从一端插入、两端删除。

**合法输出序列**：在栈中合法的序列，在输入受限的双端队列中一定也合法。

### 4. 输出受限的双端队列
只允许从两端插入、一端删除。

**合法输出序列**：在栈中合法的序列，在输出受限的双端队列中一定也合法。

## 三、深入理解

### 1. 设计思想
双端队列是栈和队列的推广，具有更大的灵活性。

**特点**：
- 可以在两端进行插入和删除
- 包含了栈和队列的所有功能

### 2. 复杂度分析
| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 前端插入 | O(1) | 需要移动元素或修改指针 |
| 后端插入 | O(1) | 直接插入 |
| 前端删除 | O(1) | 直接删除 |
| 后端删除 | O(1) | 需要移动元素或修改指针 |

## 四、真题考法分析

### 常考题型
1. **输出序列合法性判断**：给定输入序列，判断哪些输出序列是合法的
2. **概念辨析**：双端队列与栈、队列的区别
3. **变种分析**：输入受限/输出受限的双端队列

### 命题规律
- 重点考查**输出序列合法性判断**
- 常考**栈的合法序列在双端队列中一定合法**
- 注意**输入受限和输出受限**的区别

## 五、典型例题

### 概念题
**题目**：若一个双端队列的输入序列为1,2,3,4，则下列序列中不可能的输出序列是（）
A. 1,2,3,4  B. 4,3,2,1  C. 1,3,2,4  D. 4,1,2,3

**答案**：D
**解析**：
- A：从后端依次入队，从前端依次出队 ✓
- B：从后端依次入队，从后端依次出队 ✓
- C：栈的合法序列，双端队列一定合法 ✓
- D：4出队后，1,2,3在队列中，无法先出1再出2 ✗

### 计算题
**题目**：若一个输入受限的双端队列的输入序列为1,2,3,4，则合法的输出序列有多少种？

**答案**：至少14种（栈的合法序列数），具体取决于受限的方式。

## 六、易错点

1. **双端队列的特性**：
   - 双端队列可以在两端插入和删除
   - ❌ 错误：双端队列只能在一端插入

2. **输出序列合法性**：
   - 栈的合法序列在双端队列中一定合法
   - ❌ 错误：认为双端队列的合法序列更少

3. **输入受限和输出受限的区别**：
   - 输入受限：只能从一端插入
   - 输出受限：只能从一端删除
   - ❌ 混淆两种限制
