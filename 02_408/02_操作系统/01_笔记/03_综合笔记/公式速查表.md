# 操作系统公式速查表

## 一、处理机调度相关公式

### 1. 周转时间
$$周转时间 = 完成时间 - 到达时间$$
$$带权周转时间 = \frac{周转时间}{服务时间}$$
$$平均周转时间 = \frac{\sum 各作业周转时间}{作业数}$$
$$平均带权周转时间 = \frac{\sum 各作业带权周转时间}{作业数}$$

### 2. 等待时间
$$等待时间 = 周转时间 - 服务时间$$
$$平均等待时间 = \frac{\sum 各作业等待时间}{作业数}$$

### 3. 响应比（HRRN）
$$响应比 = \frac{等待时间 + 服务时间}{服务时间} = 1 + \frac{等待时间}{服务时间}$$

### 4. CPU利用率
$$CPU利用率 = \frac{CPU忙碌时间}{总时间} \times 100\%$$

### 5. 系统吞吐量
$$吞吐量 = \frac{完成的作业数}{总时间}$$

## 二、内存管理相关公式

### 1. 逻辑地址与物理地址转换（分页）
$$页号 = \frac{逻辑地址}{页面大小} \quad (\text{取整})$$
$$页内偏移 = 逻辑地址 \mod 页面大小$$
$$物理地址 = 页框号 \times 页面大小 + 页内偏移$$

### 2. 页表大小
$$页表项数量 = \frac{地址空间大小}{页面大小}$$
$$页表大小 = 页表项数量 \times 页表项大小$$

### 3. 快表（TLB）有效访问时间
$$EAT = \alpha \times (t_1 + t_2) + (1-\alpha) \times (t_1 + t_2 + t_3)$$

其中：
- $\alpha$ = TLB命中率
- $t_1$ = TLB查找时间
- $t_2$ = 内存访问时间
- $t_3$ = 页表访问时间（如果不命中，需要先访问页表再访问内存）

**简化公式（TLB查找时间忽略不计时）：**
$$EAT = \alpha \times t + (1-\alpha) \times 2t = t + (1-\alpha) \times t$$

其中 $t$ 为一次内存访问时间。

### 4. 多级页表有效访问时间（以两级页表为例）
$$EAT = \alpha \times (t_1 + t_2) + (1-\alpha) \times (t_1 + 3t_2)$$

其中：
- $\alpha$ = TLB命中率
- $t_1$ = TLB查找时间
- $t_2$ = 内存访问时间
- 3次内存访问 = 一级页表 + 二级页表 + 目标数据

### 5. 缺页率相关
$$有效访问时间 = (1-p) \times 访存时间 + p \times 缺页处理时间$$

其中：
- $p$ = 缺页率
- 缺页处理时间 = 缺页中断处理时间 + 页面置换时间 + 重新访问时间

### 6. 页面大小与页表项关系
$$页面大小 = 2^{页内偏移位数}$$
$$页表项数 = 2^{页号位数}$$

## 三、页面置换算法相关

### 1. OPT（最佳置换算法）
- 选择未来最长时间内不会被访问的页面置换
- 理论最优，实际无法实现

### 2. FIFO（先进先出）
- 选择最早进入内存的页面置换
- 可能出现 **Belady异常**（增加物理块数反而增加缺页次数）

### 3. LRU（最近最少使用）
- 选择最近最久未使用的页面置换
- 不会出现Belady异常

### 4. CLOCK（时钟置换）
- 指针循环扫描，访问位为0则置换，为1则清0并继续

### 5. 缺页次数计算
$$缺页次数 = 页面置换次数 + 初始装入页面数$$
$$缺页率 = \frac{缺页次数}{总访问次数}$$

## 四、文件管理相关公式

### 1. 混合索引分配容量计算（Unix风格）

**直接地址（10个）：**
$$容量_{直接} = 10 \times 块大小$$

**一次间接地址（1个）：**
$$间接指针数 = \frac{块大小}{指针大小}$$
$$容量_{一次间接} = 间接指针数 \times 块大小$$

**二次间接地址（1个）：**
$$容量_{二次间接} = 间接指针数^2 \times 块大小$$

**三次间接地址（1个）：**
$$容量_{三次间接} = 间接指针数^3 \times 块大小$$

**最大文件大小：**
$$最大文件大小 = 容量_{直接} + 容量_{一次间接} + 容量_{二次间接} + 容量_{三次间接}$$

### 2. 位示图大小
$$位示图大小 = \frac{磁盘块数}{每字位数}$$

### 3. 磁盘地址计算
$$柱面号 = \frac{逻辑块号}{每柱面扇区数}$$
$$磁头号 = \frac{逻辑块号 \mod 每柱面扇区数}{每道扇区数}$$
$$扇区号 = (逻辑块号 \mod 每柱面扇区数) \mod 每道扇区数$$

## 五、磁盘调度相关公式

### 1. 磁盘访问时间
$$T_a = T_s + T_r + T_t$$

其中：
- $T_a$ = 总访问时间
- $T_s$ = 寻道时间（磁头移动到目标磁道）
- $T_r$ = 旋转延迟（等待目标扇区旋转到磁头下方）
- $T_t$ = 传输时间

### 2. 平均旋转延迟
$$平均旋转延迟 = \frac{1}{2} \times \frac{60}{转速(rpm)} \text{ 秒}$$

例如：7200rpm → 平均旋转延迟 = $\frac{1}{2} \times \frac{60}{7200} = 4.17ms$

### 3. 传输时间
$$T_t = \frac{每扇区字节数}{磁盘转速 \times 每磁道字节数}$$

### 4. 磁盘调度算法

**FCFS（先来先服务）：** 按请求到达顺序处理

**SSTF（最短寻道时间优先）：** 选择离当前磁头最近的请求

**SCAN（电梯算法）：**
- 磁头先向一个方向移动，处理所有请求
- 到达边界后反向移动

**C-SCAN（循环扫描）：**
- 磁头单向移动处理请求
- 到达边界后快速返回起点，重新开始

**LOOK/C-LOOK：** SCAN/C-SCAN的改进，不需要到达边界

### 5. 寻道距离计算
$$寻道距离 = \sum_{i=1}^{n} |当前磁道 - 目标磁道_i|$$
$$平均寻道距离 = \frac{总寻道距离}{请求数}$$

## 六、缓冲管理相关公式

### 1. 单缓冲处理时间
$$T = \max(C, T) + M$$

其中：
- $C$ = CPU处理时间
- $T$ = I/O传输时间
- $M$ = 数据从缓冲区传送到用户区的时间

### 2. 双缓冲处理时间
$$T = \max(C, T)$$

当 $C < T$ 时，双缓冲可使设备连续工作。

### 3. 缓冲区利用率
$$利用率 = \frac{有效工作时间}{总时间} \times 100\%$$

## 七、信号量相关

### 1. PV操作
- **P操作（wait）：** S = S - 1，若 S < 0 则阻塞
- **V操作（signal）：** S = S + 1，若 S ≤ 0 则唤醒

### 2. 经典问题初始值

| 问题 | 信号量 | 初始值 | 含义 |
|------|--------|--------|------|
| 生产者-消费者 | mutex | 1 | 互斥访问缓冲区 |
| 生产者-消费者 | empty | n | 空缓冲区数量 |
| 生产者-消费者 | full | 0 | 满缓冲区数量 |
| 读者-写者 | rwmutex | 1 | 读写互斥 |
| 读者-写者 | mutex | 1 | 保护读者计数器 |
| 哲学家进餐 | chopstick[5] | 1 | 每根筷子 |

### 3. 死锁判断
$$死锁条件：\sum_{i=1}^{n} (Need_i - Available) > 0 \text{ 对所有进程}$$

### 4. 银行家算法
$$Need_i = Max_i - Allocation_i$$

安全性检查：找到一个安全序列使得对所有 $i$：
$$Need_i \leq Work$$
$$Work = Work + Allocation_i$$

## 快速记忆口诀

**页面置换：** OPT最理想，FIFO有Belady，LRU最常用，CLOCK是折中

**磁盘调度：** FCFS公平但慢，SSTF贪心可能饥饿，SCAN电梯双向，C-SCAN单向循环

**周转时间：** 完成减到达，周转除服务

**信号量：** P减V加，负数阻塞，非正唤醒
