### 应用题

**7.** 设有三个进程 P1, P2, P3 共享一个缓冲区 B，B 中每次只能放一个产品。P1 负责生产产品放入 B，P2 负责从 B 取产品加工，P3 负责从 B 取产品包装。试用 PV 操作实现这一同步关系。

**参考答案**：
```
semaphore mutex = 1;     // 互斥信号量
semaphore full = 0;      // 缓冲区非空计数

P1:
while (1) {
    生产产品;
    P(mutex);
    产品放入 B;
    V(mutex);
    V(full);
}

P2:
while (1) {
    P(full);
    P(mutex);
    从 B 取产品;
    V(mutex);
    加工;
}

P3:
while (1) {
    P(full);
    P(mutex);
    从 B 取产品;
    V(mutex);
    包装;
}
```

| P3 | 9,2 | 4,1 | |

### 计算题

**12.** 进程访问页面序列：1, 2, 3, 4, 1, 2, 5, 1, 2, 3, 4, 5，分配 3 个物理块。分别用 FIFO 和 LRU 算法计算缺页次数。

**FIFO 算法**：
- 访问 1：缺页（1次），内存 [1]
- 访问 2：缺页（2次），内存 [1,2]
- 访问 3：缺页（3次），内存 [1,2,3]
- 访问 4：缺页（4次），淘汰 1，[2,3,4]
- 访问 1：缺页（5次），淘汰 2，[3,4,1]
- 访问 2：缺页（6次），淘汰 3，[4,1,2]
- 访问 5：缺页（7次），淘汰 4，[1,2,5]
- 访问 1：命中
- 访问 2：命中
- 访问 3：缺页（8次），淘汰 1，[2,5,3]
- 访问 4：缺页（9次），淘汰 2，[5,3,4]
- 访问 5：命中
- **共缺页 9 次**

**LRU 算法**：
- 访问 1：缺页（1次）
- 访问 2：缺页（2次）
- 访问 3：缺页（3次）
- 访问 4：缺页（4次），淘汰 1
- 访问 1：缺页（5次），淘汰 2
- 访问 2：缺页（6次），淘汰 3
- 访问 5：缺页（7次），淘汰 4
- 访问 1：命中
- 访问 2：命中
- 访问 3：缺页（8次），淘汰 5
- 访问 4：缺页（9次），淘汰 1
- 访问 5：缺页（10次），淘汰 2
- **共缺页 10 次**
