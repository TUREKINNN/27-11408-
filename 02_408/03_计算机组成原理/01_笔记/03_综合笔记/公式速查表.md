# 计算机组成原理公式速查表

## 一、数据表示相关公式

### 1. 定点数表示范围

| 码制 | 定点整数范围 | 定点小数范围 |
|------|-------------|-------------|
| 原码 | $-(2^{n-1}-1) \sim +(2^{n-1}-1)$ | $-(1-2^{-(n-1)}) \sim +(1-2^{-(n-1)})$ |
| 反码 | $-(2^{n-1}-1) \sim +(2^{n-1}-1)$ | $-(1-2^{-(n-1)}) \sim +(1-2^{-(n-1)})$ |
| 补码 | $-2^{n-1} \sim +(2^{n-1}-1)$ | $-1 \sim +(1-2^{-(n-1)})$ |
| 移码 | $-2^{n-1} \sim +(2^{n-1}-1)$ | - |

### 2. 补码运算

**补码加法：**
$$[X+Y]_{补} = [X]_{补} + [Y]_{补}$$

**补码减法：**
$$[X-Y]_{补} = [X]_{补} + [-Y]_{补}$$

**求负数补码：** 符号位不变，数值位取反加1

### 3. 溢出检测

**单符号位法：**
$$溢出 = C_s \oplus C_1$$
其中 $C_s$ 是符号位进位，$C_1$ 是最高数值位进位

**双符号位法（模4补码）：**
- 00: 正数，无溢出
- 01: 上溢（正溢出）
- 10: 下溢（负溢出）
- 11: 负数，无溢出

**溢出条件：** 两个符号位不同

### 4. IEEE 754浮点数

**单精度格式（32位）：**
```
| 1位符号(S) | 8位阶码(E) | 23位尾数(M) |
```

**数值计算：**
$$V = (-1)^S \times 1.M \times 2^{E-127}$$

**双精度格式（64位）：**
```
| 1位符号(S) | 11位阶码(E) | 52位尾数(M) |
```

$$V = (-1)^S \times 1.M \times 2^{E-1023}$$

**特殊值：**
| 类型 | 阶码E | 尾数M | 值 |
|------|-------|-------|-----|
| 零 | 0 | 0 | $\pm 0$ |
| 非规格化数 | 0 | $\neq 0$ | $(-1)^S \times 0.M \times 2^{-126}$ |
| 无穷大 | 255 | 0 | $\pm \infty$ |
| NaN | 255 | $\neq 0$ | 非数 |

**偏置值：**
- 单精度：127 (01111111)
- 双精度：1023 (01111111111)

### 5. 浮点数加减运算步骤

```
1. 对阶：小阶向大阶看齐（阶差 = 大阶 - 小阶）
2. 尾数运算：对阶后进行加减
3. 规格化：
   - 左规：结果为0.0xxx... → 左移尾数，阶码减1
   - 右规：结果为1x.xxxx... → 右移尾数，阶码加1
4. 舍入：0舍1入 / 恒置1
5. 溢出判断：检查阶码是否溢出
```

## 二、存储系统相关公式

### 1. 存储容量计算

$$存储容量 = 存储单元数 \times 存储字长$$

**例：** 地址线20根，数据线16根
$$容量 = 2^{20} \times 16 = 1M \times 16位 = 2MB$$

### 2. 存储扩展

**位扩展（增加字长）：**
- 芯片数 = 目标字长 / 芯片字长

**字扩展（增加字数）：**
- 芯片数 = 目标字数 / 芯片字数
- 需要地址译码器

**字位同时扩展：**
- 芯片数 = (目标字数 / 芯片字数) × (目标字长 / 芯片字长)

### 3. 存储带宽

**单模块：**
$$带宽 = \frac{存储字长}{存储周期}$$

**低位交叉（m模块）：**
$$带宽 = \frac{m \times 存储字长}{存储周期}$$

**条件：** 存储周期 = m × 总线传输周期

### 4. Cache地址映射

**直接映射：**
```
主存地址 = [ 标记(Tag) | 行号(Index) | 块内偏移(Offset) ]
```
- 行号位数 = $\log_2(Cache行数)$
- 块内偏移位数 = $\log_2(块大小)$
- 标记位数 = 主存地址位数 - 行号位数 - 块内偏移位数

**组相联映射：**
```
主存地址 = [ 标记(Tag) | 组号(Index) | 块内偏移(Offset) ]
```
- 组号位数 = $\log_2(Cache组数)$
- Cache组数 = Cache行数 / 相联度

**主存块映射到Cache组号：**
$$组号 = 主存块号 \mod Cache组数$$

### 5. Cache命中率与性能

**命中率：**
$$h = \frac{命中次数}{总访问次数}$$

**平均访问时间（单级Cache）：**
$$T_{avg} = h \times T_c + (1-h) \times T_m$$
其中：
- $T_c$ = Cache访问时间
- $T_m$ = 主存访问时间

**平均访问时间（考虑缺失开销）：**
$$T_{avg} = T_c + (1-h) \times T_{penalty}$$
其中 $T_{penalty}$ = 缺失代价（访问主存时间）

**多级Cache（两级）：**
$$T_{avg} = T_{c1} + (1-h_1) \times (T_{c2} + (1-h_2) \times T_m)$$
其中：
- $h_1$ = L1命中率
- $h_2$ = L2命中率（L1缺失时）
- $T_{c1}$ = L1访问时间
- $T_{c2}$ = L2访问时间

**访问效率：**
$$e = \frac{T_c}{T_{avg}} = \frac{1}{h + (1-h) \times r}$$
其中 $r = T_m / T_c$（主存与Cache访问时间之比）

### 6. Cache行大小计算

**Cache总大小：**
$$Cache大小 = 行数 \times (标记位数 + 有效位 + 脏位 + 数据位)$$

**例：** 32KB Cache，块大小64B，直接映射
- 行数 = 32KB / 64B = 512行
- 块内偏移 = 6位
- 行号 = 9位
- 标记 = 32 - 9 - 6 = 17位
- 每行 = 17 + 1(有效) + 1(脏) + 512(数据) = 531位

## 三、指令系统相关公式

### 1. 指令字长与地址码

**定长操作码：**
$$操作码位数 = \log_2(指令数量)$$

**地址码与指令格式：**
- 三地址指令：OP A1 A2 A3
- 二地址指令：OP A1 A2
- 一地址指令：OP A
- 零地址指令：OP

### 2. 寻址方式有效地址

| 寻址方式 | 有效地址EA | 说明 |
|----------|-----------|------|
| 立即寻址 | 操作数在指令中 | 无需计算 |
| 直接寻址 | EA = A | A为地址码 |
| 间接寻址 | EA = (A) | A指向的内存单元存放EA |
| 寄存器寻址 | 操作数在Ri | 无需计算 |
| 寄存器间接寻址 | EA = (Ri) | Ri中存放地址 |
| 基址寻址 | EA = (BR) + A | BR为基址寄存器 |
| 变址寻址 | EA = (IX) + A | IX为变址寄存器 |
| 相对寻址 | EA = (PC) + A | PC为程序计数器 |
| 堆栈寻址 | EA = (SP) | SP为栈顶指针 |

### 3. 指令数量计算

**操作码位数固定：**
$$指令数 = 2^{操作码位数}$$

**操作码扩展：**
- 需要根据地址码位数和指令格式计算

## 四、流水线相关公式

### 1. 流水线执行时间

**各段时间相等（均为 $\Delta t$）：**
$$T = (n + k - 1) \times \Delta t$$
其中：
- $n$ = 任务数
- $k$ = 流水线段数

**各段时间不等：**
$$T = \sum_{i=1}^{k} \Delta t_i + (n-1) \times \max(\Delta t_i)$$

### 2. 吞吐率

**实际吞吐率：**
$$TP = \frac{n}{T} = \frac{n}{(n + k - 1) \times \Delta t}$$

**最大吞吐率（n→∞）：**
$$TP_{max} = \frac{1}{\Delta t}$$

### 3. 加速比

**实际加速比：**
$$S = \frac{T_非流水}{T_流水} = \frac{n \times k \times \Delta t}{(n + k - 1) \times \Delta t} = \frac{nk}{n + k - 1}$$

**最大加速比（n→∞）：**
$$S_{max} = k$$

### 4. 流水线效率

**实际效率：**
$$E = \frac{S}{k} = \frac{n}{n + k - 1}$$

**最大效率（n→∞）：**
$$E_{max} = 1$$

### 5. 流水线时空图计算

**步骤：**
1. 画出k段流水线
2. 标出各段时间
3. 填入n个任务
4. 计算总时间T
5. 计算TP、S、E

### 6. 流水线冲突

**数据冲突：**
- RAW（写后读）：后续指令需要前面指令的结果
- WAR（读后写）：后续指令会覆盖前面指令的读取
- WAW（写后写）：两条指令写同一位置

**解决方案：**
- 数据转发（旁路）：将结果直接传给需要的指令
- 插入气泡（暂停）：等待数据准备好
- 编译器调度：重排指令顺序

**控制冲突：**
- 分支指令导致的冲突
- 解决：分支预测、延迟分支

## 五、IO系统相关公式

### 1. 中断相关

**中断响应时间：**
$$中断响应时间 = 中断请求检测时间 + 中断判优时间 + 中断向量获取时间$$

**中断处理时间：**
$$中断处理时间 = 保护现场时间 + 中断服务时间 + 恢复现场时间$$

### 2. DMA相关

**DMA传送时间：**
$$传送时间 = \frac{数据量}{DMA传送速率}$$

**DMA与CPU冲突：**

**停止CPU访问：**
$$CPU占用率 = \frac{CPU工作时间}{总时间}$$

**周期挪用：**
$$每次挪用时间 = 1个存储周期$$

**交替访问：**
$$无需冲突，交替使用存储器$$

### 3. 通道相关

**通道流量：**
$$通道最大流量 = \frac{通道数据宽度}{通道工作周期}$$

**字节多路通道：**
$$f_{max} = \frac{1}{T_s + T_d}$$
其中：
- $T_s$ = 通道选择设备时间
- $T_d$ = 传送一个字节时间

**选择通道：**
$$f_{max} = \frac{k}{n \times T_s + k \times T_d}$$
其中：
- $n$ = 传送的字节数
- $k$ = 选择的设备数

### 4. 总线带宽

**同步总线：**
$$带宽 = \frac{数据宽度}{总线周期}$$

**异步总线：**
$$带宽 = \frac{数据宽度}{总线传输时间}$$

### 5. IO系统性能指标

**系统吞吐量：**
$$吞吐量 = \frac{完成的IO操作数}{单位时间}$$

**响应时间：**
$$响应时间 = 请求时间 + 处理时间 + 传输时间$$

## 六、快速记忆口诀

### 数据表示
- 补码范围大1个：原码±(2^n-1-1)，补码-2^(n-1)~+(2^(n-1)-1)
- IEEE 754：1+8+23，偏置127，隐藏1

### 存储系统
- Cache三要素：标记+组号+块内偏移
- 命中率越高越好，缺失率=1-命中率
- 平均时间=命中时间+缺失率×缺失代价

### 流水线
- 执行时间=(n+k-1)×Δt
- 吞吐率=n/T，最大=1/Δt
- 加速比=T非/T流，最大=k
- 效率=加速比/段数

### 寻址方式
- 立即：数在指令里
- 直接：地址在指令里
- 间接：地址指向地址
- 基址：BR+A，程序重定位
- 变址：IX+A，数组访问
- 相对：PC+A，分支跳转

### 中断 vs DMA
- 中断：每字节都打扰CPU
- DMA：只在开始结束打扰CPU
- 通道：完全不打扰CPU

## 七、常见计算题型

### 题型1：IEEE 754编码

**例题：** 将十进制数-0.75转换为IEEE 754单精度格式

**解法：**
1. 转换为二进制：-0.75 = -0.11₂
2. 规格化：-1.1 × 2⁻¹
3. 符号位S = 1（负数）
4. 阶码E = -1 + 127 = 126 = 01111110₂
5. 尾数M = 10000000000000000000000（隐藏前导1）
6. 结果：1 01111110 10000000000000000000000

### 题型2：Cache地址映射

**例题：** Cache容量16KB，块大小64B，直接映射，主存地址32位

**解法：**
1. Cache行数 = 16KB / 64B = 256行
2. 块内偏移 = log₂(64) = 6位
3. 行号 = log₂(256) = 8位
4. 标记 = 32 - 8 - 6 = 18位
5. 主存地址划分：[18位标记 | 8位行号 | 6位偏移]

### 题型3：流水线性能

**例题：** 4段流水线，每段100ns，执行100个任务

**解法：**
1. T = (100 + 4 - 1) × 100ns = 10300ns
2. TP = 100 / 10300ns ≈ 9708/s
3. T_非 = 100 × 4 × 100ns = 40000ns
4. S = 40000 / 10300 ≈ 3.88
5. E = S / 4 ≈ 97%
