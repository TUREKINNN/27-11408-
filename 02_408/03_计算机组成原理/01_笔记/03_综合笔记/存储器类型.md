# 存储器类型

> [[02_408/03_计算机组成原理/01_笔记/03_综合笔记/03-存储器层次结构]]

## ROM 类型

### ROM 分类对比

| 类型 | 写入方式 | 擦除方式 | 特点 |
|------|----------|----------|------|
| **ROM** | 出厂固化 | 不可擦除 | 只读 |
| **PROM** | 一次性编程 | 不可擦除 | 可编程一次 |
| **EPROM** | 紫外光擦除 | 整片擦除 | 可多次编程 |
| **EEPROM** | 电擦除 | 字节级擦除 | 可按字节修改 |
| **Flash** | 电擦除 | 块级擦除 | 速度快，容量大 |

### 各类 ROM 详解

#### PROM (Programmable ROM)
- 一次性可编程
- 烧断熔丝
- 适用于小批量生产

#### EPROM (Erasable PROM)
- 用紫外光擦除（15-20分钟）
- 整片擦除
- 多次编程

#### EEPROM (Electrically EPROM)
- 电擦除
- 字节级擦除和写入
- 写入速度慢

#### Flash Memory
- 块级擦除
- 读写速度快
- 分为 NOR Flash 和 NAND Flash
- 应用于 U 盘、SSD 等

### NOR vs NAND Flash

| 项目 | NOR Flash | NAND Flash |
|------|-----------|------------|
| 读取 | 字节级 | 页级 |
| 写入 | 字节级 | 页级 |
| 擦除 | 块级 | 块级 |
| 速度 | 读取快 | 写入快 |
| 容量 | 小 | 大 |
| 应用 | BIOS | U 盘、SSD |

## 双端口 RAM

### 概念

**双端口 RAM**：具有两个独立读写端口的存储器，可同时被两个处理器访问。

### 结构

- 两个独立的地址线、数据线、读写控制线
- 共享存储单元
- 可同时进行两路读写

### 用途

- CPU 与外设的并行访问
- 多处理器共享内存
- 高速数据交换

## 多模块存储器

### 单体多字

- 一个存储体，多个字同时读出
- 适合顺序访问

### 多体并行

- 多个独立的存储体
- 可同时访问多个字
- 提高带宽

#### 交叉编址

```
模块 0：地址 0, 4, 8, ...
模块 1：地址 1, 5, 9, ...
模块 2：地址 2, 6, 10, ...
模块 3：地址 3, 7, 11, ...
```

#### 特点

- 顺序访问时可流水
- 适合多字连续读写

## 虚拟存储器

### 概念

**虚拟存储器**：通过软硬件结合，把物理内存和外存统一管理，给程序提供比实际物理内存大得多的地址空间。

### 实现方式

| 方式 | 特点 |
|------|------|
| **分页** | 固定大小页面 |
| **分段** | 逻辑分段 |
| **段页式** | 先分段再分页 |

### 关键机制

1. **地址转换**：虚拟地址 → 物理地址
2. **缺页中断**：页面不在内存时触发
3. **页面置换**：选择淘汰页
4. **TLB**：快表，加速地址转换

### 性能指标

- **命中率**：访问在内存中的比例
- **平均访问时间**：命中时间 + 失效率 × 失效开销
- **颠簸（Thrashing）**：频繁换入换出
