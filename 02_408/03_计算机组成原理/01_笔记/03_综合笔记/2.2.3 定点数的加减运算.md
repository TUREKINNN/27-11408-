# 定点数的加减运算

## 一、原码的加减运算

### 1.1 原码加法规则

加法器直接对原码进行加法运算**可能出错**，需要分情况处理：

| 情况 | 操作 | 结果符号 |
|------|------|---------|
| 正 + 正 | 绝对值做加法 | 结果为正 |
| 负 + 负 | 绝对值做加法 | 结果为负 |
| 正 + 负 | 绝对值大的减绝对值小的 | 符号同绝对值大的数 |
| 负 + 正 | 绝对值大的减绝对值小的 | 符号同绝对值大的数 |

### 1.2 原码减法规则

"减数"符号取反，转变为加法：

| 转换 | 结果 |
|------|------|
| 正 - 负 | → 正 + 正 |
| 负 - 正 | → 负 + 负 |
| 正 - 正 | → 正 + 负 |
| 负 - 负 | → 负 + 正 |

> **缺点**：原码加减运算需要先判断符号，再决定做加法还是减法，硬件实现复杂。

## 二、补码的加减运算 ⭐⭐⭐

### 2.1 核心公式

$$
[A + B]_补 = [A]_补 + [B]_补
$$

$$
[A - B]_补 = [A]_补 + [-B]_补
$$

> **关键特点**：无论加法还是减法，最后都转变为加法，由加法器实现运算，**符号位也参与运算**。

### 2.2 求 $[-B]_补$ 的方法

$[B]_补$ **连同符号位一起取反加 1**

### 2.3 示例

**设机器字长 8 位（含 1 位符号位），A = 15，B = -24**

```
A = +1111    → 原码：0,0001111  → 补码：0,0001111
B = -11000   → 原码：1,0011000  → 补码：1,1101000
```

**求 $[A+B]_补$：**

$$
[A+B]_补 = [A]_补 + [B]_补 = 0,0001111 + 1,1101000 = 1,1110111
$$

转回原码：$1,0001001$，真值为 $-9$

**求 $[A-B]_补$：**

$$
[-B]_补 = 0,0011000
$$

$$
[A-B]_补 = [A]_补 + [-B]_补 = 0,0001111 + 0,0011000 = 0,0100111
$$

真值为 $+39$

## 三、溢出判断 ⭐⭐⭐⭐⭐

### 3.1 什么是溢出？

只有"**正数 + 正数**"才会上溢 → 正 + 正 = 负
只有"**负数 + 负数**"才会下溢 → 负 + 负 = 正

**示例：** 设机器字长 8 位，A = 15，C = 124

$$
[A+C]_补 = 0,0001111 + 0,1111100 = 1,0001011
$$

真值为 $-117$（错误！应该是 +139，发生了**上溢**）

### 3.2 方法一：一位符号位判断

设 A 的符号为 $A_S$，B 的符号为 $B_S$，运算结果的符号为 $S_S$

$$
V = A_S B_S \overline{S_S} + \overline{A_S} \overline{B_S} S_S
$$

- 若 $V = 0$，无溢出
- 若 $V = 1$，有溢出

**含义：**
- $A_S = 1$ 且 $B_S = 1$ 且 $S_S = 0$（负 + 负 = 正，下溢）
- $A_S = 0$ 且 $B_S = 0$ 且 $S_S = 1$（正 + 正 = 负，上溢）

### 3.3 方法二：根据进位情况判断

设符号位的进位为 $C_S$，最高数值位的进位为 $C_1$

$$
V = C_S \oplus C_1
$$

- 若 $V = 0$，无溢出（$C_S$ 和 $C_1$ 相同）
- 若 $V = 1$，有溢出（$C_S$ 和 $C_1$ 不同）

**直观理解：**

| $C_S$ | $C_1$ | 含义 |
|--------|--------|------|
| 0 | 1 | 上溢 |
| 1 | 0 | 下溢 |
| 相同 | 相同 | 无溢出 |

### 3.4 方法三：双符号位（模 4 补码）

- 正数符号为 `00`
- 负数符号为 `11`

$$
V = S_{S1} \oplus S_{S2}
$$

- 若两个符号位相同（`00` 或 `11`），无溢出
- 若两个符号位不同（`01` 或 `10`），有溢出

| 双符号位 | 含义 |
|---------|------|
| `01` | 上溢 |
| `10` | 下溢 |
| `00` | 正数，无溢出 |
| `11` | 负数，无溢出 |

> **注意**：实际存储时只存储 1 个符号位，运算时会复制一个符号位。
> - **双符号位补码**又称：**模 4 补码**
> - **单符号位补码**又称：**模 2 补码**

## 四、补码加减运算的完整流程

1. 将操作数转换为补码
2. 若是减法，将减数的补码连同符号位取反加 1
3. 补码直接相加（符号位参与运算）
4. 判断是否溢出
5. 将结果转回真值

---

## 五、408 考点总结

1. **补码加减运算**是核心考点，符号位参与运算
2. **溢出判断的三种方法**必须全部掌握，选择题和大题都可能考
3. **$[-B]_补$ 的求法**：连同符号位取反加 1
4. **模 2 补码 vs 模 4 补码**的概念
5. 上溢：正 + 正 = 负；下溢：负 + 负 = 正
6. **只有两个同号数相加才可能溢出**，异号数相加不会溢出
