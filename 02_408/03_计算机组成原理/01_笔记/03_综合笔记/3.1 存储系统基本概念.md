# 存储系统基本概念

## 一、存储器的层次化结构

### 1.1 三层存储体系

```
CPU ←→ Cache ←→ 主存 ←→ 辅存
(最快最小)              (最慢最大)
```

| 层次 | 作用 | 管理方式 |
|------|------|---------|
| **Cache — 主存** | 解决主存与 CPU 速度不匹配的问题 | 硬件自动完成 |
| **主存 — 辅存** | 实现虚拟存储系统，解决主存容量不够的问题 | 硬件 + 操作系统 |

> 辅存中的数据要调入主存后才能被 CPU 访问。

### 1.2 各层存储器的速度与价格

| 存储器 | 速度 | 容量 | 成本/位 |
|--------|------|------|--------|
| 寄存器 | 最快 | 最小 | 最高 |
| Cache（SRAM） | 很快 | 小 | 高 |
| 主存（DRAM） | 中等 | 中等 | 中等 |
| 辅存（磁盘/SSD） | 慢 | 大 | 低 |

## 二、存储器的分类

### 2.1 按存储介质分类

| 类型 | 存储介质 | 典型应用 |
|------|---------|---------|
| 半导体存储器 | 半导体器件 | 主存、Cache |
| 磁表面存储器 | 磁性材料 | 磁盘、磁带 |
| 光存储器 | 光介质 | 光盘 |

### 2.2 按存取方式分类

| 类型 | 特点 | 典型应用 |
|------|------|---------|
| **随机存取存储器（RAM）** | 读写任何单元所需时间相同，与位置无关 | 主存 |
| **顺序存取存储器（SAM）** | 读写时间取决于物理位置 | 磁带 |
| **直接存取存储器（DAM）** | 先直接选取区域，再按顺序存取 | 磁盘 |
| **相联存储器（CAM）** | 按内容检索存储位置 | 快表（TLB） |

> SAM 和 DAM 都属于"**串行访问存储器**"。

### 2.3 按信息的可更改性分类

| 类型 | 特点 |
|------|------|
| 读写存储器（R/W Memory） | 可读可写（磁盘、内存、Cache） |
| 只读存储器（ROM） | 只能读，不能写（CD-ROM、BIOS） |

### 2.4 按信息的可保存性分类

| 分类 | 含义 | 例子 |
|------|------|------|
| **易失性存储器** | 断电后信息消失 | 主存、Cache |
| **非易失性存储器** | 断电后信息保持 | 磁盘、光盘 |
| **破坏性读出** | 读出后原信息被破坏，需重写 | DRAM |
| **非破坏性读出** | 读出后原信息不被破坏 | SRAM、磁盘 |

## 三、存储器的性能指标

### 3.1 存储容量

$$
\text{存储容量} = \text{存储字数} \times \text{字长}
$$

例如：$1M \times 8$ 位，表示有 $2^{20}$ 个存储单元，每个单元 8 位。

### 3.2 单位成本

$$
\text{每位价格} = \frac{\text{总成本}}{\text{总容量}}
$$

### 3.3 存储速度

| 指标 | 含义 |
|------|------|
| **存取时间 $T_a$** | 从启动一次存储器操作到完成该操作所经历的时间 |
| **存取周期 $T_m$** | 连续两次独立访问存储器操作之间的最小时间间隔（$T_m > T_a$） |
| **主存带宽 $B_m$** | 数据传输率 = 数据宽度 / 存储周期 |

$$
B_m = \frac{\text{存储字长}}{T_m}
$$

> $T_m > T_a$ 的原因：存取周期包括存取时间 + 恢复时间（如 DRAM 的重写时间）
