# 计算机组成原理学习路径

## 学习阶段划分

### 📚 第一阶段：基础入门（2-3周）

**目标：** 掌握数据表示基础，建立计算机系统整体概念

| 学习内容 | 时间 | 重点 | 配套练习 |
|----------|------|------|----------|
| 计算机系统概述 | 1天 | 冯诺依曼体系、硬件组成 | 5道选择题 |
| 进制转换 | 1天 | 十进制/二进制/十六进制互转 | 10道计算题 |
| BCD码 | 0.5天 | 8421码、余3码 | 5道选择题 |
| 原码表示 | 1天 | 原码范围、真值转换 | 8道选择题 |
| 反码表示 | 1天 | 反码范围、与原码关系 | 8道选择题 |
| **补码表示（重点）** | 2天 | 补码范围、补码加减法 | 15道计算题 |
| 移码表示 | 0.5天 | 移码特点、与补码关系 | 5道选择题 |
| 各码制转换练习 | 1天 | 综合转换练习 | 15道计算题 |
| 溢出检测 | 1天 | 单符号位、双符号位、进位检测 | 8道计算题 |

**本阶段里程碑：** 能熟练进行各码制转换和补码运算

---

### 🔥 第二阶段：核心深入（5-6周）

**目标：** 掌握浮点数、存储系统和指令系统

#### 第2-3周：浮点数与运算

| 学习内容 | 时间 | 重点 | 配套练习 |
|----------|------|------|----------|
| 浮点数基本概念 | 1天 | 阶码/尾数、表示范围 | 5道选择题 |
| **IEEE 754单精度（重点）** | 2天 | 符号位/阶码/尾数、偏置值 | 15道计算题 |
| IEEE 754双精度 | 1天 | 64位格式、特殊值 | 8道选择题 |
| 规格化（左规/右规） | 1天 | 规格化判断、操作 | 8道计算题 |
| 浮点数溢出 | 1天 | 上溢/下溢判断 | 8道选择题 |
| 原码一位乘法 | 1天 | 手动模拟乘法过程 | 5道计算题 |
| **Booth补码乘法（重点）** | 2天 | 算法步骤、手动模拟 | 8道计算题 |
| 原码恢复余数除法 | 1天 | 手动模拟除法过程 | 5道计算题 |
| **原码加减交替除法（重点）** | 2天 | 算法步骤、手动模拟 | 8道计算题 |
| **浮点数加减运算（重点）** | 2天 | 对阶→运算→规格化→舍入 | 10道大题 |

#### 第4-5周：存储系统

| 学习内容 | 时间 | 重点 | 配套练习 |
|----------|------|------|----------|
| 存储器分类与层次 | 1天 | SRAM/DRAM、层次结构 | 5道选择题 |
| DRAM刷新方式 | 1天 | 集中/分散/异步刷新 | 5道选择题 |
| **存储容量扩展（重点）** | 2天 | 位扩展/字扩展/字位同时扩展 | 10道计算题 |
| 多模块存储器 | 1天 | 高位交叉/低位交叉 | 5道计算题 |
| 存储带宽计算 | 1天 | 带宽公式、计算方法 | 5道计算题 |
| Cache基本原理 | 1天 | 命中率、工作原理 | 5道选择题 |
| **直接映射（重点）** | 2天 | 地址划分、命中判断 | 15道计算题 |
| **全相联映射** | 1天 | 地址划分、命中判断 | 8道计算题 |
| **组相联映射（重点）** | 2天 | 地址划分、n路组相联 | 15道计算题 |
| 替换算法 | 1天 | FIFO/LRU/随机 | 5道选择题 |

#### 第6周：指令系统

## 学习方法建议

### ✅ 推荐方法

1. **手算练习**
   - IEEE 754编码要能手动完成
   - Booth乘法要能手动模拟
   - Cache地址映射要能快速判断
   - 流水线时空图要能手绘

2. **画图辅助**
   - 存储系统层次图
   - Cache地址划分图
   - 流水线时空图
   - 中断处理流程图

3. **对比记忆**
   - 原码/反码/补码/移码对比
   - 直接映射/全相联/组相联对比
   - 程序查询/中断/DMA/通道对比
   - CISC/RISC对比

4. **真题训练**
   - 每个知识点配套真题练习
   - 限时做套卷
   - 分析错题原因

### ❌ 避免方法

1. ❌ 只背公式不理解推导
2. ❌ 只看不练（计算题必须手算）
3. ❌ 忽略选择题（408选择题占比大）
4. ❌ 不做真题

## 重点难点突破

### 🔴 重点1：IEEE 754浮点数

**突破方法：**
1. 背诵格式：1位符号 + 8位阶码 + 23位尾数
2. 理解偏置值：单精度127，双精度1023
3. 大量练习转换题
4. 掌握特殊值（0/∞/NaN）

### 🔴 重点2：Cache映射

**突破方法：**
1. 理解地址划分：标记/组号/块内偏移
2. 画图辅助判断命中
3. 大量练习计算题
4. 对比三种映射方式

### 🔴 重点3：流水线性能

**突破方法：**
1. 先画时空图再计算
2. 理解吞吐率/加速比/效率的定义
3. 掌握冲突处理方法
4. 大量练习计算题
