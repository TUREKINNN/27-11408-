# SRAM 和 DRAM

## 一、基本概念

- **SRAM**（Static Random Access Memory）：静态随机存取存储器
- **DRAM**（Dynamic Random Access Memory）：动态随机存取存储器

| | SRAM | DRAM |
|--|------|------|
| **用途** | Cache | 主存 |
| **存储元** | 双稳态触发器（6 个 MOS 管） | 栅极电容 |

---

## 二、存储元对比

### 2.1 SRAM — 双稳态触发器

- **存储 1**：A 高 B 低
- **存储 0**：A 低 B 高
- 只要不断电，触发器的状态就不会改变
- **非破坏性读出**，无需重写
- 读写速度**更快**
- 每个存储元制造成本**更高**，集成度**低**，功耗**大**

### 2.2 DRAM — 栅极电容

- **存储 1**：电容内存储了电荷
- **存储 0**：电容内未存储电荷
- 电容内的电荷只能维持约 **2ms**，即便不断电也会消失
- **破坏性读出**，读出后需要**重写（再生）**
- 读写速度**更慢**
- 每个存储元制造成本**更低**，集成度**高**，功耗**低**

---

## 三、DRAM vs SRAM 对比表 ⭐⭐⭐

| 特性 | SRAM | DRAM |
|------|------|------|
| 存储信息 | 触发器 | 电容 |
| 破坏性读出 | **否** | **是** |
| 读出后需要重写 | 不需要 | **需要（再生）** |
| 运行速度 | **快** | 慢 |
| 集成度 | 低 | **高** |
| 发热量 | **大** | 小 |
| 存储成本 | **高** | 低 |
| 易失性 | 易失 | 易失 |
| 需要"刷新" | **不需要** | **需要** |
| 送行列地址 | **同时送** | **分两次送** |
| 常用作 | **Cache** | **主存** |

---

## 四、DRAM 的刷新 ⭐⭐⭐

### 4.1 刷新的基本概念

- **刷新周期**：一般为 **2ms**
- **刷新单位**：以**行**为单位，每次刷新一行存储单元
- 刷新由**存储器独立完成**，不需要 CPU 控制

### 4.2 为什么用行列地址？

存储单元排列成 $2^{n/2} \times 2^{n/2}$ 的矩阵，使用行列地址可以**减少选通线的数量**。

例如：256 个存储单元
- 不用行列地址：需要 $2^8 = 256$ 根选通线
- 用行列地址：$2^4 + 2^4 = 32$ 根选通线

### 4.3 三种刷新方式

假设 DRAM 内部结构排列成 128×128 的形式，读/写周期 0.5μs，2ms 共 4000 个周期。

#### 方式一：分散刷新

每次读写完都刷新一行。

- 系统的存取周期变为 **1μs**（前 0.5μs 正常读写，后 0.5μs 刷新）
- 优点：没有集中的"死区"
- 缺点：存取周期变长

#### 方式二：集中刷新

2ms 内集中安排时间全部刷新。

- 集中刷新需要 128 个周期（64μs）
- 剩余 3872 个周期用于正常读写
- 系统的存取周期还是 **0.5μs**
- 缺点：有一段时间专门用于刷新，存在"死区"

#### 方式三：异步刷新

2ms 内每行刷新 1 次即可，每隔 $2ms/128 = 15.6μs$ 产生一次刷新请求。

- 每 15.6μs 内有 0.5μs 的"死时间"
- 可在**译码阶段**刷新，进一步减少死区
- **实际中最常用**

---

## 五、DRAM 的地址线复用技术

DRAM 的行、列地址**分两次送**，可以**使地址线减半**，芯片引脚更少。

```
地址线 ──→ 行地址缓冲器 ──→ 行地址译码器
         ──→ 列地址缓冲器 ──→ 列地址译码器
```

> 现在的主存通常采用 **SDRAM**（Synchronous DRAM）芯片。

## 六、408 考点总结

1. **SRAM vs DRAM 的对比**是高频选择题考点
2. **破坏性读出**：只有 DRAM 是破坏性读出
3. **刷新**：只有 DRAM 需要刷新，刷新由存储器独立完成
4. **三种刷新方式**的特点和区别
5. **地址线复用技术**：DRAM 行列地址分两次送，地址线减半
6. SRAM 用于 Cache，DRAM 用于主存
