## 1. 数制与编码

### 1.1 数制的概念

<span style="background-color:#ccc1d9">**数制**</span>：用一组固定的符号和统一的规则表示数值的方法。

### 1.2 常用数制

| 数制 | 基数 | 符号 | 示例 |
|------|------|------|------|
| **二进制** | 2 | 0, 1 | 1010B |
| **八进制** | 8 | 0-7 | 12O |
| **十进制** | 10 | 0-9 | 10D |
| **十六进制** | 16 | 0-9, A-F | 0AH |

### 1.3 数制转换

#### 1.3.1 其他进制转十进制

$$
(N)_b = \sum_{i=-m}^{n-1} d_i \cdot b^i
$$

#### 1.3.2 十进制转二进制

**整数部分**：除2取余，逆序排列

**小数部分**：乘2取整，顺序排列

## 2. 定点数

### 2.1 定点数的定义

<span style="background-color:#ccc1d9">**定点数**</span>：小数点位置固定不变的数。

### 2.2 定点数的编码

#### 2.2.1 原码

```
[X]原 = 0.X1X2...Xn  （X >= 0）
[X]原 = 1.X1X2...Xn  （X < 0）
```

**范围**：$-(1 - 2^{-n}) \sim (1 - 2^{-n})$

#### 2.2.2 反码

```
[X]反 = [X]原  （X >= 0）
[X]反 = 各位取反  （X < 0）
```

#### 2.2.3 补码

```
[X]补 = [X]原  （X >= 0）
[X]补 = [X]反 + 1  （X < 0）
```

**范围**：$-1 \sim (1 - 2^{-n})$

#### 2.2.4 移码

```
[X]移 = [X]补 + 2^(n-1)
```

**特点**：便于比较大小

### 2.3 定点数的运算

#### 2.3.1 加法运算

```
[X+Y]补 = [X]补 + [Y]补
```

#### 2.3.2 减法运算

```
[X-Y]补 = [X]补 + [-Y]补
```

#### 2.3.3 溢出判断

**双符号位法**：

```
00：正，无溢出
01：正溢出
10：负溢出
11：负，无溢出
```

**进位判断法**：

```
C1 XOR C2 = 1 时有溢出
```

## 3. 浮点数

### 3.1 浮点数的定义

<span style="background-color:#ccc1d9">**浮点数**</span>：小数点位置可以浮动的数。

### 3.2 浮点数的表示

```
N = M × r^E
```

其中 $M$ 为尾数，$E$ 为阶码，$r$ 为基数。

### 3.3 IEEE 754标准

```
┌────┬──────────┬─────────────────────────────┐
│ S  │    E     │              M              │
│ 1位 │  8/11位  │        23/52位              │
└────┴──────────┴─────────────────────────────┘
```

**单精度（32位）**：

- S：符号位，0正1负
- E：阶码，偏移量127
- M：尾数，隐含首位1

**双精度（64位）**：

- S：符号位
- E：阶码，偏移量1023
- M：尾数，隐含首位1

### 3.4 浮点数的运算

#### 3.4.1 加法运算步骤

1. **对阶**：小阶向大阶看齐
2. **尾数相加**：定点数加法
3. **规格化**：调整尾数和阶码
4. **舍入**：处理多余位数
5. **溢出判断**：检查阶码是否溢出

#### 3.4.2 对阶

```
ΔE = E1 - E2
```

若 $\Delta E > 0$，则 $M2$ 右移 $\Delta E$ 位，$E2 += \Delta E$

若 $\Delta E < 0$，则 $M1$ 右移 $-\Delta E$ 位，$E1 += -\Delta E$

#### 3.4.3 规格化

**规格化形式**：

```
1/2 ≤ |M| < 1
```

**左规**：尾数左移，阶码减1

**右规**：尾数右移，阶码加1

## 4. 算术逻辑单元

### 4.1 ALU的功能

<span style="background-color:#ccc1d9">**ALU**</span>：执行算术运算和逻辑运算。

### 4.2 一位全加器

```
Si = Ai XOR Bi XOR Ci-1
Ci = Ai·Bi + (Ai XOR Bi)·Ci-1
```

### 4.3 多位加法器

#### 4.3.1 串行进位加法器

**特点**：低位进位传给高位，速度慢。

#### 4.3.2 并行进位加法器

**特点**：每位进位同时计算，速度快。

**进位公式**：

```
Ci = Gi + Pi·Ci-1
Gi = Ai·Bi（进位生成项）
Pi = Ai XOR Bi（进位传递项）
```

## 5. 乘法运算

### 5.1 原码一位乘法

**思路**：符号位单独处理，数值位逐位相乘。

```
符号：S = S1 XOR S2
数值：|P| = |X| × |Y|
```

### 5.2 补码一位乘法（Booth算法）

**思路**：根据相邻两位决定加减操作。

```
Y[i]Y[i-1]：00→+0，01→+|X|，10→-|X|，11→+0
```

### 5.3 除法运算

#### 5.3.1 原码除法（恢复余数法）

**思路**：比较被除数和除数，决定商0或1。

#### 5.3.2 原码除法（加减交替法）

**思路**：不恢复余数，而是根据余数符号决定下次操作。

## 6. 数据的存储和排列

### 6.1 大端和小端

<span style="background-color:#ccc1d9">**大端序**</span>：高位字节存低地址。

<span style="background-color:#ccc1d9">**小端序**</span>：低位字节存低地址。

### 6.2 边界对齐

<span style="background-color:#ccc1d9">**边界对齐**</span>：数据存储在地址是其大小倍数的位置。

**优点**：访问速度快

**缺点**：可能浪费空间

## 7. 典型例题

### 例题 1：定点数运算

**题目**：已知 $X = 0.1011$，$Y = -0.0101$，用补码计算 $X+Y$ 和 $X-Y$。

**解答**：

```
[X]补 = 0.1011
[Y]补 = 1.1011
[-Y]补 = 0.0101

X+Y：
  0.1011
+ 1.1011
--------
 10.0110 → 0.0110（溢出位丢弃）

X-Y：
  0.1011
+ 0.0101
--------
  0.1110
```

**总结**：补码运算简化了减法操作。

**来源**：408历年真题，考查方式：计算题

### 例题 2：IEEE 754浮点数

**题目**：将十进制数10.5转换为IEEE 754单精度浮点数。

**解答**：

```
10.5 = 1010.1B = 1.0101 × 2^3

符号位：0（正数）
阶码：3 + 127 = 130 = 10000010B
尾数：01010000000000000000000

结果：0 10000010 01010000000000000000000
     = 41280000H
```

**总结**：IEEE 754标准是浮点数的工业标准。

**来源**：408历年真题，考查方式：计算题

---

### 例题 3：浮点数加法

**题目**：已知两个浮点数 $X = 0.1101 × 2^{10}$，$Y = 0.1011 × 2^{01}$，求 $X+Y$。

**解答**：

```
步骤1：对阶
ΔE = 10 - 01 = 1
Y的尾数右移1位：0.01011 × 2^10

步骤2：尾数相加
  0.11010
+ 0.01011
--------
  1.00101

步骤3：规格化
左规：1.00101 → 0.100101 × 2^11

步骤4：舍入
结果：0.1001 × 2^11
```

**总结**：浮点数加法的关键是对阶和规格化。

**来源**：408历年真题，考查方式：计算题
