## 例题精选

### 例2.1 分布函数求参数

设随机变量 X 的分布函数为 $F(x) = \begin{cases} a + be^{-x}, & x > 0 \\ 0, & x \leqslant 0 \end{cases}$，求常数 $a, b$ 及概率 $P\{|X| < 2\}$。

**知识点**：Lecture 3：随机变量与分布函数#2 分布函数的概念及性质

**解答**：由 $F(+\infty) = 1$ 得 $a = 1$；由右连续 $a + b = 0$ 得 $b = -1$。故 $P\{|X| < 2\} = 1 - e^{-2}$。

### 例2.8 均匀分布参数求解

已知 $X \sim U(a, b)(a > 0)$，且 $P\{0 < X < 3\} = \frac{1}{4}$，$P\{X > 4\} = \frac{1}{2}$，求 X 的概率密度及 $P\{1 < X < 5\}$。

**知识点**：Lecture 5：连续型随机变量#1 均匀分布

**解答**：$a = 2, b = 6$，$f(x) = \begin{cases} \frac{1}{4}, & 2 < x < 6 \\ 0, & \text{其他} \end{cases}$，$P\{1 < X < 5\} = \frac{3}{4}$

### 例2.9 指数分布的无记忆性

某元件的工作寿命 X 服从参数为 $\lambda$ 的指数分布。
- (1) 求该元件正常工作 t 小时的概率
- (2) 已知该元件已正常工作10小时，求在此基础上再工作10小时的概率（$\lambda = 0.01$）

**知识点**：Lecture 5：连续型随机变量#2 指数分布、无记忆性

**解答**：
- (1) $P\{X > t\} = e^{-\lambda t}$
- (2) $P\{X > 20 | X > 10\} = e^{-0.1}$
