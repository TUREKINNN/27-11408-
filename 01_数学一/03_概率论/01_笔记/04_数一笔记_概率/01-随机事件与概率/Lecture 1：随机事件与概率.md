## 一、随机试验与样本空间

### 1.1 随机试验（Random Experiment）

随机试验 $E$ 需满足三个条件：
1. **可重复性**：试验可以在相同条件下重复进行
2. **可预见性**：试验的所有可能结果事先已知
3. **随机性**：每次试验出现哪一个结果事先不能确定

**常见例子：**
- 掷一枚硬币，观察正面/反面
- 掷一颗骰子，观察点数
- 记录某城市一天的交通事故次数
- 测量某零件的长度

### 1.2 样本空间（Sample Space）

**定义：** 随机试验 $E$ 的所有可能结果组成的集合称为**样本空间**，记为 $\Omega$。

$$\Omega = \{\omega\}$$

其中 $\omega$ 称为**样本点**（Sample Point）。

**示例：**
| 试验 | 样本空间 |
|------|----------|
| 掷一枚硬币 | $\Omega = \{H, T\}$ |
| 掷一颗骰子 | $\Omega = \{1, 2, 3, 4, 5, 6\}$ |
| 灯泡寿命 | $\Omega = \{t \mid t \geq 0\} = [0, +\infty)$ |
| 交通事故次数 | $\Omega = \{0, 1, 2, 3, \ldots\}$ |

> **注意：** 样本空间可以是有限集、可列无限集或不可列无限集。

## 二、随机事件及其关系与运算

### 2.1 随机事件

**定义：** 样本空间 $\Omega$ 的子集称为**随机事件**，简称事件，通常用大写字母 $A, B, C, \ldots$ 表示。

- **基本事件：** 只含一个样本点的事件
- **必然事件：** 样本空间 $\Omega$ 本身（每次试验必然发生）
- **不可能事件：** 空集 $\varnothing$（每次试验都不发生）

> **事件发生：** 当且仅当事件 $A$ 中的某一个样本点出现时，称事件 $A$ **发生**。

### 2.2 事件间的关系

| 关系 | 含义 | 集合表示 | 记号 |
|------|------|----------|------|
| **包含** | $A$ 发生 $\Rightarrow$ $B$ 发生 | $A \subseteq B$ | $A \subset B$ |
| **相等** | $A \subset B$ 且 $B \subset A$ | $A = B$ | $A = B$ |
| **互斥（互不相容）** | $A, B$ 不能同时发生 | $A \cap B = \varnothing$ | $AB = \varnothing$ |
| **对立（互逆）** | $A, B$ 必有一个且仅一个发生 | $A \cap B = \varnothing$，$A \cup B = \Omega$ | $B = \bar{A}$ |

**重要关系：**
$$\text{对立} \Rightarrow \text{互斥}，\quad \text{互斥} \nRightarrow \text{对立}$$

### 2.3 事件的运算

#### 并（和）事件
$$A \cup B = \{\omega \mid \omega \in A \text{ 或 } \omega \in B\}$$
"$A, B$ 中**至少有一个**发生"

$$\bigcup_{i=1}^{n} A_i = A_1 \cup A_2 \cup \cdots \cup A_n$$

#### 交（积）事件
$$A \cap B = \{\omega \mid \omega \in A \text{ 且 } \omega \in B\}$$
"$A, B$ **同时**发生"，简记为 $AB$

$$\bigcap_{i=1}^{n} A_i = A_1 \cap A_2 \cap \cdots \cap A_n$$

#### 差事件
$$A - B = \{\omega \mid \omega \in A \text{ 且 } \omega \notin B\}$$
"$A$ 发生但 $B$ 不发生"

$$A - B = A \bar{B} = A - AB$$

#### 对立事件（余事件）
$$\bar{A} = \Omega - A$$

### 2.4 运算律

**交换律：**
$$A \cup B = B \cup A, \quad A \cap B = B \cap A$$

**结合律：**
$$(A \cup B) \cup C = A \cup (B \cup C)$$
$$(A \cap B) \cap C = A \cap (B \cap C)$$

**分配律：**
$$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$
$$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$$

**德摩根律（De Morgan's Laws）⭐⭐⭐：**
$$\boxed{\overline{A \cup B} = \bar{A} \cap \bar{B}}$$
$$\boxed{\overline{A \cap B} = \bar{A} \cup \bar{B}}$$

推广：
$$\overline{\bigcup_{i=1}^{n} A_i} = \bigcap_{i=1}^{n} \bar{A_i}, \quad \overline{\bigcap_{i=1}^{n} A_i} = \bigcup_{i=1}^{n} \bar{A_i}$$

> **记忆口诀：** "并的补等于补的交，交的补等于补的并"——**长杠变短杠，短杠变长杠，并交互换**。

## 三、概率的定义与性质

### 3.1 概率的公理化定义（Kolmogorov公理）

设 $E$ 是随机试验，$\Omega$ 是样本空间，对 $E$ 的每一个事件 $A$，赋予一个实数 $P(A)$，满足：

**公理1（非负性）：** $\forall A \subseteq \Omega$，有
$$P(A) \geq 0$$

**公理2（规范性）：**
$$P(\Omega) = 1$$

**公理3（可列可加性）：** 若 $A_1, A_2, \ldots$ 两两互斥（$A_iA_j = \varnothing, i \neq j$），则
$$P\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(A_i)$$

则称 $P(A)$ 为事件 $A$ 的**概率**。

### 3.2 概率的基本性质

由三条公理可推出以下重要性质：

**性质1（不可能事件）：**
$$P(\varnothing) = 0$$

**性质2（有限可加性）：** 若 $A_1, A_2, \ldots, A_n$ 两两互斥，则
$$P\left(\bigcup_{i=1}^{n} A_i\right) = \sum_{i=1}^{n} P(A_i)$$

**性质3（对立事件概率）：**
$$\boxed{P(\bar{A}) = 1 - P(A)}$$

**性质4（减法公式）：**
$$\boxed{P(A - B) = P(A) - P(AB)}$$

特别地，若 $B \subset A$，则 $P(A - B) = P(A) - P(B)$

**性质5（加法公式）：**
$$\boxed{P(A \cup B) = P(A) + P(B) - P(AB)}$$

推广到三个事件：
$$P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC)$$

**性质6（单调性）：** 若 $A \subset B$，则
$$P(A) \leq P(B)$$

**性质7（概率的有界性）：**
$$0 \leq P(A) \leq 1$$

## 四、古典概型（等可能概型）

### 4.1 定义

若随机试验满足：
1. **有限性**：样本空间 $\Omega$ 只含有限个样本点，即 $\Omega = \{\omega_1, \omega_2, \ldots, \omega_n\}$
2. **等可能性**：每个基本事件发生的概率相等，即 $P(\{\omega_1\}) = P(\{\omega_2\}) = \cdots = P(\{\omega_n\}) = \dfrac{1}{n}$

则称此试验模型为**古典概型**（Classical Probability Model）。

### 4.2 古典概型中事件概率的计算

若事件 $A$ 包含 $k$ 个样本点，则：

$$\boxed{P(A) = \frac{A \text{ 包含的样本点数}}{\Omega \text{ 中样本点总数}} = \frac{k}{n}}$$

### 4.3 排列组合基础

计算古典概型需要排列组合知识：

**排列（Permutation）：**
$$A_n^m = P_n^m = \frac{n!}{(n-m)!} = n(n-1)(n-2)\cdots(n-m+1)$$

**组合（Combination）：**
$$C_n^m = \binom{n}{m} = \frac{n!}{m!(n-m)!} = \frac{A_n^m}{m!}$$

**重要公式：**
$$\binom{n}{m} = \binom{n}{n-m}$$
$$\binom{n}{m} = \binom{n-1}{m-1} + \binom{n-1}{m}$$
$$\sum_{k=0}^{n} \binom{n}{k} = 2^n$$
$$\binom{n}{0} + \binom{n}{2} + \binom{n}{4} + \cdots = 2^{n-1}$$

### 4.4 经典例题

**【例1】抽签问题（抽签公平性）**
袋中有 $a$ 个白球和 $b$ 个黑球，$n$ 个人依次不放回地各取一球，求第 $k$ 个人取到白球的概率。

**解：** 从 $a+b$ 个球中取 $n$ 个排列，第 $k$ 个位置放白球的方法数：先从 $a$ 个白球中选1个放在第 $k$ 个位置，其余 $n-1$ 个位置从剩余 $a+b-1$ 个球中排列。

$$P = \frac{a \cdot A_{a+b-1}^{n-1}}{A_{a+b}^{n}} = \frac{a}{a+b}$$

> **结论：抽签与顺序无关，每人抽到白球的概率都是 $\dfrac{a}{a+b}$。**

**【例2】生日问题**
$n$ 个人中至少有两人生日相同的概率是多少？（一年按365天计算）

**解：** 考虑对立事件——$n$ 个人生日全不相同：

$$P(\text{全不同}) = \frac{A_{365}^{n}}{365^n} = \frac{365 \times 364 \times \cdots \times (365-n+1)}{365^n}$$

$$P(\text{至少两人相同}) = 1 - \frac{A_{365}^{n}}{365^n}$$

| $n$ | 20 | 23 | 30 | 40 | 50 | 60 |
|-----|-----|-----|-----|-----|-----|-----|
| 概率 | 0.411 | 0.507 | 0.706 | 0.891 | 0.970 | 0.994 |

> **注：** 仅需23人，概率就超过50%！

**【例3】超几何概率**
$N$ 件产品中有 $M$ 件次品，从中不放回地取 $n$ 件，恰好取到 $k$ 件次品的概率：

$$P(X=k) = \frac{\binom{M}{k}\binom{N-M}{n-k}}{\binom{N}{n}}, \quad k = 0, 1, \ldots, \min(M, n)$$

## 五、几何概型

### 5.1 定义

当样本空间 $\Omega$ 是某区域（可以是一维区间、二维平面区域或三维空间区域），且具有**等可能性**（样本点落在 $\Omega$ 中任何等"大小"的子区域的概率相同）时，事件 $A$ 的概率为：

$$\boxed{P(A) = \frac{A \text{ 的几何度量（长度/面积/体积）}}{\Omega \text{ 的几何度量（长度/面积/体积）}} = \frac{S(A)}{S(\Omega)}}$$

其中 $S(\cdot)$ 表示相应的几何度量。

### 5.2 几何概型的关键要素

1. **样本空间为连续区域**
2. **等可能性**：概率与区域的"大小"（度量）成正比，与位置和形状无关
3. 几何度量可以是**长度、面积、体积**等

### 5.3 经典例题

**【例4】会面问题（约会问题）**
甲、乙两人约定在某地见面，每人等对方不超过 $T$ 分钟。求两人能见面的概率。

**解：** 设甲 $x$ 分钟到达，乙 $y$ 分钟到达，则样本空间为正方形：
$$\Omega = \{(x, y) \mid 0 \leq x \leq T, \ 0 \leq y \leq T\}$$

两人见面的条件：$|x - y| \leq T$

用正方形面积减去两个三角形面积：

$$P = \frac{T^2 - 2 \times \frac{1}{2}(T - T)^2}{T^2}$$

设等 $t$ 分钟（$t < T$）：
$$P(|x-y| \leq t) = \frac{T^2 - (T-t)^2}{T^2} = 1 - \left(1 - \frac{t}{T}\right)^2$$

特别地，当 $t = T/2$ 时，$P = 3/4$。

**【例5】蒲丰投针问题（Buffon's Needle）**
平面上画有间距为 $a$ 的平行线，向平面上随机投一枚长度为 $l$（$l < a$）的针，求针与直线相交的概率。

**解：** 设针的中点到最近直线的距离为 $x$，针与平行线的夹角为 $\theta$：
$$\Omega = \{(x, \theta) \mid 0 \leq x \leq \frac{a}{2}, \ 0 \leq \theta \leq \pi\}$$

针与直线相交的条件：$x \leq \dfrac{l}{2}\sin\theta$

$$P = \frac{\int_0^{\pi} \frac{l}{2}\sin\theta \, d\theta}{\frac{a}{2} \cdot \pi} = \frac{l}{a\pi} \times 2 = \frac{2l}{\pi a}$$

> **注：** 这个结果可以用来估算 $\pi$ 的值！$\pi = \dfrac{2l}{aP}$

**【例6】均匀分布上的几何概型**
在线段 $[0, 1]$ 上随机取两个点，求两点之间距离不超过 $1/2$ 的概率。

**解：** 设两点为 $x, y \in [0, 1]$，样本空间面积 $S(\Omega) = 1$

事件 $|x - y| \leq 1/2$ 的面积：

$$P = 1 - 2 \times \frac{1}{2} \times \left(\frac{1}{2}\right)^2 = 1 - \frac{1}{4} = \frac{3}{4}$$

## 六、重要方法与技巧总结

### 6.1 事件表示技巧

将复杂事件用基本事件的运算表示是解题的关键：

- "至少一个发生" → $A_1 \cup A_2 \cup \cdots \cup A_n = \bigcup_{i=1}^{n} A_i$
- "恰好发生 $k$ 个" → 选择 $k$ 个的并，再与其他交
- "全部发生" → $A_1 \cap A_2 \cap \cdots \cap A_n = \bigcap_{i=1}^{n} A_i$
- "都不发生" → $\bar{A_1} \cap \bar{A_2} \cap \cdots \cap \bar{A_n} = \overline{\bigcup_{i=1}^{n} A_i}$（德摩根律）

### 6.2 "至少一个"的概率

$$P(\text{至少一个发生}) = 1 - P(\text{全不发生}) = 1 - P(\bar{A_1}\bar{A_2}\cdots\bar{A_n})$$

**当事件相互独立时：**
$$P(\text{至少一个发生}) = 1 - \prod_{i=1}^{n} P(\bar{A_i}) = 1 - \prod_{i=1}^{n} [1 - P(A_i)]$$

### 6.3 古典概型计数策略

| 问题类型 | 方法 |
|----------|------|
| 有放回抽样 | $n^m$ 种（可重复排列） |
| 无放回抽样 | $A_n^m$ 种（不重复排列） |
| 分组问题 | 注意是否有序（用排列还是组合） |
| "至多/至少" | 通常考虑对立事件更简洁 |

## 七、易错点与考试陷阱

### ⚠️ 易错点1：互斥与对立的混淆

- **互斥**：$AB = \varnothing$（不能同时发生）
- **对立**：$AB = \varnothing$ **且** $A \cup B = \Omega$（恰好一个发生）
- 对立一定互斥，互斥不一定对立

### ⚠️ 易错点2：概率为0的事件不一定是不可能事件

- 在连续型样本空间中，$P(A) = 0$ 并不能推出 $A = \varnothing$
- 例如：在 $[0,1]$ 上取一个确定值（如 $x = 0.5$），$P = 0$ 但事件并非不可能

### ⚠️ 易错点3：几何概型的"等可能性"假设

- 几何概型的关键是**等可能性**，不是所有的连续问题都是几何概型
- 需要明确样本空间的几何度量是什么（长度？面积？体积？）

### ⚠️ 易错点4：排列组合中的有序与无序

- 排列 = 有序选取
- 组合 = 无序选取
- 是否放回影响总数的计算

## 九、习题精选

**习题1：** 一批产品共10件，其中3件次品，从中任取3件，求至少取到1件次品的概率。

**解：**
$$P(\text{至少1件次品}) = 1 - P(\text{全为正品}) = 1 - \frac{\binom{7}{3}}{\binom{10}{3}} = 1 - \frac{35}{120} = 1 - \frac{7}{24} = \frac{17}{24}$$

**习题2：** 将一枚骰子连续掷两次，求点数之和为6的概率。

**解：** 总数 $6^2 = 36$。和为6的组合：$(1,5),(2,4),(3,3),(4,2),(5,1)$，共5种。
$$P = \frac{5}{36}$$

**习题3：** 在区间 $(0, 2)$ 内随机取两个数 $x, y$，求 $xy < 1$ 的概率。

**解：** 样本空间面积 $= 2 \times 2 = 4$

$xy < 1$ 即 $y < 1/x$。面积 $= \int_0^2 \min(2, 1/x) \, dx$

当 $x \in (0, 1/2)$ 时，$1/x > 2$，所以 $y \in (0, 2)$，贡献面积 $= 1$
当 $x \in [1/2, 2)$ 时，$y \in (0, 1/x)$，贡献面积 $= \int_{1/2}^{2} \frac{1}{x} dx = \ln 2 - \ln(1/2) = 2\ln 2$

$$P = \frac{1 + 2\ln 2}{4} \approx \frac{1 + 1.386}{4} = \frac{2.386}{4} \approx 0.597$$
