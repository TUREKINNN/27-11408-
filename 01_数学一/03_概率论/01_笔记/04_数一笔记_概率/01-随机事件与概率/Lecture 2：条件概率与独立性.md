## 一、条件概率

### 1.1 条件概率的定义

**定义：** 设 $A, B$ 是两个事件，且 $P(B) > 0$，则在事件 $B$ 已经发生的条件下事件 $A$ 发生的**条件概率**定义为：

$$\boxed{P(A|B) = \frac{P(AB)}{P(B)}}$$

**直观理解：** 将 $B$ 作为新的样本空间，$AB$ 是我们关注的部分，条件概率就是 $AB$ 在 $B$ 中的"相对大小"。

### 1.2 条件概率的性质

条件概率 $P(\cdot|B)$ 本身满足概率的所有公理和性质：

1. **非负性**：$P(A|B) \geq 0$
2. **规范性**：$P(\Omega|B) = 1$，即 $P(B|B) = 1$
3. **可加性**：若 $A_1, A_2, \ldots$ 两两互斥，则 $P\left(\bigcup_{i=1}^{\infty} A_i \bigg| B\right) = \sum_{i=1}^{\infty} P(A_i|B)$

**重要推论：**
$$P(\bar{A}|B) = 1 - P(A|B)$$
$$P(A_1 \cup A_2 | B) = P(A_1|B) + P(A_2|B) - P(A_1A_2|B)$$

> **考试技巧：** 使用条件概率公式时，关键是确定"条件事件" $B$ 和"目标事件" $A$，并确保 $P(B) > 0$。

## 二、乘法公式

### 2.1 两个事件的乘法公式

由条件概率定义 $P(A|B) = \dfrac{P(AB)}{P(B)}$，可得：

$$\boxed{P(AB) = P(B) \cdot P(A|B) = P(A) \cdot P(B|A)}$$

> 其中要求 $P(B) > 0$（第一个等式）或 $P(A) > 0$（第二个等式）。

### 2.2 多个事件的乘法公式

**三个事件：**
$$P(ABC) = P(A) \cdot P(B|A) \cdot P(C|AB)$$

**$n$ 个事件的一般形式：**
$$\boxed{P(A_1A_2\cdots A_n) = P(A_1) \cdot P(A_2|A_1) \cdot P(A_3|A_1A_2) \cdots P(A_n|A_1A_2\cdots A_{n-1})}$$

> **记忆：** "链式法则"——每个条件都建立在前面所有事件都发生的基础上。

### 2.3 经典应用

**【例1】不放回抽样**
袋中有 $a$ 个白球和 $b$ 个红球，不放回地连续取2球，求两次都取到白球的概率。

**解：**
$$P(\text{两白}) = P(A_1A_2) = P(A_1) \cdot P(A_2|A_1) = \frac{a}{a+b} \cdot \frac{a-1}{a+b-1}$$

**【例2】产品检验**
一批产品共 $N$ 件，其中 $M$ 件次品。逐件不放回检验，求第 $k$ 次才首次发现次品的概率。

**解：** 前 $k-1$ 次取到正品，第 $k$ 次取到次品：
$$P = \frac{N-M}{N} \cdot \frac{N-M-1}{N-1} \cdots \frac{N-M-k+2}{N-k+2} \cdot \frac{M}{N-k+1}$$

## 三、全概率公式 ⭐⭐⭐

### 3.1 完备事件组（样本空间的分割）

**定义：** 设 $B_1, B_2, \ldots, B_n$ 是一组事件，满足：
1. **两两互斥**：$B_iB_j = \varnothing$（$i \neq j$）
2. **穷尽**：$\bigcup_{i=1}^{n} B_i = \Omega$

则称 $B_1, B_2, \ldots, B_n$ 为样本空间 $\Omega$ 的一个**完备事件组**（或**分割**）。

等价条件：$B_1, B_2, \ldots, B_n$ 两两互斥，且 $P(B_i) > 0$，$\sum_{i=1}^{n} P(B_i) = 1$

### 3.2 全概率公式

设 $B_1, B_2, \ldots, B_n$ 是 $\Omega$ 的一个完备事件组，$A$ 为任意事件，则：

$$\boxed{P(A) = \sum_{i=1}^{n} P(B_i) \cdot P(A|B_i)}$$

**推导：**
$$A = A\Omega = A(B_1 \cup B_2 \cup \cdots \cup B_n) = AB_1 \cup AB_2 \cup \cdots \cup AB_n$$

由于 $B_i$ 两两互斥，$AB_i$ 也两两互斥，由有限可加性：

$$P(A) = \sum_{i=1}^{n} P(AB_i) = \sum_{i=1}^{n} P(B_i)P(A|B_i)$$

### 3.3 直观理解

全概率公式的本质是**"由因溯果"**：

| 概念 | 含义 |
|------|------|
| $B_i$ | "原因"（完备事件组，所有可能的原因） |
| $P(B_i)$ | "先验概率"（各原因发生的可能性） |
| $P(A\|B_i)$ | "似然概率"（在原因 $B_i$ 下结果 $A$ 发生的概率） |
| $P(A)$ | "结果概率"（综合所有原因的结果概率） |

> **应用场景：** 已知各种"原因"发生的概率以及在每种"原因"下事件 $A$ 发生的条件概率，求 $A$ 发生的（无条件）概率。

### 3.4 经典例题

**【例3】工厂产品问题**
三个工厂生产同一产品，产量分别占总产量的 50%、30%、20%，次品率分别为 2%、3%、5%。从产品中随机取一件，求该件为次品的概率。

**解：** 设 $B_i$ 表示产品来自第 $i$ 个工厂（$i=1,2,3$），$A$ 表示产品为次品。

$$P(B_1) = 0.5, \quad P(B_2) = 0.3, \quad P(B_3) = 0.2$$
$$P(A|B_1) = 0.02, \quad P(A|B_2) = 0.03, \quad P(A|B_3) = 0.05$$

由全概率公式：
$$P(A) = 0.5 \times 0.02 + 0.3 \times 0.03 + 0.2 \times 0.05 = 0.01 + 0.009 + 0.01 = 0.029$$

**【例4】疾病的传播**
某种疾病在人群中的患病率为 1%。有一种检测方法，对患者检测阳性的概率为 95%（灵敏度），对健康人检测为阳性的概率为 3%（误诊率）。某人检测为阳性，求此人确实患病的概率。

> 此题需要贝叶斯公式，见下一节。

## 四、贝叶斯公式（逆概率公式）⭐⭐⭐⭐⭐

### 4.1 贝叶斯公式

设 $B_1, B_2, \ldots, B_n$ 是 $\Omega$ 的一个完备事件组，$P(B_i) > 0$，$A$ 为满足 $P(A) > 0$ 的事件，则：

$$\boxed{P(B_j|A) = \frac{P(B_j) \cdot P(A|B_j)}{\sum_{i=1}^{n} P(B_i) \cdot P(A|B_i)} = \frac{P(B_j) \cdot P(A|B_j)}{P(A)}}$$

其中 $j = 1, 2, \ldots, n$。

### 4.2 直观理解

贝叶斯公式实现了**"由果溯因"**——事件 $A$ 已经发生，反推是由哪个原因 $B_j$ 导致的。

| 概念 | 含义 |
|------|------|
| $P(B_j)$ | **先验概率**（Prior）——在观测到 $A$ 之前对 $B_j$ 的认知 |
| $P(B_j\|A)$ | **后验概率**（Posterior）——在观测到 $A$ 之后对 $B_j$ 的更新认知 |
| $P(A\|B_j)$ | **似然函数**（Likelihood） |
| $P(A)$ | **证据**（Evidence）——归一化常数 |

### 4.3 经典例题

**【例5】续例4——疾病的贝叶斯分析**

设 $B_1$ = 患病，$B_2$ = 健康，$A$ = 检测阳性。

$$P(B_1) = 0.01, \quad P(B_2) = 0.99$$
$$P(A|B_1) = 0.95, \quad P(A|B_2) = 0.03$$

先求 $P(A)$（全概率公式）：
$$P(A) = 0.01 \times 0.95 + 0.99 \times 0.03 = 0.0095 + 0.0297 = 0.0392$$

再求后验概率（贝叶斯公式）：
$$P(B_1|A) = \frac{0.01 \times 0.95}{0.0392} = \frac{0.0095}{0.0392} \approx 0.242$$

> **结论：** 即使检测阳性，实际患病的概率也只有约 24.2%！这说明即使检测方法看起来准确率很高，在低患病率的情况下，阳性结果的可信度可能远低于直觉。这就是著名的**"基本比率谬误"**。

**【例6】赌博问题（Monty Hall问题的推广）**
三个盒子中一个装有奖品。参赛者选定一个盒子后，主持人打开另一个空盒子。参赛者是否应该换选？

**解：** 设 $B_i$ 表示奖品在第 $i$ 个盒子，$A$ 表示主持人打开第3个盒子为空。

假设参赛者选了第1个盒子：
- $P(B_i) = 1/3$（$i = 1, 2, 3$）
- 若 $B_1$（奖品在第1盒）：主持人可开第2或第3盒，$P(A|B_1) = 1/2$
- 若 $B_2$（奖品在第2盒）：主持人只能开第3盒，$P(A|B_2) = 1$
- 若 $B_3$（奖品在第3盒）：主持人不能开第3盒，$P(A|B_3) = 0$

$$P(A) = \frac{1}{3} \times \frac{1}{2} + \frac{1}{3} \times 1 + \frac{1}{3} \times 0 = \frac{1}{6} + \frac{1}{3} = \frac{1}{2}$$

$$P(B_2|A) = \frac{\frac{1}{3} \times 1}{\frac{1}{2}} = \frac{2}{3}$$

> **结论：** 换选的概率为 $2/3$，不换为 $1/3$。**应该换！**

## 五、事件的独立性 ⭐⭐⭐⭐

### 5.1 两个事件的独立性

**定义：** 设 $A, B$ 是两个事件，若
$$\boxed{P(AB) = P(A) \cdot P(B)}$$

则称 $A$ 与 $B$ **相互独立**（Independent），简称 $A$ 与 $B$ **独立**。

> **注意：** 这是独立性的**定义式**，是判断独立性的唯一标准。

### 5.2 独立性的等价条件

当 $P(A) > 0$ 时，以下条件等价：
- $P(AB) = P(A)P(B)$
- $P(B|A) = P(B)$
- $P(B|A) = P(B|\bar{A})$

当 $P(B) > 0$ 时，以下条件等价：
- $P(AB) = P(A)P(B)$
- $P(A|B) = P(A)$
- $P(A|B) = P(A|\bar{B})$

> **直觉：** $A$ 的发生不影响 $B$ 发生的概率，反之亦然。

### 5.3 独立性的性质

**定理：** 若 $A$ 与 $B$ 独立，则以下各对事件也独立：
- $A$ 与 $\bar{B}$
- $\bar{A}$ 与 $B$
- $\bar{A}$ 与 $\bar{B}$

**证明：** 以 $A$ 与 $\bar{B}$ 独立为例：
$$P(A\bar{B}) = P(A) - P(AB) = P(A) - P(A)P(B) = P(A)[1-P(B)] = P(A)P(\bar{B}) \quad \checkmark$$

### 5.4 独立与互斥的关系

| 性质 | 独立 | 互斥 |
|------|------|------|
| 定义 | $P(AB) = P(A)P(B)$ | $AB = \varnothing$ |
| 直觉 | 一事件的发生不影响另一事件 | 两事件不能同时发生 |
| 关系 | 独立 $\nRightarrow$ 互斥 | 互斥 $\nRightarrow$ 独立 |

> **重要结论：** 若 $P(A) > 0$，$P(B) > 0$，则：
> - **互斥必不独立**：$AB = \varnothing \Rightarrow P(AB) = 0 \neq P(A)P(B) > 0$
> - **独立必不互斥**：$P(AB) = P(A)P(B) > 0 \Rightarrow AB \neq \varnothing$

### 5.5 多个事件的独立性

#### 两两独立
$A, B, C$ 两两独立是指：
$$P(AB) = P(A)P(B), \quad P(AC) = P(A)P(C), \quad P(BC) = P(B)P(C)$$

#### 相互独立
$A, B, C$ 相互独立是指**两两独立**且：
$$P(ABC) = P(A)P(B)P(C)$$

#### 一般地，$n$ 个事件相互独立
$A_1, A_2, \ldots, A_n$ 相互独立是指，对任意 $2 \leq k \leq n$ 和 $1 \leq i_1 < i_2 < \cdots < i_k \leq n$：
$$P(A_{i_1}A_{i_2}\cdots A_{i_k}) = P(A_{i_1})P(A_{i_2})\cdots P(A_{i_k})$$

> **注意：** 相互独立 $\Rightarrow$ 两两独立，但 **两两独立 $\nRightarrow$ 相互独立**！

**反例：** 四面体的三个面分别标 $1, 2, 3$，底面标 $1, 2, 3$ 中的一个。设：
- $A$ = 朝下的面含数字1
- $B$ = 朝下的面含数字2
- $C$ = 朝下的面含数字3

则 $P(A) = P(B) = P(C) = 1/2$，$P(AB) = P(A)P(B) = 1/4$，但 $P(ABC) = 1/4 \neq P(A)P(B)P(C) = 1/8$。

### 5.6 独立性的实际应用

**【例7】系统可靠性**
设系统由4个元件组成，每个元件正常工作的概率为 $p$，且各元件独立工作。

**(1) 串联系统：** 全部正常才正常
$$P(\text{系统正常}) = p^4$$

**(2) 并联系统：** 至少一个正常就正常
$$P(\text{系统正常}) = 1 - (1-p)^4$$

**(3) 混联系统（两组并联再串联）：**
$$P(\text{系统正常}) = [1-(1-p)^2]^2 = (2p-p^2)^2$$

**【例8】独立重复试验**
某射手命中率为 0.8，独立射击 3 次，求恰好命中 2 次的概率。

**解：** 这是 $n = 3$，$p = 0.8$ 的二项分布问题：
$$P(X = 2) = \binom{3}{2} \times 0.8^2 \times 0.2 = 3 \times 0.64 \times 0.2 = 0.384$$

## 六、全概率公式与贝叶斯公式的综合应用

### 6.1 多阶段试验的全概率

**【例9】传染病模型**
已知某传染病在人群中的感染率为 $p$。有一种试剂可检测，对感染者检测为阳性的概率为 $\alpha$，对非感染者检测为阳性的概率为 $\beta$。

**(1)** 求随机选一人检测为阳性的概率：
$$P(+) = p\alpha + (1-p)\beta$$

**(2)** 检测为阳性后实际感染的概率：
$$P(\text{感染}|+) = \frac{p\alpha}{p\alpha + (1-p)\beta}$$

**(3)** 如果此人再检测一次仍为阳性（两次独立），感染概率：
$$P(\text{感染}|++,++) = \frac{p\alpha^2}{p\alpha^2 + (1-p)\beta^2}$$

### 6.2 贝叶斯公式的递推更新

贝叶斯公式可以迭代使用——将上一步的后验概率作为下一步的先验概率：

$$P(B_j|A_1, A_2) = \frac{P(B_j|A_1) \cdot P(A_2|B_j, A_1)}{\sum_i P(B_i|A_1) \cdot P(A_2|B_i, A_1)}$$

当 $A_1, A_2$ 在 $B_j$ 条件下独立时：
$$P(B_j|A_1, A_2) = \frac{P(B_j|A_1) \cdot P(A_2|B_j)}{\sum_i P(B_i|A_1) \cdot P(A_2|B_i)}$$

## 七、重要公式汇总

### 7.1 条件概率公式体系

```
条件概率
├── 定义：P(A|B) = P(AB)/P(B)    [P(B)>0]
├── 乘法公式：P(AB) = P(A)·P(B|A) = P(B)·P(A|B)
│   └── 推广：P(A₁A₂...Aₙ) = P(A₁)·P(A₂|A₁)·P(A₃|A₁A₂)···P(Aₙ|A₁...Aₙ₋₁)
├── 全概率公式：P(A) = Σᵢ P(Bᵢ)·P(A|Bᵢ)    [由因溯果]
└── 贝叶斯公式：P(Bⱼ|A) = P(Bⱼ)·P(A|Bⱼ) / Σᵢ P(Bᵢ)·P(A|Bᵢ)    [由果溯因]
```

### 7.2 独立性相关公式

| 场景 | 公式 |
|------|------|
| 两事件独立 | $P(AB) = P(A)P(B)$ |
| $n$ 事件独立 | $P(A_1A_2\cdots A_n) = \prod_{i=1}^{n} P(A_i)$ |
| 至少一个发生 | $P(\bigcup A_i) = 1 - \prod[1-P(A_i)]$（独立时） |
| 恰好 $k$ 个发生 | $\binom{n}{k} p^k (1-p)^{n-k}$（独立同概率时） |

## 八、易错点与考试陷阱

### ⚠️ 易错点1：条件概率中事件的顺序

$P(A|B)$ 与 $P(B|A)$ 完全不同！
- $P(A|B) = \dfrac{P(AB)}{P(B)}$
- $P(B|A) = \dfrac{P(AB)}{P(A)}$

只有当 $P(A) = P(B)$ 时，才有 $P(A|B) = P(B|A)$。

### ⚠️ 易错点2：贝叶斯公式中先验概率的影响

当先验概率 $P(B_i)$ 非常悬殊时，后验概率 $P(B_j|A)$ 可能与直觉不符。
- 如例5中，患病率仅1%，即使检测阳性，实际患病概率也只有24%。

### ⚠️ 易错点3：独立性不能由直觉判断

- "两个不同的随机试验"的结果通常是独立的
- 同一试验的两个事件是否独立需要**验证** $P(AB) = P(A)P(B)$
- **不要混淆"互斥"和"独立"**

### ⚠️ 易错点4：全概率公式中完备事件组的选取

- 完备事件组必须**两两互斥**且**穷尽** $\Omega$
- 同一个问题可能有不同的完备事件组选取方式
- 选取原则：使得 $P(B_i)$ 和 $P(A|B_i)$ 容易计算

### ⚠️ 易错点5：独立事件的运算结果

若 $A$ 与 $B$ 独立：
- $A$ 与 $\bar{B}$ 独立 ✓
- $\bar{A}$ 与 $\bar{B}$ 独立 ✓
- $A, B$ 互斥？**不一定**（通常不互斥）

## 十、习题精选

**习题1：** 设 $P(A) = 0.4$，$P(B) = 0.3$，$P(A \cup B) = 0.6$。求 $P(A|B)$ 和 $P(B|A)$。

**解：**
$$P(AB) = P(A) + P(B) - P(A \cup B) = 0.4 + 0.3 - 0.6 = 0.1$$
$$P(A|B) = \frac{P(AB)}{P(B)} = \frac{0.1}{0.3} = \frac{1}{3}$$
$$P(B|A) = \frac{P(AB)}{P(A)} = \frac{0.1}{0.4} = \frac{1}{4}$$

**习题2：** 设 $P(A) = 1/3$，$P(B|A) = 1/4$，$P(A|B) = 1/2$。求 $P(A \cup B)$。

**解：**
$$P(AB) = P(A) \cdot P(B|A) = \frac{1}{3} \times \frac{1}{4} = \frac{1}{12}$$
$$P(B) = \frac{P(AB)}{P(A|B)} = \frac{1/12}{1/2} = \frac{1}{6}$$
$$P(A \cup B) = P(A) + P(B) - P(AB) = \frac{1}{3} + \frac{1}{6} - \frac{1}{12} = \frac{4+2-1}{12} = \frac{5}{12}$$

**习题3（考研经典）：** 设有来自三个地区的各10名、15名、25名考生的报名表，其中女生的报名表分别为3份、7份、5份。随机取一个地区的报名表，再从中先后取两份。

**(1)** 求先取到的一份是女生表的概率 $p$。
**(2)** 已知后取到的一份是男生表，求先取到的一份是女生表的概率 $q$。

**解：** 设 $H_i$ 表示选中第 $i$ 个地区的表（$i=1,2,3$），$A_j$ 表示第 $j$ 次取到女生表。

$$P(H_i) = \frac{1}{3}$$

**(1)** 全概率公式：
$$p = P(A_1) = \frac{1}{3}\left(\frac{3}{10} + \frac{7}{15} + \frac{5}{25}\right) = \frac{1}{3}\left(\frac{3}{10} + \frac{7}{15} + \frac{1}{5}\right)$$
$$= \frac{1}{3} \times \frac{9 + 14 + 6}{30} = \frac{29}{90}$$

**(2)** 需要求 $P(A_1|\bar{A_2}) = \dfrac{P(A_1\bar{A_2})}{P(\bar{A_2})}$

$$P(A_1\bar{A_2}|H_1) = \frac{3}{10} \times \frac{7}{9} = \frac{7}{30}$$
$$P(A_1\bar{A_2}|H_2) = \frac{7}{15} \times \frac{8}{14} = \frac{4}{15} = \frac{8}{30}$$
$$P(A_1\bar{A_2}|H_3) = \frac{5}{25} \times \frac{20}{24} = \frac{1}{6} = \frac{5}{30}$$

$$P(A_1\bar{A_2}) = \frac{1}{3}\left(\frac{7}{30} + \frac{8}{30} + \frac{5}{30}\right) = \frac{1}{3} \times \frac{20}{30} = \frac{2}{9}$$

$$P(\bar{A_2}) = 1 - P(A_2)$$

由对称性，$P(A_2) = P(A_1) = 29/90$，所以 $P(\bar{A_2}) = 61/90$。

$$q = \frac{P(A_1\bar{A_2})}{P(\bar{A_2})} = \frac{2/9}{61/90} = \frac{20}{61}$$
