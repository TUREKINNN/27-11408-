## 一、协方差的定义与计算 ⭐⭐⭐⭐⭐

### 1.1 协方差的定义

**定义：** 设随机变量 $X, Y$ 的期望 $E(X)$、$E(Y)$ 存在，则：

$$\\boxed{\\text{Cov}(X, Y) = E\\{[X - E(X)][Y - E(Y)]\\}}$$

称为 $X$ 与 $Y$ 的**协方差**（Covariance）。

### 1.2 协方差的计算公式 ⭐⭐⭐⭐⭐

$$\\boxed{\\text{Cov}(X, Y) = E(XY) - E(X) \\cdot E(Y)}$$

**推导：**
$$\\text{Cov}(X, Y) = E[(X - EX)(Y - EY)]$$
$$= E[XY - X \\cdot EY - Y \\cdot EX + EX \\cdot EY]$$
$$= E(XY) - EX \\cdot EY - EY \\cdot EX + EX \\cdot EY$$
$$= E(XY) - EX \\cdot EY$$

> **这是考研中最常用的协方差计算公式！先求 $E(XY)$，再减去 $EX \\cdot EY$。**

### 1.3 协方差的本质

协方差衡量两个随机变量之间的**线性相关程度**：
- $\\text{Cov}(X, Y) > 0$：$X, Y$ 正相关（一个增大，另一个也倾向于增大）
- $\\text{Cov}(X, Y) < 0$：$X, Y$ 负相关
- $\\text{Cov}(X, Y) = 0$：$X, Y$ 不相关（无线性关系）

---

## 二、协方差的性质 ⭐⭐⭐⭐

### 2.1 基本性质

**(1) 对称性：**
$$\\boxed{\\text{Cov}(X, Y) = \\text{Cov}(Y, X)}$$

**(2) 自协方差等于方差：**
$$\\boxed{\\text{Cov}(X, X) = D(X)}$$

**(3) 线性性：**
$$\\boxed{\\text{Cov}(aX + b, \, cY + d) = ac \\cdot \\text{Cov}(X, Y)}$$

常数平移 $b, d$ 不影响协方差。

**(4) 双线性性：**
$$\\text{Cov}(X_1 + X_2, \, Y) = \\text{Cov}(X_1, Y) + \\text{Cov}(X_2, Y)$$

**(5) 与常数的协方差为零：**
$$\\text{Cov}(X, C) = 0$$

### 2.2 方差与协方差的关系 ⭐⭐⭐⭐⭐

$$\\boxed{D(X \\pm Y) = D(X) + D(Y) \\pm 2\\text{Cov}(X, Y)}$$

**推广到 $n$ 个随机变量：**

$$D\\left(\\sum_{i=1}^n X_i\\right) = \\sum_{i=1}^n D(X_i) + 2\\sum_{1 \\leq i < j \\leq n} \\text{Cov}(X_i, X_j)$$

---

## 三、相关系数 ⭐⭐⭐⭐⭐

### 3.1 相关系数的定义

**定义：** 设 $D(X) > 0$，$D(Y) > 0$，则：

$$\\boxed{\\rho_{XY} = \\frac{\\text{Cov}(X, Y)}{\\sqrt{D(X)} \\cdot \\sqrt{D(Y)}} = \\frac{\\text{Cov}(X, Y)}{\\sigma_X \\cdot \\sigma_Y}}$$

称为 $X$ 与 $Y$ 的**相关系数**（Correlation Coefficient）或**Pearson相关系数**。

> **相关系数是标准化的协方差**，取值范围为 $[-1, 1]$，无量纲。

### 3.2 相关系数的性质

**(1) 有界性：**
$$\\boxed{|\\rho_{XY}| \\leq 1}$$

**(2) 达到极值的条件：**
$$|\\rho_{XY}| = 1 \\iff Y = aX + b \\quad (a \\neq 0)$$

- $\\rho_{XY} = 1 \\iff Y = aX + b$，$a > 0$（完全正相关）
- $\\rho_{XY} = -1 \\iff Y = aX + b$，$a < 0$（完全负相关）

**(3) 相关系数的意义：**

| $\\rho_{XY}$ 的值 | 含义 |
|-------------------|------|
| $\\rho = 1$ | 完全正线性相关 |
| $0 < \\rho < 1$ | 正线性相关 |
| $\\rho = 0$ | 不相关（无**线性**关系） |
| $-1 < \\rho < 0$ | 负线性相关 |
| $\\rho = -1$ | 完全负线性相关 |

### 3.3 相关系数为零的等价条件

$$\\rho_{XY} = 0 \\iff \\text{Cov}(X, Y) = 0 \\iff E(XY) = E(X) \\cdot E(Y)$$

这三种表述完全等价，都称为 "$X$ 与 $Y$ **不相关**"。

---

## 四、不相关与独立的关系 ⭐⭐⭐⭐⭐

### 4.1 核心结论

$$\\boxed{X, Y \\text{ 独立} \\Rightarrow X, Y \\text{ 不相关}（即 } \\rho_{XY} = 0 \\text{）}}$$

$$\\boxed{X, Y \\text{ 不相关} \\not\\Rightarrow X, Y \\text{ 独立}}$$

即：
- **独立一定不相关** ✓
- **不相关不一定独立** ✗

> **"不相关≠独立"是考研概率论最高频的考点之一！几乎每年选择题都会涉及。**

### 4.2 为什么独立一定不相关

若 $X, Y$ 独立，则 $E(XY) = E(X) \\cdot E(Y)$，所以 $\\text{Cov}(X, Y) = 0$，$\\rho_{XY} = 0$。

### 4.3 为什么不相关不一定独立

**反例：** 设 $X \\sim U(-1, 1)$，$Y = X^2$。

显然 $Y$ 完全由 $X$ 决定，$X, Y$ 不独立。但：

$$E(X) = 0, \quad E(XY) = E(X^3) = \\int_{-1}^1 x^3 \\cdot \\frac{1}{2} dx = 0$$

$$\\text{Cov}(X, Y) = E(XY) - E(X) \\cdot E(Y) = 0 - 0 \\cdot E(Y) = 0$$

所以 $\\rho_{XY} = 0$，$X, Y$ 不相关但不独立。

> **关键理解：** 相关系数只衡量**线性**关系。$Y = X^2$ 是非线性关系，相关系数无法检测。

### 4.4 不相关与独立等价的特殊情况

**(1) 二维正态分布：** 若 $(X, Y) \\sim N(\\mu_1, \\mu_2, \\sigma_1^2, \\sigma_2^2, \\rho)$，则：

$$\\boxed{X, Y \\text{ 不相关} \\iff X, Y \\text{ 独立}}$$

> **二维正态分布中，不相关与独立等价！这是二维正态分布的特殊性质。**

**(2) 一般情形：** 独立 ⟹ 不相关，反之不成立。

## 五、协方差在方差计算中的应用

### 5.1 独立随机变量线性组合的方差

若 $X_1, X_2, \\ldots, X_n$ 相互独立，则：

$$D\\left(\\sum_{i=1}^n c_i X_i\\right) = \\sum_{i=1}^n c_i^2 D(X_i)$$

### 5.2 非独立随机变量线性组合的方差

一般地：

$$D\\left(\\sum_{i=1}^n c_i X_i\\right) = \\sum_{i=1}^n c_i^2 D(X_i) + 2\\sum_{i < j} c_i c_j \\text{Cov}(X_i, X_j)$$

## 六、矩与协方差矩阵（了解）

### 6.1 矩的概念

**$k$ 阶原点矩：** $E(X^k)$

**$k$ 阶中心矩：** $E[(X - EX)^k]$

- 一阶原点矩 = 期望：$E(X)$
- 二阶中心矩 = 方差：$E[(X - EX)^2] = D(X)$

### 6.2 协方差矩阵

设 $(X, Y)$ 为二维随机变量，协方差矩阵为：

$$\\Sigma = \\begin{pmatrix} D(X) & \\text{Cov}(X, Y) \\\\ \\text{Cov}(Y, X) & D(Y) \\end{pmatrix} = \\begin{pmatrix} \\sigma_1^2 & \\rho\\sigma_1\\sigma_2 \\\\ \\rho\\sigma_1\\sigma_2 & \\sigma_2^2 \\end{pmatrix}$$

## 七、经典例题

**【例1】（协方差与相关系数的基本计算）** 设 $(X, Y)$ 的联合分布律为：

| $X \\backslash Y$ | $0$ | $1$ |
|------|------|------|
| $0$ | $0.1$ | $0.3$ |
| $1$ | $0.3$ | $0.3$ |

求 $\\text{Cov}(X, Y)$ 和 $\\rho_{XY}$。

**解：**

边缘分布：$P\\{X=0\\}=0.4$，$P\\{X=1\\}=0.6$，$P\\{Y=0\\}=0.4$，$P\\{Y=1\\}=0.6$

$$E(X) = 0.6, \quad E(Y) = 0.6, \quad E(XY) = 1 \\cdot 1 \\cdot 0.3 = 0.3$$

$$\\text{Cov}(X, Y) = 0.3 - 0.6 \\times 0.6 = 0.3 - 0.36 = -0.06$$

$$D(X) = 0.6 - 0.36 = 0.24, \quad D(Y) = 0.24$$

$$\\rho_{XY} = \\frac{-0.06}{\\sqrt{0.24 \\times 0.24}} = \\frac{-0.06}{0.24} = -0.25$$

**【例2】（独立与不相关）** 设 $X \\sim N(0, 1)$，$Y = X^2$，证明 $X, Y$ 不相关但不独立。

**证明：**

不相关：$E(X) = 0$，$E(XY) = E(X^3) = 0$（标准正态的三阶原点矩为0）

$$\\text{Cov}(X, Y) = E(XY) - E(X)E(Y) = 0$$

不独立：$Y = X^2$ 是确定性函数关系，$P\\{Y \\leq 1 | X = 2\\} = 0 \\neq P\\{Y \\leq 1\\}$。

**【例3】（二维正态分布的相关系数）** 设 $(X, Y) \\sim N(1, 4, 9, 16, 0.5)$，求 $\\text{Cov}(X, Y)$。

**解：**
$$\\text{Cov}(X, Y) = \\rho \\sigma_X \\sigma_Y = 0.5 \\times 3 \\times 4 = 6$$

**【例4】（方差的综合计算）** 设 $X_1, X_2, X_3$ 满足 $E(X_i) = 0$，$D(X_i) = 1$，$\\text{Cov}(X_i, X_j) = 0.2$（$i \\neq j$），求 $D(X_1 + X_2 + X_3)$。

**解：**
$$D(X_1+X_2+X_3) = \\sum_{i=1}^3 D(X_i) + 2\\sum_{i<j}\\text{Cov}(X_i,X_j) = 3 + 2 \\times 3 \\times 0.2 = 3 + 1.2 = 4.2$$

## 八、易错点与考试陷阱

### ⚠️ 易错点1：不相关 ≠ 独立

这是考研最高频的易错点。记住：只有在**二维正态分布**中，不相关才等价于独立。

### ⚠️ 易错点2：协方差公式

$\\text{Cov}(X, Y) = E(XY) - E(X) \\cdot E(Y)$，容易忘记减去 $E(X) \\cdot E(Y)$。

### ⚠️ 易错点3：相关系数的符号

$\\rho_{XY}$ 的符号由 $\\text{Cov}(X, Y)$ 决定。$\\rho > 0$ 不代表因果关系。

### ⚠️ 易错点4：$D(X-Y)$ 在独立时

$D(X - Y) = D(X) + D(Y)$（独立时），不是 $D(X) - D(Y)$。

### ⚠️ 易错点5：相关系数只衡量线性关系

$\\rho_{XY} = 0$ 只说明没有线性关系，可能存在非线性关系（如 $Y = X^2$）。

## 十、习题精选

**习题1：** 设 $X \\sim U(0, \\pi)$，$Y = \\sin X$，求 $\\text{Cov}(X, Y)$。

**解：**
$$E(X) = \\frac{\\pi}{2}, \quad E(Y) = E(\\sin X) = \\frac{1}{\\pi}\\int_0^\\pi \\sin x \, dx = \\frac{2}{\\pi}$$
$$E(XY) = \\frac{1}{\\pi}\\int_0^\\pi x \\sin x \, dx = \\frac{1}{\\pi}[-x\\cos x + \\sin x]_0^\\pi = \\frac{1}{\\pi} \\cdot \\pi = 1$$
$$\\text{Cov}(X, Y) = 1 - \\frac{\\pi}{2} \\cdot \\frac{2}{\\pi} = 1 - 1 = 0$$

$X, Y$ 不相关！（但显然 $Y = \\sin X$ 不独立）

**习题2：** 设 $X_1, X_2$ 独立同分布于 $N(0, 1)$，令 $Y_1 = X_1 + X_2$，$Y_2 = X_1 - X_2$，证明 $Y_1, Y_2$ 不相关且独立。

**证明：**
$$\\text{Cov}(Y_1, Y_2) = \\text{Cov}(X_1+X_2, X_1-X_2) = D(X_1) - D(X_2) = 1 - 1 = 0$$

$Y_1, Y_2$ 都是独立正态变量的线性组合，所以 $(Y_1, Y_2)$ 服从二维正态分布。在二维正态中不相关即独立。

**习题3：** 设 $D(X) = 4$，$D(Y) = 9$，$\\rho_{XY} = -0.5$，求 $D(2X - 3Y + 1)$。

**解：**
$$\\text{Cov}(X, Y) = \\rho_{XY}\\sqrt{D(X)}\\sqrt{D(Y)} = -0.5 \\times 2 \\times 3 = -3$$
$$D(2X - 3Y + 1) = 4 \\times 4 + 9 \\times 9 + 2 \\times 2 \\times (-3) \\times (-3)$$
$$= 16 + 81 + 2 \\times (-6) \\times (-3) = 16 + 81 + 36 = 133$$
