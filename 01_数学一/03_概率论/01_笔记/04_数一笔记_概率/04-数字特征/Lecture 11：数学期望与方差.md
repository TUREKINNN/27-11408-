## 一、数学期望的定义 ⭐⭐⭐⭐⭐

### 1.1 离散型随机变量的期望

**定义：** 设离散型随机变量 $X$ 的分布律为 $P\\{X = x_k\\} = p_k$（$k = 1, 2, \\ldots$），若级数 $\\sum_{k=1}^{\\infty} x_k p_k$ 绝对收敛，则称：

$$\\boxed{E(X) = \\sum_{k=1}^{\\infty} x_k p_k}$$

为 $X$ 的**数学期望**（Expected Value），简称**期望**或**均值**。

> **注意：** 期望存在的条件是级数**绝对收敛**，即 $\\sum |x_k| p_k < \\infty$。若级数条件收敛，则期望不存在。

### 1.2 连续型随机变量的期望

**定义：** 设连续型随机变量 $X$ 的概率密度函数为 $f(x)$，若积分 $\\int_{-\\infty}^{+\\infty} x f(x) dx$ 绝对收敛，则称：

$$\\boxed{E(X) = \\int_{-\\infty}^{+\\infty} x f(x) \, dx}$$

为 $X$ 的**数学期望**。

### 1.3 期望的本质

数学期望是随机变量取值的**加权平均**，权为各取值的概率（离散型）或密度（连续型）。它是随机变量最重要的数字特征，反映了随机变量取值的"中心位置"。

---

## 二、随机变量函数的期望 ⭐⭐⭐⭐⭐

### 2.1 一维随机变量函数的期望

**定理：** 设 $Y = g(X)$，$g$ 为连续函数，则：

**离散型：**
$$\\boxed{E(Y) = E[g(X)] = \\sum_{k} g(x_k) p_k}$$

**连续型：**
$$\\boxed{E(Y) = E[g(X)] = \\int_{-\\infty}^{+\\infty} g(x) f(x) \, dx}$$

> **重要：** 计算 $E[g(X)]$ **不需要**先求 $Y$ 的分布，直接用 $X$ 的分布即可！这是期望计算最常用的方法。

### 2.2 二维随机变量函数的期望

设 $Z = g(X, Y)$，则：

**离散型：**
$$E(Z) = \\sum_i \\sum_j g(x_i, y_j) P\\{X = x_i, Y = y_j\\}$$

**连续型：**
$$E(Z) = \\int_{-\\infty}^{+\\infty} \\int_{-\\infty}^{+\\infty} g(x, y) f(x, y) \, dx \, dy$$

## 三、数学期望的性质 ⭐⭐⭐⭐⭐

设以下涉及的期望均存在，则：

### 3.1 常数的期望

$$\\boxed{E(C) = C}$$

其中 $C$ 为常数。

### 3.2 线性性

$$\\boxed{E(aX + b) = aE(X) + b}$$

其中 $a, b$ 为常数。

### 3.3 和的期望

$$\\boxed{E(aX + bY) = aE(X) + bE(Y)}$$

> **重要：此性质不要求 $X, Y$ 独立！** 这是期望最基本的性质。

**推广到 $n$ 个随机变量：**

$$E\\left(\\sum_{i=1}^n c_i X_i\\right) = \\sum_{i=1}^n c_i E(X_i)$$

### 3.4 独立性与乘积的期望

若 $X, Y$ **相互独立**，则：

$$\\boxed{E(XY) = E(X) \\cdot E(Y)}$$

**反之不成立：** $E(XY) = E(X) \\cdot E(Y)$ 不能推出 $X, Y$ 独立。

## 四、方差的定义与计算 ⭐⭐⭐⭐⭐

### 4.1 方差的定义

**定义：** 设随机变量 $X$ 的期望 $E(X)$ 存在，则称：

$$\\boxed{D(X) = \\text{Var}(X) = E\\{[X - E(X)]^2\\}}$$

为 $X$ 的**方差**（Variance），$\\sqrt{D(X)}$ 为 $X$ 的**标准差**（Standard Deviation）。

方差反映了随机变量围绕其期望的**离散程度**。方差越大，随机变量的取值越分散。

### 4.2 方差的计算公式 ⭐⭐⭐⭐⭐

$$\\boxed{D(X) = E(X^2) - [E(X)]^2}$$

**推导：**
$$D(X) = E[(X - EX)^2] = E[X^2 - 2X \\cdot EX + (EX)^2]$$
$$= E(X^2) - 2EX \\cdot EX + (EX)^2 = E(X^2) - (EX)^2$$

> **这个公式是考研中最常用的方差计算公式，必须熟练掌握！**

### 4.3 方差的性质

**(1)** 常数的方差为零：
$$\\boxed{D(C) = 0}$$

**(2)** 线性变换的方差：
$$\\boxed{D(aX + b) = a^2 D(X)}$$

> **注意：** 是 $a^2$ 不是 $a$！常数平移 $b$ 不影响方差。

**(3)** 和的方差：
$$\\boxed{D(X \\pm Y) = D(X) + D(Y) \\pm 2\\text{Cov}(X, Y)}$$

当 $X, Y$ **独立**（或不相关）时：
$$\\boxed{D(X + Y) = D(X) + D(Y)}$$
$$\\boxed{D(X - Y) = D(X) + D(Y)}$$

> **独立时减法也是加！** 这是一个重要且易错的结论。

**(4)** 若 $X, Y$ 独立，推广到 $n$ 个：
$$D\\left(\\sum_{i=1}^n X_i\\right) = \\sum_{i=1}^n D(X_i)$$
$$D\\left(\\sum_{i=1}^n c_i X_i\\right) = \\sum_{i=1}^n c_i^2 D(X_i)$$

## 五、常见分布的期望与方差汇总 ⭐⭐⭐⭐⭐

### 5.1 七种重要分布的期望与方差

| 分布 | 记号 | 期望 $E(X)$ | 方差 $D(X)$ |
|------|------|-------------|-------------|
| **0-1分布** | $B(1, p)$ | $p$ | $p(1-p)$ |
| **二项分布** | $B(n, p)$ | $np$ | $np(1-p)$ |
| **泊松分布** | $P(\\lambda)$ | $\\lambda$ | $\\lambda$ |
| **均匀分布** | $U(a, b)$ | $\\dfrac{a+b}{2}$ | $\\dfrac{(b-a)^2}{12}$ |
| **指数分布** | $E(\\lambda)$ | $\\dfrac{1}{\\lambda}$ | $\\dfrac{1}{\\lambda^2}$ |
| **正态分布** | $N(\\mu, \\sigma^2)$ | $\\mu$ | $\\sigma^2$ |
| **几何分布** | $G(p)$ | $\\dfrac{1}{p}$ | $\\dfrac{1-p}{p^2}$ |

> **泊松分布：期望 = 方差 = $\\lambda$**，这是一个特殊性质。

### 5.2 各分布期望方差的推导要点

**0-1分布：** $E(X) = 1 \\cdot p + 0 \\cdot (1-p) = p$，$E(X^2) = p$，$D(X) = p - p^2 = p(1-p)$

**二项分布：** $X = \\sum_{i=1}^n X_i$，其中 $X_i \\sim B(1, p)$ 独立。$E(X) = np$，$D(X) = np(1-p)$

**泊松分布：** $E(X) = \\sum_{k=0}^\\infty k \\cdot \\frac{\\lambda^k e^{-\\lambda}}{k!} = \\lambda$，$E(X^2) = \\lambda^2 + \\lambda$，$D(X) = \\lambda$

**几何分布：** $E(X) = \\frac{1}{p}$，$D(X) = \\frac{1-p}{p^2}$

## 六、经典例题

**【例1】（期望的基本计算）** 设 $X$ 的分布律为：

| $X$ | $-1$ | $0$ | $1$ | $2$ |
|-----|------|-----|-----|-----|
| $P$ | $0.2$ | $0.3$ | $0.3$ | $0.2$ |

求 $E(X)$，$E(X^2)$，$D(X)$。

**解：**
$$E(X) = (-1)(0.2) + 0(0.3) + 1(0.3) + 2(0.2) = -0.2 + 0 + 0.3 + 0.4 = 0.5$$
$$E(X^2) = 1(0.2) + 0(0.3) + 1(0.3) + 4(0.2) = 0.2 + 0 + 0.3 + 0.8 = 1.3$$
$$D(X) = E(X^2) - [E(X)]^2 = 1.3 - 0.25 = 1.05$$

**【例2】（函数的期望）** 设 $X \\sim E(1)$，求 $E(2X + 3)$ 和 $E(e^{-2X})$。

**解：**
$$E(2X + 3) = 2E(X) + 3 = 2(1) + 3 = 5$$
$$E(e^{-2X}) = \\int_0^{+\\infty} e^{-2x} \\cdot e^{-x} \, dx = \\int_0^{+\\infty} e^{-3x} \, dx = \\frac{1}{3}$$

**【例3】（独立随机变量和的方差）** 设 $X \\sim N(1, 4)$，$Y \\sim P(3)$，$X, Y$ 独立，求 $E(3X - 2Y + 1)$ 和 $D(3X - 2Y + 1)$。

**解：**
$$E(3X - 2Y + 1) = 3(1) - 2(3) + 1 = 3 - 6 + 1 = -2$$
$$D(3X - 2Y + 1) = 9D(X) + 4D(Y) = 9 \\times 4 + 4 \\times 3 = 36 + 12 = 48$$

> 注意：$D(3X - 2Y + 1) = 9D(X) + 4D(Y)$，减法变加法！常数 $1$ 不影响方差。

**【例4】（综合应用）** 设 $X \\sim U(0, 2)$，求 $D(X)$ 和 $E(e^X)$。

**解：**
$$D(X) = \\frac{(2-0)^2}{12} = \\frac{1}{3}$$
$$E(e^X) = \\int_0^2 e^x \\cdot \\frac{1}{2} \, dx = \\frac{1}{2}(e^2 - 1)$$

## 七、期望方差的重要应用

### 7.1 切比雪夫不等式（预告）

设 $E(X) = \\mu$，$D(X) = \\sigma^2$，则对任意 $\\varepsilon > 0$：

$$P\\{|X - \\mu| \\geq \\varepsilon\\} \\leq \\frac{\\sigma^2}{\\varepsilon^2}$$

详见 Lecture 13。

### 7.2 标准化

若 $E(X) = \\mu$，$D(X) = \\sigma^2 > 0$，则 $X^* = \\frac{X - \\mu}{\\sigma}$ 满足 $E(X^*) = 0$，$D(X^*) = 1$。

## 八、易错点与考试陷阱

### ⚠️ 易错点1：方差公式 $D(X) = E(X^2) - [E(X)]^2$

容易误写为 $D(X) = [E(X)]^2 - E(X^2)$，注意顺序！

### ⚠️ 易错点2：$D(aX + b) = a^2 D(X)$

是 $a^2$ 不是 $a$！常数 $b$ 不影响方差。$D(2X + 3) = 4D(X)$，不是 $2D(X) + 3$。

### ⚠️ 易错点3：减法的方差

$D(X - Y) = D(X) + D(Y)$（独立时），**不是** $D(X) - D(Y)$。

### ⚠️ 易错点4：独立是乘积期望的条件

$E(XY) = E(X)E(Y)$ 需要独立条件。但 $E(aX + bY) = aE(X) + bE(Y)$ **不需要**独立条件。

### ⚠️ 易错点5：期望不存在的情况

柯西分布 $f(x) = \\frac{1}{\\pi(1+x^2)}$ 的期望不存在（积分不绝对收敛）。

```
数学期望与方差
├── 数学期望 E(X) ⭐⭐⭐⭐⭐
│   ├── 离散型：E(X) = Σxₖpₖ
│   ├── 连续型：E(X) = ∫xf(x)dx
│   ├── 函数期望：E[g(X)] = ∫g(x)f(x)dx
│   └── 性质
│       ├── 线性性：E(aX+bY) = aEX+bEY（不需要独立）
│       └── 独立乘积：E(XY) = EX·EY（需要独立）
├── 方差 D(X) = E(X²)-[EX]² ⭐⭐⭐⭐⭐
│   ├── 性质
│   │   ├── D(C) = 0
│   │   ├── D(aX+b) = a²D(X)
│   │   └── D(X±Y) = DX+DY（独立时减法也是加）
│   └── 计算：先求E(X)，再求E(X²)，代入公式
└── 七种分布期望方差汇总 ⭐⭐⭐⭐⭐
    ├── 0-1：p, p(1-p)
    ├── 二项：np, np(1-p)
    ├── 泊松：λ, λ（期望=方差）
    ├── 均匀：(a+b)/2, (b-a)²/12
    ├── 指数：1/λ, 1/λ²
    ├── 正态：μ, σ²
    └── 几何：1/p, (1-p)/p²
```

## 十、习题精选

**习题1：** 设 $X$ 的密度为 $f(x) = 2x$（$0 < x < 1$），求 $E(X)$ 和 $D(X)$。

**解：**
$$E(X) = \\int_0^1 x \\cdot 2x \, dx = \\frac{2}{3}$$
$$E(X^2) = \\int_0^1 x^2 \\cdot 2x \, dx = \\frac{1}{2}$$
$$D(X) = \\frac{1}{2} - \\frac{4}{9} = \\frac{1}{18}$$

**习题2：** 设 $X \\sim B(10, 0.3)$，求 $E(2X - 1)$ 和 $D(2X - 1)$。

**解：**
$$E(2X - 1) = 2 \\times 3 - 1 = 5, \quad D(2X - 1) = 4 \\times 2.1 = 8.4$$

**习题3：** 设 $X_1, X_2, \\ldots, X_{10}$ 独立同分布于 $N(0, 1)$，求 $E(\\bar{X})$ 和 $D(\\bar{X})$，其中 $\\bar{X} = \\frac{1}{10}\\sum_{i=1}^{10} X_i$。

**解：**
$$E(\\bar{X}) = \\frac{1}{10} \\sum_{i=1}^{10} E(X_i) = 0$$
$$D(\\bar{X}) = \\frac{1}{100} \\sum_{i=1}^{10} D(X_i) = \\frac{10}{100} = \\frac{1}{10}$$
