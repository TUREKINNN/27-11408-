# 双正态总体的参数估计与假设检验

> [[01_数学一/03_概率论/01_笔记/04_数一笔记_概率/06-数理统计与参数估计]] | [[01_数学一/03_概率论/01_笔记/04_数一笔记_概率/06-数理统计与参数估计]]

## 概述

当从两个正态总体 $X \sim N(\mu_1, \sigma_1^2)$，$Y \sim N(\mu_2, \sigma_2^2)$ 中抽样时，需要比较它们的参数。

## 抽样分布

设 $X_1, ..., X_{n_1} \sim N(\mu_1, \sigma_1^2)$，$Y_1, ..., Y_{n_2} \sim N(\mu_2, \sigma_2^2)$，相互独立。

**样本均值**：
$$\bar{X} = \frac{1}{n_1}\sum X_i, \quad \bar{Y} = \frac{1}{n_2}\sum Y_i$$

**样本方差**：
$$S_1^2 = \frac{1}{n_1-1}\sum(X_i - \bar{X})^2, \quad S_2^2 = \frac{1}{n_2-1}\sum(Y_i - \bar{Y})^2$$

## 参数估计

### 1. 均值差 $\mu_1 - \mu_2$ 的估计

**点估计**：
$$\hat{\mu_1 - \mu_2} = \bar{X} - \bar{Y}$$

#### 情形 1：$\sigma_1^2 = \sigma_2^2 = \sigma^2$ 已知

置信区间：
$$(\bar{X} - \bar{Y}) \pm u_{\alpha/2}\sqrt{\frac{\sigma^2}{n_1} + \frac{\sigma^2}{n_2}}$$

#### 情形 2：$\sigma_1^2 = \sigma_2^2 = \sigma^2$ 未知

**统计量**：
$$T = \frac{(\bar{X} - \bar{Y}) - (\mu_1 - \mu_2)}{S_w\sqrt{\frac{1}{n_1} + \frac{1}{n_2}}} \sim t(n_1 + n_2 - 2)$$

其中 $S_w = \sqrt{\frac{(n_1-1)S_1^2 + (n_2-1)S_2^2}{n_1+n_2-2}}$（合并方差）

#### 情形 3：$\sigma_1^2, \sigma_2^2$ 均未知且不等

近似用 $t$ 分布（**Behrens-Fisher 问题**）：
$$T \approx \frac{(\bar{X} - \bar{Y}) - (\mu_1 - \mu_2)}{\sqrt{\frac{S_1^2}{n_1} + \frac{S_2^2}{n_2}}} \sim t(f)$$

其中自由度 $f$ 用 Welch 公式：
$$f = \frac{(S_1^2/n_1 + S_2^2/n_2)^2}{\frac{(S_1^2/n_1)^2}{n_1-1} + \frac{(S_2^2/n_2)^2}{n_2-1}}$$

### 2. 方差比 $\sigma_1^2 / \sigma_2^2$ 的估计

**统计量**：
$$F = \frac{S_1^2}{S_2^2} \sim F(n_1-1, n_2-1)$$

**置信区间**：
$$\left(\frac{S_1^2}{S_2^2} \cdot \frac{1}{F_{\alpha/2}(n_1-1, n_2-1)}, \frac{S_1^2}{S_2^2} \cdot F_{\alpha/2}(n_2-1, n_1-1)\right)$$

## 假设检验

### 1. 均值差检验

#### 检验 $H_0: \mu_1 = \mu_2$ vs $H_1: \mu_1 \neq \mu_2$

**情形 1**：$\sigma_1^2 = \sigma_2^2$ 已知

用 $U$ 检验：
$$U = \frac{\bar{X} - \bar{Y}}{\sigma\sqrt{\frac{1}{n_1} + \frac{1}{n_2}}} \sim N(0, 1)$$

**情形 2**：$\sigma_1^2 = \sigma_2^2$ 未知

用 $t$ 检验（合并方差）：
$$T = \frac{\bar{X} - \bar{Y}}{S_w\sqrt{\frac{1}{n_1} + \frac{1}{n_2}}} \sim t(n_1+n_2-2)$$

#### 配对样本检验

对配对数据 $(X_i, Y_i)$，令 $Z_i = X_i - Y_i$，转化为**单正态总体**的均值检验。

### 2. 方差比检验（F 检验）

**检验 $H_0: \sigma_1^2 = \sigma_2^2$ vs $H_1: \sigma_1^2 \neq \sigma_2^2$**

用 $F$ 检验：
$$F = \frac{S_1^2}{S_2^2} \sim F(n_1-1, n_2-1)$$

**拒绝域**：$F > F_{\alpha/2}(n_1-1, n_2-1)$ 或 $F < F_{1-\alpha/2}(n_1-1, n_2-1)$

## 检验流程

```
1. 明确原假设 H0 和备择假设 H1
2. 选择检验统计量（U/T/F）
3. 确定显著性水平 α
4. 计算统计量值
5. 确定拒绝域
6. 判断：若统计量在拒绝域内，拒绝 H0
```

## 考研真题示例

### 例题（2010）

设两个工厂生产同一种产品，假设某项指标 $X \sim N(\mu_1, \sigma^2)$，$Y \sim N(\mu_2, \sigma^2)$。从两个工厂分别抽取 $n_1 = 10$，$n_2 = 15$ 件产品，测得：
- $\bar{X} = 82$，$S_1^2 = 5.2$
- $\bar{Y} = 78$，$S_2^2 = 4.8$

在 $\alpha = 0.05$ 下，检验两个工厂的均值是否相等。

**解**：
- $H_0: \mu_1 = \mu_2$，$H_1: \mu_1 \neq \mu_2$
- 合并方差：$S_w^2 = \frac{9 \times 5.2 + 14 \times 4.8}{23} = \frac{46.8 + 67.2}{23} \approx 4.96$
- $S_w \approx 2.23$
- $T = \frac{82 - 78}{2.23 \times \sqrt{1/10 + 1/15}} = \frac{4}{2.23 \times 0.408} \approx 4.40$
- 临界值 $t_{0.025}(23) \approx 2.069$
- $|T| > 2.069$，拒绝 $H_0$
- **结论**：两个工厂的均值有显著差异

## 重要公式汇总

| 检验类型 | 统计量 | 分布 |
|----------|--------|------|
| $\sigma$ 已知，均值差 | $U = \frac{\bar{X}-\bar{Y}}{\sigma\sqrt{1/n_1+1/n_2}}$ | $N(0,1)$ |
| $\sigma$ 未知相等，均值差 | $T$（合并方差） | $t(n_1+n_2-2)$ |
| $\sigma$ 未知不等，均值差 | $T$（Welch） | $t(f)$（近似） |
| 方差比 | $F = S_1^2/S_2^2$ | $F(n_1-1, n_2-1)$ |
