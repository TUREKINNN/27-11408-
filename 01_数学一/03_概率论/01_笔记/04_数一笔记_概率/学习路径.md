# 概率论学习路径

---

## 第一阶段：基础夯实（2 周）

### 第 1 周：随机事件与概率
- [ ] 样本空间、事件的关系与运算
- [ ] 概率的定义与性质
- [ ] 古典概型与几何概型
- [ ] 条件概率与乘法公式
- [ ] 全概率公式与贝叶斯公式
- [ ] 事件的独立性

### 第 2 周：随机变量及其分布
- [ ] 分布函数的定义与性质
- [ ] 离散型随机变量（二项、泊松、几何分布）
- [ ] 连续型随机变量（均匀、指数、正态分布）
- [ ] 正态分布的标准化
- [ ] 随机变量函数的分布

---

## 第二阶段：核心突破（2 周）

### 第 3 周：多维随机变量
- [ ] 联合分布函数与联合密度
- [ ] 边缘分布与条件分布
- [ ] 二维随机变量的独立性
- [ ] 两个随机变量函数的分布（$Z=X+Y$、$\max$、$\min$）
- [ ] 卷积公式

### 第 4 周：数字特征
- [ ] 期望的定义、性质与计算
- [ ] 方差的定义、性质与计算
- [ ] 协方差与相关系数
- [ ] 常见分布的期望与方差汇总记忆

---

## 第三阶段：进阶提升（2 周）

### 第 5 周：大数定律与中心极限定理
- [ ] 切比雪夫不等式
- [ ] 各种大数定律的条件与结论
- [ ] 中心极限定理及应用

### 第 6 周：数理统计
- [ ] $\chi^2$、$t$、$F$ 三大抽样分布
- [ ] 正态总体的抽样分布定理
- [ ] 矩估计与最大似然估计
- [ ] 置信区间与假设检验

---

## 第四阶段：冲刺综合（1-2 周）

- [ ] 历年真题分类训练（重点关注：全概率/贝叶斯、分布函数、期望方差、参数估计）
- [ ] 综合题型训练（概率 + 分布 + 数字特征联合考查）
- [ ] 错题回顾
- [ ] 模拟套卷限时训练

---

## 💡 学习建议

1. **概率论 ≠ 排列组合**：核心是分布函数和密度函数的分析
2. **记清分布表**：6 个离散分布 + 3 个连续分布的期望方差必须烂熟
3. **全概率 + 贝叶斯**是每年必考内容，务必掌握拆分事件的方法
4. **最大似然估计**步骤固定，属于送分题型，务必拿满分
5. **数理统计**难度较低但概念多，建议最后集中记忆
