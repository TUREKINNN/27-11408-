## 一、随机变量（Random Variable）

### 1.1 随机变量的定义

**定义：** 设随机试验的样本空间为 $\Omega = \{\omega\}$，定义在 $\Omega$ 上的实值函数 $X = X(\omega)$（$\omega \in \Omega$）称为**随机变量**（Random Variable），简记为 $X$。

$$X: \Omega \rightarrow \mathbb{R}$$

**直观理解：** 随机变量就是将样本空间中的每一个结果（样本点）映射为一个实数的函数。

### 1.2 引入随机变量的意义

引入随机变量后，我们可以用**数量**来描述随机现象，从而：
1. 利用**数学分析**的工具研究概率问题
2. 用**统一的框架**处理不同类型的随机试验
3. 建立概率模型——**分布**

### 1.3 随机变量的分类

| 类型 | 特征 | 例子 |
|------|------|------|
| **离散型** | 取值为有限个或可列无穷多个 | 掷骰子的点数 $X \in \{1,2,3,4,5,6\}$ |
| **连续型** | 取值为连续区间（不可列无穷多） | 零件长度 $X \in [a, b]$ |
| **混合型** | 既非离散也非连续 | 保险理赔金额（含零点的概率质量） |

> **考研重点：** 离散型和连续型两大类。

### 1.4 随机事件与随机变量的关系

随机变量使得事件可以用"变量取值范围"来表示：

$$\{X = a\} = \{\omega \in \Omega : X(\omega) = a\}$$
$$\{a < X \leq b\} = \{\omega \in \Omega : a < X(\omega) \leq b\}$$
$$\{X \leq x\} = \{\omega \in \Omega : X(\omega) \leq x\}$$

## 二、分布函数（Cumulative Distribution Function, CDF）

### 2.1 分布函数的定义

**定义：** 设 $X$ 是一个随机变量，$x$ 是任意实数，函数

$$\boxed{F(x) = P\{X \leq x\}, \quad -\infty < x < +\infty}$$

称为 $X$ 的**分布函数**（Distribution Function）。

> **注意：** 分布函数对所有随机变量（离散型、连续型、混合型）都有定义。

### 2.2 分布函数的几何意义

$F(x)$ 表示随机变量 $X$ 的取值落在 $(-\infty, x]$ 区间内的概率。

在数轴上，$F(x)$ 表示 $X$ 的概率分布"累积"到 $x$ 点的值。

### 2.3 分布函数的重要应用

利用分布函数可以计算任意区间的概率：

$$P\{a < X \leq b\} = F(b) - F(a)$$

$$P\{X = a\} = F(a) - F(a^-)$$

其中 $F(a^-) = \lim_{x \to a^-} F(x)$ 表示 $F$ 在 $a$ 点的**左极限**。

$$P\{X > a\} = 1 - F(a) = 1 - P\{X \leq a\}$$

$$P\{X \geq a\} = 1 - P\{X < a\} = 1 - F(a^-)$$

$$P\{a \leq X \leq b\} = F(b) - F(a^-)$$

$$P\{a < X < b\} = F(b^-) - F(a)$$

$$P\{a \leq X < b\} = F(b^-) - F(a^-)$$

> **考研记忆口诀：** $P\{a \leq X \leq b\}$ 用 $F(b) - F(a^-)$；$P\{a < X \leq b\}$ 用 $F(b) - F(a)$。
> **左闭取左极限，右闭取右值。**

### 2.4 分布函数的三条基本性质 ⭐⭐⭐

**定理：** 任意随机变量 $X$ 的分布函数 $F(x)$ 都具有以下三条性质：

**性质1（单调不减性）：**
$$\text{若 } x_1 < x_2, \text{ 则 } F(x_1) \leq F(x_2)$$

**性质2（有界性与归一性）：**
$$\boxed{0 \leq F(x) \leq 1, \quad \forall x \in \mathbb{R}}$$
$$\boxed{F(-\infty) = \lim_{x \to -\infty} F(x) = 0}$$
$$\boxed{F(+\infty) = \lim_{x \to +\infty} F(x) = 1}$$

**性质3（右连续性）：**
$$\boxed{F(x^+) = F(x), \quad \forall x \in \mathbb{R}}$$

即 $F(x)$ 在每个点 $x$ 处**右连续**。

> **重要结论：** 满足以上三条性质的函数 $F(x)$ 一定是某个随机变量的分布函数。即这三条性质是分布函数的**充要条件**。

### 2.5 三条性质的证明

**性质1的证明：**
设 $x_1 < x_2$，则 $\{X \leq x_1\} \subset \{X \leq x_2\}$

由概率的单调性：
$$F(x_1) = P\{X \leq x_1\} \leq P\{X \leq x_2\} = F(x_2)$$

**性质2的证明：**
- $F(x) \geq 0$：由概率的非负性
- $F(x) \leq 1$：由概率 $\leq 1$
- $F(-\infty) = 0$：事件 $\{X \leq -\infty\}$ 是不可能事件
- $F(+\infty) = 1$：事件 $\{X \leq +\infty\}$ 是必然事件

**性质3的证明：**（略，需要用到概率的连续性定理）

## 三、离散型随机变量的分布函数

### 3.1 分布律（概率分布）

**定义：** 离散型随机变量 $X$ 的取值为 $x_1, x_2, \ldots, x_n, \ldots$（有限或可列），对应的概率为：

$$P\{X = x_k\} = p_k, \quad k = 1, 2, 3, \ldots$$

满足：
1. $p_k \geq 0$（非负性）
2. $\sum_k p_k = 1$（归一性）

则称 $\{p_k\}$ 为 $X$ 的**分布律**或**概率分布**。

常用表格或公式表示：

| $X$ | $x_1$ | $x_2$ | $\cdots$ | $x_k$ | $\cdots$ |
|-----|--------|--------|----------|--------|----------|
| $P$ | $p_1$ | $p_2$ | $\cdots$ | $p_k$ | $\cdots$ |

### 3.2 由分布律求分布函数

$$F(x) = P\{X \leq x\} = \sum_{x_k \leq x} p_k$$

$F(x)$ 是一个**阶梯函数**，在每个 $x_k$ 处有跳跃，跳跃高度为 $p_k$。

### 3.3 示例

**【例1】** 设 $X$ 的分布律为：

| $X$ | 0 | 1 | 2 |
|-----|---|---|---|
| $P$ | 0.3 | 0.5 | 0.2 |

求 $F(x)$ 的表达式。

**解：**
$$F(x) = \begin{cases} 0, & x < 0 \\ 0.3, & 0 \leq x < 1 \\ 0.8, & 1 \leq x < 2 \\ 1, & x \geq 2 \end{cases}$$

这是一个阶梯函数，在 $x = 0, 1, 2$ 处有跳跃，跳跃高度分别为 $0.3, 0.5, 0.2$。

## 四、连续型随机变量的分布函数

### 4.1 概率密度函数（PDF）

**定义：** 若存在非负函数 $f(x)$，使得对任意实数 $x$，随机变量 $X$ 的分布函数可以表示为：

$$\boxed{F(x) = \int_{-\infty}^{x} f(t) \, dt}$$

则称 $X$ 为**连续型随机变量**，$f(x)$ 为 $X$ 的**概率密度函数**（Probability Density Function, PDF）。

### 4.2 概率密度函数的性质

**性质1（非负性）：**
$$f(x) \geq 0, \quad \forall x \in \mathbb{R}$$

**性质2（归一性）：**
$$\boxed{\int_{-\infty}^{+\infty} f(x) \, dx = 1}$$

**性质3：** 在 $f(x)$ 的连续点处：
$$F'(x) = f(x)$$

**性质4：** 连续型随机变量取任意单点值的概率为零：
$$P\{X = a\} = 0, \quad \forall a \in \mathbb{R}$$

> **重要推论：** 对连续型随机变量，"开区间"与"闭区间的"概率相等：
> $$P\{a < X < b\} = P\{a \leq X \leq b\} = P\{a < X \leq b\} = P\{a \leq X < b\} = F(b) - F(a)$$

**性质5：**
$$P\{a < X \leq b\} = F(b) - F(a) = \int_a^b f(x) \, dx$$

> **注意：** $f(x)$ 可以大于1！$f(x)$ 不是概率，而是概率密度。$f(x)$ 的值反映的是 $X$ 在 $x$ 附近取值的"密集程度"。

### 4.3 概率的几何意义

$P\{a < X < b\}$ 等于 $f(x)$ 在 $[a, b]$ 上与 $x$ 轴所围成的面积。

## 五、分布函数的进一步性质与应用

### 5.1 利用分布函数判断随机变量类型

- **离散型**：$F(x)$ 是阶梯函数，在某些点处有跳跃
- **连续型**：$F(x)$ 是连续函数（无跳跃），且几乎处处可导

### 5.2 分布函数的充要条件

一个函数 $F(x)$ 是某个随机变量的分布函数，当且仅当：
1. 单调不减
2. $F(-\infty) = 0$，$F(+\infty) = 1$
3. 右连续

> **考试题型：** 给出一个函数 $F(x)$，判断它是否可以作为分布函数。检验以上三条即可。

### 5.3 分布函数在边界处的行为

| 随机变量类型 | 在 $x = a$ 处 | $F(a^-)$ | $F(a)$ | $P\{X = a\}$ |
|-------------|-------------|----------|--------|--------------|
| 离散型（$a$ 为取值点） | 有跳跃 | $F(a) - p_a$ | $F(a)$ | $p_a > 0$ |
| 连续型 | 连续 | $F(a)$ | $F(a)$ | $0$ |

## 六、常见分布函数的图形特征

### 6.1 离散型分布函数

**阶梯函数特征：**
- 在每个取值点 $x_k$ 处有跳跃，跳跃高度为 $p_k$
- 在相邻取值点之间为常数
- 跳跃从 $0$ 开始，最终趋于 $1$

### 6.2 连续型分布函数

**连续函数特征：**
- 整个曲线是连续的（无跳跃点）
- 从 $0$ 开始单调上升（或不变）至 $1$
- 具体形状取决于密度函数 $f(x)$

## 七、综合例题

**【例2】** 设随机变量 $X$ 的分布函数为：
$$F(x) = \begin{cases} 0, & x < 0 \\ A + B\arctan x, & x \geq 0 \end{cases}$$

求常数 $A, B$。

**解：** 由 $F(+\infty) = 1$：
$$A + B \cdot \frac{\pi}{2} = 1$$

由 $F(-\infty) = 0$：$0 = 0$ ✓

由右连续性，在 $x = 0$ 处：$F(0) = A + B \cdot 0 = A$，$F(0^-) = 0$，需要 $F(0) = F(0^+) = F(0)$（右连续自动满足）。

但我们还需要左极限 $F(0^-) = 0$，而 $F(0) = A$，由左连续性（$F$ 需要右连续，左极限可能不等于 $F(0)$）。

再检查：$F(0^-) = 0$，$F(0) = A$。对于分布函数，$P\{X = 0\} = F(0) - F(0^-) = A$。

如果 $X$ 是连续型，则 $A = 0$，从而 $B = 2/\pi$。

$$F(x) = \begin{cases} 0, & x < 0 \\ \frac{2}{\pi}\arctan x, & x \geq 0 \end{cases}$$

**【例3】** 设 $F(x) = A + B\arctan x$（$-\infty < x < +\infty$），求 $A, B$ 及 $P\{-1 < X \leq 1\}$。

**解：** 
$$F(-\infty) = A - \frac{\pi}{2}B = 0$$
$$F(+\infty) = A + \frac{\pi}{2}B = 1$$

解得：$A = 1/2$，$B = 1/\pi$

$$F(x) = \frac{1}{2} + \frac{1}{\pi}\arctan x$$

$$P\{-1 < X \leq 1\} = F(1) - F(-1) = \left(\frac{1}{2} + \frac{1}{\pi} \cdot \frac{\pi}{4}\right) - \left(\frac{1}{2} + \frac{1}{\pi} \cdot \left(-\frac{\pi}{4}\right)\right) = \frac{3}{4} - \frac{1}{4} = \frac{1}{2}$$

**【例4】** 设连续型随机变量 $X$ 的分布函数为：
$$F(x) = \begin{cases} 0, & x \leq 0 \\ Ax^2, & 0 < x \leq 1 \\ 1, & x > 1 \end{cases}$$

求：(1) 常数 $A$；(2) $X$ 的密度函数 $f(x)$；(3) $P\{0.5 < X < 2\}$。

**解：**

**(1)** $F(x)$ 在 $x = 1$ 处需连续：$A \cdot 1^2 = 1$，所以 $A = 1$。

**(2)** $f(x) = F'(x)$：

$$f(x) = \begin{cases} 2x, & 0 < x < 1 \\ 0, & \text{其他} \end{cases}$$

验证：$\int_{-\infty}^{+\infty} f(x)dx = \int_0^1 2x \, dx = x^2 \Big|_0^1 = 1$ ✓

**(3)** 
$$P\{0.5 < X < 2\} = F(2) - F(0.5) = 1 - 0.25 = 0.75$$

**【例5】** 设 $X$ 的分布律为 $P\{X = k\} = \dfrac{C}{k(k+1)}$，$k = 1, 2, 3, \ldots$，求常数 $C$。

**解：** 由 $\sum_{k=1}^{\infty} p_k = 1$：

$$\sum_{k=1}^{\infty} \frac{C}{k(k+1)} = C \sum_{k=1}^{\infty} \left(\frac{1}{k} - \frac{1}{k+1}\right) = C \cdot 1 = C$$

（利用裂项求和，这是望远镜级数）

所以 $C = 1$。

## 八、易错点与考试陷阱

### ⚠️ 易错点1：$F(x)$ 的右连续性

分布函数是**右连续**的，即 $F(x) = F(x^+)$。对于离散型随机变量，在取值点 $x_k$ 处：
- $F(x_k)$ 包含 $p_k$
- $F(x_k^-)$ 不包含 $p_k$

### ⚠️ 易错点2：概率密度可以大于1

$f(x)$ 是密度函数，不是概率。$f(x)$ 的值可以大于1，但 $\int f(x) dx = 1$。

例如均匀分布 $U(0, 0.5)$：$f(x) = 2$（$0 < x < 0.5$），大于1，但积分面积为1。

### ⚠️ 易错点3：连续型随机变量取单点值的概率为0

$P\{X = a\} = 0$ 对连续型随机变量成立，但"$X = a$"并非不可能事件！

### ⚠️ 易错点4：分布函数的判定

判断一个函数是否是分布函数，必须同时验证**三条性质**。任何一条不满足都不是分布函数。

### ⚠️ 易错点5：混合型随机变量

有些随机变量既非纯离散也非纯连续（混合型），它们也有分布函数，但不能用分布律或概率密度完整描述。

## 十、习题精选

**习题1：** 设 $F(x) = \begin{cases} 0, & x < -1 \\ 0.3, & -1 \leq x < 1 \\ 0.7, & 1 \leq x < 2 \\ 1, & x \geq 2 \end{cases}$

求 $X$ 的分布律。

**解：** 跳跃点为 $x = -1, 1, 2$：

| $X$ | $-1$ | $1$ | $2$ |
|-----|-------|------|------|
| $P$ | $0.3$ | $0.4$ | $0.3$ |

**习题2：** 设 $f(x) = \begin{cases} cx^2, & 0 < x < 2 \\ 0, & \text{其他} \end{cases}$

求常数 $c$ 和 $F(x)$。

**解：** $\int_0^2 cx^2 dx = \frac{c \cdot 8}{3} = 1$，$c = \frac{3}{8}$

$$F(x) = \begin{cases} 0, & x \leq 0 \\ \frac{x^3}{8}, & 0 < x < 2 \\ 1, & x \geq 2 \end{cases}$$

**习题3（考研真题型）：** 设 $X$ 是连续型随机变量，其分布函数 $F(x)$ 在 $x = 0$ 处可导，$F(0) = 0$，且当 $x > 0$ 时 $F(x) = 1 - e^{-\lambda x}$。求 $P\{X > 1/\lambda\}$。

**解：** 当 $x > 0$ 时：
$$P\{X > 1/\lambda\} = 1 - F(1/\lambda) = 1 - (1 - e^{-\lambda \cdot 1/\lambda}) = e^{-1} = \frac{1}{e}$$
