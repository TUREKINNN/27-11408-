## 一、事件与概率（第1-4页）

### §1 事件（画圆圈○）

#### 一、事件的关系

**（1）包含**
- A⊂B 表示事件A发生必然导致事件B发生

**（2）互不相容（互斥）**
- AB=∅，A与B不能同时发生

**（3）对立（互逆）**
- A∪B=Ω 且 AB=∅
- B=Ā，A与B有且仅有一个发生
- 注意：对立一定互斥，互斥不一定对立

**（4）独立**
- P(AB)=P(A)P(B)
- 若P(A)>0, P(B)>0，则互斥与独立不能同时成立
- 四对事件独立性：A与B独立 ⟺ A与B̄独立 ⟺ Ā与B独立 ⟺ Ā与B̄独立

#### 二、事件的运算

**（1）事件的运算**
- 并：A∪B（至少一个发生）
- 交：A∩B=AB（同时发生）
- 差：A-B=AB̄（A发生且B不发生）

**（2）事件的运算律**
- 交换律、结合律、分配律
- **De Morgan定律**：
  - A∪B的对立 = Ā∩B̄（并的对立 = 对立的交）
  - A∩B的对立 = Ā∪B̄（交的对立 = 对立的并）

### §2 概率（○的面积）

#### 一、基本概率公式
- P(A) = 1 - P(Ā)
- P(A∪B) = P(A) + P(B) - P(AB)
- P(A-B) = P(A) - P(AB) = P(AB̄)

#### 二、三大概率公式

**（1）加法公式**
- P(A∪B∪C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC)

**（2）减法公式**
- P(A-B) = P(A) - P(AB)

**（3）条件概率与乘法公式**
- P(B|A) = P(AB)/P(A)，其中P(A) > 0
- P(AB) = P(A)P(B|A) = P(B)P(A|B)

#### 三、三大概型

**（1）古典概型**
- 有限个等可能结果
- P(A) = A包含的基本事件数 / 总的基本事件数

**（2）几何概型**
- 样本空间为区域，等可能性
- P(A) = A的度量 / Ω的度量

**（3）伯努利概型（n重Bernoulli试验）**
- 独立重复试验，每次只有"成功"和"失败"两种结果
- P(X=k) = C(n,k) · p^k · (1-p)^(n-k)，k=0,1,...,n

#### 四、两个认识
- **全概率公式**：P(B) = ΣP(Aᵢ)P(B|Aᵢ)，其中Aᵢ构成完备事件组
- **贝叶斯公式（逆概率公式）**：P(Aⱼ|B) = P(Aⱼ)P(B|Aⱼ) / ΣP(Aᵢ)P(B|Aᵢ)

#### 五、一个重要不等式
- P(AB) ≥ P(A) + P(B) - 1
- 若A⊂B，则P(A) ≤ P(B)

## 二、一维随机变量（第5-17页）

### §1 老三样

#### 一、分布函数 F

**（1）定义**
- F(x) = P{X ≤ x}，-∞ < x < +∞

**（2）性质**
- 单调不减
- 右连续：F(x₀⁺) = F(x₀)
- F(-∞) = 0，F(+∞) = 1
- P{a < X ≤ b} = F(b) - F(a)

**（3）离散与连续的F**
- 离散：阶梯函数（跳跃间断点）
- 连续：连续函数

#### 二、概率密度 f

**（1）定义**
- F(x) = ∫₋∞ˣ f(t)dt

**（2）性质**
- f(x) ≥ 0
- ∫₋∞^{+∞} f(x)dx = 1
- f(x)连续点处，F'(x) = f(x)
- P{X = a} = 0（连续型）

#### 三、概率分布

**（1）定义**
- P{X = xₖ} = pₖ，k = 1, 2, ...

**（2）性质**
- pₖ ≥ 0
- Σpₖ = 1

### §2 F(x) 和 f(x) 判定

#### 一、各自
**（1）运算**
- 线性组合：aX+b 的分布
- 乘积

**（2）复合**
- F(x)复合：F(g(x))
- f(x)复合

#### 二、混搭

### §3 八大分布（○+□）

#### 一、离散型（5种）

**（1）Bernoulli分布族**

**① 0-1分布**
- X~B(1, p)
- P{X=1}=p, P{X=0}=1-p
- E(X)=p, D(X)=p(1-p)

**② 二项分布**
- X~B(n, p)
- P{X=k}=C(n,k)p^k(1-p)^(n-k), k=0,1,...,n
- E(X)=np, D(X)=np(1-p)
- **可加性**：X₁~B(n₁,p), X₂~B(n₂,p) 独立，则 X₁+X₂~B(n₁+n₂, p)

**③ 几何分布**
- X~Ge(p)
- P{X=k}=(1-p)^(k-1)·p, k=1,2,...
- E(X)=1/p, D(X)=(1-p)/p²
- **无记忆性**：P{X>m+n|X>m}=P{X=n}

**（2）泊松分布（自然数）**
- X~P(λ)
- P{X=k}=λ^k·e^(-λ)/k!, k=0,1,2,...
- E(X)=λ, D(X)=λ
- **二项分布的Poisson近似**：n大p小时，B(n,p)≈P(np)

**（3）超几何分布（古典概型）**
- P{X=k}=C(M,k)·C(N-M,n-k)/C(N,n)
- E(X)=nM/N

#### 二、连续型（3种）

**（1）均匀分布**
- X~U[a,b]
- f(x)=1/(b-a), a≤x≤b
- E(X)=(a+b)/2, D(X)=(b-a)²/12

**（2）指数分布（使用寿命）**
- X~E(λ)
- f(x)=λe^(-λx), x>0
- E(X)=1/λ, D(X)=1/λ²
- F(x)=1-e^(-λx), x>0
- **无记忆性**：P{X>s+t|X>s}=P{X>t}

**（3）正态分布（Gauss分布）**
- X~N(μ, σ²)
- f(x)=(1/√(2πσ²))·exp{-(x-μ)²/(2σ²)}
- E(X)=μ, D(X)=σ²

**① 正态分布的标准化**
- 若X~N(μ,σ²)，则 Z=(X-μ)/σ ~ N(0,1)
- Φ(-x) = 1-Φ(x)
- P{a<X<b} = Φ((b-μ)/σ) - Φ((a-μ)/σ)

**② 正态定参**
- 正态分布由E(X)和D(X)唯一确定

**③ 正态分布函数相加**
- X₁~N(μ₁,σ₁²), X₂~N(μ₂,σ₂²) 独立
- X₁+X₂ ~ N(μ₁+μ₂, σ₁²+σ₂²)
- aX+b ~ N(aμ+b, a²σ²)

#### 三、总结
| 分布 | 期望 | 方差 |
|------|------|------|
| 0-1分布 B(1,p) | p | p(1-p) |
| 二项分布 B(n,p) | np | np(1-p) |
| 几何分布 Ge(p) | 1/p | (1-p)/p² |
| 泊松分布 P(λ) | λ | λ |
| 均匀分布 U[a,b] | (a+b)/2 | (b-a)²/12 |
| 指数分布 E(λ) | 1/λ | 1/λ² |
| 正态分布 N(μ,σ²) | μ | σ² |

#### 四、无记忆性
- **几何分布**：P{X>m+n|X>m}=P{X>n}
- **指数分布**：P{X>s+t|X>s}=P{X>t}
- 注：只有这两个离散/连续分布有无记忆性

#### 五、参数可加性
- 二项分布：B(n₁,p)+B(n₂,p)=B(n₁+n₂,p)
- 泊松分布：P(λ₁)+P(λ₂)=P(λ₁+λ₂)
- 正态分布：N(μ₁,σ₁²)+N(μ₂,σ₂²)=N(μ₁+μ₂,σ₁²+σ₂²)

### §4 一维随机变量函数的分布

#### 一、连续型随机变量函数的分布

**（1）分布函数法（三大步）**
1. 写出Y=g(X)的值域
2. 求F_Y(y)=P{Y≤y}=P{g(X)≤y}
3. 求导得f_Y(y)=F'_Y(y)

**（2）公式法**
- 若y=g(x)单调，x=h(y)为其反函数
- f_Y(y) = f_X(h(y))·|h'(y)|

#### 二、F(X)的分布
- 若X是连续型随机变量，F(x)是其分布函数
- 则 Y=F(X) ~ U(0,1)

### §5 三种特殊积分

#### 一、伽马函数
- Γ(α) = ∫₀^∞ x^(α-1)e^(-x)dx, α>0
- Γ(1)=1, Γ(1/2)=√π
- Γ(α+1)=α·Γ(α)
- Γ(n+1)=n!（n为正整数）

#### 二、泊松积分（Poisson）
- ∫₋∞^{+∞} e^(-x²)dx = √π
- ∫₀^{+∞} e^(-x²)dx = √π/2

#### 三、指数零到无穷积分
- ∫₀^{+∞} x^n·e^(-x)dx = n!

### §6 求最值函数取值的概率
- M=max(X₁,...,Xₙ), m=min(X₁,...,Xₙ)
- F_M(y)=[F(y)]^n（Xᵢ独立同分布）
- F_m(y)=1-[1-F(y)]^n

## 三、二维随机变量（第18-27页）

### §1 分布函数

#### 一、联合分布函数
- F(x,y) = P{X≤x, Y≤y}

#### 二、边缘分布函数
- F_X(x) = F(x, +∞) = lim_{y→+∞} F(x,y)
- F_Y(y) = F(+∞, y) = lim_{x→+∞} F(x,y)

### §2 概率分布

#### 一、联合概率分布
- P{X=xᵢ, Y=yⱼ} = pᵢⱼ

#### 二、边缘概率分布
- P{X=xᵢ} = Σⱼ pᵢⱼ = pᵢ.
- P{Y=yⱼ} = Σᵢ pᵢⱼ = p.ⱼ

#### 三、条件概率分布
- P{X=xᵢ|Y=yⱼ} = pᵢⱼ/p.ⱼ

### §3 概率密度

#### 一、联合概率密度
- F(x,y) = ∫₋∞ˣ ∫₋∞ʸ f(u,v)dudv

#### 二、边缘概率密度
- f_X(x) = ∫₋∞^{+∞} f(x,y)dy
- f_Y(y) = ∫₋∞^{+∞} f(x,y)dx

#### 三、条件概率密度
- f_{X|Y}(x|y) = f(x,y)/f_Y(y)
- f_{Y|X}(y|x) = f(x,y)/f_X(x)

### §4 随机变量的独立性
- X与Y独立 ⟺ F(x,y)=F_X(x)·F_Y(y)
- 离散：pᵢⱼ=pᵢ.·p.ⱼ 对所有i,j成立
- 连续：f(x,y)=f_X(x)·f_Y(y) 几乎处处成立
- **重要结论**：若X与Y独立，g(X)与h(Y)也独立

### §5 二维均匀分布
- f(x,y)=1/S, (x,y)∈D
- 其中S为区域D的面积

### §6 二维正态分布

#### 一、定义
- (X,Y)~N(μ₁,μ₂,σ₁²,σ₂²,ρ)
- 边缘分布：X~N(μ₁,σ₁²), Y~N(μ₂,σ₂²)

#### 二、性质
1. X与Y独立 ⟺ ρ=0
2. aX+bY仍服从正态分布
3. (X,Y)正态 ⟹ X与Y的任何线性组合服从正态分布
4. 若ρ=0，则X,Y独立

### §7 二维随机变量函数的分布

#### 一、二维连续型随机变量函数的分布
- Z=X+Y 的卷积公式：f_Z(z)=∫f(x,z-x)dx
- Z=max(X,Y), Z=min(X,Y) 的分布

#### 二、一离散一连续型随机变量函数的分布
- 全概率公式结合

#### 三、最值函数的分布
- M=max(X,Y): F_M(z)=F_X(z)·F_Y(z)（独立时）
- m=min(X,Y): F_m(z)=1-[1-F_X(z)]·[1-F_Y(z)]（独立时）

## 四、数字特征（第28-35页）

### §1 期望

#### 一、定义
- **一维离散**：E(X)=Σxₖpₖ
- **二维离散**：E[g(X,Y)]=ΣᵢΣⱼg(xᵢ,yⱼ)pᵢⱼ
- **一维连续**：E(X)=∫xf(x)dx
- **二维连续**：E[g(X,Y)]=∫∫g(x,y)f(x,y)dxdy

#### 二、性质
- E(C)=C
- E(aX+bY)=aE(X)+bE(Y)
- 若X,Y独立：E(XY)=E(X)·E(Y)
- 注意：E(XY)=E(X)E(Y) **不意味着** X,Y独立

#### 三、轮换对称性求期望
- 若(X,Y)中X与Y可交换（如联合密度关于y=x对称）
- 则E(X)=E(Y)

#### 四、最大最小值函数求期望
- max(X,Y)+min(X,Y)=X+Y
- E[max(X,Y)]+E[min(X,Y)]=E(X)+E(Y)

#### 五、分布函数的线性组合求期望
- E(X)=∫₀^{+∞}[1-F(x)]dx（X≥0时）
- E(X)=∫₀^{+∞}P{X>x}dx（X≥0时）

### §2 方差
- D(X)=E[X-E(X)]²=E(X²)-[E(X)]²
- D(aX+b)=a²D(X)
- D(X±Y)=D(X)+D(Y)±2Cov(X,Y)
- 若X,Y独立：D(X+Y)=D(X)+D(Y)

### §3 八大分布的期望与方差

| 分布 | E(X) | D(X) |
|------|------|------|
| 0-1分布 B(1,p) | p | p(1-p) |
| 二项分布 B(n,p) | np | np(1-p) |
| 几何分布 Ge(p) | 1/p | (1-p)/p² |
| 泊松分布 P(λ) | λ | λ |
| 超几何分布 | nM/N | — |
| 均匀分布 U[a,b] | (a+b)/2 | (b-a)²/12 |
| 指数分布 E(λ) | 1/λ | 1/λ² |
| 正态分布 N(μ,σ²) | μ | σ² |

#### 一、正态分布绝对值的期望方差
- X~N(0,σ²)，E(|X|)=σ√(2/π)
- D(|X|)=σ²(1-2/π)

#### 二、标准正态分布的期望
- Z~N(0,1)，E(Z)=0，D(Z)=1
- E(Z²)=1，E(Z³)=0，E(Z⁴)=3

#### 三、Pascal分布
- 负二项分布，第r次成功时的试验次数

### §4 协方差

#### 一、定义
- Cov(X,Y)=E[(X-E(X))(Y-E(Y))]=E(XY)-E(X)E(Y)

#### 二、性质
- Cov(X,Y)=Cov(Y,X)
- Cov(aX+b, cY+d)=ac·Cov(X,Y)
- Cov(X₁+X₂, Y)=Cov(X₁,Y)+Cov(X₂,Y)
- Cov(X,X)=D(X)
- D(X±Y)=D(X)+D(Y)±2Cov(X,Y)
- **独立 ⟹ 不相关**（反之不一定，但正态分布中等价）

### §5 相关系数

#### 一、定义
- ρ_{XY} = Cov(X,Y)/√(D(X)·D(Y))

#### 二、性质
- |ρ_{XY}| ≤ 1
- |ρ_{XY}|=1 ⟺ Y=aX+b 几乎必然
- ρ_{XY}=0 称为不相关

#### 三、已知XY相关系数求UV相关系数
- 若U=aX+b, V=cY+d，则ρ_{UV}=sign(ac)·ρ_{XY}

#### 四、X+Y与X-Y的相关系数
- 若X,Y独立同分布~B(n,p)
- ρ(X+Y, X-Y) = [D(X)-D(Y)]/√[D(X+Y)·D(X-Y)] = 0

## 五、大数定律与中心极限定理（第36-40页）

### §1 考试大纲
- 了解切比雪夫不等式
- 了解大数定律（切比雪夫、辛钦、伯努利）
- 了解中心极限定理（林德伯格-列维、棣莫弗-拉普拉斯）

### §2 切比雪夫不等式
- P{|X-E(X)|≥ε} ≤ D(X)/ε²
- 或：P{|X-E(X)|<ε} ≥ 1-D(X)/ε²
- 意义：方差越小，随机变量越集中在期望附近

### §3 依概率收敛
- Xₙ →ᵖ a：对任意ε>0，lim_{n→∞}P{|Xₙ-a|≥ε}=0
- 性质：若Xₙ→ᵖ a, Yₙ→ᵖ b，则Xₙ+Yₙ→ᵖ a+b, XₙYₙ→ᵖ ab

### §4 大数定律

**（1）切比雪夫大数定律**
- 条件：X₁,X₂,...相互独立，E(Xᵢ)=μ, D(Xᵢ)≤C
- 结论：(1/n)ΣXᵢ →ᵖ μ

**（2）辛钦大数定律（独立同分布）**
- 条件：X₁,X₂,...独立同分布，E(Xᵢ)=μ
- 结论：(1/n)ΣXᵢ →ᵖ μ
- 意义：算术平均值依概率收敛于期望

**（3）伯努利大数定律**
- n次试验中事件A发生的频率 fₙ →ᵖ p=P(A)

### §5 中心极限定理

**（1）林德伯格-列维（Lindeberg-Lévy）**
- 条件：X₁,X₂,...独立同分布，E(Xᵢ)=μ, D(Xᵢ)=σ²
- 结论：(ΣXᵢ-nμ)/(σ√n) →ᵈ N(0,1)
- 即 ΣXᵢ 近似服从 N(nμ, nσ²)

**（2）棣莫弗-拉普拉斯（De Moivre-Laplace）**
- 条件：X~B(n,p)
- 结论：(X-np)/√(np(1-p)) →ᵈ N(0,1)
- 即 B(n,p) 近似服从 N(np, np(1-p))

## 六、统计初步（第41-48页）

### §1 统计基础知识

#### 一、概念
- **总体**：研究对象的全体
- **样本**：(X₁,X₂,...,Xₙ)，独立同分布
- **统计量**：样本的函数，不含未知参数

#### 二、常用统计量
- 样本均值：X̄=(1/n)ΣXᵢ
- 样本方差：S²=(1/(n-1))Σ(Xᵢ-X̄)²
- 样本标准差：S=√S²
- 样本k阶矩：Aₖ=(1/n)ΣXᵢᵏ
- 样本k阶中心矩：Bₖ=(1/n)Σ(Xᵢ-X̄)ᵏ

#### 三、样本均值与样本方差的数字特征
- E(X̄)=μ, D(X̄)=σ²/n
- E(S²)=σ²
- 注意：E(S)≠σ

#### 四、求统计量的数字特征
- 利用期望和方差的性质

### §2 三大抽样分布

#### 一、χ²分布（卡方）

**（1）定义**
- X₁,...,Xₙ独立同分布~N(0,1)
- χ²=X₁²+X₂²+...+Xₙ² ~ χ²(n)
- E(χ²)=n, D(χ²)=2n

**（2）性质**
- 可加性：χ²(n₁)+χ²(n₂)=χ²(n₁+n₂)

#### 二、F分布

**（1）定义**
- X~χ²(m), Y~χ²(n) 独立
- F=(X/m)/(Y/n) ~ F(m,n)

**（2）性质**
- 1/F(n,m) ~ F(m,n)
- 若T~t(n)，则T²~F(1,n)

#### 三、t分布

**（1）定义**
- X~N(0,1), Y~χ²(n) 独立
- T=X/√(Y/n) ~ t(n)
- E(T)=0 (n>1), D(T)=n/(n-2) (n>2)

**（2）性质**
- t分布关于0对称
- n→∞时，t(n)→N(0,1)

### §3 六大统计量

#### 一、单正态总体（X~N(μ,σ²)）
- (X̄-μ)/(σ/√n) ~ N(0,1)
- (n-1)S²/σ² ~ χ²(n-1)
- X̄与S²独立
- (X̄-μ)/(S/√n) ~ t(n-1)

#### 二、双正态总体
- X₁~N(μ₁,σ₁²), X₂~N(μ₂,σ₂²)
- (X̄-Ȳ-(μ₁-μ₂))/√(σ₁²/n₁+σ₂²/n₂) ~ N(0,1)
- S₁²σ₂²/(S₂²σ₁²) ~ F(n₁-1, n₂-1)

### §4 参数估计

#### 一、估计量
- 用样本统计量估计总体参数

#### 二、矩估计
- 用样本矩替换总体矩
- 步骤：令样本k阶矩=总体k阶矩，解方程

#### 三、最大似然估计（MLE）
- 步骤：
  1. 写似然函数 L(θ)=∏f(xᵢ;θ)
  2. 取对数 lnL(θ)
  3. 求导令 d(lnL)/dθ=0
  4. 解方程得 θ̂_MLE

#### 四、估计量的评价标准
- **无偏性**：E(θ̂)=θ
- **有效性**：D(θ̂₁)<D(θ̂₂) 则 θ̂₁ 更有效
- **一致性（相合性）**：θ̂ →ᵖ θ

### §5 置信区间与假设检验

#### 一、置信区间
- 1-α置信区间：P{θ₁<θ<θ₂}=1-α
- 常见置信水平：90%（α=0.1）、95%（α=0.05）、99%（α=0.01）

#### 二、假设检验
- 步骤：
  1. 提出H₀和H₁
  2. 选择检验统计量
  3. 确定拒绝域
  4. 做出判断
- **两类错误**：
  - 第Ⅰ类（弃真）：H₀为真却拒绝，概率α
  - 第Ⅱ类（取伪）：H₀为假却接受，概率β
- 单侧检验和双侧检验
