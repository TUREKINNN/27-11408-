## 一、边缘分布的基本概念

### 1.1 什么是边缘分布？

**"边缘"的由来：** 在联合分布律的表格中，将各列（或各行）的概率相加，得到的结果写在表格的边缘（最右边列或最下行），因此称为**边缘分布**（Marginal Distribution）。

**核心思想：** 边缘分布是从二维联合分布中"提取"出单个变量的分布信息，相当于"忘掉"另一个变量。

> **重要：** 仅由边缘分布**不能**唯一确定联合分布！

### 1.2 边缘分布函数

设 $(X, Y)$ 的联合分布函数为 $F(x, y)$，则 $X$ 和 $Y$ 各自的**边缘分布函数**为：

$$\boxed{F_X(x) = F(x, +\infty) = \lim_{y \to +\infty} F(x, y)}$$

$$\boxed{F_Y(y) = F(+\infty, y) = \lim_{x \to +\infty} F(x, y)}$$

**几何理解：** $F_X(x) = P\{X \leq x\} = P\{X \leq x, Y < +\infty\}$，即在 $Y$ 方向上取全部范围。

## 二、离散型的边缘分布律 ⭐⭐⭐⭐

### 2.1 定义

设 $(X, Y)$ 的联合分布律为 $P\{X = x_i, Y = y_j\} = p_{ij}$，则：

**$X$ 的边缘分布律：**
$$\boxed{P\{X = x_i\} = \sum_{j} p_{ij} \triangleq p_{i\cdot}, \quad i = 1, 2, 3, \ldots}$$

**$Y$ 的边缘分布律：**
$$\boxed{P\{Y = y_j\} = \sum_{i} p_{ij} \triangleq p_{\cdot j}, \quad j = 1, 2, 3, \ldots}$$

即 $X$ 的边缘分布律是联合分布律按**行求和**，$Y$ 的边缘分布律是按**列求和**。

### 2.2 表格表示

| $X \backslash Y$ | $y_1$ | $y_2$ | $\cdots$ | $y_j$ | $\cdots$ | $P\{X=x_i\}$ |
|------------------|--------|--------|----------|--------|----------|--------------|
| $x_1$ | $p_{11}$ | $p_{12}$ | $\cdots$ | $p_{1j}$ | $\cdots$ | $p_{1\cdot}$ |
| $x_2$ | $p_{21}$ | $p_{22}$ | $\cdots$ | $p_{2j}$ | $\cdots$ | $p_{2\cdot}$ |
| $\vdots$ | $\vdots$ | $\vdots$ | $\ddots$ | $\vdots$ | $\ddots$ | $\vdots$ |
| $P\{Y=y_j\}$ | $p_{\cdot 1}$ | $p_{\cdot 2}$ | $\cdots$ | $p_{\cdot j}$ | $\cdots$ | $1$ |

### 2.3 经典例题

**【例1】** 设 $(X, Y)$ 的联合分布律为：

| $X \backslash Y$ | $0$ | $1$ | $2$ | $P\{X=x_i\}$ |
|------------------|-----|-----|-----|--------------|
| $0$ | $\frac{1}{10}$ | $\frac{3}{10}$ | $\frac{1}{10}$ | $\frac{5}{10}$ |
| $1$ | $\frac{2}{10}$ | $\frac{2}{10}$ | $0$ | $\frac{4}{10}$ |
| $2$ | $0$ | $\frac{1}{10}$ | $0$ | $\frac{1}{10}$ |
| $P\{Y=y_j\}$ | $\frac{3}{10}$ | $\frac{6}{10}$ | $\frac{1}{10}$ | $1$ |

**$X$ 的边缘分布律：**

| $X$ | $0$ | $1$ | $2$ |
|-----|-----|-----|-----|
| $P$ | $\frac{1}{2}$ | $\frac{2}{5}$ | $\frac{1}{10}$ |

**$Y$ 的边缘分布律：**

| $Y$ | $0$ | $1$ | $2$ |
|-----|-----|-----|-----|
| $P$ | $\frac{3}{10}$ | $\frac{3}{5}$ | $\frac{1}{10}$ |

> **注意：** 同样的边缘分布可以对应不同的联合分布！例如上面的联合分布改为：

| $X \backslash Y$ | $0$ | $1$ | $2$ | $P\{X=x_i\}$ |
|------------------|-----|-----|-----|--------------|
| $0$ | $\frac{2}{10}$ | $\frac{2}{10}$ | $\frac{1}{10}$ | $\frac{5}{10}$ |
| $1$ | $\frac{1}{10}$ | $\frac{3}{10}$ | $0$ | $\frac{4}{10}$ |
| $2$ | $0$ | $\frac{1}{10}$ | $0$ | $\frac{1}{10}$ |
| $P\{Y=y_j\}$ | $\frac{3}{10}$ | $\frac{6}{10}$ | $\frac{1}{10}$ | $1$ |

边缘分布完全相同，但联合分布不同！

## 三、连续型的边缘密度函数 ⭐⭐⭐⭐⭐

### 3.1 边缘密度函数的定义

设 $(X, Y)$ 的联合密度函数为 $f(x, y)$，则：

**$X$ 的边缘密度函数：**
$$\boxed{f_X(x) = \int_{-\infty}^{+\infty} f(x, y) \, dy}$$

**$Y$ 的边缘密度函数：**
$$\boxed{f_Y(y) = \int_{-\infty}^{+\infty} f(x, y) \, dx}$$

即边缘密度是对联合密度在另一个变量上积分（"消去"另一个变量）。

### 3.2 由边缘密度求边缘分布函数

$$F_X(x) = \int_{-\infty}^x f_X(t) \, dt, \quad F_Y(y) = \int_{-\infty}^y f_Y(t) \, dt$$

### 3.3 推导

$$F_X(x) = P\{X \leq x\} = P\{X \leq x, Y < +\infty\} = \int_{-\infty}^x \int_{-\infty}^{+\infty} f(s, t) \, dt \, ds$$

所以 $f_X(x) = \dfrac{d}{dx} F_X(x) = \displaystyle\int_{-\infty}^{+\infty} f(x, y) \, dy$。

### 3.4 经典例题

**【例2】** 设 $(X, Y)$ 的联合密度为：

$$f(x, y) = \begin{cases} 6xy^2, & 0 < x < 1, \, 0 < y < 1 \\ 0, & \text{其他} \end{cases}$$

求 $f_X(x)$ 和 $f_Y(y)$。

**解：**

$$f_X(x) = \int_{-\infty}^{+\infty} f(x,y) \, dy = \int_0^1 6xy^2 \, dy = 6x \cdot \frac{y^3}{3}\bigg|_0^1 = 2x, \quad 0 < x < 1$$

$$f_Y(y) = \int_{-\infty}^{+\infty} f(x,y) \, dx = \int_0^1 6xy^2 \, dx = 6y^2 \cdot \frac{x^2}{2}\bigg|_0^1 = 3y^2, \quad 0 < y < 1$$

所以：
$$f_X(x) = \begin{cases} 2x, & 0 < x < 1 \\ 0, & \text{其他} \end{cases}, \quad f_Y(y) = \begin{cases} 3y^2, & 0 < y < 1 \\ 0, & \text{其他} \end{cases}$$

**【例3】** 设 $(X, Y)$ 的联合密度为：

$$f(x, y) = \begin{cases} 2, & 0 < x < y < 1 \\ 0, & \text{其他} \end{cases}$$

求 $f_X(x)$ 和 $f_Y(y)$。

**解：** 支撑区域为三角形 $0 < x < y < 1$。

$$f_X(x) = \int_x^1 2 \, dy = 2(1-x), \quad 0 < x < 1$$

$$f_Y(y) = \int_0^y 2 \, dx = 2y, \quad 0 < y < 1$$

> **注意：** 积分限时要根据支撑区域确定，不能简单地写 $\int_{-\infty}^{+\infty}$。

**【例4】（二维正态的边缘分布）**

设 $(X, Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$，则：

$$\boxed{X \sim N(\mu_1, \sigma_1^2), \quad Y \sim N(\mu_2, \sigma_2^2)}$$

即二维正态分布的边缘分布仍然是一维正态分布，且**边缘分布不依赖于 $\rho$**。

> **重要结论：** 不同的 $\rho$ 值给出不同的二维正态分布，但它们的边缘分布都相同！这再次说明：边缘分布不能唯一确定联合分布。

## 四、条件分布 ⭐⭐⭐⭐⭐

### 4.1 离散型的条件分布律

**定义：** 设 $(X, Y)$ 的联合分布律为 $P\{X=x_i, Y=y_j\} = p_{ij}$，$P\{Y=y_j\} = p_{\cdot j} > 0$，则：

$$\boxed{P\{X = x_i | Y = y_j\} = \frac{P\{X = x_i, Y = y_j\}}{P\{Y = y_j\}} = \frac{p_{ij}}{p_{\cdot j}}, \quad i = 1, 2, 3, \ldots}$$

称为在 $Y = y_j$ 条件下 $X$ 的**条件分布律**。

类似地：
$$P\{Y = y_j | X = x_i\} = \frac{p_{ij}}{p_{i\cdot}}, \quad j = 1, 2, 3, \ldots$$

> **验证：** $\displaystyle\sum_i P\{X=x_i | Y=y_j\} = \sum_i \frac{p_{ij}}{p_{\cdot j}} = \frac{1}{p_{\cdot j}} \sum_i p_{ij} = \frac{p_{\cdot j}}{p_{\cdot j}} = 1$ ✓

### 4.2 离散型条件分布例题

**【例5】** 利用例1的数据，求在 $Y = 1$ 条件下 $X$ 的条件分布律。

**解：** $P\{Y = 1\} = 6/10 = 3/5$

$$P\{X=0 | Y=1\} = \frac{p_{12}}{p_{\cdot 2}} = \frac{3/10}{6/10} = \frac{1}{2}$$
$$P\{X=1 | Y=1\} = \frac{p_{22}}{p_{\cdot 2}} = \frac{2/10}{6/10} = \frac{1}{3}$$
$$P\{X=2 | Y=1\} = \frac{p_{32}}{p_{\cdot 2}} = \frac{1/10}{6/10} = \frac{1}{6}$$

验证：$\dfrac{1}{2} + \dfrac{1}{3} + \dfrac{1}{6} = 1$ ✓

### 4.3 连续型的条件密度函数

**定义：** 设 $(X, Y)$ 的联合密度函数为 $f(x, y)$，$f_Y(y) > 0$，则在 $Y = y$ 条件下 $X$ 的**条件密度函数**为：

$$\boxed{f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)}}$$

类似地，当 $f_X(x) > 0$ 时：

$$\boxed{f_{Y|X}(y|x) = \frac{f(x, y)}{f_X(x)}}$$

### 4.4 条件密度函数的性质

$f_{X|Y}(x|y)$ 本身是一个密度函数：
1. $f_{X|Y}(x|y) \geq 0$
2. $\displaystyle\int_{-\infty}^{+\infty} f_{X|Y}(x|y) \, dx = \int_{-\infty}^{+\infty} \frac{f(x,y)}{f_Y(y)} \, dx = \frac{1}{f_Y(y)} \int_{-\infty}^{+\infty} f(x,y) \, dx = \frac{f_Y(y)}{f_Y(y)} = 1$ ✓

### 4.5 连续型条件分布例题

**【例6】** 利用例3的联合密度 $f(x,y) = 2$（$0 < x < y < 1$），求 $f_{X|Y}(x|y)$。

**解：** 由例3，$f_Y(y) = 2y$（$0 < y < 1$）

$$f_{X|Y}(x|y) = \frac{f(x,y)}{f_Y(y)} = \frac{2}{2y} = \frac{1}{y}, \quad 0 < x < y$$

> **解读：** 在给定 $Y = y$ 的条件下，$X$ 在区间 $(0, y)$ 上服从均匀分布 $U(0, y)$。

**【例7】** 设 $(X, Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$，求 $f_{X|Y}(x|y)$。

**解：** 可以证明（推导较复杂），在二维正态分布下：

$$\boxed{f_{X|Y}(x|y) = \frac{1}{\sqrt{2\pi}\sigma_1\sqrt{1-\rho^2}} \exp\left\{-\frac{1}{2\sigma_1^2(1-\rho^2)}\left[x - \left(\mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2)\right)\right]^2\right\}}$$

即：
$$X | Y=y \sim N\left(\mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2), \quad \sigma_1^2(1-\rho^2)\right)$$

> **重要结论：** 二维正态的条件分布仍然是正态分布！
> - 条件均值：$E(X|Y=y) = \mu_1 + \rho\dfrac{\sigma_1}{\sigma_2}(y - \mu_2)$（是 $y$ 的线性函数）
> - 条件方差：$\text{Var}(X|Y=y) = \sigma_1^2(1-\rho^2)$（与 $y$ 无关）
> - 当 $\rho = 0$ 时，条件分布退化为边缘分布（$X$ 与 $Y$ 独立）

## 五、条件分布的计算公式汇总

### 5.1 离散型

$$P\{X=x_i | Y=y_j\} = \frac{P\{X=x_i, Y=y_j\}}{P\{Y=y_j\}} = \frac{p_{ij}}{p_{\cdot j}}$$

### 5.2 连续型

$$f_{X|Y}(x|y) = \frac{f(x,y)}{f_Y(y)}$$

### 5.3 由条件密度求概率

$$P\{a < X \leq b | Y = y\} = \int_a^b f_{X|Y}(x|y) \, dx$$

## 六、联合密度与边缘密度、条件密度的关系 ⭐⭐⭐⭐

### 6.1 联合 = 边缘 × 条件

由条件密度的定义 $f_{X|Y}(x|y) = \dfrac{f(x,y)}{f_Y(y)}$，可得：

$$\boxed{f(x, y) = f_Y(y) \cdot f_{X|Y}(x|y) = f_X(x) \cdot f_{Y|X}(y|x)}$$

> **记忆：** 联合密度 = 边缘密度 × 条件密度。这是求联合密度的重要方法之一。

### 6.2 全概率公式的密度版本

$$f_X(x) = \int_{-\infty}^{+\infty} f_{X|Y}(x|y) \cdot f_Y(y) \, dy$$

这是一维全概率公式在密度函数中的推广。

### 6.3 贝叶斯公式的密度版本

$$f_{Y|X}(y|x) = \frac{f_{X|Y}(x|y) \cdot f_Y(y)}{\int_{-\infty}^{+\infty} f_{X|Y}(x|t) \cdot f_Y(t) \, dt}$$

## 七、综合例题

**【例8】** 设 $(X, Y)$ 的联合密度为 $f(x,y) = 8xy$（$0 < x < y < 1$），求 $f_X(x), f_Y(y), f_{X|Y}(x|y), f_{Y|X}(y|x)$。

**解：**

**边缘密度：**

$$f_X(x) = \int_x^1 8xy \, dy = 8x \cdot \frac{y^2}{2}\bigg|_x^1 = 4x(1-x^2), \quad 0 < x < 1$$

$$f_Y(y) = \int_0^y 8xy \, dx = 8y \cdot \frac{x^2}{2}\bigg|_0^y = 4y^3, \quad 0 < y < 1$$

**条件密度：**

$$f_{X|Y}(x|y) = \frac{8xy}{4y^3} = \frac{2x}{y^2}, \quad 0 < x < y$$

$$f_{Y|X}(y|x) = \frac{8xy}{4x(1-x^2)} = \frac{2y}{1-x^2}, \quad x < y < 1$$

**【例9】（考研经典）** 设 $(X, Y)$ 的联合密度为：

$$f(x,y) = \begin{cases} \dfrac{1}{y} e^{-(x+y)/y}, & x > 0, \, y > 0 \\ 0, & \text{其他} \end{cases}$$

求 $f_{X|Y}(x|y)$。

**解：** 先求 $f_Y(y)$：

$$f_Y(y) = \int_0^{+\infty} \frac{1}{y} e^{-(x+y)/y} \, dx = \frac{1}{y} e^{-1} \int_0^{+\infty} e^{-x/y} \, dx = \frac{1}{y} e^{-1} \cdot y = e^{-1}, \quad y > 0$$

等等，$f_Y(y) = e^{-1}$ 对所有 $y > 0$？这意味着 $\int_0^\infty e^{-1} dy = \infty$，不满足归一性。

让我重新计算：
$$f_Y(y) = \int_0^{+\infty} \frac{1}{y} e^{-x/y} \cdot e^{-1} \, dx = \frac{e^{-1}}{y} \cdot y = e^{-1}$$

这确实有问题，因为 $e^{-1}$ 是常数，$\int_0^\infty e^{-1} dy = \infty$。

让我检查联合密度：$f(x,y) = \frac{1}{y}e^{-(x+y)/y}$。注意 $(x+y)/y = x/y + 1$，所以：

$$f(x,y) = \frac{1}{y}e^{-x/y-1} = \frac{e^{-1}}{y}e^{-x/y}$$

$$\int_0^\infty \int_0^\infty \frac{e^{-1}}{y}e^{-x/y} dx \, dy = \int_0^\infty \frac{e^{-1}}{y} \cdot y \, dy = \int_0^\infty e^{-1} dy = \infty$$

密度不满足归一性！让我修改例题。

**【例9】（修正）** 设 $(X,Y)$ 的联合密度为：

$$f(x,y) = \begin{cases} e^{-y}, & 0 < x < y \\ 0, & \text{其他} \end{cases}$$

求 $f_X(x), f_Y(y), f_{X|Y}(x|y)$。

**解：**

$$f_Y(y) = \int_0^y e^{-t} \, dt = ye^{-y} \cdot \frac{1}{y} \cdot y$$

等等，让我重新来：

$$f_Y(y) = \int_0^y e^{-y} \, dx = ye^{-y}, \quad y > 0$$

验证归一性：$\int_0^\infty ye^{-y} dy = 1$（Gamma(2)=1）✓

$$f_X(x) = \int_x^{+\infty} e^{-y} \, dy = e^{-x}, \quad x > 0$$

所以 $X \sim E(1)$！

$$f_{X|Y}(x|y) = \frac{e^{-y}}{ye^{-y}} = \frac{1}{y}, \quad 0 < x < y$$

即在 $Y = y$ 条件下，$X \sim U(0, y)$。

> **结论：** $X \sim E(1)$，$Y|X=x \sim$ 从 $x$ 到 $+\infty$ 的截断指数分布，而 $X|Y=y \sim U(0,y)$。

## 八、易错点与考试陷阱

### ⚠️ 易错点1：边缘密度的积分限

求 $f_X(x)$ 时，是对 $y$ 积分，但积分限取决于**支撑区域**的形状。

例如 $0 < x < y < 1$ 时：$f_X(x) = \int_x^1 f(x,y) \, dy$（$y$ 从 $x$ 到 $1$，不是从 $0$ 到 $1$）

### ⚠️ 易错点2：条件密度中分母为零的情况

$f_{X|Y}(x|y) = \dfrac{f(x,y)}{f_Y(y)}$，当 $f_Y(y) = 0$ 时条件密度不存在。使用时要先确认 $f_Y(y) > 0$。

### ⚠️ 易错点3：边缘分布不能唯一确定联合分布

相同的边缘分布可以对应不同的联合分布——这是选择题的常见考点。

### ⚠️ 易错点4：条件分布仍是分布

条件分布 $f_{X|Y}(x|y)$ 是关于 $x$ 的函数（$y$ 固定），它本身满足密度函数的所有性质。

### ⚠️ 易错点5：二维正态的条件分布公式

记住条件均值和条件方差的公式：
$$E(X|Y=y) = \mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2)$$
$$D(X|Y=y) = \sigma_1^2(1-\rho^2)$$

## 十、习题精选

**习题1：** 设 $(X, Y)$ 的联合分布律为：

| $X \backslash Y$ | $1$ | $2$ | $3$ |
|------------------|-----|-----|-----|
| $1$ | $1/9$ | $2/9$ | $0$ |
| $2$ | $2/9$ | $2/9$ | $1/9$ |
| $3$ | $0$ | $1/9$ | $0$ |

求 $P\{X=2 | Y=2\}$。

**解：** $P\{Y=2\} = 2/9 + 2/9 + 1/9 = 5/9$

$$P\{X=2|Y=2\} = \frac{2/9}{5/9} = \frac{2}{5}$$

**习题2：** 设 $(X,Y)$ 的联合密度为 $f(x,y) = c(x^2+y^2)$（$0<x<1, 0<y<1$），求 $c$ 和 $f_{X|Y}(x|y)$。

**解：** 归一性：
$$c\int_0^1\int_0^1 (x^2+y^2) \, dy \, dx = c\left(\frac{1}{3}+\frac{1}{3}\right) = \frac{2c}{3} = 1 \Rightarrow c = \frac{3}{2}$$

$$f_Y(y) = \frac{3}{2}\int_0^1 (x^2+y^2) \, dx = \frac{3}{2}\left(\frac{1}{3}+y^2\right) = \frac{1}{2}+\frac{3y^2}{2}$$

$$f_{X|Y}(x|y) = \frac{\frac{3}{2}(x^2+y^2)}{\frac{1}{2}+\frac{3y^2}{2}} = \frac{3(x^2+y^2)}{1+3y^2}, \quad 0 < x < 1$$

**习题3：** 设 $(X,Y) \sim N(1, 0, 4, 1, 0.5)$，求 $f_{X|Y}(x|0)$。

**解：** $\mu_1=1, \mu_2=0, \sigma_1=2, \sigma_2=1, \rho=0.5$

条件均值：$E(X|Y=0) = 1 + 0.5 \times \frac{2}{1} \times (0-0) = 1$

条件方差：$D(X|Y=0) = 4 \times (1-0.25) = 3$

$$X|Y=0 \sim N(1, 3)$$
