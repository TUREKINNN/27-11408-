## 一、随机变量独立性的定义

### 1.1 定义

**定义：** 设 $F(x, y)$ 是二维随机变量 $(X, Y)$ 的联合分布函数，$F_X(x)$ 和 $F_Y(y)$ 分别为边缘分布函数。若对任意实数 $x, y$，有：

$$\boxed{F(x, y) = F_X(x) \cdot F_Y(y)}$$

则称 $X$ 与 $Y$ **相互独立**（Mutually Independent）。

### 1.2 独立性的直观含义

$X$ 与 $Y$ 独立意味着：
- $X$ 的取值**不影响** $Y$ 的分布，反之亦然
- 知道 $X$ 的信息**不改变**对 $Y$ 的概率判断
- 数学上：$P\{X \leq x, Y \leq y\} = P\{X \leq x\} \cdot P\{Y \leq y\}$

### 1.3 独立性与事件独立的关系

两个事件 $A, B$ 独立：$P(AB) = P(A) \cdot P(B)$

两个随机变量独立：$P\{X \leq x, Y \leq y\} = P\{X \leq x\} \cdot P\{Y \leq y\}$（对所有 $x, y$）

> 随机变量独立是事件独立的推广——要求对**所有** $x, y$ 都成立。

## 二、离散型随机变量的独立性判别 ⭐⭐⭐⭐

### 2.1 等价条件

**定理：** 设 $(X, Y)$ 是离散型随机变量，可能取值为 $(x_i, y_j)$（$i, j = 1, 2, \ldots$），则：

$$\boxed{X \text{ 与 } Y \text{ 独立} \iff P\{X=x_i, Y=y_j\} = P\{X=x_i\} \cdot P\{Y=y_j\}, \quad \forall i, j}$$

即 $p_{ij} = p_{i\cdot} \cdot p_{\cdot j}$ 对所有 $i, j$ 成立。

### 2.2 判别方法

1. 写出联合分布律表格
2. 求出边缘分布律（行和与列和）
3. 验证每一对 $(i, j)$ 是否满足 $p_{ij} = p_{i\cdot} \cdot p_{\cdot j}$
4. **只要有一对不满足**，就不独立

### 2.3 经典例题

**【例1】** 设 $(X, Y)$ 的联合分布律为：

| $X \backslash Y$ | $0$ | $1$ | $P\{X=x_i\}$ |
|------------------|-----|-----|--------------|
| $0$ | $0.1$ | $0.4$ | $0.5$ |
| $1$ | $0.3$ | $0.2$ | $0.5$ |
| $P\{Y=y_j\}$ | $0.4$ | $0.6$ | $1$ |

判断 $X$ 与 $Y$ 是否独立。

**解：** 验证 $p_{ij} = p_{i\cdot} \cdot p_{\cdot j}$：

- $p_{11} = 0.1$，$p_{1\cdot} \cdot p_{\cdot 1} = 0.5 \times 0.4 = 0.2$，$0.1 \neq 0.2$ ✗

**不独立**。

**【例2】** 设 $X \sim B(1, 1/2)$，$Y \sim B(1, 1/3)$，且 $X$ 与 $Y$ 独立，求 $(X, Y)$ 的联合分布律。

**解：** $X$ 的分布律：$P\{X=0\} = 1/2$，$P\{X=1\} = 1/2$

$Y$ 的分布律：$P\{Y=0\} = 2/3$，$P\{Y=1\} = 1/3$

由独立性 $p_{ij} = p_{i\cdot} \cdot p_{\cdot j}$：

| $X \backslash Y$ | $0$ | $1$ | $P\{X=x_i\}$ |
|------------------|-----|-----|--------------|
| $0$ | $\frac{1}{3}$ | $\frac{1}{6}$ | $\frac{1}{2}$ |
| $1$ | $\frac{1}{3}$ | $\frac{1}{6}$ | $\frac{1}{2}$ |
| $P\{Y=y_j\}$ | $\frac{2}{3}$ | $\frac{1}{3}$ | $1$ |

## 三、连续型随机变量的独立性判别 ⭐⭐⭐⭐⭐

### 3.1 等价条件

**定理：** 设 $(X, Y)$ 是连续型随机变量，联合密度函数为 $f(x, y)$，边缘密度函数为 $f_X(x)$ 和 $f_Y(y)$，则：

$$\boxed{X \text{ 与 } Y \text{ 独立} \iff f(x, y) = f_X(x) \cdot f_Y(y), \text{ 几乎处处成立}}$$

> "几乎处处"意味着在密度函数的连续点处成立即可，个别点的取值不影响。

### 3.2 判别方法

**方法一：** 求出 $f_X(x)$ 和 $f_Y(y)$，验证 $f(x,y) = f_X(x) \cdot f_Y(y)$

**方法二（实用技巧）：** 如果联合密度可以写成 $f(x,y) = g(x) \cdot h(y)$ 的形式，且支撑区域可以分解为 $A \times B$（即 $\{(x,y): x \in A, y \in B\}$），则 $X$ 与 $Y$ 独立。

> **关键：** 支撑区域必须是**矩形区域**（可以分解为两个一维区域的乘积）。三角形区域（如 $0 < x < y < 1$）上不可能独立。

### 3.3 经典例题

**【例3】** 设 $(X, Y)$ 的联合密度为：

$$f(x, y) = \begin{cases} 6xy^2, & 0 < x < 1, \, 0 < y < 1 \\ 0, & \text{其他} \end{cases}$$

判断 $X$ 与 $Y$ 是否独立。

**解：** 方法一：

$$f_X(x) = \int_0^1 6xy^2 \, dy = 2x, \quad 0 < x < 1$$
$$f_Y(y) = \int_0^1 6xy^2 \, dx = 3y^2, \quad 0 < y < 1$$

$$f_X(x) \cdot f_Y(y) = 2x \cdot 3y^2 = 6xy^2 = f(x, y) \quad \checkmark$$

$X$ 与 $Y$ **独立**。

方法二（快速判断）：

$f(x,y) = 6xy^2 = (6x)(y^2)$，支撑区域 $0 < x < 1, 0 < y < 1$ 是矩形。可以直接判断独立。

**【例4】** 设 $(X, Y)$ 的联合密度为：

$$f(x, y) = \begin{cases} 2, & 0 < x < y < 1 \\ 0, & \text{其他} \end{cases}$$

判断 $X$ 与 $Y$ 是否独立。

**解：** 支撑区域为三角形 $0 < x < y < 1$，**不是矩形区域**。

直接结论：$X$ 与 $Y$ **不独立**。

验证：$f_X(x) = 2(1-x)$（$0 < x < 1$），$f_Y(y) = 2y$（$0 < y < 1$）

$$f_X(x) \cdot f_Y(y) = 4y(1-x) \neq 2 = f(x, y) \quad \text{（不等）}$$

确实不独立。

**【例5】** 设 $(X, Y)$ 的联合密度为：

$$f(x, y) = \begin{cases} \dfrac{1}{2}e^{-x/2}, & x > 0, \, 0 < y < 1 \\ 0, & \text{其他} \end{cases}$$

判断 $X$ 与 $Y$ 是否独立。

**解：** $f(x,y) = \frac{1}{2}e^{-x/2} = g(x) \cdot h(y)$，其中 $g(x) = e^{-x/2}$（$x > 0$），$h(y) = 1$（$0 < y < 1$）。

支撑区域 $\{x > 0, 0 < y < 1\}$ 是矩形。$X$ 与 $Y$ **独立**。

$X \sim E(1/2)$，$Y \sim U(0, 1)$。

## 四、二维正态分布的独立性 ⭐⭐⭐⭐⭐

### 4.1 核心定理

**定理：** 设 $(X, Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$，则：

$$\boxed{X \text{ 与 } Y \text{ 独立} \iff \rho = 0 \iff X \text{ 与 } Y \text{ 不相关}}$$

> **这是二维正态分布特有的性质！** 对于一般的随机变量：
> - 独立 $\Rightarrow$ 不相关 ✓（恒成立）
> - 不相关 $\Rightarrow$ 独立 ✗（一般不成立，但二维正态例外）

### 4.2 不相关与独立的关系

**不相关：** $\text{Cov}(X, Y) = 0$ 或 $\rho_{XY} = 0$

**独立：** $F(x,y) = F_X(x) \cdot F_Y(y)$

**一般关系：**
$$\text{独立} \Rightarrow \text{不相关} \quad \text{（总是成立）}$$
$$\text{不相关} \not\Rightarrow \text{独立} \quad \text{（一般情况下不成立）}$$

**二维正态的特殊性：**
$$\text{不相关} \iff \text{独立} \quad \text{（二维正态中成立）}$$

### 4.3 证明：二维正态中 $\rho=0 \Rightarrow$ 独立

当 $\rho = 0$ 时，二维正态的联合密度函数为：

$$f(x,y) = \frac{1}{2\pi\sigma_1\sigma_2} \exp\left\{-\frac{1}{2}\left[\frac{(x-\mu_1)^2}{\sigma_1^2} + \frac{(y-\mu_2)^2}{\sigma_2^2}\right]\right\}$$

$$= \frac{1}{\sqrt{2\pi}\sigma_1} e^{-\frac{(x-\mu_1)^2}{2\sigma_1^2}} \cdot \frac{1}{\sqrt{2\pi}\sigma_2} e^{-\frac{(y-\mu_2)^2}{2\sigma_2^2}} = f_X(x) \cdot f_Y(y) \quad \checkmark$$

### 4.4 重要推论

对于二维正态 $(X, Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$：

1. $X$ 与 $Y$ 独立 $\iff$ $\rho = 0$
2. $X$ 与 $Y$ 不相关 $\iff$ $\rho = 0$
3. 因此：$X$ 与 $Y$ 独立 $\iff$ $X$ 与 $Y$ 不相关（**仅在二维正态中成立**）

> **考试高频考点：** 题目给出 $(X, Y)$ 服从二维正态分布，问 $X$ 与 $Y$ 是否独立？答：看 $\rho$ 是否为零。

## 五、独立性的性质与应用

### 5.1 独立性的基本性质

**性质1：** 若 $X$ 与 $Y$ 独立，则 $g(X)$ 与 $h(Y)$ 也独立（$g, h$ 为任意函数）。

**性质2：** 若 $X$ 与 $Y$ 独立，则条件分布等于边缘分布：
$$f_{X|Y}(x|y) = f_X(x), \quad f_{Y|X}(y|x) = f_Y(y)$$

**性质3：** 若 $X$ 与 $Y$ 独立，则：
$$E(XY) = E(X) \cdot E(Y)$$
$$\text{Cov}(X, Y) = 0$$

### 5.2 独立性的判断流程

```
判断 X 与 Y 是否独立
│
├── 离散型？
│   ├── 验证 pᵢⱼ = pᵢ· × p·ⱼ 对所有 i,j
│   └── 有一个不成立则不独立
│
├── 连续型？
│   ├── 支撑区域是否为矩形？
│   │   ├── 否 → 一定不独立
│   │   └── 是 → 继续检验
│   ├── f(x,y) 能否分解为 g(x)·h(y)？
│   │   ├── 否 → 不独立
│   │   └── 是 → 独立
│   └── 或者：验证 f(x,y) = f_X(x)·f_Y(y)
│
└── 二维正态？
    └── ρ = 0 ⟺ 独立 ⟺ 不相关
```

### 5.3 独立同分布（i.i.d.）

**定义：** 若 $X_1, X_2, \ldots, X_n$ **相互独立**，且服从**相同的分布**，则称它们为**独立同分布**（independent and identically distributed, i.i.d.），记为 $X_1, X_2, \ldots, X_n \stackrel{\text{i.i.d.}}{\sim} F$。

> **考研中常见表述：** "设 $X_1, X_2, \ldots, X_n$ 是来自总体 $X$ 的简单随机样本" 等价于 $X_1, X_2, \ldots, X_n \stackrel{\text{i.i.d.}}{\sim} F_X$。

## 六、独立性的反例

### 6.1 不相关但不独立的例子

**【例6】** 设 $X \sim N(0, 1)$，$Y = X^2$。

$\text{Cov}(X, Y) = E(XY) - E(X)E(Y) = E(X^3) - 0 = 0$（因为 $X \sim N(0,1)$，$E(X^3)=0$）

所以 $X$ 与 $Y$ 不相关。

但 $Y = X^2$，$X$ 完全决定了 $Y$，显然不独立。

### 6.2 独立但有函数关系的误区

> **注意：** 如果 $Y = g(X)$（$g$ 不是常数函数），则 $X$ 与 $Y$ **一定不独立**。因为知道 $X$ 的值就完全确定了 $Y$。

### 6.3 从联合密度直接判断

**【例7】** 设联合密度为 $f(x,y) = \frac{1}{2\pi} e^{-(x^2+y^2)/2}(1 + \sin x \sin y)$，判断 $X$ 与 $Y$ 是否独立。

**解：** 虽然 $\sin x \sin y$ 可以分解为 $\sin x \cdot \sin y$，但 $f(x,y)$ 不能写成 $g(x) \cdot h(y)$ 的形式（因为 $e^{-(x^2+y^2)/2}$ 是 $e^{-x^2/2} \cdot e^{-y^2/2}$，但乘上 $(1+\sin x \sin y)$ 后整体不能分解）。

所以 $X$ 与 $Y$ **不独立**。

## 七、综合例题

**【例8】（考研经典）**

设 $(X, Y)$ 的联合密度为：

$$f(x,y) = \begin{cases} \dfrac{6}{7}\left(x^2 + \dfrac{xy}{2}\right), & 0 < x < 1, \, 0 < y < 2 \\ 0, & \text{其他} \end{cases}$$

判断 $X$ 与 $Y$ 是否独立。

**解：** 支撑区域为矩形 $0 < x < 1, 0 < y < 2$。

$$f_X(x) = \int_0^2 \frac{6}{7}\left(x^2 + \frac{xy}{2}\right) dy = \frac{6}{7}\left(2x^2 + x\right), \quad 0 < x < 1$$

$$f_Y(y) = \int_0^1 \frac{6}{7}\left(x^2 + \frac{xy}{2}\right) dx = \frac{6}{7}\left(\frac{1}{3} + \frac{y}{4}\right), \quad 0 < y < 2$$

检验：$f_X(x) \cdot f_Y(y) = \frac{36}{49}(2x^2+x)\left(\frac{1}{3}+\frac{y}{4}\right)$

而 $f(x,y) = \frac{6}{7}\left(x^2+\frac{xy}{2}\right) = \frac{6}{7}x\left(x+\frac{y}{2}\right)$

显然 $f_X(x) \cdot f_Y(y) \neq f(x,y)$。

$X$ 与 $Y$ **不独立**。

**【例9】（独立性的应用——概率计算）**

设 $X \sim U(0, 1)$，$Y \sim U(0, 2)$，且 $X$ 与 $Y$ 独立，求 $P\{X < Y\}$。

**解：** 由独立性，$f(x,y) = f_X(x) \cdot f_Y(y) = 1 \times \frac{1}{2} = \frac{1}{2}$（$0 < x < 1, 0 < y < 2$）

$$P\{X < Y\} = \iint_{x < y} \frac{1}{2} \, dx \, dy$$

积分区域：$0 < x < 1, 0 < y < 2, x < y$

$$= \int_0^1 \int_x^2 \frac{1}{2} \, dy \, dx = \int_0^1 \frac{2-x}{2} \, dx = \frac{1}{2}\left[2x - \frac{x^2}{2}\right]_0^1 = \frac{1}{2}\left(2 - \frac{1}{2}\right) = \frac{3}{4}$$

**【例10】（考研高频——二维正态的独立性判断）**

设 $(X, Y) \sim N(1, -2, 4, 9, -1/3)$。问：$X$ 与 $Y$ 是否独立？$2X + Y$ 服从什么分布？

**解：** $\rho = -1/3 \neq 0$，所以 $X$ 与 $Y$ **不独立**。

对于二维正态，$aX + bY$ 仍服从正态分布：
$$E(2X+Y) = 2 \times 1 + (-2) = 0$$
$$D(2X+Y) = 4 \times 4 + 9 + 2 \times 2 \times 1 \times \text{Cov}(X,Y)$$
$$\text{Cov}(X,Y) = \rho\sigma_1\sigma_2 = -\frac{1}{3} \times 2 \times 3 = -2$$
$$D(2X+Y) = 16 + 9 + 4 \times (-2) = 17$$

所以 $2X + Y \sim N(0, 17)$。

## 八、易错点与考试陷阱

### ⚠️ 易错点1：不相关 $\not\Rightarrow$ 独立（一般情况）

$X$ 与 $Y$ 不相关（$\rho_{XY} = 0$）不能推出 $X$ 与 $Y$ 独立。**只有在二维正态分布中**，不相关等价于独立。

### ⚠️ 易错点2：支撑区域的重要性

即使 $f(x,y) = g(x) \cdot h(y)$，如果支撑区域不是矩形（如 $0 < x < y$），仍然不独立。

### ⚠️ 易错点3：独立性与条件分布

$X$ 与 $Y$ 独立 $\iff$ $f_{X|Y}(x|y) = f_X(x)$（条件分布等于边缘分布）

### ⚠️ 易错点4：独立与可加性

$X$ 与 $Y$ 独立时，$D(X+Y) = D(X) + D(Y)$

$X$ 与 $Y$ 不独立时，$D(X+Y) = D(X) + D(Y) + 2\text{Cov}(X,Y)$

### ⚠️ 易错点5：二维正态中的 $\rho$

$(X, Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$ 中：
- $\rho$ 与"不相关"和"独立"**完全等价**
- 题目说"不相关"就等价于 $\rho = 0$
- 题目说"独立"就等价于 $\rho = 0$

```
随机变量的独立性

│   ├── g(X) 与 h(Y) 独立
│   ├── 条件分布 = 边缘分布

    ├── 独立 → 不相关 ✓
    ├── 不相关 → 独立 ✗（一般）
    └── 不相关 ⟺ 独立（二维正态）
```

## 十、习题精选

**习题1：** 设 $X$ 与 $Y$ 独立，$X \sim P(1)$，$Y \sim P(2)$，求 $P\{X + Y = 2\}$。

**解：** $X + Y \sim P(1+2) = P(3)$（泊松分布的可加性，需独立）

$$P\{X+Y = 2\} = \frac{3^2 e^{-3}}{2!} = \frac{9e^{-3}}{2} \approx 0.2240$$

**习题2：** 设 $X \sim E(1)$，$Y \sim E(1)$，$X$ 与 $Y$ 独立，求 $P\{X > 2Y\}$。

**解：** $f(x,y) = e^{-x} \cdot e^{-y} = e^{-(x+y)}$（$x > 0, y > 0$）

$$P\{X > 2Y\} = \int_0^{+\infty} \int_{2y}^{+\infty} e^{-(x+y)} \, dx \, dy = \int_0^{+\infty} e^{-y} \left[-e^{-x}\right]_{2y}^{+\infty} dy = \int_0^{+\infty} e^{-y} \cdot e^{-2y} \, dy$$
$$= \int_0^{+\infty} e^{-3y} \, dy = \frac{1}{3}$$

**习题3：** 设 $(X,Y) \sim N(0, 0, 1, 1, \rho)$，问 $X+Y$ 与 $X-Y$ 是否独立？

**解：** $(X+Y, X-Y)$ 仍服从二维正态。计算协方差：

$$\text{Cov}(X+Y, X-Y) = D(X) - D(Y) + \text{Cov}(Y,X) - \text{Cov}(X,Y) = 1 - 1 = 0$$

（因为 $D(X) = D(Y) = 1$，$\text{Cov}(X,Y) = \text{Cov}(Y,X)$）

不相关，在二维正态中等价于独立。$X+Y$ 与 $X-Y$ **独立**。
