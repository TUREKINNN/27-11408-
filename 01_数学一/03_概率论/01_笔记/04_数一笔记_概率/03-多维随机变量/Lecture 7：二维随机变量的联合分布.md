## 一、二维随机变量的概念

### 1.1 定义

**定义：** 设 $X, Y$ 是定义在同一样本空间 $\Omega$ 上的两个随机变量，则称 $(X, Y)$ 为一个**二维随机变量**（或**二维随机向量**）。

> **直观理解：** 每次试验的结果可以用平面上的一个点 $(X(\omega), Y(\omega))$ 来描述。例如：
> - 身高 $X$ 和体重 $Y$ 构成 $(X, Y)$
> - 某地区的经度 $X$ 和纬度 $Y$
> - 某产品的长度 $X$ 和直径 $Y$

### 1.2 为什么要研究联合分布？

仅有 $X$ 的分布和 $Y$ 的分布（各自的一维分布）**不足以**确定 $(X, Y)$ 的联合行为。例如：
- $X$ 和 $Y$ 可能是独立的
- $X$ 和 $Y$ 可能有某种相关性
- $X$ 和 $Y$ 可能有函数关系（如 $Y = X^2$）

**联合分布**包含了 $X$ 和 $Y$ 各自的分布信息以及它们之间的相互关系信息。

## 二、联合分布函数 ⭐⭐⭐⭐⭐

### 2.1 定义

设 $(X, Y)$ 是二维随机变量，对任意实数 $x, y$，定义：

$$\boxed{F(x, y) = P\{X \leq x, \, Y \leq y\} = P\{(X \leq x) \cap (Y \leq y)\}}$$

为 $(X, Y)$ 的**联合分布函数**（Joint Cumulative Distribution Function, Joint CDF）。

### 2.2 联合分布函数的几何意义

$F(x, y)$ 表示随机点 $(X, Y)$ 落在以 $(x, y)$ 为右上顶点的无穷矩形区域 $(-\infty, x] \times (-\infty, y]$ 内的概率。

```
         y
         |_________(x,y)
         |         |
         |  概率    |
         | F(x,y)  |
    -----+---------+----- x
         |         |
    ←—————  -∞,−∞  —————→
```

### 2.3 联合分布函数的性质 ⭐⭐⭐⭐

$F(x, y)$ 具有以下性质：

**性质1（单调性）：** $F(x, y)$ 关于 $x$ 和 $y$ 分别单调不减，即：
- 固定 $y$，$F(x_1, y) \leq F(x_2, y)$（当 $x_1 < x_2$）
- 固定 $x$，$F(x, y_1) \leq F(x, y_2)$（当 $y_1 < y_2$）

**性质2（右连续性）：** $F(x, y)$ 关于 $x$ 和 $y$ 分别右连续

**性质3（有界性）：**
$$F(-\infty, y) = 0, \quad F(x, -\infty) = 0, \quad F(-\infty, -\infty) = 0$$
$$F(+\infty, +\infty) = 1$$

**性质4（非负性）：** 对任意 $x_1 < x_2$，$y_1 < y_2$：

$$\boxed{P\{x_1 < X \leq x_2, \, y_1 < Y \leq y_2\} = F(x_2, y_2) - F(x_2, y_1) - F(x_1, y_2) + F(x_1, y_1) \geq 0}$$

> **注意：** 性质4是二维分布函数特有的"交叉项"公式，一维情形没有这个复杂度。

### 2.4 由联合分布函数求概率

**矩形区域的概率：**

$$P\{a < X \leq b, \, c < Y \leq d\} = F(b, d) - F(b, c) - F(a, d) + F(a, c)$$

**半平面的概率：**

$$P\{X \leq x\} = F(x, +\infty) = \lim_{y \to +\infty} F(x, y)$$
$$P\{Y \leq y\} = F(+\infty, y) = \lim_{x \to +\infty} F(x, y)$$

## 三、二维离散型随机变量（联合分布律）⭐⭐⭐⭐

### 3.1 定义

若 $(X, Y)$ 的所有可能取值为有限对或可列无穷对 $(x_i, y_j)$，则称 $(X, Y)$ 为**二维离散型随机变量**。

$$\boxed{P\{X = x_i, \, Y = y_j\} = p_{ij}, \quad i, j = 1, 2, 3, \ldots}$$

称为 $(X, Y)$ 的**联合分布律**（Joint PMF），满足：
1. $p_{ij} \geq 0$
2. $\displaystyle\sum_{i} \sum_{j} p_{ij} = 1$

### 3.2 联合分布律的表格表示

| $X \backslash Y$ | $y_1$ | $y_2$ | $\cdots$ | $y_j$ | $\cdots$ | **行和** |
|------------------|--------|--------|----------|--------|----------|---------|
| $x_1$ | $p_{11}$ | $p_{12}$ | $\cdots$ | $p_{1j}$ | $\cdots$ | $p_{1\cdot}$ |
| $x_2$ | $p_{21}$ | $p_{22}$ | $\cdots$ | $p_{2j}$ | $\cdots$ | $p_{2\cdot}$ |
| $\vdots$ | $\vdots$ | $\vdots$ | $\ddots$ | $\vdots$ | $\ddots$ | $\vdots$ |
| $x_i$ | $p_{i1}$ | $p_{i2}$ | $\cdots$ | $p_{ij}$ | $\cdots$ | $p_{i\cdot}$ |
| $\vdots$ | $\vdots$ | $\vdots$ | $\ddots$ | $\vdots$ | $\ddots$ | $\vdots$ |
| **列和** | $p_{\cdot 1}$ | $p_{\cdot 2}$ | $\cdots$ | $p_{\cdot j}$ | $\cdots$ | $1$ |

其中 $p_{i\cdot} = \sum_j p_{ij}$，$p_{\cdot j} = \sum_i p_{ij}$。

### 3.3 经典例题

**【例1】** 设袋中有 3 个红球和 2 个白球，不放回地取 2 次，每次取 1 个。令 $X$ 为第 1 次取到红球的个数（0或1），$Y$ 为第 2 次取到红球的个数（0或1）。求 $(X, Y)$ 的联合分布律。

**解：** $(X, Y)$ 的可能取值为 $(0,0), (0,1), (1,0), (1,1)$

$$P\{X=0, Y=0\} = \frac{2}{5} \times \frac{1}{4} = \frac{2}{20} = \frac{1}{10}$$
$$P\{X=0, Y=1\} = \frac{2}{5} \times \frac{3}{4} = \frac{6}{20} = \frac{3}{10}$$
$$P\{X=1, Y=0\} = \frac{3}{5} \times \frac{2}{4} = \frac{6}{20} = \frac{3}{10}$$
$$P\{X=1, Y=1\} = \frac{3}{5} \times \frac{2}{4} = \frac{6}{20} = \frac{3}{10}$$

联合分布律表格：

| $X \backslash Y$ | $0$ | $1$ | **行和** |
|------------------|-----|-----|---------|
| $0$ | $\frac{1}{10}$ | $\frac{3}{10}$ | $\frac{4}{10}$ |
| $1$ | $\frac{3}{10}$ | $\frac{3}{10}$ | $\frac{6}{10}$ |
| **列和** | $\frac{4}{10}$ | $\frac{6}{10}$ | $1$ |

**【例2】（多项分布）** 设一枚均匀骰子投掷 3 次，$X$ 为出现 1 点的次数，$Y$ 为出现 2 点的次数。求 $(X, Y)$ 的联合分布律。

**解：** $X, Y$ 的可能取值都是 $0, 1, 2, 3$，且 $X + Y \leq 3$。

这是一个多项分布问题，$P\{X = i, Y = j\} = \dfrac{3!}{i! \, j! \, (3-i-j)!} \left(\dfrac{1}{6}\right)^i \left(\dfrac{1}{6}\right)^j \left(\dfrac{4}{6}\right)^{3-i-j}$

例如：
$$P\{X=1, Y=1\} = \frac{3!}{1! \cdot 1! \cdot 1!} \cdot \frac{1}{6} \cdot \frac{1}{6} \cdot \frac{4}{6} = 6 \times \frac{4}{216} = \frac{24}{216} = \frac{1}{9}$$

## 四、二维连续型随机变量（联合密度函数）⭐⭐⭐⭐⭐

### 4.1 定义

设 $(X, Y)$ 的联合分布函数为 $F(x, y)$，若存在非负可积函数 $f(x, y) \geq 0$，使得：

$$\boxed{F(x, y) = \int_{-\infty}^{x} \int_{-\infty}^{y} f(s, t) \, ds \, dt}$$

则称 $(X, Y)$ 为**二维连续型随机变量**，$f(x, y)$ 称为 $(X, Y)$ 的**联合概率密度函数**（Joint PDF）。

### 4.2 联合密度函数的性质

1. **非负性：** $f(x, y) \geq 0$，对一切 $(x, y) \in \mathbb{R}^2$
2. **归一性：** $\displaystyle\int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} f(x, y) \, dx \, dy = 1$

### 4.3 由联合密度函数求概率

$$\boxed{P\{(X, Y) \in D\} = \iint_D f(x, y) \, dx \, dy}$$

即概率等于联合密度函数在区域 $D$ 上的二重积分。

**矩形区域：**
$$P\{a \leq X \leq b, \, c \leq Y \leq d\} = \int_a^b \int_c^d f(x, y) \, dy \, dx$$

**一般区域：** 确定积分限后计算二重积分。

### 4.4 联合密度与联合分布函数的关系

- **由密度求分布：** $F(x, y) = \displaystyle\int_{-\infty}^{x} \int_{-\infty}^{y} f(s, t) \, ds \, dt$
- **由分布求密度：** 在 $f(x, y)$ 的连续点处，$f(x, y) = \dfrac{\partial^2 F(x, y)}{\partial x \, \partial y}$

## 五、常见二维连续型联合分布

### 5.1 二维均匀分布

若 $(X, Y)$ 在平面有界区域 $G$（面积为 $S_G$）上服从均匀分布，则：

$$f(x, y) = \begin{cases} \dfrac{1}{S_G}, & (x, y) \in G \\ 0, & (x, y) \notin G \end{cases}$$

记为 $(X, Y) \sim U(G)$。

**性质：** 落在 $G$ 的子区域 $D$ 中的概率只与 $D$ 的面积有关：

$$P\{(X, Y) \in D\} = \frac{S_D}{S_G}, \quad D \subseteq G$$

**【例3】** 设 $(X, Y)$ 在以 $(0,0), (1,0), (0,1)$ 为顶点的三角形区域上服从均匀分布，求 $P\{X + Y \leq 0.5\}$。

**解：** 三角形面积 $S_G = 1/2$。

$\{X + Y \leq 0.5\}$ 对应的小三角形面积 $S_D = 0.5 \times 0.5 \times 0.5 = 0.125$

$$P\{X + Y \leq 0.5\} = \frac{0.125}{0.5} = \frac{1}{4}$$

### 5.2 二维正态分布 ⭐⭐⭐⭐⭐

$(X, Y)$ 服从参数为 $(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$ 的**二维正态分布**，记为：

$$(X, Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$$

其中 $\mu_1, \mu_2 \in \mathbb{R}$，$\sigma_1, \sigma_2 > 0$，$|\rho| < 1$。

联合密度函数为：

$$f(x,y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left\{-\frac{1}{2(1-\rho^2)}\left[\frac{(x-\mu_1)^2}{\sigma_1^2} - \frac{2\rho(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}\right]\right\}$$

> **注意：** 这个公式不需要记忆，但要知道五个参数的含义。

**重要性质（将在后续 Lecture 中详细讨论）：**
1. $X \sim N(\mu_1, \sigma_1^2)$，$Y \sim N(\mu_2, \sigma_2^2)$
2. $\rho$ 是 $X$ 与 $Y$ 的相关系数
3. $\rho = 0 \Leftrightarrow X$ 与 $Y$ 独立（**二维正态的特殊性质**）
4. $aX + bY$ 仍服从正态分布

### 5.3 其他常见联合密度

**【例4】** 设 $(X, Y)$ 的联合密度为：

$$f(x, y) = \begin{cases} 2, & 0 < x < y < 1 \\ 0, & \text{其他} \end{cases}$$

验证归一性并求 $P\{X + Y < 1\}$。

**解：** 验证归一性：

$$\int_0^1 \int_0^y 2 \, dx \, dy = \int_0^1 2y \, dy = 1 \quad \checkmark$$

求 $P\{X + Y < 1\}$：

积分区域为 $0 < x < y < 1$ 且 $x + y < 1$，即 $0 < x < y < 1$ 且 $y < 1-x$。

由于 $x < y$ 且 $y < 1-x$，所以 $x < 1-x$，即 $x < 1/2$。

$$P\{X + Y < 1\} = \int_0^{1/2} \int_x^{1-x} 2 \, dy \, dx = \int_0^{1/2} 2(1-2x) \, dx = 2\left[x - x^2\right]_0^{1/2} = 2 \times \frac{1}{4} = \frac{1}{2}$$

## 六、联合分布函数的完整总结

### 6.1 二维 vs 一维分布函数

| | 一维 | 二维 |
|---|---|---|
| 定义 | $F(x) = P\{X \leq x\}$ | $F(x,y) = P\{X \leq x, Y \leq y\}$ |
| 变量数 | 1 | 2 |
| 单调性 | 关于 $x$ 不减 | 关于 $x, y$ 各自不减 |
| 极限 | $F(-\infty)=0, F(+\infty)=1$ | $F(-\infty,y)=0, F(x,-\infty)=0, F(+\infty,+\infty)=1$ |
| 求概率 | $P\{a<X\leq b\}=F(b)-F(a)$ | 需要"交叉项"公式 |

### 6.2 二维联合分布的知识体系

```
二维随机变量 (X,Y)
├── 离散型：联合分布律 p_{ij} = P{X=x_i, Y=y_j}
│   └── 表格表示
├── 连续型：联合密度函数 f(x,y)
│   └── F(x,y) = ∫∫ f(s,t) ds dt
├── 联合分布函数 F(x,y) = P{X≤x, Y≤y}
│   ├── 性质：单调、右连续、有界、非负
│   └── 求概率：交叉项公式
└── 常见二维分布
    ├── 二维均匀分布 U(G)
    └── 二维正态分布 N(μ₁,μ₂,σ₁²,σ₂²,ρ)
```

## 七、易错点与考试陷阱

### ⚠️ 易错点1：联合分布函数求概率的交叉项

$$P\{x_1 < X \leq x_2, y_1 < Y \leq y_2\} = F(x_2,y_2) - F(x_1,y_2) - F(x_2,y_1) + F(x_1,y_1)$$

容易忘记最后一项 $F(x_1, y_1)$ 或搞错正负号。

**记忆方法：** 类似于二维容斥原理——"右上角减左边减下边加左下角"。

### ⚠️ 易错点2：联合密度函数积分区域的确定

在计算 $P\{(X,Y) \in D\}$ 时，正确画出积分区域 $D$ 与联合密度函数支撑集的交集是关键。

### ⚠️ 易错点3：二维均匀分布的区域

$S_G$ 是**整个支撑区域** $G$ 的面积，不是整个平面的面积。

### ⚠️ 易错点4：二维正态分布的参数

二维正态有**5个参数** $(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$，不要与一维正态的2个参数混淆。其中 $\rho$ 是相关系数，$|\rho| < 1$。

### ⚠️ 易错点5：一维分布不能唯一确定二维分布

$(X, Y)$ 的联合分布不能仅由 $X$ 的边缘分布和 $Y$ 的边缘分布确定——它们之间的关系（相关性、独立性）也是重要信息。

## 八、习题精选

**习题1：** 设 $(X, Y)$ 的联合分布律为：

| $X \backslash Y$ | $0$ | $1$ | $2$ |
|------------------|-----|-----|-----|
| $0$ | $1/6$ | $1/3$ | $1/12$ |
| $1$ | $1/4$ | $1/6$ | $0$ |

求 $P\{X + Y \leq 1\}$ 和 $P\{X = 1 | Y = 0\}$。

**解：** 
$$P\{X+Y \leq 1\} = P\{X=0,Y=0\} + P\{X=0,Y=1\} + P\{X=1,Y=0\}$$
$$= \frac{1}{6} + \frac{1}{3} + \frac{1}{4} = \frac{2+4+3}{12} = \frac{9}{12} = \frac{3}{4}$$

$$P\{X=1|Y=0\} = \frac{P\{X=1,Y=0\}}{P\{Y=0\}} = \frac{1/4}{1/6+1/4} = \frac{1/4}{5/12} = \frac{3}{5}$$

**习题2：** 设 $(X,Y)$ 的联合密度为 $f(x,y) = kxy$（$0 < x < 1, 0 < y < 1$），求 $k$。

**解：** 归一性：
$$\int_0^1 \int_0^1 kxy \, dy \, dx = k \int_0^1 x \, dx \int_0^1 y \, dy = k \cdot \frac{1}{2} \cdot \frac{1}{2} = \frac{k}{4} = 1 \Rightarrow k = 4$$

**习题3：** 设 $(X,Y)$ 在区域 $D = \{(x,y): 0 < x < 1, 0 < y < x\}$ 上服从均匀分布，求 $P\{Y < 0.5\}$。

**解：** $S_D = 1/2$，$f(x,y) = 2$（$(x,y) \in D$）

$\{Y < 0.5\}$ 在 $D$ 中的区域：$0 < x < 1, 0 < y < \min(x, 0.5)$

$$P\{Y < 0.5\} = \int_0^1 \int_0^{\min(x,0.5)} 2 \, dy \, dx$$

当 $x < 0.5$ 时，$y$ 的上限为 $x$；当 $x \geq 0.5$ 时，上限为 $0.5$：

$$= \int_0^{0.5} 2x \, dx + \int_{0.5}^1 1 \, dx = [x^2]_0^{0.5} + 0.5 = 0.25 + 0.5 = 0.75$$
