## 一、Z = X + Y 的分布（卷积公式）⭐⭐⭐⭐⭐

### 1.1 离散型情形

设 $(X, Y)$ 为二维离散型随机变量，$Z = X + Y$，则 $Z$ 的分布律为：

$$\boxed{P\\{Z = z_k\\} = \\sum_{i} P\\{X = x_i, \\, Y = z_k - x_i\\}}$$

即对所有可能的 $x_i$，将满足 $x_i + y_j = z_k$ 的联合概率求和。

**特别地，当 $X, Y$ 独立时：**

$$P\\{Z = z_k\\} = \\sum_{i} P\\{X = x_i\\} \\cdot P\\{Y = z_k - x_i\\}$$

### 1.2 连续型情形（卷积公式）⭐⭐⭐⭐⭐

设 $(X, Y)$ 为二维连续型随机变量，联合密度为 $f(x, y)$，$Z = X + Y$，则 $Z$ 的概率密度函数为：

$$\\boxed{f_Z(z) = \\int_{-\\infty}^{+\\infty} f(x, \, z - x) \, dx}$$

**特别地，当 $X, Y$ 相互独立时（卷积公式）：**

$$\\boxed{f_Z(z) = \\int_{-\\infty}^{+\\infty} f_X(x) \\cdot f_Y(z - x) \, dx}$$

其中 $f_X(x)$、$f_Y(y)$ 分别为 $X$、$Y$ 的边缘密度。

**等价形式（换元后）：**

$$f_Z(z) = \\int_{-\\infty}^{+\\infty} f_X(z - y) \\cdot f_Y(y) \, dy$$

### 1.3 卷积公式的推导思路

**分布函数法推导：**

$$F_Z(z) = P\\{X + Y \\leq z\\} = \\iint_{x+y \\leq z} f(x, y) \, dx \, dy$$

对 $x+y \\leq z$ 的积分区域，固定 $x$，$y$ 从 $-\\infty$ 到 $z-x$：

$$F_Z(z) = \\int_{-\\infty}^{+\\infty} \\left[ \\int_{-\\infty}^{z-x} f(x, y) \, dy \\right] dx$$

令 $y = t - x$（换元），则：

$$F_Z(z) = \\int_{-\\infty}^{+\\infty} \\left[ \\int_{-\\infty}^{z} f(x, t - x) \, dt \\right] dx = \\int_{-\\infty}^{z} \\left[ \\int_{-\\infty}^{+\\infty} f(x, t - x) \, dx \\right] dt$$

求导得密度：

$$f_Z(z) = F_Z'(z) = \\int_{-\\infty}^{+\\infty} f(x, z - x) \, dx$$

### 1.4 卷积公式的适用条件与注意事项

1. **独立性假设：** 标准卷积公式要求 $X, Y$ 相互独立
2. **积分限的确定：** 实际计算时，积分限由 $f_X(x)$ 和 $f_Y(z-x)$ 的非零区域共同决定
3. **分段讨论：** 由于 $f_X(x)$ 和 $f_Y(y)$ 通常是分段函数，需要对 $z$ 进行分段讨论

## 二、Z = X + Y 卷积公式的经典应用

### 2.1 两个独立均匀分布之和

**【例1】** 设 $X \\sim U(0, 1)$，$Y \\sim U(0, 1)$，$X, Y$ 独立，求 $Z = X + Y$ 的密度函数。

**解：** $f_X(x) = 1$（$0 \\leq x \\leq 1$），$f_Y(y) = 1$（$0 \\leq y \\leq 1$）

$$f_Z(z) = \\int_{-\\infty}^{+\\infty} f_X(x) f_Y(z-x) \, dx$$

$f_X(x)$ 非零条件：$0 \\leq x \\leq 1$

$f_Y(z-x)$ 非零条件：$0 \\leq z - x \\leq 1$，即 $z - 1 \\leq x \\leq z$

两个条件取交集，分三种情况：

**(1) 当 $0 < z \\leq 1$ 时：** $\\max(0, z-1) = 0$，$\\min(1, z) = z$

$$f_Z(z) = \\int_0^z 1 \\cdot 1 \, dx = z$$

**(2) 当 $1 < z < 2$ 时：** $\\max(0, z-1) = z-1$，$\\min(1, z) = 1$

$$f_Z(z) = \\int_{z-1}^1 1 \\cdot 1 \, dx = 2 - z$$

**(3) 其他情况：** $f_Z(z) = 0$

**结论（三角分布）：**

$$f_Z(z) = \\begin{cases} z, & 0 < z \\leq 1 \\\\ 2 - z, & 1 < z < 2 \\\\ 0, & \\text{其他} \\end{cases}$$

### 2.2 两个独立正态分布之和 ⭐⭐⭐⭐⭐

**【定理】** 设 $X \\sim N(\\mu_1, \\sigma_1^2)$，$Y \\sim N(\\mu_2, \\sigma_2^2)$，$X, Y$ 独立，则：

$$\\boxed{Z = X + Y \\sim N(\\mu_1 + \\mu_2, \, \\sigma_1^2 + \\sigma_2^2)}$$

**推广：** 若 $X_i \\sim N(\\mu_i, \\sigma_i^2)$（$i = 1, 2, \\ldots, n$）相互独立，则：

$$\\sum_{i=1}^n X_i \\sim N\\left(\\sum_{i=1}^n \\mu_i, \, \\sum_{i=1}^n \\sigma_i^2\\right)$$

特别地，$n$ 个独立同分布 $N(\\mu, \\sigma^2)$ 的随机变量之和：

$$\\sum_{i=1}^n X_i \\sim N(n\\mu, \\, n\\sigma^2)$$

> **正态分布的可加性是正态分布最重要的性质之一，考研必背！**

### 2.3 两个独立指数分布之和（Gamma分布）

**【例2】** 设 $X \\sim E(\\lambda)$，$Y \\sim E(\\lambda)$，$X, Y$ 独立，求 $Z = X + Y$ 的密度。

**解：**
$$f_Z(z) = \\int_0^z \\lambda e^{-\\lambda x} \\cdot \\lambda e^{-\\lambda(z-x)} \, dx = \\lambda^2 e^{-\\lambda z} \\int_0^z dx = \\lambda^2 z e^{-\\lambda z}, \quad z > 0$$

即 $Z \\sim \\text{Gamma}(2, \\lambda)$。

### 2.4 两个独立泊松分布之和

设 $X \\sim P(\\lambda_1)$，$Y \\sim P(\\lambda_2)$，$X, Y$ 独立，则：

$$\\boxed{Z = X + Y \\sim P(\\lambda_1 + \\lambda_2)}$$

> **泊松分布的可加性同样非常重要！**

### 2.5 两个独立二项分布之和

设 $X \\sim B(n, p)$，$Y \\sim B(m, p)$，$X, Y$ 独立（注意参数 $p$ 必须相同），则：

$$\\boxed{Z = X + Y \\sim B(n + m, \, p)}$$

## 三、Z = max{X, Y} 与 Z = min{X, Y} 的分布 ⭐⭐⭐⭐⭐

### 3.1 Z = max{X, Y} 的分布

设 $X, Y$ 相互独立，分布函数分别为 $F_X(x)$、$F_Y(y)$，令 $Z = \\max\\{X, Y\\}$。

**分布函数：**

$$\\boxed{F_{\\max}(z) = P\\{\\max(X, Y) \\leq z\\} = P\\{X \\leq z\\} \\cdot P\\{Y \\leq z\\} = F_X(z) \\cdot F_Y(z)}$$

**密度函数（若存在）：**

$$f_{\\max}(z) = F_X(z) f_Y(z) + f_X(z) F_Y(z)$$

**推广到 $n$ 个独立随机变量：** 若 $X_1, X_2, \\ldots, X_n$ 独立，分布函数分别为 $F_1, F_2, \\ldots, F_n$，令 $M = \\max\\{X_1, X_2, \\ldots, X_n\\}$，则：

$$\\boxed{F_M(z) = \\prod_{i=1}^n F_i(z)}$$

特别地，当 $X_1, \\ldots, X_n$ 独立同分布（i.i.d.），共同分布函数为 $F(x)$ 时：

$$F_M(z) = [F(z)]^n$$

### 3.2 Z = min{X, Y} 的分布

设 $X, Y$ 相互独立，分布函数分别为 $F_X(x)$、$F_Y(y)$，令 $Z = \\min\\{X, Y\\}$。

**分布函数：**

$$\\boxed{F_{\\min}(z) = P\\{\\min(X, Y) \\leq z\\} = 1 - P\\{X > z\} \\cdot P\\{Y > z\\} = 1 - [1 - F_X(z)][1 - F_Y(z)]}$$

**密度函数（若存在）：**

$$f_{\\min}(z) = [1 - F_Y(z)] f_X(z) + [1 - F_X(z)] f_Y(z)$$

**推广到 $n$ 个独立随机变量：** 令 $N = \\min\\{X_1, X_2, \\ldots, X_n\\}$，则：

$$\\boxed{F_N(z) = 1 - \\prod_{i=1}^n [1 - F_i(z)]}$$

特别地，当 $X_1, \\ldots, X_n$ 独立同分布时：

$$F_N(z) = 1 - [1 - F(z)]^n$$

### 3.3 独立同分布指数分布的极值分布

设 $X_1, X_2, \\ldots, X_n$ 独立同分布于 $E(\\lambda)$。

**最大值：**

$$F_M(z) = (1 - e^{-\\lambda z})^n, \quad z > 0$$

**最小值：**

$$F_N(z) = 1 - e^{-n\\lambda z}, \quad z > 0$$

> **结论：** $n$ 个独立同分布指数分布 $E(\\lambda)$ 的最小值服从 $E(n\\lambda)$！这是一个非常重要的结论。

## 四、其他常见函数的分布

### 4.1 Z = X - Y 的分布

设 $(X, Y)$ 为连续型，联合密度 $f(x, y)$，$Z = X - Y$：

$$\\boxed{f_Z(z) = \\int_{-\\infty}^{+\\infty} f(x, \, x - z) \, dx}$$

当 $X, Y$ 独立时：

$$f_Z(z) = \\int_{-\\infty}^{+\\infty} f_X(x) \\cdot f_Y(x - z) \, dx$$

### 4.2 Z = XY 的分布

$$f_Z(z) = \\int_{-\\infty}^{+\\infty} \\frac{1}{|x|} f\\left(x, \, \\frac{z}{x}\right) dx$$

### 4.3 Z = X/Y 的分布

$$\\boxed{f_Z(z) = \\int_{-\\infty}^{+\\infty} |x| \, f(x, \, xz) \, dx}$$

当 $X, Y$ 独立时：

$$f_Z(z) = \\int_{-\\infty}^{+\\infty} |x| \, f_X(x) \\cdot f_Y(xz) \, dx$$

## 五、分布函数法求二维随机变量函数分布的一般步骤

### 5.1 分布函数法（通用方法）⭐⭐⭐⭐⭐

**步骤：**

1. **写出 $Z = g(X, Y)$ 的分布函数定义：**
$$F_Z(z) = P\\{g(X, Y) \\leq z\\}$$

2. **转化为二重积分：**
$$F_Z(z) = \\iint_{g(x,y) \\leq z} f(x, y) \, dx \, dy$$

3. **确定积分区域 $\\{(x, y) : g(x, y) \\leq z\\}$**，画出积分区域图

4. **计算积分**（通常需要对 $z$ 分段讨论）

5. **求导得密度：**
$$f_Z(z) = F_Z'(z)$$

> **分布函数法是万能方法！当卷积公式等特殊情况不适用时，使用分布函数法。**

## 六、经典例题

**【例3】（独立正态之和）** 设 $X \\sim N(1, 4)$，$Y \\sim N(2, 9)$，$X, Y$ 独立，求 $Z = X + Y$ 的分布及 $P\\{Z < 5\\}$。

**解：** $Z \\sim N(1+2, 4+9) = N(3, 13)$

$$P\\{Z < 5\\} = \\Phi\\left(\\frac{5-3}{\\sqrt{13}}\\right) = \\Phi\\left(\\frac{2}{\\sqrt{13}}\\right) = \\Phi(0.5547) \\approx 0.7105$$

**【例4】（极值分布）** 设系统由两个独立元件串联而成，元件寿命分别为 $X \\sim E(0.01)$，$Y \\sim E(0.02)$。求系统寿命 $T = \\min\\{X, Y\\}$ 的分布及 $E(T)$。

**解：**
$$F_T(t) = 1 - [1 - F_X(t)][1 - F_Y(t)] = 1 - e^{-0.01t} \\cdot e^{-0.02t} = 1 - e^{-0.03t}$$

所以 $T \\sim E(0.03)$，$E(T) = \\frac{1}{0.03} \\approx 33.3$ 小时。

**【例5】（卷积公式综合）** 设 $X \\sim N(0, 1)$，$Y \\sim N(0, 1)$，$X, Y$ 独立，求 $Z = X - Y$ 的分布。

**解：** $Z = X + (-Y)$，其中 $-Y \\sim N(0, 1)$（正态分布关于原点对称）

由正态分布的可加性：$Z \\sim N(0 + 0, 1 + 1) = N(0, 2)$

**【例6】（最大值分布应用）** 设某设备有 $n$ 个独立同分布的零件，每个零件的寿命 $X_i \\sim E(\\lambda)$。求至少一个零件在 $t$ 时刻之前损坏的概率。

**解：** 至少一个损坏 = 不是全部存活：

$$P = 1 - P\\{\\min(X_1, \\ldots, X_n) > t\\} = 1 - [P\\{X_1 > t\\}]^n = 1 - e^{-n\\lambda t}$$

## 七、易错点与考试陷阱

### ⚠️ 易错点1：卷积公式的积分限

卷积公式 $f_Z(z) = \\int_{-\\infty}^{+\\infty} f_X(x) f_Y(z-x) dx$ 的积分限必须考虑 $f_X(x)$ 和 $f_Y(z-x)$ 同时非零的区域。通常需要对 $z$ 分段讨论。

### ⚠️ 易错点2：正态分布可加性要求独立

$X + Y \\sim N(\\mu_1 + \\mu_2, \\sigma_1^2 + \\sigma_2^2)$ **仅在 $X, Y$ 独立时成立**。若不独立，需要知道联合分布或相关系数。

### ⚠️ 易错点3：二项分布可加性要求 $p$ 相同

$B(n, p) + B(m, p) = B(n+m, p)$ 要求两个二项分布的参数 $p$ 相同。若 $p$ 不同，则和不服从二项分布。

### ⚠️ 易错点4：最小值分布公式

$F_{\\min}(z) = 1 - \\prod[1 - F_i(z)]$，容易误写为 $F_{\\min}(z) = \\prod F_i(z)$（那是最大值的公式）。

### ⚠️ 易错点5：$Z = X/Y$ 的密度公式

$f_Z(z) = \\int |x| f_X(x) f_Y(xz) dx$，注意有 $|x|$ 的因子，不能漏掉。

## 九、习题精选

**习题1：** 设 $X \\sim U(0, 2)$，$Y \\sim U(0, 2)$，$X, Y$ 独立，求 $Z = X + Y$ 的密度函数。

**解：**
$$f_Z(z) = \\begin{cases} \\frac{z}{4}, & 0 < z \\leq 2 \\\\ \\frac{4-z}{4}, & 2 < z < 4 \\\\ 0, & \\text{其他} \\end{cases}$$

**习题2：** 设 $X_1, X_2, X_3$ 独立同分布于 $E(2)$，求 $T = \\min\\{X_1, X_2, X_3\\}$ 的分布和 $E(T)$。

**解：** $T \\sim E(2 \\times 3) = E(6)$，$E(T) = \\frac{1}{6}$

**习题3：** 设 $X \\sim N(1, 1)$，$Y \\sim N(2, 4)$，$X, Y$ 独立，求 $Z = 2X - Y + 3$ 的分布。

**解：**
$$E(Z) = 2 \\times 1 - 2 + 3 = 3$$
$$D(Z) = 4 \\times 1 + 4 = 8$$
$$Z \\sim N(3, 8)$$

**习题4：** 设 $X, Y$ 独立同分布于 $U(0, 1)$，求 $Z = \\max\\{X, Y\\}$ 的密度函数。

**解：** $F_Z(z) = z^2$（$0 < z < 1$），$f_Z(z) = 2z$（$0 < z < 1$）
