# 概率论易错点汇总

---

## 一、概率计算易错点

### 1. 条件概率理解
- $P(B|A)$ 的样本空间已缩小为 $A$，不是原来的 $\Omega$
- $P(AB) \neq P(A) \cdot P(B)$，除非 $A, B$ 独立

### 2. 全概率公式使用
- 必须验证 $\{A_i\}$ 构成**完备事件组**（互斥且并集为 $\Omega$）
- 常见错误：忘记检验互斥性或遗漏事件

### 3. 独立性误区
- 两两独立 $\neq$ 相互独立
- $A, B$ 独立 $\implies$ $A$ 与 $\bar{B}$、$\bar{A}$ 与 $B$、$\bar{A}$ 与 $\bar{B}$ 也独立
- $P(AB) = 0$ 不能推出 $A, B$ 互斥

### 4. 对立事件 vs 互斥事件
- 互斥：$AB = \emptyset$
- 对立：$AB = \emptyset$ 且 $A \cup B = \Omega$
- 对立必互斥，互斥不一定对立

---

## 二、分布函数易错点

### 1. 分布函数性质
- $F(x)$ **右连续**，不是左连续
- $0 \le F(x) \le 1$，但 $F(x)$ 不是概率密度
- $P(X = a) = F(a) - F(a^-)$（连续型时为 0）

### 2. 密度函数性质
- $f(x) \ge 0$ 且 $\int_{-\infty}^{+\infty} f(x)dx = 1$
- $f(x)$ 可以 $> 1$（不像概率）
- $P(a < X \le b) = F(b) - F(a) = \int_a^b f(x)dx$

### 3. 正态分布
- $\Phi(-x) = 1 - \Phi(x)$，容易忘记
- $X \sim N(\mu, \sigma^2)$ 则 $\frac{X-\mu}{\sigma} \sim N(0,1)$
- $\Phi(0) = 0.5$，不是 0

---

## 三、多维随机变量易错点

### 1. 边缘分布不能确定联合分布
- 已知边缘分布 $f_X, f_Y$，不能确定 $f(x,y)$（除非独立）
- 相同的边缘分布可以对应不同的联合分布

### 2. 独立性判定
- $X, Y$ 独立 $\iff$ $f(x,y) = f_X(x) \cdot f_Y(y)$（**处处成立**）
- 仅在个别点成立不算独立
- 判定离散型：验证**所有** $(x_i, y_j)$ 组合

### 3. $Z = X+Y$ 的卷积公式
- 积分限容易出错：要结合 $f_X, f_Y$ 的支撑集确定
- 独立同分布的指数分布之和不再是指数分布（是 Erlang 分布）

### 4. $\max$ 和 $\min$
- $P(\max \le z) = P(X \le z)P(Y \le z)$（独立时）
- $P(\min > z) = P(X > z)P(Y > z)$
- 不要混淆 $P(\max \le z)$ 与 $P(\max > z)$

---

## 四、数字特征易错点

### 1. 期望不存在的情况
- 柯西分布的期望不存在（积分发散）
- 并非所有随机变量都有期望

### 2. 方差公式
- $D(X) = E(X^2) - [E(X)]^2$，不是 $E(X^2)$
- $D(X) \ge 0$，等号成立 $\iff$ $X$ 几乎处处为常数

### 3. 不相关 vs 独立
- 独立 $\implies$ 不相关（$\rho = 0$）
- 不相关 $\nRightarrow$ 独立（一般情况）
- **二维正态分布**：不相关 $\iff$ 独立（特例！）
- $E(XY) = E(X)E(Y)$ 只能推出不相关，不能推出独立

### 4. 协方差计算
- $\text{Cov}(X,X) = D(X)$
- $\text{Cov}(X,Y) = \text{Cov}(Y,X)$
- $D(X+Y) = D(X) + D(Y) + 2\text{Cov}(X,Y)$

---

## 五、数理统计易错点

### 1. 样本方差公式
- $S^2 = \frac{1}{n-1}\sum(X_i - \bar{X})^2$，分母是 $n-1$ 不是 $n$
- $E(S^2) = \sigma^2$（无偏性），$E(\frac{1}{n}\sum(X_i-\bar{X})^2) \neq \sigma^2$

### 2. 三大分布
- $\chi^2(n)$：$n$ 个独立标准正态变量的平方和
- $t(n)$：标准正态 $/\sqrt{\chi^2/n}$
- $F(m,n)$：$(\chi^2_1/m) / (\chi^2_2/n)$
- $t^2(n) \sim F(1,n)$

### 3. 最大似然估计
- 似然函数是**乘积**形式，取对数后变求和
- 若 $f(x;\theta)$ 关于 $\theta$ 单调，则 MLE 在参数边界取值
- MLE 具有不变性：若 $\hat{\theta}$ 是 MLE，则 $g(\hat{\theta})$ 是 $g(\theta)$ 的 MLE

### 4. 置信区间
- 置信水平 $1-\alpha$ 不是"参数落在区间内的概率为 $1-\alpha$"
- 正确理解：反复抽样，$100(1-\alpha)\%$ 的区间会包含真值

---

## 六、综合易错点

### 1. 泊松近似的条件
- $n$ 大、$p$ 小、$np$ 适中（一般 $n \ge 20, p \le 0.05$）
- 近似：$B(n,p) \approx P(np)$

### 2. 切比雪夫不等式
- 只给出**上界**，不给出精确概率
- 适用条件：期望和方差存在

### 3. 常见分布记忆混淆
- 指数分布的无记忆性：$P(X > s+t | X > s) = P(X > t)$
- 几何分布也有无记忆性
- 泊松过程的到达间隔服从指数分布
