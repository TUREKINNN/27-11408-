### 概率公式

| 公式 | 位置 |
|------|------|
| $P(A \cup B) = P(A) + P(B) - P(AB)$ | Lecture 01 |
| $P(B \mid A) = \frac{P(AB)}{P(A)}$ | Lecture 01 |
| $P(AB) = P(A)P(B)$（独立） | Lecture 01 |
| $P(A) = \sum_{i=1}^{n} P(B_i)P(A \mid B_i)$（全概率） | Lecture 01 |
| $P(B_i \mid A) = \frac{P(B_i)P(A \mid B_i)}{\sum_{j} P(B_j)P(A \mid B_j)}$（贝叶斯） | Lecture 01 |

|------|------|
| $F(x) = P\{X \leq x\}$ | Lecture 02 |

| $P\{X = k\} = \frac{\lambda^k e^{-\lambda}}{k!}$（泊松） | Lecture 02 |
| $f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$（正态） | Lecture 02 |
