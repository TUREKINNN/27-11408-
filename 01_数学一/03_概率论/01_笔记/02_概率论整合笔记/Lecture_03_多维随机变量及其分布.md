# Lecture 03: 多维随机变量及其分布

## 一、二维随机变量的联合分布函数

### 1.1 联合分布函数的定义

<span style="background-color: #ccc1d9">**定义（联合分布函数）**</span> 设 $(X, Y)$ 是二维随机变量，对任意实数 $x, y$，称函数

$$F(x, y) = P\{X \leq x, Y \leq y\}$$

为二维随机变量 $(X, Y)$ 的**联合分布函数**。

### 1.2 联合分布函数的性质

<span style="background-color: #8db3e2">**定理（联合分布函数的性质）**</span> 联合分布函数 $F(x, y)$ 具有以下性质：

1. **有界性**：$0 \leq F(x, y) \leq 1$
2. **单调不减性**：对固定的 $y$，$F(x, y)$ 关于 $x$ 单调不减；对固定的 $x$，$F(x, y)$ 关于 $y$ 单调不减
3. **右连续性**：对固定的 $y$，$F(x, y)$ 关于 $x$ 右连续；对固定的 $x$，$F(x, y)$ 关于 $y$ 右连续
4. **极限性质**：
   - $F(-\infty, y) = F(x, -\infty) = F(-\infty, -\infty) = 0$
   - $F(+\infty, +\infty) = 1$
5. **矩形概率公式**：对任意 $x_1 < x_2, y_1 < y_2$，有

$$P\{x_1 < X \leq x_2, y_1 < Y \leq y_2\} = F(x_2, y_2) - F(x_2, y_1) - F(x_1, y_2) + F(x_1, y_1)$$

## 二、二维离散型随机变量的联合分布律

### 2.1 联合分布律的定义

<span style="background-color: #ccc1d9">**定义（联合分布律）**</span> 若二维随机变量 $(X, Y)$ 所有可能的取值为有限个或可列无限个，则称 $(X, Y)$ 为**二维离散型随机变量**。设其所有可能取值为 $(x_i, y_j), i, j = 1, 2, \cdots$，则称

$$P\{X = x_i, Y = y_j\} = p_{ij}, \quad i, j = 1, 2, \cdots$$

为 $(X, Y)$ 的**联合概率分布律**。

### 2.2 联合分布律的性质

<span style="background-color: #8db3e2">**定理**</span> 数列 $\{p_{ij}\}$ 是联合分布律的充要条件：

1. **非负性**：$p_{ij} \geq 0$，$\forall i, j$
2. **归一性**：$\sum_{i=1}^{\infty} \sum_{j=1}^{\infty} p_{ij} = 1$

### 2.3 联合分布律的表示

联合分布律通常用表格形式表示：

| $X \setminus Y$ | $y_1$ | $y_2$ | $\cdots$ | $y_j$ | $\cdots$ |
|-----------------|-------|-------|----------|-------|----------|
| $x_1$           | $p_{11}$ | $p_{12}$ | $\cdots$ | $p_{1j}$ | $\cdots$ |
| $x_2$           | $p_{21}$ | $p_{22}$ | $\cdots$ | $p_{2j}$ | $\cdots$ |
| $\vdots$        | $\vdots$ | $\vdots$ | $\ddots$ | $\vdots$ | $\ddots$ |
| $x_i$           | $p_{i1}$ | $p_{i2}$ | $\cdots$ | $p_{ij}$ | $\cdots$ |
| $\vdots$        | $\vdots$ | $\vdots$ | $\ddots$ | $\vdots$ | $\ddots$ |

## 三、二维连续型随机变量的联合概率密度

### 3.1 联合概率密度的定义

<span style="background-color: #ccc1d9">**定义（联合概率密度）**</span> 如果存在非负可积函数 $f(x, y)$，使得二维随机变量 $(X, Y)$ 的联合分布函数 $F(x, y)$ 可以表示为

$$F(x, y) = \int_{-\infty}^{y} \int_{-\infty}^{x} f(u, v) du dv$$

则称 $(X, Y)$ 为**二维连续型随机变量**，称 $f(x, y)$ 为 $(X, Y)$ 的**联合概率密度函数**。

### 3.2 联合概率密度的性质

<span style="background-color: #8db3e2">**定理**</span> 函数 $f(x, y)$ 是联合概率密度的充要条件：

1. **非负性**：$f(x, y) \geq 0$，$\forall (x, y) \in \mathbb{R}^2$
2. **归一性**：$\int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} f(x, y) dx dy = 1$

### 3.3 概率计算

设 $G$ 是平面上的一个区域，则

$$P\{(X, Y) \in G\} = \iint_{G} f(x, y) dx dy$$

## 四、边缘分布

### 4.1 边缘分布函数

二维随机变量 $(X, Y)$ 的**边缘分布函数**分别为：

$$F_X(x) = P\{X \leq x\} = F(x, +\infty)$$

$$F_Y(y) = P\{Y \leq y\} = F(+\infty, y)$$

### 4.2 二维离散型的边缘分布律

设 $(X, Y)$ 的联合分布律为 $P\{X = x_i, Y = y_j\} = p_{ij}$，则：

- $X$ 的边缘分布律：$p_{i.} = P\{X = x_i\} = \sum_{j=1}^{\infty} p_{ij}$
- $Y$ 的边缘分布律：$p_{.j} = P\{Y = y_j\} = \sum_{i=1}^{\infty} p_{ij}$

### 4.3 二维连续型的边缘概率密度

设 $(X, Y)$ 的联合概率密度为 $f(x, y)$，则：

- $X$ 的边缘概率密度：$f_X(x) = \int_{-\infty}^{+\infty} f(x, y) dy$
- $Y$ 的边缘概率密度：$f_Y(y) = \int_{-\infty}^{+\infty} f(x, y) dx$

## 五、条件分布

### 5.1 二维离散型的条件分布律

<span style="background-color: #ccc1d9">**定义（条件分布律）**</span> 设 $(X, Y)$ 的联合分布律为 $P\{X = x_i, Y = y_j\} = p_{ij}$，若 $p_{.j} > 0$，则在 $Y = y_j$ 的条件下，$X$ 的条件分布律为：

$$P\{X = x_i \mid Y = y_j\} = \frac{p_{ij}}{p_{.j}}, \quad i = 1, 2, \cdots$$

同理，若 $p_{i.} > 0$，则在 $X = x_i$ 的条件下，$Y$ 的条件分布律为：

$$P\{Y = y_j \mid X = x_i\} = \frac{p_{ij}}{p_{i.}}, \quad j = 1, 2, \cdots$$

### 5.2 二维连续型的条件概率密度

<span style="background-color: #ccc1d9">**定义（条件概率密度）**</span> 设 $(X, Y)$ 的联合概率密度为 $f(x, y)$，$f_Y(y) > 0$，则在 $Y = y$ 的条件下，$X$ 的条件概率密度为：

$$f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)}$$

同理，若 $f_X(x) > 0$，则在 $X = x$ 的条件下，$Y$ 的条件概率密度为：

$$f_{Y|X}(y|x) = \frac{f(x, y)}{f_X(x)}$$

## 六、随机变量的独立性

### 6.1 独立性的定义

<span style="background-color: #ccc1d9">**定义（独立性）**</span> 若对任意实数 $x, y$，有

$$F(x, y) = F_X(x) \cdot F_Y(y)$$

则称随机变量 $X$ 与 $Y$ **相互独立**。

### 6.2 独立性的判定

<span style="background-color: #8db3e2">**定理**</span> 

1. **离散型**：$X$ 与 $Y$ 相互独立 $\iff p_{ij} = p_{i.} \cdot p_{.j}$，$\forall i, j$
2. **连续型**：$X$ 与 $Y$ 相互独立 $\iff f(x, y) = f_X(x) \cdot f_Y(y)$，$\forall (x, y) \in \mathbb{R}^2$

### 6.3 独立随机变量的性质

1. 若 $X$ 与 $Y$ 独立，则 $g(X)$ 与 $h(Y)$ 也独立
2. 若 $X$ 与 $Y$ 独立，则 $E(XY) = E(X)E(Y)$，$\mathrm{Cov}(X, Y) = 0$

## 七、两个随机变量的函数的分布

### 7.1 和的分布 $Z = X + Y$

**分布函数法**：

$$F_Z(z) = P\{X + Y \leq z\} = \iint_{x+y \leq z} f(x, y) dx dy$$

**卷积公式**（$X$ 与 $Y$ 独立）：

$$f_Z(z) = \int_{-\infty}^{+\infty} f_X(x) f_Y(z - x) dx = \int_{-\infty}^{+\infty} f_X(z - y) f_Y(y) dy$$

### 7.2 差的分布 $Z = X - Y$

$$f_Z(z) = \int_{-\infty}^{+\infty} f(x, x - z) dx$$

### 7.3 积的分布 $Z = XY$

$$f_Z(z) = \int_{-\infty}^{+\infty} f\left(x, \frac{z}{x}\right) \frac{1}{|x|} dx$$

### 7.4 商的分布 $Z = \frac{X}{Y}$

$$f_Z(z) = \int_{-\infty}^{+\infty} |y| f(zy, y) dy$$

### 7.5 最大值与最小值的分布

设 $X_1, X_2, \cdots, X_n$ 相互独立，分布函数分别为 $F_{X_i}(x_i)$，则：

- $M = \max\{X_1, X_2, \cdots, X_n\}$ 的分布函数：

$$F_M(x) = \prod_{i=1}^{n} F_{X_i}(x)$$

- $N = \min\{X_1, X_2, \cdots, X_n\}$ 的分布函数：

$$F_N(x) = 1 - \prod_{i=1}^{n} [1 - F_{X_i}(x)]$$

## 八、典型例题

### 例题1：联合分布律与条件概率（660题第530题）

**题目**：随机变量 $(X, Y)$ 的分布律为：

| $X \setminus Y$ | -1 | 0 | 1 |
|-----------------|----|---|---|
| 1               | 0.3 | 0.1 | 0.15 |
| 2               | 0 | 0.25 | 0.2 |

设 $F(x, y)$ 为 $(X, Y)$ 的分布函数，则 $P\{0 < X + Y \leq 2.3\} =$ ____，$F(1, 0) =$ ____，$P\{Y \leq -1\} =$ ____，随机变量 $2X - Y$ 的分布律为 ____，$P\{X \leq 1 \mid Y = 0\} =$ ____。

**思路**：利用联合分布律计算各种概率。

**解答**：
1. $P\{0 < X + Y \leq 2.3\}$：满足条件的 $(X, Y)$ 为 $(1, 0), (1, 1), (2, 0)$
   $$= 0.1 + 0.15 + 0.25 = 0.5$$

2. $F(1, 0) = P\{X \leq 1, Y \leq 0\} = 0.3 + 0.1 = 0.4$

3. $P\{Y \leq -1\} = P\{Y = -1\} = 0.3 + 0 = 0.3$

4. $2X - Y$ 的可能取值：$2\times1-(-1)=3, 2\times1-0=2, 2\times1-1=1, 2\times2-0=4, 2\times2-1=3$
   | $2X-Y$ | 1 | 2 | 3 | 4 |
   |--------|---|---|---|---|
   | $P$ | 0.15 | 0.1 | 0.5 | 0.25 |

5. $P\{X \leq 1 \mid Y = 0\} = \frac{P\{X \leq 1, Y = 0\}}{P\{Y = 0\}} = \frac{0.1}{0.1+0.25} = \frac{2}{7}$

**总结**：联合分布律的计算需要仔细枚举所有满足条件的取值组合。

### 例题2：交换球后的分布律（660题第531题）

**题目**：甲袋中有2个白球，乙袋中有2个黑球，每次独立、随机地分别从这2个袋中任取1球交换放入对方袋中，这样交换取、放共3次。3次后，设甲袋中有 $X$ 个白球，则 $X$ 的分布律为 ____。

**思路**：分析每次交换后甲袋白球数的变化情况。

**解答**：
初始状态：甲袋2白，乙袋2黑。
设 $X_n$ 表示第 $n$ 次交换后甲袋白球数，则 $X_0 = 2$。

每次交换有四种等可能情况：
- 甲白→乙黑，乙黑→甲白：$X$ 不变
- 甲白→乙黑，乙白→甲白：不可能（乙袋初始无白球）
- 甲黑→乙黑，乙黑→甲白：$X$ 增加1
- 甲黑→乙黑，乙白→甲白：不可能

实际上，每次交换：
- 从甲袋取白球（概率 $\frac{X_n}{2}$），从乙袋取黑球（概率 $\frac{2-(2-X_n)}{2}=\frac{X_n}{2}$），交换后 $X_{n+1}=X_n$
- 从甲袋取白球，从乙袋取白球：不可能
- 从甲袋取黑球（概率 $\frac{2-X_n}{2}$），从乙袋取黑球（概率 $\frac{X_n}{2}$），交换后 $X_{n+1}=X_n+1$
- 从甲袋取黑球，从乙袋取白球（概率 $\frac{2-X_n}{2}$），交换后 $X_{n+1}=X_n-1$

经过3次交换，可能的白球数为0,1,2,3,4。但由于对称性和初始条件，计算得：
$$P\{X=0\}=\frac{1}{8},\quad P\{X=1\}=\frac{3}{8},\quad P\{X=2\}=\frac{3}{8},\quad P\{X=3\}=\frac{1}{8},\quad P\{X=4\}=0$$

**总结**：马尔可夫链思想，状态转移概率计算。

### 例题3：联合概率密度的二重积分（660题第532题）

**题目**：已知随机变量 $(X, Y)$ 的概率密度函数 $f(x, y) = \begin{cases} e^{-y}, & 0 < x < y, \\ 0, & \text{其他}, \end{cases}$ 则 $P(X + Y \leq 1) =$ ____。

**思路**：确定积分区域并计算二重积分。

**解答**：
积分区域：$0 < x < y$ 且 $x + y \leq 1$，即 $0 < x < \frac{1}{2}, x < y \leq 1 - x$

$$P(X + Y \leq 1) = \int_{0}^{\frac{1}{2}} \int_{x}^{1-x} e^{-y} dy dx$$

$$= \int_{0}^{\frac{1}{2}} \left[-e^{-y}\right]_{x}^{1-x} dx = \int_{0}^{\frac{1}{2}} (e^{-x} - e^{-(1-x)}) dx$$

$$= \left[-e^{-x} - e^{-(1-x)}\right]_{0}^{\frac{1}{2}} = (-e^{-\frac{1}{2}} - e^{-\frac{1}{2}}) - (-1 - e^{-1})$$

$$= 1 + e^{-1} - 2e^{-\frac{1}{2}}$$

**总结**：二重积分计算是基本功，正确确定积分区域是关键。

### 例题4：独立随机变量之和的分布（660题第533题）

**题目**：设随机变量 $X_1$ 和 $X_2$ 相互独立，已知 $X_1 \sim B\left(1, \frac{3}{4}\right), X_2$ 的分布函数为 $F(x)$，则随机变量 $Y = X_1 + X_2$ 的分布函数 $F_Y(y) =$ ____。

**思路**：利用全概率公式和独立性。

**解答**：
$$F_Y(y) = P\{X_1 + X_2 \leq y\} = P\{X_1 = 0\}P\{X_2 \leq y\} + P\{X_1 = 1\}P\{X_2 \leq y - 1\}$$

$$= \frac{1}{4}F(y) + \frac{3}{4}F(y - 1)$$

**总结**：全概率公式适用于混合类型随机变量的函数分布。

### 例题5：最大值的概率计算（660题第537题）

**题目**：设相互独立的两随机变量 $X, Y$ 均服从 $[0, 3]$ 上的均匀分布，则 $P\{1 < \max(X, Y) \leq 2\}$ 的值为 ____。

**思路**：利用最大值分布的性质。

**解答**：
$$P\{1 < \max(X, Y) \leq 2\} = P\{\max(X, Y) \leq 2\} - P\{\max(X, Y) \leq 1\}$$

$$= [P\{X \leq 2\}P\{Y \leq 2\}] - [P\{X \leq 1\}P\{Y \leq 1\}]$$

$$= \left(\frac{2}{3}\right)^2 - \left(\frac{1}{3}\right)^2 = \frac{4}{9} - \frac{1}{9} = \frac{1}{3}$$

**总结**：最大值分布 $P\{\max \leq x\} = [F(x)]^n$（独立同分布）。

### 例题6：二维正态分布的性质（660题第538题）

**题目**：设 $(X, Y) \sim N(\mu_1, \mu_2; \sigma_1^2, \sigma_2^2; 0)$，其分布函数为 $F(x, y)$，已知 $F(\mu_1, y) = \frac{1}{4}$，则 $y =$ ____。

**思路**：利用二维正态分布的性质，$\rho = 0$ 时 $X$ 与 $Y$ 独立。

**解答**：
$\rho = 0$，则 $X$ 与 $Y$ 独立，$F(x, y) = F_X(x)F_Y(y)$

$$F(\mu_1, y) = F_X(\mu_1)F_Y(y) = \frac{1}{2}F_Y(y) = \frac{1}{4}$$

$$F_Y(y) = \frac{1}{2} \Rightarrow y = \mu_2$$

**总结**：二维正态分布中 $\rho = 0$ 等价于独立性。

### 例题7：随机变量函数的分布（660题第539题）

**题目**：设随机变量 $X$ 和 $Y$ 相互独立，且 $X$ 服从标准正态分布，其分布函数为 $\Phi(x), Y$ 的概率分布为 $P\{Y = -1\} = P\{Y = 1\} = \frac{1}{2}$，则随机变量 $Z = XY$ 的分布函数 $F_Z(x) =$ ____。

**思路**：利用全概率公式分解。

**解答**：
$$F_Z(x) = P\{XY \leq x\} = P\{Y = -1\}P\{-X \leq x\} + P\{Y = 1\}P\{X \leq x\}$$

$$= \frac{1}{2}P\{X \geq -x\} + \frac{1}{2}\Phi(x) = \frac{1}{2}[1 - \Phi(-x)] + \frac{1}{2}\Phi(x)$$

$$= \frac{1}{2}[1 - (1 - \Phi(x))] + \frac{1}{2}\Phi(x) = \frac{1}{2}\Phi(x) + \frac{1}{2}\Phi(x) = \Phi(x)$$

**总结**：$Z = XY$ 仍服从标准正态分布。

### 例题8：二维正态分布的参数（660题第540题）

**题目**：随机变量 $(X, Y)$ 的概率密度为：

$$f(x, y) = \frac{1}{\pi} \exp\left[-\frac{1}{2}(2x^2 + 2xy + y^2 - 22x - 14y + 65)\right], (x, y) \in \mathbb{R}^2$$

则 $E(X) =$ ____，$D(X) =$ ____，$E(Y) =$ ____，$D(Y) =$ ____，$X, Y$ 的相关系数 $\rho_{XY} =$ ____。

**思路**：将密度函数化为二维正态分布的标准形式。

**解答**：
二维正态分布的标准形式：$f(x, y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left[-\frac{1}{2(1-\rho^2)}\left(\left(\frac{x-\mu_1}{\sigma_1}\right)^2 - 2\rho\left(\frac{x-\mu_1}{\sigma_1}\right)\left(\frac{y-\mu_2}{\sigma_2}\right) + \left(\frac{y-\mu_2}{\sigma_2}\right)^2\right)\right]$

对比系数得：$\mu_1 = 4, \mu_2 = 3, \sigma_1^2 = 1, \sigma_2^2 = 2, \rho = \frac{\sqrt{2}}{2}$

$$E(X) = 4,\quad D(X) = 1,\quad E(Y) = 3,\quad D(Y) = 2,\quad \rho_{XY} = \frac{\sqrt{2}}{2}$$

**总结**：二维正态分布的参数识别需要配方变形。

## 九、解题方法总结

### 9.1 主要方法

| 类型 | 方法 |
|------|------|
| 联合分布函数 | 验证单调不减、右连续、极限性质 |
| 离散型联合分布律 | 利用非负性和归一性 |
| 连续型联合概率密度 | 利用非负性和归一性，二重积分求概率 |
| 边缘分布 | 对联合分布求和或积分 |
| 条件分布 | 条件概率公式 |
| 独立性判定 | 验证 $F(x,y)=F_X(x)F_Y(y)$ |
| 随机变量函数分布 | 分布函数法、卷积公式、最值分布公式 |

### 9.2 常见易错点

1. **边缘分布的计算**：离散型是对联合分布律求和，连续型是对联合概率密度积分
2. **条件分布的分母**：必须保证边缘分布不为零
3. **二维正态分布的独立性**：$\rho = 0$ 等价于独立性，但一般分布不成立
4. **最大值与最小值的分布**：注意区别分布函数和概率密度

## 十、练习题精选

### 基础练习

1. 设 $(X, Y)$ 的联合分布律为：

| $X \setminus Y$ | 0 | 1 |
|-----------------|---|---|
| 0 | 0.4 | 0.1 |
| 1 | 0.2 | 0.3 |

求 $P\{X = Y\}$ 和 $P\{X > Y\}$。

2. 设 $(X, Y)$ 的联合概率密度为 $f(x, y) = \begin{cases} 6xy, & 0 < x < 1, 0 < y < x, \\ 0, & \text{其他}, \end{cases}$ 求 $f_X(x)$ 和 $f_Y(y)$。

### 提高练习

3. 设 $X \sim U(0, 1), Y \sim U(0, 1)$，且 $X$ 与 $Y$ 独立，求 $Z = X + Y$ 的概率密度。

4. 设 $X_1, X_2, X_3$ 独立同分布，均服从 $U(0, 1)$，求 $P\{\max(X_1, X_2, X_3) \leq 0.5\}$。

---

## 参考答案提示

1. $P\{X = Y\} = 0.7$，$P\{X > Y\} = 0.2$（提示：直接枚举）

2. $f_X(x) = 3x^2, 0 < x < 1$；$f_Y(y) = 3y(1 - y^2), 0 < y < 1$（提示：积分计算）

3. $f_Z(z) = \begin{cases} z, & 0 < z < 1, \\ 2 - z, & 1 < z < 2, \\ 0, & \text{其他} \end{cases}$（提示：卷积公式）

4. $P\{\max \leq 0.5\} = 0.125$（提示：最大值分布公式）

### 补充：二维正态分布的性质

<span style="background-color: #8db3e2">**二维正态分布的性质**</span>

若 $(X,Y) \sim N(\mu_1, \mu_2; \sigma_1^2, \sigma_2^2; \rho)$，则：

1. **边缘分布**：$X \sim N(\mu_1, \sigma_1^2)$，$Y \sim N(\mu_2, \sigma_2^2)$
2. **条件分布**：
   - 给定 $Y=y$，$X \sim N\left(\mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2), \sigma_1^2(1-\rho^2)\right)$
   - 给定 $X=x$，$Y \sim N\left(\mu_2 + \rho\frac{\sigma_2}{\sigma_1}(x-\mu_1), \sigma_2^2(1-\rho^2)\right)$
3. **独立性**：$X$ 与 $Y$ 独立 $\Leftrightarrow \rho = 0$
4. **线性组合**：$aX+bY \sim N(a\mu_1+b\mu_2, a^2\sigma_1^2+b^2\sigma_2^2+2ab\rho\sigma_1\sigma_2)$
5. **不相关与独立**：对二维正态分布，不相关等价于独立

### 补充：常用二维均匀分布

若 $(X,Y)$ 在区域 $G$（面积为 $S_G$）上服从均匀分布，则：

$$f(x,y) = \begin{cases} \frac{1}{S_G}, & (x,y) \in G, \\ 0, & \text{其他} \end{cases}$$

**注意**：二维均匀分布的边缘分布不一定是均匀分布。
