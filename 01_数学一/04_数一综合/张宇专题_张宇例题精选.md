# 张宇例题精选

## 第一章 函数极限连续（张宇零基础）

### 例1
**题目**：求 $\lim_{x \to 0} \frac{\sin 3x}{x}$

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★ |
| 来源 | 张宇零基础通关讲义-第一章 |
| 章节 | 函数极限连续 |
| 知识点 | 重要极限 |

**解析**：
$$\lim_{x \to 0} \frac{\sin 3x}{x} = \lim_{x \to 0} 3 \cdot \frac{\sin 3x}{3x} = 3$$

---

### 例2
**题目**：求 $\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^{2x}$

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★★ |
| 来源 | 张宇零基础通关讲义-第一章 |
| 章节 | 函数极限连续 |
| 知识点 | 第二重要极限 |

**解析**：
$$\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^{2x} = \lim_{x \to \infty} \left[\left(1 + \frac{1}{x}\right)^x\right]^2 = e^2$$

---

### 例3
**题目**：已知 $\lim_{x \to 0} \frac{f(x)}{x^2} = 1$，且 $f(x)$ 连续，求 $f(0)$

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★★★ |
| 来源 | 张宇零基础通关讲义-第一章 |
| 章节 | 函数极限连续 |
| 知识点 | 无穷小阶 |

**解析**：
由 $\lim_{x \to 0} \frac{f(x)}{x^2} = 1$ 知，当 $x \to 0$ 时，$f(x) \sim x^2$

由连续性，$f(0) = \lim_{x \to 0} f(x) = \lim_{x \to 0} x^2 = 0$

## 第二章 导数与微分（张宇零基础）

### 例4
**题目**：设 $y = e^{x^2}$，求 $y'$

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★★ |
| 来源 | 张宇零基础通关讲义-第二章 |
| 章节 | 导数与微分 |
| 知识点 | 复合函数求导 |

**解析**：
$$y' = e^{x^2} \cdot 2x = 2x e^{x^2}$$

---

### 例5
**题目**：设 $f(x) = x|x|$，讨论 $f(x)$ 的可导性

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★★★ |
| 来源 | 张宇零基础通关讲义-第二章 |
| 章节 | 导数与微分 |
| 知识点 | 绝对值函数可导性 |

**解析**：
$f(x) = x|x| = \begin{cases} x^2 & x \geq 0 \\ -x^2 & x < 0 \end{cases}$

$f'_+(0) = \lim_{x \to 0^+} \frac{x^2 - 0}{x} = 0$

$f'_-(0) = \lim_{x \to 0^-} \frac{-x^2 - 0}{x} = 0$

所以 $f(x)$ 在 $x = 0$ 处可导，且 $f'(0) = 0$

---

## 第三章 微分中值定理（张宇零基础）

### 例6
**题目**：证明 $\arctan x + \text{arccot } x = \frac{\pi}{2}$

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★★★ |
| 来源 | 张宇零基础通关讲义-第三章 |
| 章节 | 微分中值定理 |
| 知识点 | 拉格朗日中值定理 |

**解析**：
设 $f(x) = \arctan x + \text{arccot } x$，则 $f'(x) = \frac{1}{1+x^2} - \frac{1}{1+x^2} = 0$

所以 $f(x) = C$，取 $x = 0$，得 $f(0) = \frac{\pi}{2}$，故 $C = \frac{\pi}{2}$

---

### 例7
**题目**：用泰勒公式证明 $\cos x \leq 1 - \frac{x^2}{2} + \frac{x^4}{24}$

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★★★ |
| 来源 | 张宇零基础通关讲义-第三章 |
| 章节 | 微分中值定理 |
| 知识点 | 泰勒公式 |

**解析**：
$\cos x = 1 - \frac{x^2}{2} + \frac{x^4}{24} - \frac{\cos \xi}{5!} x^5$，其中 $\xi \in (0, x)$

当 $x > 0$ 时，$-\frac{\cos \xi}{120} x^5 \leq 0$，所以 $\cos x \leq 1 - \frac{x^2}{2} + \frac{x^4}{24}$

## 第四章 不定积分（张宇零基础）

### 例8
**题目**：求 $\int \tan x dx$

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★★ |
| 来源 | 张宇零基础通关讲义-第四章 |
| 章节 | 不定积分 |
| 知识点 | 积分公式 |

**解析**：
$$\int \tan x dx = \int \frac{\sin x}{\cos x} dx = -\ln|\cos x| + C$$

---

### 例9
**题目**：求 $\int \sec x dx$

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★★★ |
| 来源 | 张宇零基础通关讲义-第四章 |
| 章节 | 不定积分 |
| 知识点 | 积分公式 |

**解析**：
$$\int \sec x dx = \int \sec x \cdot \frac{\sec x + \tan x}{\sec x + \tan x} dx = \ln|\sec x + \tan x| + C$$

---

## 第五章 定积分（张宇零基础）

### 例10
**题目**：设 $f(x) = \int_0^x \sin t^2 dt$，求 $\frac{df}{dx}$

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★★ |
| 来源 | 张宇零基础通关讲义-第五章 |
| 章节 | 定积分 |
| 知识点 | 变上限积分 |

**解析**：
由变上限积分求导公式：$\frac{df}{dx} = \sin x^2$

---

### 例11
**题目**：求 $\int_0^1 x^2 \sqrt{1-x^2} dx$

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★★★ |
| 来源 | 张宇零基础通关讲义-第五章 |
| 章节 | 定积分 |
| 知识点 | 换元积分法 |

**解析**：
令 $x = \sin t$，则 $dx = \cos t dt$，当 $x = 0$ 时 $t = 0$；当 $x = 1$ 时 $t = \frac{\pi}{2}$

$$\int_0^1 x^2 \sqrt{1-x^2} dx = \int_0^{\frac{\pi}{2}} \sin^2 t \cdot \cos^2 t dt = \frac{\pi}{16}$$

## 第六章 微分方程（张宇零基础）

### 例12
**题目**：求 $\frac{dy}{dx} = y$ 的通解

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★★ |
| 来源 | 张宇零基础通关讲义-第六章 |
| 章节 | 微分方程 |
| 知识点 | 可分离变量方程 |

**解析**：
$$\frac{dy}{y} = dx \Rightarrow \ln|y| = x + C \Rightarrow y = Ce^x$$

## 第七章 多元函数微分学（张宇零基础）

### 例14
**题目**：设 $z = x^y$，求 $\frac{\partial z}{\partial x}$ 和 $\frac{\partial z}{\partial y}$

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★★★ |
| 来源 | 张宇零基础通关讲义-第七章 |
| 章节 | 多元函数微分学 |
| 知识点 | 偏导数计算 |

**解析**：
$$\frac{\partial z}{\partial x} = y x^{y-1}$$

$$\frac{\partial z}{\partial y} = x^y \ln x$$

---

### 例15
**题目**：设 $z = f(xy, x^2 + y^2)$，其中 $f$ 可微，求 $\frac{\partial z}{\partial x}$

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★★★★ |
| 来源 | 张宇零基础通关讲义-第七章 |
| 章节 | 多元函数微分学 |
| 知识点 | 多元复合函数偏导数 |

**解析**：
令 $u = xy, v = x^2 + y^2$，则
$$\frac{\partial z}{\partial x} = \frac{\partial f}{\partial u} \cdot \frac{\partial u}{\partial x} + \frac{\partial f}{\partial v} \cdot \frac{\partial v}{\partial x} = y f'_u + 2x f'_v$$

---

## 第八章 二重积分（张宇零基础）

### 例16
**题目**：计算 $\iint_D x y d\sigma$，其中 $D: 0 \leq x \leq 1, 0 \leq y \leq 1$

| 属性 | 值 |
|------|-----|
| 题型 | 例题 |
| 难度 | ★★ |
| 来源 | 张宇零基础通关讲义-第八章 |
| 章节 | 二重积分 |
| 知识点 | 二重积分计算 |

**解析**：
$$\iint_D x y d\sigma = \int_0^1 x dx \int_0^1 y dy = \frac{1}{2} \cdot \frac{1}{2} = \frac{1}{4}$$
