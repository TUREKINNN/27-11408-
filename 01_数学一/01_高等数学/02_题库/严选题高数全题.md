# 严选题高数全题

## 第一章 函数极限连续

### 第1题
**题目**：求极限 $\lim_{x \to 0} \frac{\sin x - x}{x^3}$

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★ |
| 来源 | 严选题-高数篇-第1题 |
| 章节 | 第一章 函数极限连续 |
| 知识点 | 洛必达法则 |

**参考答案**：
使用洛必达法则三次：
$$\lim_{x \to 0} \frac{\sin x - x}{x^3} = \lim_{x \to 0} \frac{\cos x - 1}{3x^2} = \lim_{x \to 0} \frac{-\sin x}{6x} = \lim_{x \to 0} \frac{-\cos x}{6} = -\frac{1}{6}$$

---

### 第2题
**题目**：求极限 $\lim_{x \to \infty} \left(1 + \frac{2}{x}\right)^{x+3}$

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★ |
| 来源 | 严选题-高数篇-第2题 |
| 章节 | 第一章 函数极限连续 |
| 知识点 | 第二重要极限 |

**参考答案**：
$$\lim_{x \to \infty} \left(1 + \frac{2}{x}\right)^{x+3} = \lim_{x \to \infty} \left(1 + \frac{2}{x}\right)^x \cdot \left(1 + \frac{2}{x}\right)^3 = e^2 \cdot 1 = e^2$$

---

### 第3题
**题目**：已知 $\lim_{x \to 0} \frac{\sin 6x + xf(x)}{x^3} = 0$，求 $f(x)$ 使其满足 $\lim_{x \to 0} \frac{6 + f(x)}{x^2}$ 存在

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★ |
| 来源 | 严选题-高数篇-第3题 |
| 章节 | 第一章 函数极限连续 |
| 知识点 | 泰勒公式 |

**参考答案**：
由 $\sin 6x = 6x - \frac{(6x)^3}{3!} + o(x^3) = 6x - 36x^3 + o(x^3)$

要使 $\lim_{x \to 0} \frac{\sin 6x + xf(x)}{x^3} = 0$，即 $\lim_{x \to 0} \frac{6x - 36x^3 + xf(x) + o(x^3)}{x^3} = 0$

所以 $6 + f(x) = 36x^2 + o(x^2)$，即 $f(x) = 36x^2 + o(x^2) - 6$

### 第4题
**题目**：设函数 $f(x) = \begin{cases} \frac{\sin x}{x} & x \neq 0 \\ 1 & x = 0 \end{cases}$，讨论 $f(x)$ 的连续性

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★ |
| 来源 | 严选题-高数篇-第4题 |
| 章节 | 第一章 函数极限连续 |
| 知识点 | 连续性判定 |

**参考答案**：
当 $x \neq 0$ 时，$f(x) = \frac{\sin x}{x}$ 连续
当 $x = 0$ 时，$\lim_{x \to 0} \frac{\sin x}{x} = 1 = f(0)$，所以 $f(x)$ 在 $x = 0$ 处连续
因此 $f(x)$ 在 $\mathbb{R}$ 上连续

### 第5题
**题目**：求极限 $\lim_{x \to 0} \frac{\tan x - \sin x}{x^3}$

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★ |
| 来源 | 严选题-高数篇-第5题 |
| 章节 | 第一章 函数极限连续 |
| 知识点 | 等价无穷小替换 |

**参考答案**：
$\tan x - \sin x = \sin x(\frac{1}{\cos x} - 1) = \sin x \frac{1 - \cos x}{\cos x} \sim x \cdot \frac{x^2}{2} = \frac{x^3}{2}$

所以 $\lim_{x \to 0} \frac{\tan x - \sin x}{x^3} = \frac{1}{2}$

## 第二章 导数与微分

### 第6题
**题目**：设 $y = e^{\sin x} \cdot \cos(\ln x)$，求 $y'$

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★ |
| 来源 | 严选题-高数篇-第6题 |
| 章节 | 第二章 导数与微分 |
| 知识点 | 复合函数求导 |

**参考答案**：
$$y' = e^{\sin x} \cos x \cdot \cos(\ln x) + e^{\sin x} \cdot [-\sin(\ln x)] \cdot \frac{1}{x}$$
$$= e^{\sin x} \left[\cos x \cos(\ln x) - \frac{\sin(\ln x)}{x}\right]$$

---

### 第7题
**题目**：设 $f(x) = x(x-1)(x-2)...(x-100)$，求 $f'(0)$

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★ |
| 来源 | 严选题-高数篇-第7题 |
| 章节 | 第二章 导数与微分 |
| 知识点 | 导数定义与乘积法则 |

**参考答案**：
由导数定义 $f'(0) = \lim_{x \to 0} \frac{f(x) - f(0)}{x} = \lim_{x \to 0} \frac{x(x-1)(x-2)...(x-100)}{x}$

当 $x \to 0$ 时，$f'(0) = (-1)(-2)...(-100) = 100!$

---

### 第8题
**题目**：设 $y = \frac{x^2}{1-x}$，求 $y^{(n)}$

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★★ |
| 来源 | 严选题-高数篇-第8题 |
| 章节 | 第二章 导数与微分 |
| 知识点 | 高阶导数 |

**参考答案**：
$y = \frac{x^2}{1-x} = -x - 1 + \frac{1}{1-x}$

由 $(x^n)^{(n)} = n!$，$(1-x)^{-1}$ 的 $n$ 阶导数为 $n!(1-x)^{-1-n}$

所以 $y^{(n)} = n!(1-x)^{-1-n}$（$n \geq 2$）

---

### 第9题
**题目**：设 $f(x) = \begin{cases} x^2 \sin \frac{1}{x} & x \neq 0 \\ 0 & x = 0 \end{cases}$，讨论 $f(x)$ 在 $x = 0$ 处的可导性

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★ |
| 来源 | 严选题-高数篇-第9题 |
| 章节 | 第二章 导数与微分 |
| 知识点 | 导数定义 |

**参考答案**：
$f'(0) = \lim_{x \to 0} \frac{x^2 \sin \frac{1}{x} - 0}{x} = \lim_{x \to 0} x \sin \frac{1}{x} = 0$

所以 $f(x)$ 在 $x = 0$ 处可导，且 $f'(0) = 0$

---

### 第10题
**题目**：设 $y = \arctan x$，求 $y^{(n)}(0)$

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★★ |
| 来源 | 严选题-高数篇-第10题 |
| 章节 | 第二章 导数与微分 |
| 知识点 | 高阶导数 |

**参考答案**：
$y' = \frac{1}{1+x^2}$，$y'' = \frac{-2x}{(1+x^2)^2}$

由幂级数展开：$\arctan x = \sum_{k=0}^{\infty} (-1)^k \frac{x^{2k+1}}{2k+1}$

所以 $y^{(2k)}(0) = 0$，$y^{(2k+1)}(0) = (2k)! (-1)^k$

## 第三章 微分中值定理与导数应用

### 第11题
**题目**：证明 $\arcsin x + \arccos x = \frac{\pi}{2}$

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★ |
| 来源 | 严选题-高数篇-第11题 |
| 章节 | 第三章 微分中值定理与导数应用 |
| 知识点 | 微分中值定理应用 |

**参考答案**：
设 $f(x) = \arcsin x + \arccos x$，则 $f'(x) = \frac{1}{\sqrt{1-x^2}} - \frac{1}{\sqrt{1-x^2}} = 0$

所以 $f(x) = C$（常数），取 $x = 0$，得 $f(0) = \frac{\pi}{2}$，故 $C = \frac{\pi}{2}$

---

### 第12题
**题目**：设 $f(x)$ 在 $[a, b]$ 上连续，在 $(a, b)$ 内可导，证明：存在 $\xi \in (a, b)$ 使 $f'(\xi) = \frac{f(b) - f(a)}{b - a}$

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★ |
| 来源 | 严选题-高数篇-第12题 |
| 章节 | 第三章 微分中值定理与导数应用 |
| 知识点 | 拉格朗日中值定理 |

**参考答案**：
构造辅助函数 $F(x) = f(x) - f(a) - \frac{f(b) - f(a)}{b - a}(x - a)$

验证 $F(a) = F(b) = 0$，由罗尔定理，存在 $\xi \in (a, b)$ 使 $F'(\xi) = 0$

即 $f'(\xi) = \frac{f(b) - f(a)}{b - a}$

---

### 第13题
**题目**：求函数 $f(x) = x^3 - 3x^2 + 2$ 的单调区间和极值

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★ |
| 来源 | 严选题-高数篇-第13题 |
| 章节 | 第三章 微分中值定理与导数应用 |
| 知识点 | 导数应用 |

**参考答案**：
$f'(x) = 3x^2 - 6x = 3x(x - 2)$

令 $f'(x) = 0$，得 $x = 0, 2$

单调递增区间：$(-\infty, 0) \cup (2, +\infty)$
单调递减区间：$(0, 2)$

极值：$f(0) = 2$（极大值），$f(2) = -2$（极小值）

---

### 第14题
**题目**：求曲线 $y = \frac{x^2}{1+x}$ 的渐近线

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★ |
| 来源 | 严选题-高数篇-第14题 |
| 章节 | 第三章 微分中值定理与导数应用 |
| 知识点 | 曲线的渐近线 |

**参考答案**：
$\lim_{x \to -1} \frac{x^2}{1+x} = \infty$，所以 $x = -1$ 是垂直渐近线

$\lim_{x \to \infty} \frac{f(x)}{x} = \lim_{x \to \infty} \frac{x}{1+x} = 1 = a$

$\lim_{x \to \infty} [f(x) - ax] = \lim_{x \to \infty} [\frac{x^2}{1+x} - x] = -1 = b$

所以斜渐近线为 $y = x - 1$

### 第15题
**题目**：证明 $e^x > 1 + x + \frac{x^2}{2}$（$x > 0$）

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★ |
| 来源 | 严选题-高数篇-第15题 |
| 章节 | 第三章 微分中值定理与导数应用 |
| 知识点 | 泰勒公式 |

**参考答案**：
$e^x = 1 + x + \frac{x^2}{2} + \frac{e^\xi}{3!} x^3$（$\xi \in (0, x)$）

当 $x > 0$ 时，$\frac{e^\xi}{3!} x^3 > 0$，所以 $e^x > 1 + x + \frac{x^2}{2}$

## 第四章 不定积分

### 第16题
**题目**：求 $\int \frac{1}{x^2 + x + 1} dx$

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★ |
| 来源 | 严选题-高数篇-第16题 |
| 章节 | 第四章 不定积分 |
| 知识点 | 积分公式 |

**参考答案**：
$\int \frac{1}{x^2 + x + 1} dx = \int \frac{1}{(x+\frac{1}{2})^2 + \frac{3}{4}} dx = \frac{2}{\sqrt{3}} \arctan \frac{2x+1}{\sqrt{3}} + C$

### 第17题
**题目**：求 $\int \frac{x+1}{x^2 - 2x + 5} dx$

| 属性 | 值 |
|------|-----|
| 题型 | 解答题 |
| 难度 | ★★★ |
| 来源 | 严选题-高数篇-第17题 |
| 章节 | 第四章 不定积分 |
| 知识点 | 分式积分 |

**参考答案**：
$\int \frac{x+1}{x^2 - 2x + 5} dx = \int \frac{\frac{1}{2}(2x-2) + 2}{(x-1)^2 + 4} dx$

$= \frac{1}{2} \ln((x-1)^2 + 4) + \arctan \frac{x-1}{2} + C$
