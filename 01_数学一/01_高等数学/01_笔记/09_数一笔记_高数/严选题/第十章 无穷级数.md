## 二、填空题

**1.** 已知级数 $\sum_{n=1}^{\infty} \frac{\sqrt{n+1} \arctan(n+1)}{n^{\alpha} + 1}$ 收敛，则 $\alpha$ 应满足____

**答案：** $\alpha > \frac{3}{2}$

**2.** 幂级数 $\sum_{n=1}^{\infty}\frac{x^{n}}{2n+1}$ 的收敛域为____

## 三、解答题

**1.** 将 $f(x) = \frac{x - 1}{4 - x}$ 在 $x = 1$ 处展开为幂级数。
> 涉及知识点：[[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/09-无穷级数/Lecture 51：函数展开成幂级数|间接展开法]]

**答案：** $\frac{1}{3}\left[(x - 1) + \frac{(x - 1)^2}{3} +\frac{(x - 1)^3}{3^2} +\dots\right]$，$|x - 1| < 3$

---

**2.** 将 $f(x) = \frac{1}{x^2 + 5x + 6}$ 展开为 $x$ 的幂级数。
> 涉及知识点：[[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/09-无穷级数/Lecture 51：函数展开成幂级数|部分分式展开]]

**答案：** $\sum_{n=0}^{\infty} (-1)^{n} \left( \frac{1}{2^{n+1}} - \frac{1}{3^{n+1}} \right) x^{n}$，$|x| < 2$

---

**3.** 将 $f(x) = \arctan x$ 展开为 $x$ 的幂级数。
> 涉及知识点：[[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/09-无穷级数/Lecture 51：函数展开成幂级数|逐项积分法]]

**答案：** $\sum_{n=0}^{\infty} \frac{(-1)^n}{2n+1} x^{2n+1}$，$|x| \leqslant 1$

---

**4.** 将 $f(x) = \arctan \frac{1 + x}{1 - x}$ 展开为 $x$ 的幂级数。
> 涉及知识点：[[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/09-无穷级数/Lecture 51：函数展开成幂级数|展开式的选取]]

**答案：** $\frac{\pi}{4} + \sum_{n=0}^{\infty} \frac{(-1)^n}{2n+1} x^{2n+1}$，$-1 \leqslant x < 1$

---

**5.** 将 $f(x) = \ln(1 + x + x^{2})$ 展开为 $x$ 的幂级数。
> 涉及知识点：[[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/09-无穷级数/Lecture 51：函数展开成幂级数|对数函数展开]]

**答案：** $-\sum_{n=1}^{\infty} \frac{x^{3n}}{n} + \sum_{n=1}^{\infty} \frac{x^n}{n}$，$-1 \leqslant x < 1$

---

**6.** 将 $f(x) = \frac{x}{x^2 + x - 2}$ 在 $x = -1$ 处展开为幂级数。
> 涉及知识点：[[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/09-无穷级数/Lecture 51：函数展开成幂级数|在指定点展开]]

**答案：** $-\frac{1}{6}\sum_{n=0}^{\infty} \frac{(x+1)^{n}}{2^{n}} + \frac{2}{3}\sum_{n=0}^{\infty} (-1)^{n}(x+1)^{n}$，$-2 < x < 0$

---

**7.** 将 $f(x) = \frac{1}{(x + 1)^2}$ 在 $x = -2$ 处展开为幂级数。
> 涉及知识点：[[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/09-无穷级数/Lecture 51：函数展开成幂级数|逐项求导法]]

**答案：** $\sum_{n=1}^{\infty} n(x + 2)^{n-1}$，$-3 < x < -1$

---

**8.** 将 $f(x) = \ln (2 + x - 3x^2)$ 在 $x = 0$ 处展开为幂级数。
> 涉及知识点：[[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/09-无穷级数/Lecture 51：函数展开成幂级数|对数函数展开]]

**答案：** $-\sum_{n=1}^{\infty} \frac{x^{n}}{n} + \ln 2 + \sum_{n=1}^{\infty} (-1)^{n-1}\frac{3^{n}}{2^{n}}\frac{x^{n}}{n}$，$-\frac{2}{3} < x \leqslant \frac{2}{3}$

---

**9.** 求幂级数 $\sum_{n=1}^{\infty}\frac{x^{n}}{n(n+1)}$ 的收敛域及和函数。
> 涉及知识点：[[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/09-无穷级数/Lecture 50：幂级数|幂级数求和函数]]

**答案：** 收敛域 $[-1,1]$；$S(x) = \begin{cases} \left(\frac{1}{x} - 1\right)\ln(1-x) + 1, & -1 \leqslant x < 0, 0 < x < 1 \\ 0, & x = 0 \\ 1, & x = 1 \end{cases}$

---

**10.** 求 $\sum_{n=1}^{\infty}\frac{(-1)^{n-1}}{n(2n-1)}\frac{1}{3^{n}}$ 的和。
> 涉及知识点：[[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/09-无穷级数/Lecture 50：幂级数|幂级数求和]]

**答案：** $\frac{2}{\sqrt{3}}\arctan\frac{1}{\sqrt{3}} - \ln\left(1 + \frac{1}{3}\right)$

---

**11.** 将 $f(x) = \pi^2 - x^2$（$-\pi \leqslant x \leqslant \pi$）展开成傅里叶级数。
> 涉及知识点：[[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/09-无穷级数/Lecture 52：傅里叶级数|傅里叶级数展开]]

**答案：** $\pi^{2} - x^{2} = \frac{2\pi^{2}}{3} + 4\sum_{n=1}^{\infty}\frac{(-1)^{n+1}}{n^{2}}\cos nx$

---

> - 正项级数审敛法要能灵活选择（比较、比值、根值）

> - 函数展开常用间接展开法，注意收敛域的确定
> - 幂级数求和需要掌握逐项积分/求导的技巧
