## 一、基本概念

### 1.1 原函数与不定积分

**原函数**：若在区间 $I$ 上，$F'(x) = f(x)$，则称 $F(x)$ 为 $f(x)$ 在 $I$ 上的一个**原函数**。

**不定积分**：$f(x)$ 的全体原函数称为 $f(x)$ 的不定积分，记作

$$\boxed{\int f(x) \, dx = F(x) + C}$$

其中 $C$ 为任意常数。

> **基本关系**：$\left(\int f(x)dx\right)' = f(x)$，$\int F'(x)dx = F(x) + C$。
>
> **存在性**：连续函数必有原函数（但原函数未必能用初等函数表示）。

### 1.2 定积分的定义

设 $f(x)$ 在 $[a,b]$ 上有定义，将 $[a,b]$ 任意分成 $n$ 个小区间 $[x_{i-1}, x_i]$（$i = 1, 2, \ldots, n$），取 $\xi_i \in [x_{i-1}, x_i]$，令 $\lambda = \max\{\Delta x_i\}$，若极限

$$\lim_{\lambda \to 0} \sum_{i=1}^{n} f(\xi_i) \Delta x_i$$

存在且与分法和 $\xi_i$ 的取法无关，则称此极限为 $f(x)$ 在 $[a,b]$ 上的**定积分**，记作

$$\int_a^b f(x) \, dx$$

> **可积条件**：$f(x)$ 在 $[a,b]$ 上连续 $\Rightarrow$ $f(x)$ 在 $[a,b]$ 上可积。有界且只有有限个间断点也保证可积。

## 二、核心知识

### 2.1 基本积分公式表

| 积分 | 结果 | 积分 | 结果 |
|------|------|------|------|
| $\int x^n dx$ | $\frac{x^{n+1}}{n+1}+C$ $(n\ne-1)$ | $\int \frac{1}{x}dx$ | $\ln\|x\|+C$ |
| $\int e^x dx$ | $e^x + C$ | $\int a^x dx$ | $\frac{a^x}{\ln a}+C$ |
| $\int \sin x \, dx$ | $-\cos x + C$ | $\int \cos x \, dx$ | $\sin x + C$ |
| $\int \sec^2 x \, dx$ | $\tan x + C$ | $\int \csc^2 x \, dx$ | $-\cot x + C$ |
| $\int \sec x \tan x \, dx$ | $\sec x + C$ | $\int \csc x \cot x \, dx$ | $-\csc x + C$ |
| $\int \frac{dx}{\sqrt{1-x^2}}$ | $\arcsin x + C$ | $\int \frac{dx}{1+x^2}$ | $\arctan x + C$ |
| $\int \tan x \, dx$ | $-\ln\|\cos x\|+C$ | $\int \cot x \, dx$ | $\ln\|\sin x\|+C$ |
| $\int \sec x \, dx$ | $\ln\|\sec x+\tan x\|+C$ | $\int \csc x \, dx$ | $\ln\|\csc x-\cot x\|+C$ |
| $\int \frac{dx}{x^2+a^2}$ | $\frac{1}{a}\arctan\frac{x}{a}+C$ | $\int \frac{dx}{x^2-a^2}$ | $\frac{1}{2a}\ln\left\|\frac{x-a}{x+a}\right\|+C$ |
| $\int \frac{dx}{\sqrt{a^2-x^2}}$ | $\arcsin\frac{x}{a}+C$ | $\int \frac{dx}{\sqrt{x^2+a^2}}$ | $\ln(x+\sqrt{x^2+a^2})+C$ |

### 2.2 第一类换元法（凑微分法）

$$\boxed{\int f[\varphi(x)]\varphi'(x) \, dx = \int f(u) \, du \Big|_{u=\varphi(x)}}$$

**核心思想**：识别被积函数中 $g(x)$ 与 $g'(x)$ 的关系，令 $u = g(x)$。

**常见凑微分模式**：

| 积分形式 | 凑微分 |
|----------|--------|
| $\int f(ax+b)dx$ | $= \frac{1}{a}\int f(ax+b)d(ax+b)$ |
| $\int x^n f(x^{n+1})dx$ | $= \frac{1}{n+1}\int f(x^{n+1})d(x^{n+1})$ |
| $\int f(\ln x)\frac{1}{x}dx$ | $= \int f(\ln x)d(\ln x)$ |
| $\int f(e^x)e^x dx$ | $= \int f(e^x)d(e^x)$ |
| $\int f(\sin x)\cos x \, dx$ | $= \int f(\sin x)d(\sin x)$ |
| $\int \frac{f'(x)}{f(x)}dx$ | $= \ln\|f(x)\| + C$ |

### 2.3 第二类换元法

$$\int f(x)dx = \int f[\varphi(t)]\varphi'(t)dt \Big|_{t = \varphi^{-1}(x)}$$

**常用换元**：

| 换元类型 | 适用情形 | 换元方式 |
|----------|----------|----------|
| 三角换元 | $\sqrt{a^2-x^2}$ | $x = a\sin t$ |
| 三角换元 | $\sqrt{a^2+x^2}$ | $x = a\tan t$ |
| 三角换元 | $\sqrt{x^2-a^2}$ | $x = a\sec t$ |
| 倒代换 | 分母次数高于分子 | $x = \frac{1}{t}$ |
| 根号换元 | $\sqrt[n]{ax+b}$ | $t = \sqrt[n]{ax+b}$ |

### 2.4 分部积分法

$$\boxed{\int u \, dv = uv - \int v \, du}$$

**口诀**：**反对幂指三**（选择 $u$ 的优先级：反三角 > 对数 > 幂函数 > 指数 > 三角函数）。

> 即：被积函数是两种不同类型函数乘积时，优先选排在前面的类型作为 $u$。

**常见分部积分模式**：

| 被积函数 | $u$ | $dv$ |
|----------|-----|------|
| $x^n e^{ax}$ | $x^n$ | $e^{ax}dx$ |
| $x^n \sin ax$ | $x^n$ | $\sin ax \, dx$ |
| $x^n \ln x$ | $\ln x$ | $x^n dx$ |
| $x^n \arctan x$ | $\arctan x$ | $x^n dx$ |
| $e^{ax}\sin bx$ | 任取均可（需两次分部后回代） | — |

### 2.5 有理函数积分

**部分分式分解**：设 $\frac{P(x)}{Q(x)}$ 为真分式，则可分解为

$$\frac{P(x)}{Q(x)} = \sum \frac{A}{(x-a)^k} + \sum \frac{Bx+C}{(x^2+px+q)^s}$$

其中 $x^2+px+q$ 为不可约二次因式（$\Delta < 0$）。

**积分方法**：
- $\int \frac{A}{x-a}dx = A\ln|x-a| + C$
- $\int \frac{A}{(x-a)^k}dx = \frac{A}{(1-k)(x-a)^{k-1}} + C$（$k \ge 2$）
- $\int \frac{Bx+C}{x^2+px+q}dx$ 先配方再拆分

### 2.6 定积分的性质

| 性质 | 表达式 |
|------|--------|
| **线性** | $\int_a^b [\alpha f + \beta g]dx = \alpha\int_a^b f \, dx + \beta\int_a^b g \, dx$ |
| **区间可加性** | $\int_a^b f \, dx = \int_a^c f \, dx + \int_c^b f \, dx$ |
| **保号性** | $f(x) \ge 0 \Rightarrow \int_a^b f \, dx \ge 0$（$a < b$） |
| **保序性** | $f(x) \le g(x) \Rightarrow \int_a^b f \, dx \le \int_a^b g \, dx$ |
| **估值定理** | $m(b-a) \le \int_a^b f(x)dx \le M(b-a)$，$m, M$ 为最小最大值 |
| **中值定理** | $\exists \xi \in [a,b]$，$\int_a^b f(x)dx = f(\xi)(b-a)$ |
| **绝对值不等式** | $\left\|\int_a^b f \, dx\right\| \le \int_a^b |f| \, dx$ |

### 2.7 变限积分求导

$$\boxed{\frac{d}{dx}\int_a^{g(x)} f(t) \, dt = f[g(x)] \cdot g'(x)}$$

$$\frac{d}{dx}\int_{h(x)}^{g(x)} f(t) \, dt = f[g(x)] \cdot g'(x) - f[h(x)] \cdot h'(x)$$

> **重要结论**：变限积分是连接微分学和积分学的桥梁。变限积分在被积函数连续的条件下可导。

### 2.8 牛顿-莱布尼茨公式

$$\boxed{\int_a^b f(x) \, dx = F(b) - F(a)}$$

其中 $F(x)$ 是 $f(x)$ 的任一原函数，$f(x)$ 在 $[a,b]$ 上连续。

> 这是计算定积分的基本公式，将定积分转化为求原函数的值。

## 三、深入理解

### 3.1 反常积分

#### 无穷区间上的反常积分

$$\int_a^{+\infty} f(x)dx = \lim_{b \to +\infty} \int_a^b f(x)dx$$

若极限存在则**收敛**，否则**发散**。

#### 无界函数的反常积分

若 $f(x)$ 在 $x = b$ 处无界：

$$\int_a^b f(x)dx = \lim_{\varepsilon \to 0^+} \int_a^{b-\varepsilon} f(x)dx$$

#### 常用判别法

**比较判别法**（非负函数）：$0 \le f(x) \le g(x)$
- $\int g$ 收敛 $\Rightarrow$ $\int f$ 收敛
- $\int f$ 发散 $\Rightarrow$ $\int g$ 发散

**极限判别法**：$\lim_{x \to +\infty} \frac{f(x)}{g(x)} = c$（$0 < c < +\infty$），则 $\int f$ 与 $\int g$ 同敛散。

**$p$-积分**：
- $\int_1^{+\infty} \frac{1}{x^p}dx$：$p > 1$ 收敛，$p \le 1$ 发散
- $\int_0^1 \frac{1}{x^p}dx$：$p < 1$ 收敛，$p \ge 1$ 发散

### 3.2 定积分的应用

#### 平面图形面积

**直角坐标**：

$$S = \int_a^b |f(x) - g(x)| \, dx$$

**极坐标**：

$$S = \frac{1}{2}\int_\alpha^\beta r^2(\theta) \, d\theta$$

#### 旋转体体积

**绕 $x$ 轴旋转**：

$$\boxed{V_x = \pi \int_a^b [f(x)]^2 \, dx}$$

**绕 $y$ 轴旋转**（壳法）：

$$\boxed{V_y = 2\pi \int_a^b x|f(x)| \, dx}$$

**绕 $y$ 轴旋转**（截面法）：

$$V_y = \pi \int_c^d [g(y)]^2 \, dy$$

其中 $x = g(y)$ 为 $y = f(x)$ 的反函数。

#### 弧长

$$L = \int_a^b \sqrt{1 + [f'(x)]^2} \, dx$$

参数方程：$L = \int_\alpha^\beta \sqrt{[x'(t)]^2 + [y'(t)]^2} \, dt$

极坐标：$L = \int_\alpha^\beta \sqrt{r^2 + [r'(\theta)]^2} \, d\theta$

#### 旋转体侧面积

$$S = 2\pi \int_a^b |f(x)| \sqrt{1 + [f'(x)]^2} \, dx$$

## 五、典型例题

### 例题 1：不定积分计算

**题目**：求 $\int \frac{x \arctan x}{(1+x^2)^{3/2}} dx$。

**思路**：含 $\arctan x$，令 $u = \arctan x$，用分部积分法。

**解答**：

令 $u = \arctan x$，$dv = \frac{x}{(1+x^2)^{3/2}}dx$。

由 $\int \frac{x}{(1+x^2)^{3/2}}dx = \frac{1}{2}\int (1+x^2)^{-3/2}d(1+x^2) = -\frac{1}{\sqrt{1+x^2}}$

故 $v = -\frac{1}{\sqrt{1+x^2}}$，$du = \frac{1}{1+x^2}dx$。

$$\int \frac{x \arctan x}{(1+x^2)^{3/2}}dx = -\frac{\arctan x}{\sqrt{1+x^2}} + \int \frac{dx}{(1+x^2)^{3/2}}$$

对 $\int \frac{dx}{(1+x^2)^{3/2}}$，令 $x = \tan t$：

$$\int \frac{\sec^2 t}{\sec^3 t}dt = \int \cos t \, dt = \sin t + C = \frac{x}{\sqrt{1+x^2}} + C$$

故原式 $= -\frac{\arctan x}{\sqrt{1+x^2}} + \frac{x}{\sqrt{1+x^2}} + C = \frac{x - \arctan x}{\sqrt{1+x^2}} + C$。

**总结**：分部积分法中 $dv$ 的选择要使 $v$ 容易求出。三角换元是处理 $(1+x^2)^{3/2}$ 的标准方法。

---

### 例题 2：定积分计算（利用对称性）

**题目**：求 $\int_{-\pi/2}^{\pi/2} \frac{\cos x}{1+e^x} dx$。

**思路**：被积函数没有明显奇偶性，利用 $x \to -x$ 配对。

**解答**：

令 $I = \int_{-\pi/2}^{\pi/2} \frac{\cos x}{1+e^x}dx$。

令 $x = -t$：$I = \int_{-\pi/2}^{\pi/2} \frac{\cos t}{1+e^{-t}}dt = \int_{-\pi/2}^{\pi/2} \frac{e^t \cos t}{1+e^t}dt$

两式相加：$2I = \int_{-\pi/2}^{\pi/2} \frac{\cos x(1+e^x)}{1+e^x}dx = \int_{-\pi/2}^{\pi/2} \cos x \, dx = 2$

故 $I = 1$。

**总结**：形如 $\int_{-a}^{a} \frac{f(x)}{1+e^{g(x)}}dx$（$f$ 为偶函数，$g$ 为奇函数）的积分，用 $x \to -x$ 配对后相加可大幅简化。

---

### 例题 3：变限积分求导

**题目**：设 $F(x) = \int_0^{x^2} (x^2 - t)f(t)dt$，其中 $f(t)$ 连续，求 $F'(x)$。

**思路**：变限积分中既含 $x$ 又含 $t$，先将 $x$ 提到积分号外。

**解答**：

$$F(x) = x^2 \int_0^{x^2} f(t)dt - \int_0^{x^2} tf(t)dt$$

$$F'(x) = 2x\int_0^{x^2}f(t)dt + x^2 \cdot f(x^2) \cdot 2x - x^2f(x^2) \cdot 2x$$

$$= 2x\int_0^{x^2}f(t)dt$$

**总结**：变限积分求导的关键：(1) 先把与积分变量无关的 $x$ 提到积分号外；(2) 用乘法法则和链式法则；(3) 注意 $\frac{d}{dx}\int_0^{g(x)}f(t)dt = f(g(x)) \cdot g'(x)$。

---

### 例题 4：旋转体体积

**题目**：求由 $y = \sin x$（$0 \le x \le \pi$）与 $x$ 轴所围区域绕 $y$ 轴旋转所得旋转体的体积。

**思路**：用壳法（柱壳法）。

**解答**：

$$V_y = 2\pi \int_0^\pi x \sin x \, dx$$

用分部积分，令 $u = x$，$dv = \sin x \, dx$：

$$V_y = 2\pi \left[-x\cos x \Big|_0^\pi + \int_0^\pi \cos x \, dx\right]$$

$$= 2\pi\left[-(\pi \cdot (-1) - 0) + \sin x \Big|_0^\pi\right]$$

$$= 2\pi[\pi + 0] = 2\pi^2$$

**总结**：绕 $y$ 轴旋转优先考虑壳法 $V = 2\pi \int x f(x) dx$，比用截面法（需先求反函数再分段积分）更简洁。

## 六、易错点

1. **不定积分别忘 $+C$**：不定积分的结果是原函数的全体。

2. **定积分换元必须换限**：令 $x = \varphi(t)$ 时，积分上下限要对应变为 $\varphi^{-1}(a)$ 和 $\varphi^{-1}(b)$。

3. **分部积分的 $uv$ 项**：在定积分中，$uv\big|_a^b$ 不能忘记在端点处取值。

4. **变限积分中 $x$ 与 $t$ 的区别**：$x$ 是对它求导的变量，$t$ 是积分变量；对 $x$ 求导时 $t$ 被"积掉"。

5. **反常积分与定积分的区别**：反常积分是极限，不能直接用牛顿-莱布尼茨公式中的端点代入（需用极限）。

6. **等价无穷小替换在积分中不适用**：积分中不能用等价无穷小替换被积函数。

7. **旋转体体积公式的选用**：绕 $x$ 轴用 $V = \pi\int y^2 dx$（圆盘法），绕 $y$ 轴用 $V = 2\pi\int xy \, dx$（壳法）。
