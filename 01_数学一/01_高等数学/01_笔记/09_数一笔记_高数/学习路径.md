# 高等数学学习路径

## 学习目标
掌握高等数学的基本概念、理论和方法，能够运用数学知识解决实际问题，为考研数学一做好准备。

## 阶段一：函数与极限（2-3周）

### 第1周：函数基础
**学习内容：**
- 函数的概念与性质
- 基本初等函数
- 复合函数与初等函数

**学习目标：**
- [ ] 理解函数的定义和表示方法
- [ ] 掌握函数的性质（有界性、单调性、奇偶性、周期性）
- [ ] 熟悉基本初等函数的图像和性质
- [ ] 理解复合函数的概念

**练习：**
1. 求函数的定义域和值域
2. 判断函数的性质
3. 分析复合函数的结构

### 第2周：极限基础
**学习内容：**
- 数列极限
- 函数极限
- 极限的运算法则

**学习目标：**
- [ ] 理解极限的ε-N和ε-δ定义
- [ ] 掌握极限的运算法则
- [ ] 会求简单的极限

**练习：**
1. 用定义证明极限
2. 计算数列和函数的极限
3. 应用极限运算法则

### 第3周：极限进阶
**学习内容：**
- 极限存在准则
- 两个重要极限
- 无穷小与无穷大
- 无穷小的比较

**学习目标：**
- [ ] 掌握夹逼准则和单调有界准则
- [ ] 熟练应用两个重要极限
- [ ] 理解无穷小和无穷大的概念
- [ ] 掌握无穷小的比较方法

**练习：**
1. 应用极限存在准则求极限
2. 使用两个重要极限求极限
3. 比较无穷小的阶

## 学习建议

### 学习方法
1. **理解为主**：理解概念和定理，不要死记硬背
2. **多做练习**：通过练习巩固知识，提高解题能力
3. **画图理解**：通过画图理解函数的性质和几何意义
4. **总结归纳**：定期总结知识点，建立知识体系
5. **联系实际**：将数学知识与实际问题联系起来

### 时间安排
- 每天学习3-4小时
- 每周完成一章的学习
- 每两周进行一次复习

### 学习资源
1. **教材**：同济大学高等数学教材
2. **视频**：张宇高等数学课程、汤家凤高等数学课程
3. **习题**：历年考研真题、模拟试题
4. **参考书**：张宇高等数学18讲、汤家凤高等数学辅导讲义

### 注意事项
1. **循序渐进**：按照学习路径逐步学习，不要跳跃
2. **及时复习**：定期复习已学内容，避免遗忘
3. **做好笔记**：记录重点内容和难点问题
4. **多问问题**：遇到问题及时请教，不要积累问题
5. **保持耐心**：高等数学内容较多，需要耐心学习
