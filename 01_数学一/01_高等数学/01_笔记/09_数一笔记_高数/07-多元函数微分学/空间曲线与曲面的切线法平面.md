# 空间曲线与曲面的切线/法平面/切平面/法线

## 空间曲线的切线与法平面

### 情形 1：参数方程

设曲线 $\Gamma$ 由参数方程给出：
$$\begin{cases} x = \varphi(t) \\ y = \psi(t) \\ z = \omega(t) \end{cases}$$

在 $t = t_0$ 对应点 $M_0(x_0, y_0, z_0)$，**切向量**为：
$$\vec{T} = (\varphi'(t_0), \psi'(t_0), \omega'(t_0))$$

**切线方程**：
$$\frac{x - x_0}{\varphi'(t_0)} = \frac{y - y_0}{\psi'(t_0)} = \frac{z - z_0}{\omega'(t_0)}$$

**法平面方程**（过 $M_0$ 且垂直于切向量）：
$$\varphi'(t_0)(x - x_0) + \psi'(t_0)(y - y_0) + \omega'(t_0)(z - z_0) = 0$$

### 情形 2：两曲面交线

曲线 $\Gamma$ 为两曲面 $F(x, y, z) = 0$ 和 $G(x, y, z) = 0$ 的交线。

切向量 = 两曲面法向量的叉积：
$$\vec{T} = \vec{n}_F \times \vec{n}_G = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ F_x & F_y & F_z \\ G_x & G_y & G_z \end{vmatrix}$$

## 空间曲面的切平面与法线

### 情形 1：显式方程 $z = f(x, y)$

曲面上点 $M_0(x_0, y_0, z_0)$ 处的**法向量**：
$$\vec{n} = (f_x(x_0, y_0), f_y(x_0, y_0), -1)$$

**切平面方程**：
$$f_x(x_0, y_0)(x - x_0) + f_y(x_0, y_0)(y - y_0) - (z - z_0) = 0$$

**法线方程**（过 $M_0$ 且平行于 $\vec{n}$）：
$$\frac{x - x_0}{f_x(x_0, y_0)} = \frac{y - y_0}{f_y(x_0, y_0)} = \frac{z - z_0}{-1}$$

### 情形 2：隐式方程 $F(x, y, z) = 0$

**法向量**：
$$\vec{n} = (F_x, F_y, F_z)\bigg|_{M_0}$$

**切平面方程**：
$$F_x(M_0)(x - x_0) + F_y(M_0)(y - y_0) + F_z(M_0)(z - z_0) = 0$$

**法线方程**：
$$\frac{x - x_0}{F_x(M_0)} = \frac{y - y_0}{F_y(M_0)} = \frac{z - z_0}{F_z(M_0)}$$

## 二元函数的泰勒公式

### 形式

若 $f(x, y)$ 在 $(x_0, y_0)$ 的某邻域有 $n+1$ 阶连续偏导，则
$$f(x_0 + h, y_0 + k) = \sum_{i=0}^{n} \frac{1}{i!}\left(h\frac{\partial}{\partial x} + k\frac{\partial}{\partial y}\right)^i f(x_0, y_0) + R_n$$

其中余项：
$$R_n = \frac{1}{(n+1)!}\left(h\frac{\partial}{\partial x} + k\frac{\partial}{\partial y}\right)^{n+1} f(x_0 + \theta h, y_0 + \theta k)$$

### 展开式

#### 二阶泰勒公式

$$f(x, y) \approx f(x_0, y_0) + f_x(x_0, y_0)(x - x_0) + f_y(x_0, y_0)(y - y_0)$$
$$+ \frac{1}{2}\left[f_{xx}(x_0, y_0)(x - x_0)^2 + 2f_{xy}(x_0, y_0)(x - x_0)(y - y_0) + f_{yy}(x_0, y_0)(y - y_0)^2\right]$$

#### 佩亚诺余项

$$f(x, y) = \sum_{i=0}^{n} \frac{1}{i!}\left(h\frac{\partial}{\partial x} + k\frac{\partial}{\partial y}\right)^i f(x_0, y_0) + o(\rho^n)$$

其中 $\rho = \sqrt{h^2 + k^2}$。

### 常用应用

#### 求极值的充分条件

设 $f(x_0, y_0)$ 是驻点，$A = f_{xx}, B = f_{xy}, C = f_{yy}$。

| 条件 | 结论 |
|------|------|
| $AC - B^2 > 0, A > 0$ | 极小值 |
| $AC - B^2 > 0, A < 0$ | 极大值 |
| $AC - B^2 < 0$ | 非极值 |
| $AC - B^2 = 0$ | 需进一步判断 |

#### 近似计算

**例**：求 $\sqrt{1.02^3 + 1.97^3}$ 的近似值

**解**：
设 $f(x, y) = \sqrt{x^3 + y^3}$，$(x_0, y_0) = (1, 2)$，$(h, k) = (0.02, -0.03)$

$$f(1, 2) = \sqrt{9} = 3$$

$$f_x = \frac{3x^2}{2\sqrt{x^3 + y^3}}\bigg|_{(1,2)} = \frac{3}{2 \cdot 3} = \frac{1}{2}$$

$$f_y = \frac{3y^2}{2\sqrt{x^3 + y^3}}\bigg|_{(1,2)} = \frac{12}{6} = 2$$

$$f \approx 3 + \frac{1}{2}(0.02) + 2(-0.03) = 3 + 0.01 - 0.06 = 2.95$$
