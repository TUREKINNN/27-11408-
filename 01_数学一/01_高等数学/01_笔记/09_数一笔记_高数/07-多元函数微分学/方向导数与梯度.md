# 方向导数与梯度

> [[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/04-多元微积分]] | [[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/04-多元微积分]]

## 方向导数

### 定义

设函数 $z = f(x, y)$ 在点 $P_0(x_0, y_0)$ 的某邻域内有定义，$\vec{l}$ 是给定的方向向量，$P(x_0 + \Delta x, y_0 + \Delta y)$ 是 $\vec{l}$ 方向上的点。

若极限
$$\lim_{\rho \to 0} \frac{f(x_0 + \Delta x, y_0 + \Delta y) - f(x_0, y_0)}{\rho}$$
存在，则称此极限为函数在 $P_0$ 点沿方向 $\vec{l}$ 的**方向导数**，记作：
$$\frac{\partial f}{\partial \vec{l}}\bigg|_{P_0} \quad \text{或} \quad \frac{\partial f}{\partial l}\bigg|_{P_0}$$

### 计算公式

若 $f(x, y)$ 在 $P_0(x_0, y_0)$ 可微，$\vec{l}$ 方向余弦为 $(\cos\alpha, \cos\beta)$，则
$$\frac{\partial f}{\partial l}\bigg|_{P_0} = f_x(P_0)\cos\alpha + f_y(P_0)\cos\beta$$

### 关键性质

1. **方向导数存在 ≠ 可微**
2. **可微 ⟹ 方向导数存在**
3. **可微时所有方向的方向导数都存在**
4. 沿反方向：$\frac{\partial f}{\partial(-\vec{l})} = -\frac{\partial f}{\partial \vec{l}}$

## 梯度

### 定义

设 $f(x, y)$ 在 $P_0(x_0, y_0)$ 可微，则
$$\text{grad } f(P_0) = \nabla f(P_0) = (f_x(P_0), f_y(P_0))$$
称为 $f$ 在 $P_0$ 点的**梯度**。

三维情形：
$$\text{grad } f(P_0) = (f_x, f_y, f_z)$$

### 几何意义

- 梯度方向：函数值增长最快的方向
- 梯度模：最大方向导数的大小

$$\max \frac{\partial f}{\partial \vec{l}} = |\text{grad } f|$$

方向：$\vec{l}^0 = \frac{\text{grad } f}{|\text{grad } f|}$

### 关系

$$\frac{\partial f}{\partial \vec{l}} = \text{grad } f \cdot \vec{l}^0 = |\text{grad } f| \cos\theta$$

其中 $\theta$ 是梯度方向与 $\vec{l}$ 方向的夹角。

## 等值面（线）

### 等值面

$f(x, y, z) = C$ 是函数 $f$ 的等值面。

**重要性质**：梯度方向是等值面的**法向量**。

### 等高线

在二维情形，$f(x, y) = C$ 是等高线，梯度垂直于等高线。

## 多元微分应用

### 方向导数计算示例

设 $f(x, y) = x^2 + y^2$，$P_0 = (1, 1)$，$\vec{l}$ 方向为 $(1, 1)$。

```
f_x = 2x, f_y = 2y
在 P_0: f_x = 2, f_y = 2

方向余弦：cos α = 1/√2, cos β = 1/√2

∂f/∂l = 2 × 1/√2 + 2 × 1/√2 = 2√2

验证：沿 (1,1) 方向，f 增长最快，模 |grad f| = √(4+4) = 2√2 ✓
```
