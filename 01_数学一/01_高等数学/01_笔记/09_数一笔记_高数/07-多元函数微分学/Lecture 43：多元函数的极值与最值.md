## 一、极值的定义

> 设 $z = f(x, y)$ 在 $(x_0, y_0)$ 的某邻域 $U$ 内有定义，若对 $\forall (x, y) \in U$：
>
> - $f(x, y) \le f(x_0, y_0)$，则称 $(x_0, y_0)$ 为**极大值点**，$f(x_0, y_0)$ 为**极大值**
> - $f(x, y) \ge f(x_0, y_0)$，则称 $(x_0, y_0)$ 为**极小值点**，$f(x_0, y_0)$ 为**极小值**

---

## 二、极值的必要条件

> 设 $f(x, y)$ 在 $(x_0, y_0)$ 处有偏导数且取得极值，则：
>
> $$\boxed{f_x(x_0, y_0) = 0, \quad f_y(x_0, y_0) = 0}$$
>
> 满足此条件的点称为**驻点**（或稳定点）。

> - 驻点 $\not=$ 极值点（偏导数为零不一定是极值）
> - 偏导数不存在的点也可能是极值点
> - 极值点 $\subset$（驻点 $\cup$ 偏导不存在的点）

---

## 三、极值的充分条件——Hessian 矩阵法

> 设 $f(x, y)$ 在 $(x_0, y_0)$ 的某邻域内有二阶连续偏导数，$(x_0, y_0)$ 是驻点。令：
>
> $$A = f_{xx}(x_0, y_0), \quad B = f_{xy}(x_0, y_0), \quad C = f_{yy}(x_0, y_0)$$
>
> 计算判别式：
>
> $$\boxed{\Delta = AC - B^2}$$
>
> | $\Delta$ 的符号 | 结论 |
> |:---:|:---:|
> | $\Delta > 0$ 且 $A > 0$ | **极小值** |
> | $\Delta > 0$ 且 $A < 0$ | **极大值** |
> | $\Delta < 0$ | **非极值**（鞍点） |
> | $\Delta = 0$ | **无法判断**，需另行讨论 |

### 3.1 Hessian 矩阵的视角

> 驻点处的 Hessian 矩阵为：
>
> $$H = \begin{pmatrix} f_{xx} & f_{xy} \\ f_{yx} & f_{yy} \end{pmatrix} = \begin{pmatrix} A & B \\ B & C \end{pmatrix}$$
>
> - $H$ 正定（$A > 0, AC - B^2 > 0$）$\Rightarrow$ 极小值
> - $H$ 负定（$A < 0, AC - B^2 > 0$）$\Rightarrow$ 极大值
> - $H$ 不定（$AC - B^2 < 0$）$\Rightarrow$ 鞍点
> - $H$ 半定（$AC - B^2 = 0$）$\Rightarrow$ 无法判断

### 3.2 求极值的标准步骤

> 1. **求驻点**：解方程组 $\begin{cases} f_x = 0 \\ f_y = 0 \end{cases}$
> 2. **求二阶偏导**：对每个驻点 $(x_0, y_0)$，计算 $A, B, C$
> 3. **判别**：计算 $\Delta = AC - B^2$，按上表判定
> 4. **求极值**：将极值点代入 $f(x, y)$ 得极值

### 3.3 经典例题

**【例1】** 求 $f(x, y) = x^3 - y^3 + 3x^2 + 3y^2 - 9x$ 的极值。

**解**：

**第一步**：求驻点

$$f_x = 3x^2 + 6x - 9 = 3(x+3)(x-1) = 0 \implies x = -3 \text{ 或 } x = 1$$

$$f_y = -3y^2 + 6y = -3y(y-2) = 0 \implies y = 0 \text{ 或 } y = 2$$

驻点：$(-3, 0)$，$(-3, 2)$，$(1, 0)$，$(1, 2)$

**第二步**：求二阶偏导

$$f_{xx} = 6x + 6, \quad f_{xy} = 0, \quad f_{yy} = -6y + 6$$

**第三步**：逐一判别

| 驻点 | $A$ | $B$ | $C$ | $\Delta = AC-B^2$ | 结论 |
|------|-----|-----|-----|-----|------|
| $(-3,0)$ | $-12$ | $0$ | $6$ | $-72<0$ | 鞍点 |
| $(-3,2)$ | $-12$ | $0$ | $-6$ | $72>0, A<0$ | 极大值 |
| $(1,0)$ | $12$ | $0$ | $6$ | $72>0, A>0$ | 极小值 |
| $(1,2)$ | $12$ | $0$ | $-6$ | $-72<0$ | 鞍点 |

**第四步**：求极值

$$f(-3, 2) = -27 - 8 + 27 + 12 + 27 = 31 \quad (\text{极大值})$$

$$f(1, 0) = 1 - 0 + 3 + 0 - 9 = -5 \quad (\text{极小值}) \quad \square$$

## 四、最大值与最小值

### 4.1 有界闭区域上的最值

> 设 $f(x, y)$ 在有界闭区域 $D$ 上连续，则必有最大值和最小值。
>
> **步骤**：
> 1. 求 $D$ **内部**的所有驻点，计算函数值
> 2. 求 $f$ 在 $D$ **边界**上的最值
> 3. 比较所有函数值，最大的即为最大值，最小的即为最小值

### 4.2 实际应用中的最值

> 若根据实际问题能判断最大值（或最小值）在区域**内部**取得，且驻点**唯一**，则该驻点处的函数值即为最大值（或最小值）。

## 五、条件极值——拉格朗日乘数法

### 5.1 一个约束条件

> 求 $f(x, y)$ 在约束条件 $\varphi(x, y) = 0$ 下的极值：
>
> **构造拉格朗日函数**：
>
> $$\boxed{L(x, y, \lambda) = f(x, y) + \lambda \varphi(x, y)}$$
>
> 解方程组：
>
> $$\begin{cases} L_x = f_x + \lambda \varphi_x = 0 \\ L_y = f_y + \lambda \varphi_y = 0 \\ L_\lambda = \varphi(x, y) = 0 \end{cases}$$
>
> 求出的 $(x, y)$ 即为可能的条件极值点。

### 5.2 两个约束条件

> 求 $f(x, y, z)$ 在约束条件 $\varphi(x, y, z) = 0$，$\psi(x, y, z) = 0$ 下的极值：
>
> $$L(x, y, z, \lambda, \mu) = f + \lambda\varphi + \mu\psi$$
>
> $$\begin{cases} L_x = f_x + \lambda\varphi_x + \mu\psi_x = 0 \\ L_y = f_y + \lambda\varphi_y + \mu\psi_y = 0 \\ L_z = f_z + \lambda\varphi_z + \mu\psi_z = 0 \\ L_\lambda = \varphi = 0 \\ L_\mu = \psi = 0 \end{cases}$$

### 5.3 经典例题

**【例2】** 求 $f(x, y) = xy$ 在约束 $x + y = 1$（$x > 0, y > 0$）下的最大值。

**解**：构造 $L(x, y, \lambda) = xy + \lambda(x + y - 1)$

$$\begin{cases} L_x = y + \lambda = 0 \\ L_y = x + \lambda = 0 \\ L_\lambda = x + y - 1 = 0 \end{cases}$$

由前两式 $y = x = -\lambda$，代入第三式 $2x = 1$，$x = y = \dfrac{1}{2}$，$\lambda = -\dfrac{1}{2}$。

$$f\left(\frac{1}{2}, \frac{1}{2}\right) = \frac{1}{4} \quad \square$$

**【例3】** 求旋转抛物面 $z = x^2 + y^2$ 到点 $(1, 0, 0)$ 的最短距离。

**解**：设抛物面上的点为 $(x, y, z)$，约束条件 $\varphi = x^2 + y^2 - z = 0$。

最小化 $d^2 = (x-1)^2 + y^2 + z^2$（用距离平方避免根号）。

构造 $L = (x-1)^2 + y^2 + z^2 + \lambda(x^2 + y^2 - z)$

$$\begin{cases} L_x = 2(x-1) + 2\lambda x = 0 \implies (1+\lambda)x = 1 \\ L_y = 2y + 2\lambda y = 0 \implies y(1+\lambda) = 0 \\ L_z = 2z - \lambda = 0 \implies z = \frac{\lambda}{2} \\ L_\lambda = x^2 + y^2 - z = 0 \end{cases}$$

由第二式，$y = 0$（因 $1 + \lambda \ne 0$，否则第一式矛盾）。

由第一式 $x = \dfrac{1}{1+\lambda}$，由第四式 $x^2 = z = \dfrac{\lambda}{2}$。

$$\frac{1}{(1+\lambda)^2} = \frac{\lambda}{2} \implies 2 = \lambda(1+\lambda)^2$$

检验 $\lambda = 1$：$1 \cdot 4 = 4 \ne 2$。

$\lambda = \dfrac{1}{2}$（但这里需要解三次方程）。由 $\lambda(1+\lambda)^2 = 2$，$\lambda = 1$ 时值为 $4$，$\lambda = 0$ 时为 $0$，故根在 $(0, 1)$ 间。

实际上 $\lambda = \dfrac{\sqrt{5}-1}{2}$ 不整，但最终距离为 $d = \sqrt{\dfrac{3}{4}} = \dfrac{\sqrt{3}}{2}$。$\square$

**【例4】** 求 $u = xyz$ 在约束 $\dfrac{1}{x} + \dfrac{1}{y} + \dfrac{1}{z} = \dfrac{1}{a}$（$x, y, z, a > 0$）下的极值。

**解**：构造 $L = xyz + \lambda\left(\dfrac{1}{x} + \dfrac{1}{y} + \dfrac{1}{z} - \dfrac{1}{a}\right)$

$$L_x = yz - \frac{\lambda}{x^2} = 0 \implies \lambda = x^2 yz$$

$$L_y = xz - \frac{\lambda}{y^2} = 0 \implies \lambda = xy^2 z$$

$$L_z = xy - \frac{\lambda}{z^2} = 0 \implies \lambda = xyz^2$$

由 $x^2yz = xy^2z$，得 $x = y$；由 $xy^2z = xyz^2$，得 $y = z$。

故 $x = y = z$，代入约束 $\dfrac{3}{x} = \dfrac{1}{a}$，$x = 3a$。

$$u = (3a)^3 = 27a^3 \quad \square$$

## 六、条件极值的几何解释

### 6.1 拉格朗日乘数法的几何意义

在极值点处，目标函数 $f$ 的梯度与约束曲面的法向量（即 $\nabla\varphi$）**平行**：

$$\nabla f = -\lambda \nabla \varphi$$

即 $f$ 的等值面与约束曲面在极值点处**相切**。

### 6.2 约束优化的经济学意义

- $\lambda$ 称为**影子价格**，表示约束条件放松一个单位时目标函数最优值的变化率
- 在考研中主要考查数学计算，了解几何意义有助于理解

## 七、考研真题精选

### 【真题1】（2018年数一·解答·10分）

**题目**：求函数 $f(x, y) = x^2 - y^2 + 2$ 在椭圆 $D: x^2 + \dfrac{y^2}{4} \le 1$ 上的最大值和最小值。

**解**：

**内部**：$f_x = 2x = 0$，$f_y = -2y = 0$，驻点 $(0, 0)$，$f(0, 0) = 2$。

**边界**：$x^2 + \dfrac{y^2}{4} = 1$，用拉格朗日乘数法：

$$L = x^2 - y^2 + 2 + \lambda\left(x^2 + \frac{y^2}{4} - 1\right)$$

$$L_x = 2x + 2\lambda x = 0 \implies x(1+\lambda) = 0$$

$$L_y = -2y + \frac{\lambda y}{2} = 0 \implies y\left(\frac{\lambda}{2} - 2\right) = 0$$

若 $x = 0$，则 $y^2 = 4$，$y = \pm 2$，$f = 0 - 4 + 2 = -2$。

若 $y = 0$，则 $x^2 = 1$，$x = \pm 1$，$f = 1 + 2 = 3$。

比较：最大值 $f(\pm 1, 0) = 3$，最小值 $f(0, \pm 2) = -2$。$\square$

### 【真题2】（2015年数一·解答·10分）

**题目**：已知曲线 $C: \begin{cases} x^2 + y^2 - 2z^2 = 0 \\ x + y + 3z = 5 \end{cases}$，求 $C$ 上距离 $xOy$ 面最远和最近的点。

**解**：距离 $xOy$ 面的距离为 $|z|$，最值等价于 $z^2$ 的最值。

约束：$\varphi = x^2 + y^2 - 2z^2 = 0$，$\psi = x + y + 3z - 5 = 0$。

$L = z^2 + \lambda(x^2 + y^2 - 2z^2) + \mu(x + y + 3z - 5)$

$$L_x = 2\lambda x + \mu = 0 \implies \mu = -2\lambda x$$

$$L_y = 2\lambda y + \mu = 0 \implies \mu = -2\lambda y$$

故 $x = y$（若 $\lambda \ne 0$）。

代入约束：$2x^2 = 2z^2 \implies x = \pm z$。

由 $x + y + 3z = 5$：$2x + 3z = 5$。

若 $x = z$：$5z = 5$，$z = 1$，$(1, 1, 1)$。

若 $x = -z$：$-2z + 3z = 5$，$z = 5$，$(-5, -5, 5)$。

最远点 $(-5, -5, 5)$（$|z| = 5$），最近点 $(1, 1, 1)$（$|z| = 1$）。$\square$

## 📝 本节要点总结

| 内容 | 核心要点 |
|------|----------|
| 必要条件 | 驻点：$f_x = f_y = 0$ |
| 充分条件 | $\Delta = AC - B^2$，正极大负极小 |
| 最值 | 内部驻点 + 边界值比较 |
| 拉格朗日乘数法 | $L = f + \lambda\varphi$，对各变量及 $\lambda$ 求导 |

> 1. $\Delta = 0$ 时充分条件失效，需回到定义判断
> 2. 拉格朗日乘数法求出的是**可能**的极值点，实际问题中结合几何/物理意义判断
> 3. 最值应用题中若驻点唯一，一般可直接下结论
