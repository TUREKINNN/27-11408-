## 一、基本概念

### 1.1 多元函数的基本概念

**定义**：设 $D$ 是 $\mathbb{R}^2$ 的一个非空子集，若对每个 $(x,y) \in D$，按法则 $f$ 都有**唯一确定**的实数 $z$ 与之对应，则称 $f$ 为定义在 $D$ 上的**二元函数**，记作

$$z = f(x, y), \quad (x, y) \in D$$

### 1.2 平面点集的基本概念

设 $P_0(x_0, y_0)$ 为平面上一点，$\delta > 0$：

- **邻域**：$U(P_0, \delta) = \{(x,y) \mid \sqrt{(x-x_0)^2 + (y-y_0)^2} < \delta\}$
- **去心邻域**：$\mathring{U}(P_0, \delta) = \{(x,y) \mid 0 < \sqrt{(x-x_0)^2 + (y-y_0)^2} < \delta\}$
- **内点**：若 $\exists \delta > 0$，使 $U(P_0, \delta) \subset D$，则 $P_0$ 是 $D$ 的内点
- **边界点**：若 $P_0$ 的任意邻域内既有 $D$ 中的点，又有不在 $D$ 中的点，则 $P_0$ 是 $D$ 的**边界点**
- **开集**：若 $D$ 中每一点都是内点，则 $D$ 为开集
- **闭集**：若 $D$ 的补集为开集，则 $D$ 为闭集（等价于 $D$ 包含其所有边界点）
- **连通区域**：若 $D$ 中任意两点都可用完全含于 $D$ 的折线连接，则 $D$ 为**连通区域**（简称区域）

> **考研要点**：开区域 = 开集 + 连通；闭区域 = 开区域 + 边界。

### 1.3 多元函数的极限

**定义**：设 $z = f(x,y)$ 在 $P_0(x_0, y_0)$ 的某去心邻域内有定义，若 $\forall \varepsilon > 0, \exists \delta > 0$，当 $0 < \sqrt{(x-x_0)^2 + (y-y_0)^2} < \delta$ 时，$|f(x,y) - A| < \varepsilon$，则称

$$\lim_{(x,y) \to (x_0, y_0)} f(x,y) = A$$

> **关键区别**：二元极限要求点 $(x,y)$ 以**任意方式**趋近 $(x_0,y_0)$，因此可用路径法判断极限不存在，但不能用路径法证明极限存在。

**计算方法**：
1. **夹逼准则**：将 $|f(x,y)-A|$ 适当放缩
2. **变量替换**：令 $x = x_0 + r\cos\theta$，$y = y_0 + r\sin\theta$，化为 $r \to 0$
3. **等价无穷小替换**：同一元，但需注意方向
4. **有界量 × 无穷小 = 无穷小**

### 1.4 多元函数的连续性

**定义**：若 $\lim\limits_{(x,y) \to (x_0,y_0)} f(x,y) = f(x_0, y_0)$，则 $f(x,y)$ 在 $(x_0, y_0)$ 处**连续**。

**有界性与最值定理**：有界闭区域上的连续函数必有界，且能取到最大值和最小值。

**介值定理**：有界闭区域上的连续函数能取到最大值与最小值之间的一切值。

## 二、核心知识

### 2.1 偏导数

**定义**：设 $z = f(x,y)$ 在 $(x_0, y_0)$ 的某邻域内有定义，

$$\boxed{\frac{\partial f}{\partial x}\bigg|_{(x_0,y_0)} = \lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x, y_0) - f(x_0, y_0)}{\Delta x}}$$

$$\frac{\partial f}{\partial y}\bigg|_{(x_0,y_0)} = \lim_{\Delta y \to 0} \frac{f(x_0, y_0 + \Delta y) - f(x_0, y_0)}{\Delta y}$$

**计算方法**：求 $\frac{\partial f}{\partial x}$ 时，将 $y$ 视为常数，对 $x$ 求导即可。

**二阶偏导数**（假设连续时，混合偏导与次序无关）：

$$\frac{\partial^2 f}{\partial x \partial y} = \frac{\partial^2 f}{\partial y \partial x}$$

### 2.2 全微分

**定义**：若 $z = f(x,y)$ 在 $(x_0,y_0)$ 处的全增量可表示为

$$\Delta z = A \cdot \Delta x + B \cdot \Delta y + o(\rho), \quad \rho = \sqrt{(\Delta x)^2 + (\Delta y)^2}$$

则称 $f$ 在 $(x_0,y_0)$ 处**可微**，全微分为

$$\boxed{dz = \frac{\partial f}{\partial x} dx + \frac{\partial f}{\partial y} dy}$$

其中 $A = f_x(x_0,y_0)$，$B = f_y(x_0,y_0)$。

### 2.3 可微的条件链（重要！）

$$\boxed{\text{偏导连续} \Rightarrow \text{可微} \Rightarrow \begin{cases} \text{偏导存在} \\ \text{连续} \end{cases}}$$

> **注意**：以上箭头**不可逆**！偏导存在 $\not\Rightarrow$ 连续，偏导存在 $\not\Rightarrow$ 可微。
>
> **可微的充分条件**：偏导数 $\frac{\partial f}{\partial x}$，$\frac{\partial f}{\partial y}$ 在该点**连续**，则可微。
>
> **可微的充要条件**：$\lim\limits_{\rho \to 0} \frac{\Delta z - f_x \Delta x - f_y \Delta y}{\rho} = 0$。

### 2.4 方向导数与梯度

**方向导数**：设 $l$ 为从 $P_0$ 出发的方向（方向角 $\alpha, \beta$），则

$$\frac{\partial f}{\partial l} = \lim_{t \to 0^+} \frac{f(x_0 + t\cos\alpha, y_0 + t\cos\beta) - f(x_0, y_0)}{t}$$

**计算公式**（$f$ 可微时）：

$$\frac{\partial f}{\partial l} = f_x \cos\alpha + f_y \cos\beta$$

三维类似：$\frac{\partial f}{\partial l} = f_x \cos\alpha + f_y \cos\beta + f_z \cos\gamma$

**梯度**：

$$\boxed{\nabla f = \left(\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z}\right)}$$

**核心性质**：
- 梯度方向是方向导数**取最大值**的方向，最大值为 $|\nabla f|$
- 梯度的反方向是方向导数取最小值的方向
- 梯度垂直于等值面 $f(x,y,z) = C$

### 2.5 多元复合函数求导（链式法则）

**树形图法**：画出因变量 → 中间变量 → 自变量的树形图，每条路径贡献一项，各路径求和。

**典型情形**：

**情形1**：$z = f(u,v)$，$u = \varphi(t)$，$v = \psi(t)$

$$\frac{dz}{dt} = \frac{\partial f}{\partial u} \cdot \frac{du}{dt} + \frac{\partial f}{\partial v} \cdot \frac{dv}{dt}$$

**情形2**：$z = f(u,v)$，$u = \varphi(x,y)$，$v = \psi(x,y)$

$$\frac{\partial z}{\partial x} = \frac{\partial f}{\partial u} \cdot \frac{\partial u}{\partial x} + \frac{\partial f}{\partial v} \cdot \frac{\partial v}{\partial x}$$

$$\frac{\partial z}{\partial y} = \frac{\partial f}{\partial u} \cdot \frac{\partial u}{\partial y} + \frac{\partial f}{\partial v} \cdot \frac{\partial v}{\partial y}$$

### 2.6 隐函数求导

**由 $F(x,y) = 0$ 确定 $y = y(x)$**：

$$\boxed{\frac{dy}{dx} = -\frac{F_x}{F_y} \quad (F_y \ne 0)}$$

**推导**：$F(x, y(x)) = 0$ 两边对 $x$ 求导：$F_x + F_y \cdot y' = 0$。

**由 $F(x,y,z) = 0$ 确定 $z = z(x,y)$**：

$$\boxed{\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}, \quad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z} \quad (F_z \ne 0)}$$

### 2.7 空间曲线与曲面的几何应用

#### 空间曲线的切线与法平面

设曲线参数方程 $x = x(t)$，$y = y(t)$，$z = z(t)$，在 $t = t_0$ 处：

**切线方程**：

$$\frac{x - x_0}{x'(t_0)} = \frac{y - y_0}{y'(t_0)} = \frac{z - z_0}{z'(t_0)}$$

**法平面方程**：

$$x'(t_0)(x - x_0) + y'(t_0)(y - y_0) + z'(t_0)(z - z_0) = 0$$

#### 曲面的切平面与法线

设曲面 $F(x,y,z) = 0$，在点 $(x_0, y_0, z_0)$ 处：

**切平面方程**：

$$F_x(x-x_0) + F_y(y-y_0) + F_z(z-z_0) = 0$$

**法线方程**：

$$\frac{x - x_0}{F_x} = \frac{y - y_0}{F_y} = \frac{z - z_0}{F_z}$$

### 2.8 多元函数极值

#### 无条件极值

**必要条件**：若 $f(x,y)$ 在 $(x_0,y_0)$ 处取极值，且偏导存在，则

$$\boxed{f_x(x_0, y_0) = 0, \quad f_y(x_0, y_0) = 0}$$

**充分条件**：设 $f_x = f_y = 0$，记 $A = f_{xx}$，$B = f_{xy}$，$C = f_{yy}$，$\Delta = AC - B^2$（Hessian 矩阵判别式）：

| 条件 | 结论 |
|------|------|
| $\Delta > 0$ 且 $A > 0$ | 极小值 |
| $\Delta > 0$ 且 $A < 0$ | 极大值 |
| $\Delta < 0$ | 非极值（鞍点） |
| $\Delta = 0$ | 无法判断，需另行分析 |

#### 条件极值与拉格朗日乘数法

求 $f(x,y,z)$ 在约束 $\varphi(x,y,z) = 0$ 下的极值：

**步骤**：
1. 构造 Lagrange 函数：$F(x,y,z,\lambda) = f(x,y,z) + \lambda \varphi(x,y,z)$
2. 解方程组：$F_x = 0, F_y = 0, F_z = 0, F_\lambda = 0$（即 $\varphi = 0$）
3. 判断驻点是否为极值点（通常由实际问题判断）

多个约束条件时，每个约束引入一个乘数。

### 2.9 二重积分

#### 定义与性质

$$\iint_D f(x,y) \, d\sigma = \lim_{\lambda \to 0} \sum_{i=1}^{n} f(\xi_i, \eta_i) \Delta \sigma_i$$

**性质**：
- 线性：$\iint_D [\alpha f + \beta g] = \alpha \iint_D f + \beta \iint_D g$
- 区域可加性：$D = D_1 \cup D_2$ 时 $\iint_D = \iint_{D_1} + \iint_{D_2}$
- 若 $f(x,y) \le g(x,y)$，则 $\iint_D f \le \iint_D g$
- 若在 $D$ 上 $f(x,y) = 1$，则 $\iint_D d\sigma = S_D$（面积）

#### 直角坐标计算

**X 型区域**（先 $y$ 后 $x$）：$D: a \le x \le b, \, \varphi_1(x) \le y \le \varphi_2(x)$

$$\iint_D f(x,y) \, d\sigma = \int_a^b dx \int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y) \, dy$$

**Y 型区域**（先 $x$ 后 $y$）：$D: c \le y \le d, \, \psi_1(y) \le x \le \psi_2(y)$

$$\iint_D f(x,y) \, d\sigma = \int_c^d dy \int_{\psi_1(y)}^{\psi_2(y)} f(x,y) \, dx$$

#### 极坐标计算

适用于：被积函数含 $x^2 + y^2$，积分区域为圆域或扇形。

$$\boxed{\iint_D f(x,y) \, d\sigma = \iint_D f(r\cos\theta, r\sin\theta) \, r \, dr \, d\theta}$$

#### 二重积分换元（Jacobian）

设变换 $x = x(u,v)$，$y = y(u,v)$，则

$$\iint_D f(x,y) \, dx \, dy = \iint_{D'} f(x(u,v), y(u,v)) \, |J| \, du \, dv$$

其中 **Jacobian 行列式**：

$$J = \frac{\partial(x,y)}{\partial(u,v)} = \begin{vmatrix} \dfrac{\partial x}{\partial u} & \dfrac{\partial x}{\partial v} \\[6pt] \dfrac{\partial y}{\partial u} & \dfrac{\partial y}{\partial v} \end{vmatrix}$$

### 2.10 三重积分

$$\iiint_\Omega f(x,y,z) \, dV$$

**直角坐标**（先一后二）：

$$\iiint_\Omega f \, dV = \iint_D dx \, dy \int_{z_1(x,y)}^{z_2(x,y)} f(x,y,z) \, dz$$

**柱面坐标** $(r, \theta, z)$：$x = r\cos\theta$，$y = r\sin\theta$，$z = z$

$$\iiint_\Omega f \, dV = \iiint_\Omega f(r\cos\theta, r\sin\theta, z) \, r \, dr \, d\theta \, dz$$

**球面坐标** $(r, \varphi, \theta)$：$x = r\sin\varphi\cos\theta$，$y = r\sin\varphi\sin\theta$，$z = r\cos\varphi$

$$\iiint_\Omega f \, dV = \iiint_\Omega f(\cdots) \, r^2 \sin\varphi \, dr \, d\varphi \, d\theta$$

### 2.11 曲线积分

#### 第一类曲线积分（对弧长）

$$\int_L f(x,y) \, ds$$

参数化 $L: x = x(t), y = y(t), t \in [\alpha, \beta]$：

$$\int_L f \, ds = \int_\alpha^\beta f(x(t), y(t)) \sqrt{x'^2(t) + y'^2(t)} \, dt$$

> 注意 $ds = \sqrt{x'^2 + y'^2} \, dt$ 始终 $> 0$，与方向无关。

#### 第二类曲线积分（对坐标）

$$\int_L P \, dx + Q \, dy$$

参数化：

$$\int_L P \, dx + Q \, dy = \int_\alpha^\beta [P(x(t),y(t)) x'(t) + Q(x(t),y(t)) y'(t)] \, dt$$

> **注意**：第二类曲线积分**与方向有关**，反向积分变号。

### 2.12 格林公式

设 $D$ 为平面有界闭区域，$L = \partial D$ 为正向边界（左侧为 $D$ 内部），$P, Q$ 在 $D$ 上有连续偏导数，则

$$\boxed{\oint_L P \, dx + Q \, dy = \iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dx \, dy}$$

**应用**：
1. 简化曲线积分计算（将曲线积分化为二重积分）
2. 计算平面区域面积：$S = \frac{1}{2} \oint_L x \, dy - y \, dx$

### 2.13 曲线积分与路径无关的条件

以下四条等价（$D$ 为**单连通区域**，$P, Q$ 有连续偏导）：

1. $\int_L P \, dx + Q \, dy$ 与路径无关
2. $\oint_L P \, dx + Q \, dy = 0$（沿任意闭曲线）
3. 存在 $u(x,y)$ 使 $du = P \, dx + Q \, dy$
4. $$\boxed{\frac{\partial Q}{\partial x} = \frac{\partial P}{\partial y}}$$（在单连通区域内）

### 2.14 曲面积分

#### 第一类曲面积分（对面积）

$$\iint_\Sigma f(x,y,z) \, dS$$

设 $\Sigma: z = z(x,y)$，投影到 $xOy$ 面的区域为 $D_{xy}$：

$$\iint_\Sigma f \, dS = \iint_{D_{xy}} f(x, y, z(x,y)) \sqrt{1 + z_x^2 + z_y^2} \, dx \, dy$$

#### 第二类曲面积分（对坐标）

$$\iint_\Sigma P \, dy \, dz + Q \, dz \, dx + R \, dx \, dy$$

**投影法**（计算 $\iint_\Sigma R \, dx \, dy$）：将 $\Sigma$ 投影到 $xOy$ 面得 $D_{xy}$，

$$\iint_\Sigma R \, dx \, dy = \pm \iint_{D_{xy}} R(x, y, z(x,y)) \, dx \, dy$$

取上侧为 $+$，下侧为 $-$（取决于法向量与 $z$ 轴正向的夹角）。

### 2.15 高斯公式

设 $\Omega$ 为空间有界闭区域，$\Sigma = \partial \Omega$ 为外侧，$P, Q, R$ 有连续偏导数，则

$$\boxed{\oiint_\Sigma P \, dy \, dz + Q \, dz \, dx + R \, dx \, dy = \iiint_\Omega \left(\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\right) dV}$$

### 2.16 斯托克斯公式

$$\oint_L P \, dx + Q \, dy + R \, dz = \iint_\Sigma \begin{vmatrix} dydz & dzdx & dxdy \\ \dfrac{\partial}{\partial x} & \dfrac{\partial}{\partial y} & \dfrac{\partial}{\partial z} \\ P & Q & R \end{vmatrix}$$

$$= \iint_\Sigma \left(\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}\right) dydz + \left(\frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}\right) dzdx + \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dxdy$$

其中 $L = \partial \Sigma$，方向与 $\Sigma$ 的法向量成右手系。

### 2.17 散度与旋度

**散度**（标量）：

$$\text{div} \, \mathbf{F} = \nabla \cdot \mathbf{F} = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}$$

> 高斯公式可简写为 $\oiint_\Sigma \mathbf{F} \cdot d\mathbf{S} = \iiint_\Omega \text{div} \, \mathbf{F} \, dV$

**旋度**（向量）：

$$\text{rot} \, \mathbf{F} = \nabla \times \mathbf{F} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ \dfrac{\partial}{\partial x} & \dfrac{\partial}{\partial y} & \dfrac{\partial}{\partial z} \\ P & Q & R \end{vmatrix}$$

## 三、深入理解

### 3.1 偏导存在但不可微的经典反例

$$f(x,y) = \begin{cases} \dfrac{xy}{x^2 + y^2}, & (x,y) \ne (0,0) \\ 0, & (x,y) = (0,0) \end{cases}$$

此函数在原点偏导存在（$f_x(0,0) = 0$，$f_y(0,0) = 0$），但不连续（沿 $y = kx$ 极限与 $k$ 有关），故不可微。

### 3.2 二重积分的对称性

**偶零奇半**：

- 若 $D$ 关于 $y$ 轴对称，$f(x,y)$ 关于 $x$ 为偶函数：$\iint_D f = 2\iint_{D_{x \ge 0}} f$
- 若 $D$ 关于 $y$ 轴对称，$f(x,y)$ 关于 $x$ 为奇函数：$\iint_D f = 0$

**轮换对称性**：若 $D$ 关于 $y = x$ 对称，则

$$\iint_D f(x,y) \, d\sigma = \iint_D f(y,x) \, d\sigma$$

### 3.3 格林公式的本质

格林公式是**牛顿-莱布尼茨公式在二维的推广**：

- 一维：$\int_a^b f'(x) \, dx = f(b) - f(a)$
- 二维：$\iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) d\sigma = \oint_L P \, dx + Q \, dy$

## 五、典型例题

### 例题 1：二重积分计算

**题目**：计算 $\iint_D (x^2 + y^2) \, d\sigma$，其中 $D$ 为圆域 $x^2 + y^2 \le 2x$。

**思路**：积分区域为圆，被积函数含 $x^2 + y^2$，适合用极坐标。

**解答**：

圆 $x^2 + y^2 = 2x$ 化为极坐标：$r^2 = 2r\cos\theta$，即 $r = 2\cos\theta$。

$D$：$-\dfrac{\pi}{2} \le \theta \le \dfrac{\pi}{2}$，$0 \le r \le 2\cos\theta$。

$$\iint_D (x^2 + y^2) \, d\sigma = \int_{-\pi/2}^{\pi/2} d\theta \int_0^{2\cos\theta} r^2 \cdot r \, dr = \int_{-\pi/2}^{\pi/2} d\theta \int_0^{2\cos\theta} r^3 \, dr$$

$$= \int_{-\pi/2}^{\pi/2} \frac{r^4}{4}\bigg|_0^{2\cos\theta} d\theta = \int_{-\pi/2}^{\pi/2} 4\cos^4\theta \, d\theta$$

$$= 4 \cdot 2 \int_0^{\pi/2} \cos^4\theta \, d\theta = 8 \cdot \frac{3}{4} \cdot \frac{1}{2} \cdot \frac{\pi}{2} = \frac{3\pi}{2}$$

**总结**：被积函数含 $x^2 + y^2$、积分区域为圆/扇形时，优先考虑极坐标变换。公式 $\int_0^{\pi/2} \cos^n\theta \, d\theta$ 的 Wallis 公式要熟练。

---

### 例题 2：格林公式计算曲线积分

**题目**：计算 $\oint_L (x^2 - y) \, dx + (x + y^2) \, dy$，其中 $L$ 为正方形 $|x| + |y| = 1$ 的正向边界。

**思路**：$L$ 是闭曲线，检查是否满足格林公式条件。

**解答**：

$P = x^2 - y$，$Q = x + y^2$，$\frac{\partial Q}{\partial x} = 1$，$\frac{\partial P}{\partial y} = -1$。

由格林公式：

$$\oint_L P \, dx + Q \, dy = \iint_D (1 - (-1)) \, d\sigma = 2 \iint_D d\sigma = 2 \times S_D$$

正方形 $|x| + |y| \le 1$ 的面积为 $2$，故

$$\text{原式} = 2 \times 2 = 4$$

**总结**：闭曲线上的第二类曲线积分，优先考虑格林公式。若 $\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}$ 为常数，则积分 = 常数 × 面积。

---

### 例题 3：高斯公式计算曲面积分

**题目**：计算 $\oiint_\Sigma x^3 \, dydz + y^3 \, dzdx + z^3 \, dxdy$，其中 $\Sigma$ 为球面 $x^2 + y^2 + z^2 = a^2$ 的外侧。

**思路**：闭曲面 + 外侧 → 高斯公式。

**解答**：

$P = x^3$，$Q = y^3$，$R = z^3$。

$$\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z} = 3x^2 + 3y^2 + 3z^2 = 3(x^2 + y^2 + z^2)$$

由高斯公式：

$$\text{原式} = \iiint_\Omega 3(x^2 + y^2 + z^2) \, dV$$

用球面坐标：

$$= 3 \int_0^{2\pi} d\theta \int_0^\pi \sin\varphi \, d\varphi \int_0^a r^2 \cdot r^2 \, dr = 3 \cdot 2\pi \cdot 2 \cdot \frac{a^5}{5} = \frac{12\pi a^5}{5}$$

**总结**：闭曲面上的第二类曲面积分，优先用高斯公式化为三重积分。球域三重积分用球面坐标。

### 例题 4：多元极值与拉格朗日乘数法

**题目**：求函数 $f(x,y) = xy$ 在约束条件 $x + y = 1$（$x, y > 0$）下的最大值。

**思路**：条件极值，用拉格朗日乘数法。

**解答**：

构造 $F(x, y, \lambda) = xy + \lambda(x + y - 1)$。

$$F_x = y + \lambda = 0, \quad F_y = x + \lambda = 0, \quad F_\lambda = x + y - 1 = 0$$

由前两式：$y = -\lambda$，$x = -\lambda$，故 $x = y$。

代入第三式：$2x = 1$，$x = y = \dfrac{1}{2}$。

$$f_{\max} = \frac{1}{2} \times \frac{1}{2} = \frac{1}{4}$$

**总结**：拉格朗日乘数法步骤固定：构造 $F$ → 求偏导令为零 → 解方程组 → 判断。对于简单的二元条件极值，也可以代入化为一元函数求极值。

## 六、易错点

1. **极坐标积分忘乘 $r$**：$d\sigma = r \, dr \, d\theta$，漏掉 $r$ 是最常见错误。

2. **格林公式的边界方向**：公式要求 $L$ 为**正向**（逆时针），若 $L$ 为顺时针则需加负号。

3. **第二类曲面积分的正负号**：投影法计算时，法向量与坐标轴正向同向取正、反向取负，极易出错。

4. **可微条件链的方向**：偏导连续 $\Rightarrow$ 可微 $\Rightarrow$ 偏导存在 + 连续，**箭头不可逆**。偏导存在不一定连续。

5. **隐函数求导公式中的偏导**：$\frac{dy}{dx} = -\frac{F_x}{F_y}$，注意 $F_x, F_y$ 是对 $F(x,y) = 0$ 中的 $x, y$ 分别求偏导，而非对最终的 $y(x)$ 求导。

6. **二重积分换序时画图**：改变积分次序时一定要画出积分区域，不能只凭代数变换。

7. **球面坐标中 $\varphi$ 的范围**：$\varphi$ 是与 $z$ 轴正向的夹角，范围为 $[0, \pi]$，不是 $[0, 2\pi]$。

8. **高斯公式的封闭性**：若曲面不封闭，需**补面**后用高斯公式再减去补面的积分。
