## 一、费马引理

> 设 $f(x)$ 在 $x_0$ 处可导，且在 $x_0$ 的某邻域内 $f(x) \le f(x_0)$（或 $f(x) \ge f(x_0)$），则：
>
> $$\boxed{f'(x_0) = 0}$$
>
> 即**可导函数在极值点处的导数为零**。

**证明**（以 $f(x) \le f(x_0)$ 为例）：

当 $x > x_0$ 时，$\dfrac{f(x) - f(x_0)}{x - x_0} \le 0$，故 $f'_+(x_0) \le 0$

当 $x < x_0$ 时，$\dfrac{f(x) - f(x_0)}{x - x_0} \ge 0$，故 $f'_-(x_0) \ge 0$

由可导知 $f'_-(x_0) = f'_+(x_0)$，因此 $f'(x_0) = 0$。$\blacksquare$

---

## 二、罗尔定理

> 设 $f(x)$ 满足以下三个条件：
> 1. 在闭区间 $[a, b]$ 上**连续**
> 2. 在开区间 $(a, b)$ 内**可导**
> 3. $f(a) = f(b)$
>
> 则至少存在一点 $\xi \in (a, b)$，使得：
>
> $$\boxed{f'(\xi) = 0}$$

**证明思路**：

由闭区间上连续函数的最值定理，$f(x)$ 在 $[a, b]$ 上取得最大值 $M$ 和最小值 $m$。

- 若 $M = m$，则 $f(x) \equiv$ 常数，$f'(\xi) = 0$ 对任意 $\xi$ 成立。
- 若 $M \ne m$，由 $f(a) = f(b)$ 知 $M$ 和 $m$ 中至少有一个在 $(a, b)$ 内取得，设 $f(\xi) = M$（$\xi \in (a, b)$），由费马引理，$f'(\xi) = 0$。$\blacksquare$

**几何意义**：满足条件的曲线在 $[a, b]$ 上至少有一点的切线平行于 $x$ 轴。

```
  y
  |    ╭──╮
  |   ╱    ╲
  |──╱──────╲──     ← f(a)=f(b) 的水平线
  | ╱        ╲
  |╱    ↑     ╲
  +----ξ-------→ x
    f'(ξ)=0（切线水平）
```

## 三、拉格朗日中值定理

> 设 $f(x)$ 满足：
> 1. 在闭区间 $[a, b]$ 上**连续**
> 2. 在开区间 $(a, b)$ 内**可导**
>
> 则至少存在一点 $\xi \in (a, b)$，使得：
>
> $$\boxed{f'(\xi) = \frac{f(b) - f(a)}{b - a}}$$

**等价形式**：

$$f(b) - f(a) = f'(\xi)(b - a) \quad (\xi \text{ 在 } a, b \text{ 之间})$$

$$f(x + \Delta x) - f(x) = f'(\xi)\Delta x \quad (\xi \text{ 在 } x \text{ 与 } x+\Delta x \text{ 之间})$$

**几何意义**：曲线 $y = f(x)$ 在 $(a, b)$ 内至少有一点的切线平行于连接两端点的割线。

```
  y
  |         ╱
  |    ╭──╱──
  |   ╱  ↑╱
  |  ╱  ξ╱       ← 在ξ处切线平行于割线
  | ╱  ╱
  |╱──╱
  +──────→ x
  a      b
```

**证明**：构造辅助函数

$$F(x) = f(x) - f(a) - \frac{f(b)-f(a)}{b-a}(x-a)$$

验证 $F(x)$ 满足罗尔定理的三个条件：

1. $F(x)$ 在 $[a, b]$ 上连续 ✓
2. $F(x)$ 在 $(a, b)$ 内可导 ✓
3. $F(a) = 0$，$F(b) = f(b) - f(a) - [f(b)-f(a)] = 0$，故 $F(a) = F(b)$ ✓

由罗尔定理，$\exists\, \xi \in (a, b)$，$F'(\xi) = 0$，即 $f'(\xi) = \dfrac{f(b)-f(a)}{b-a}$。$\blacksquare$

### 推论

> - **推论 1**：若 $f'(x) \equiv 0$（$x \in I$），则 $f(x) \equiv C$（常数）
> - **推论 2**：若 $f'(x) = g'(x)$（$x \in I$），则 $f(x) = g(x) + C$

## 四、柯西中值定理

> 设 $f(x)$、$g(x)$ 满足：
> 1. 在闭区间 $[a, b]$ 上均**连续**
> 2. 在开区间 $(a, b)$ 内均**可导**
> 3. $g'(x) \ne 0$，$x \in (a, b)$
>
> 则至少存在一点 $\xi \in (a, b)$，使得：
>
> $$\boxed{\frac{f'(\xi)}{g'(\xi)} = \frac{f(b) - f(a)}{g(b) - g(a)}}$$

**几何意义**：将两个函数视为参数方程 $\begin{cases} X = g(x) \\ Y = f(x) \end{cases}$，柯西中值定理表明参数曲线上存在一点的切线平行于连接两端点的割线。

**证明**：构造辅助函数

$$F(x) = [f(b)-f(a)]g(x) - [g(b)-g(a)]f(x)$$

验证 $F(a) = F(b)$，由罗尔定理得证。

## 五、中值定理的证明技巧

### 5.1 辅助函数构造法

> **将要证明的等式变形为 $F'(\xi) = 0$ 的形式**，然后"反向求导"找出 $F(x)$。

常见辅助函数：

| 目标形式 | 辅助函数 $F(x)$ |
|:---:|:---:|
| $f'(\xi) = 0$ | $F(x) = f(x)$ |
| $f'(\xi) + g'(\xi) = 0$ | $F(x) = f(x) + g(x)$ |
| $f'(\xi)g(\xi) + f(\xi)g'(\xi) = 0$ | $F(x) = f(x)g(x)$ |
| $f'(\xi)g(\xi) - f(\xi)g'(\xi) = 0$ | $F(x) = \dfrac{f(x)}{g(x)}$ |
| $f'(\xi) - kf(\xi) = 0$ | $F(x) = e^{-kx}f(x)$ |
| $f'(\xi) + f(\xi)\varphi'(\xi) = 0$ | $F(x) = e^{\varphi(x)}f(x)$ |

### 5.2 双中值问题

> 思路：在 $[a, b]$ 上**使用两次中值定理**，分别在不同的子区间上。
>
> 或者：一个用拉格朗日，另一个用柯西。

## 六、经典例题

### 【例1】罗尔定理的直接应用

**题目**：设 $f(x)$ 在 $[0, 3]$ 上连续，在 $(0, 3)$ 内可导，$f(0) + f(1) + f(2) = 3$，$f(3) = 1$。证明 $\exists\, \xi \in (0, 3)$，$f'(\xi) = 0$。

**证明**：

由 $f(0) + f(1) + f(2) = 3$，知 $f(0), f(1), f(2)$ 的平均值为 $1$。

由介值定理，$f(0), f(1), f(2)$ 中至少有一个 $\le 1$，至少有一个 $\ge 1$。

**情形1**：若 $f(0) = f(1) = f(2) = 1 = f(3)$

在 $[0, 3]$ 的任意子区间上用罗尔定理，$\exists\, \xi$，$f'(\xi) = 0$。

**情形2**：$f(0), f(1), f(2)$ 不全为 $1$，但均值为 $1$，则必有 $f(k) < 1$ 和 $f(j) > 1$。

由介值定理，在 $f(k)$ 和 $f(j)$ 之间的某个 $x_0$ 处，$f(x_0) = 1 = f(3)$。

在 $[x_0, 3]$ 上用罗尔定理，$\exists\, \xi \in (x_0, 3) \subset (0, 3)$，$f'(\xi) = 0$。$\blacksquare$

### 【例2】构造辅助函数

**题目**：设 $f(x)$ 在 $[0, 1]$ 上连续，在 $(0, 1)$ 内可导，$f(0) = 0$，$f(1) = 1$。证明 $\exists\, \xi, \eta \in (0, 1)$，$\xi \ne \eta$，使得 $\dfrac{1}{f'(\xi)} + \dfrac{1}{f'(\eta)} = 2$。

**证明**：

取 $c \in (0, 1)$，在 $[0, c]$ 和 $[c, 1]$ 上分别用拉格朗日中值定理：

$$f'(\xi) = \frac{f(c) - f(0)}{c - 0} = \frac{f(c)}{c}, \quad \xi \in (0, c)$$

$$f'(\eta) = \frac{f(1) - f(c)}{1 - c} = \frac{1 - f(c)}{1 - c}, \quad \eta \in (c, 1)$$

则 $\dfrac{1}{f'(\xi)} + \dfrac{1}{f'(\eta)} = \dfrac{c}{f(c)} + \dfrac{1-c}{1-f(c)}$

取 $c = \dfrac{1}{2}$，若 $f\left(\dfrac{1}{2}\right) = \dfrac{1}{2}$，则和 $= 1 + 1 = 2$。

若 $f\left(\dfrac{1}{2}\right) \ne \dfrac{1}{2}$，由连续函数的介值性质，可以调整 $c$ 使得和为 $2$（详细的连续性论证略）。$\blacksquare$

### 【例3】柯西中值定理的应用

**题目**：设 $f(x)$ 在 $[a, b]$ 上连续，在 $(a, b)$ 内可导，$0 < a < b$。证明 $\exists\, \xi \in (a, b)$，使得 $f(b) - f(a) = \xi f'(\xi)\ln\dfrac{b}{a}$。

**证明**：

取 $g(x) = \ln x$，$g'(x) = \dfrac{1}{x} \ne 0$（$x \in (a, b)$，$a > 0$）。

由柯西中值定理：

$$\frac{f'(\xi)}{g'(\xi)} = \frac{f(b) - f(a)}{g(b) - g(a)} = \frac{f(b) - f(a)}{\ln b - \ln a}$$

$$\frac{f'(\xi)}{1/\xi} = \frac{f(b) - f(a)}{\ln(b/a)}$$

$$\xi f'(\xi) = \frac{f(b) - f(a)}{\ln(b/a)}$$

$$f(b) - f(a) = \xi f'(\xi)\ln\frac{b}{a} \quad \blacksquare$$

## 七、考研真题精选

### 【真题1】（2019年数一·解答·12分）

**题目**：设 $f(x)$ 在 $[0, 1]$ 上连续，在 $(0, 1)$ 内可导，$f(0) = 0$，$f(1) = \dfrac{1}{3}$。证明 $\exists\, \xi \in \left(0, \dfrac{1}{2}\right)$，$\eta \in \left(\dfrac{1}{2}, 1\right)$，使得 $f'(\xi) + f'(\eta) = \xi^2 + \eta^2$。

**证明**：

构造 $F(x) = f(x) - \dfrac{x^3}{3}$

在 $\left[0, \dfrac{1}{2}\right]$ 上用拉格朗日：$F'(\xi) = \dfrac{F\left(\frac{1}{2}\right) - F(0)}{\frac{1}{2} - 0}$，$\xi \in \left(0, \dfrac{1}{2}\right)$

即 $f'(\xi) - \xi^2 = 2\left[f\left(\dfrac{1}{2}\right) - \dfrac{1}{24}\right]$

在 $\left[\dfrac{1}{2}, 1\right]$ 上用拉格朗日：$F'(\eta) = \dfrac{F(1) - F\left(\frac{1}{2}\right)}{1 - \frac{1}{2}}$，$\eta \in \left(\dfrac{1}{2}, 1\right)$

即 $f'(\eta) - \eta^2 = 2\left[\dfrac{1}{3} - \dfrac{1}{3} - f\left(\dfrac{1}{2}\right) + \dfrac{1}{24}\right] = -2\left[f\left(\dfrac{1}{2}\right) - \dfrac{1}{24}\right]$

两式相加：

$$f'(\xi) + f'(\eta) - \xi^2 - \eta^2 = 0$$

即 $f'(\xi) + f'(\eta) = \xi^2 + \eta^2$。$\blacksquare$

## 📝 本节要点总结

| 定理 | 条件 | 结论 |
|------|------|------|
| 费马引理 | 可导 + 极值 | $f'(x_0) = 0$ |
| 罗尔定理 | 连续 + 可导 + $f(a)=f(b)$ | $f'(\xi) = 0$ |
| 拉格朗日 | 连续 + 可导 | $f'(\xi) = \frac{f(b)-f(a)}{b-a}$ |
| 柯西 | 两函数均满足 + $g' \ne 0$ | $\frac{f'(\xi)}{g'(\xi)} = \frac{f(b)-f(a)}{g(b)-g(a)}$ |

> 中值定理证明题的关键是**构造辅助函数**。记住辅助函数构造表，遇到新题型时先将目标等式与表中的形式对照。
