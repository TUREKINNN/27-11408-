# 重积分的应用

> [[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/04-多元微积分]] | [[01_数学一/01_高等数学/01_笔记/09_数一笔记_高数/04-多元微积分]]

## 概述

重积分的应用包括求面积、体积、质心、转动惯量、引力等。

## 1. 平面面积

### 二重积分

**平面区域 $D$ 的面积**：
$$S = \iint_D 1 \,d\sigma = \iint_D d\sigma$$

**曲边梯形 / 扇形面积**用极坐标更简单：
$$S = \iint_D r\,dr\,d\theta$$

### 例题

**例**：求由 $r = a\cos\theta$ 围成的面积（圆心在 $(a/2, 0)$，半径 $a/2$）

$$S = \int_{-\pi/2}^{\pi/2} d\theta \int_0^{a\cos\theta} r\,dr = \int_{-\pi/2}^{\pi/2} \frac{a^2\cos^2\theta}{2}\,d\theta = \frac{\pi a^2}{4}$$

## 2. 体积

### 立体体积

**曲顶柱体体积**（底 $D$，顶 $z = f(x, y) \geq 0$）：
$$V = \iint_D f(x, y)\,d\sigma$$

### 例题

**例**：求 $z = x^2 + y^2$ 与 $z = 1$ 围成立体的体积

**解**：
$$V = \iint_{x^2+y^2 \leq 1} (1 - x^2 - y^2)\,d\sigma = \int_0^{2\pi}d\theta \int_0^1 (1-r^2)r\,dr = \frac{\pi}{2}$$

## 3. 质心（形心）

### 平面薄片质心

**面密度 $\rho(x, y)$ 的薄片 $D$ 质心**：

$$\bar{x} = \frac{\iint_D x\rho(x,y)\,d\sigma}{\iint_D \rho(x,y)\,d\sigma}, \quad \bar{y} = \frac{\iint_D y\rho(x,y)\,d\sigma}{\iint_D \rho(x,y)\,d\sigma}$$

**均匀薄片**（$\rho$ 为常数）：

$$\bar{x} = \frac{1}{S}\iint_D x\,d\sigma, \quad \bar{y} = \frac{1}{S}\iint_D y\,d\sigma$$

### 立体质心

**体密度 $\rho(x, y, z)$ 的立体 $\Omega$ 质心**：

$$\bar{x} = \frac{\iiint_\Omega x\rho\,dV}{\iiint_\Omega \rho\,dV}, \quad \bar{y} = \frac{\iiint_\Omega y\rho\,dV}{\iiint_\Omega \rho\,dV}, \quad \bar{z} = \frac{\iiint_\Omega z\rho\,dV}{\iiint_\Omega \rho\,dV}$$

## 4. 转动惯量

### 平面薄片

**面密度 $\rho(x, y)$ 的薄片 $D$ 对 $x$ 轴、$y$ 轴、原点的转动惯量**：

$$I_x = \iint_D y^2\rho(x,y)\,d\sigma$$
$$I_y = \iint_D x^2\rho(x,y)\,d\sigma$$
$$I_O = I_x + I_y = \iint_D (x^2+y^2)\rho(x,y)\,d\sigma$$

### 立体

**对 $x$ 轴、$y$ 轴、$z$ 轴的转动惯量**：

$$I_x = \iiint_\Omega (y^2+z^2)\rho\,dV$$
$$I_y = \iiint_\Omega (x^2+z^2)\rho\,dV$$
$$I_z = \iiint_\Omega (x^2+y^2)\rho\,dV$$

## 5. 引力

**立体 $\Omega$（密度 $\rho$）对质点 $M_0(x_0, y_0, z_0)$ 的引力**：

**$x$ 方向分量**：
$$F_x = \iiint_\Omega \frac{G\rho(x-x_0)}{r^3}\,dV$$

其中 $r = \sqrt{(x-x_0)^2 + (y-y_0)^2 + (z-z_0)^2}$。

## 常用公式

### 立体几何

| 几何量 | 公式 |
|--------|------|
| 球体体积 | $V = \frac{4}{3}\pi R^3$ |
| 球体表面积 | $S = 4\pi R^2$ |
| 圆柱体体积 | $V = \pi R^2 h$ |
| 圆锥体体积 | $V = \frac{1}{3}\pi R^2 h$ |

### 转动惯量（均匀质量 $m$）

| 几何体 | 转动惯量 |
|--------|----------|
| 细杆（长 $l$） | $I = \frac{1}{12}ml^2$ |
| 圆环 | $I = mR^2$ |
| 圆盘 | $I = \frac{1}{2}mR^2$ |
| 球壳 | $I = \frac{2}{3}mR^2$ |
| 实心球 | $I = \frac{2}{5}mR^2$ |
