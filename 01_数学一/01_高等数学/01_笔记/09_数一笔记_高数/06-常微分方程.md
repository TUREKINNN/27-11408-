## 一、基本概念

### 1.1 微分方程的定义

**微分方程**：含有未知函数及其导数（或微分）的方程。

- **阶**：方程中出现的未知函数的**最高阶导数的阶数**
  - $y' + y = 0$：一阶
  - $y'' + 2y' + y = 0$：二阶

### 1.2 解、通解与特解

- **解**：代入微分方程使之成为恒等式的函数
- **通解**：含有与方程阶数相同个数的**独立任意常数**的解
  - $n$ 阶方程的通解含 $n$ 个独立常数 $C_1, C_2, \ldots, C_n$
- **特解**：由初值条件确定了任意常数后的解
- **初值条件**：用于确定特解的附加条件
  - 一阶：$y(x_0) = y_0$
  - 二阶：$y(x_0) = y_0, \, y'(x_0) = y_0'$

## 二、核心知识

### 2.1 一阶微分方程

#### 可分离变量方程

**形式**：$\frac{dy}{dx} = f(x) \cdot g(y)$

**解法**：

$$\boxed{\frac{dy}{g(y)} = f(x) \, dx \quad \Rightarrow \quad \int \frac{dy}{g(y)} = \int f(x) \, dx + C}$$

> 注意 $g(y) = 0$ 的常数解可能被遗漏。

#### 齐次方程

**形式**：$\frac{dy}{dx} = \varphi\!\left(\frac{y}{x}\right)$

**解法**：令 $u = \frac{y}{x}$（即 $y = ux$），则

$$\frac{dy}{dx} = u + x \frac{du}{dx}$$

代入得 $u + x\frac{du}{dx} = \varphi(u)$，即 $x\frac{du}{dx} = \varphi(u) - u$，为可分离变量方程。

$$\int \frac{du}{\varphi(u) - u} = \int \frac{dx}{x} + C$$

#### 一阶线性方程

**形式**：$y' + P(x) y = Q(x)$

**通解公式**（**必须牢记**）：

$$\boxed{y = e^{-\int P(x) \, dx} \left[\int Q(x) \cdot e^{\int P(x) \, dx} \, dx + C\right]}$$

**推导**（常数变易法）：
1. 先解齐次方程 $y' + P(x)y = 0$：$\frac{dy}{y} = -P(x) \, dx$，得 $y = Ce^{-\int P \, dx}$
2. 将常数 $C$ 换为 $C(x)$：$y = C(x) e^{-\int P \, dx}$
3. 代入原方程求 $C(x)$

#### 伯努利方程

**形式**：$y' + P(x) y = Q(x) y^n \quad (n \ne 0, 1)$

**解法**：令 $z = y^{1-n}$，则 $z' = (1-n) y^{-n} y'$。

原方程两边除以 $y^n$ 并化简，得关于 $z$ 的一阶线性方程：

$$\frac{z'}{1-n} + P(x) z = Q(x)$$

化为标准形式后用一阶线性通解公式求解。

#### 全微分方程（恰当方程）

**形式**：$P(x,y) \, dx + Q(x,y) \, dy = 0$

**判断条件**：若 $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$（在单连通区域内），则为全微分方程。

**解法**：存在 $u(x,y)$ 使 $du = P \, dx + Q \, dy$，通解为

$$u(x,y) = C$$

其中

$$u(x,y) = \int_{x_0}^{x} P(t, y_0) \, dt + \int_{y_0}^{y} Q(x, t) \, dt$$

或用**凑微分法**直接求 $u(x,y)$。

### 2.2 可降阶的高阶方程

#### 类型一：$y^{(n)} = f(x)$

直接积分 $n$ 次即可。

$$y^{(n-1)} = \int f(x) \, dx + C_1, \quad \ldots$$

#### 类型二：$y'' = f(x, y')$（不显含 $y$）

令 $p = y'$，则 $y'' = p' = \frac{dp}{dx}$，代入得

$$p' = f(x, p)$$

为关于 $p$ 的一阶方程。解出 $p = p(x, C_1)$ 后再积分 $y = \int p \, dx + C_2$。

#### 类型三：$y'' = f(y, y')$（不显含 $x$）

令 $p = y'$，则 $y'' = p \frac{dp}{dy}$（注意 $\frac{dp}{dx} = \frac{dp}{dy} \cdot \frac{dy}{dx} = p \frac{dp}{dy}$），代入得

$$p \frac{dp}{dy} = f(y, p)$$

为关于 $p$ 和 $y$ 的一阶方程。

### 2.3 线性微分方程解的结构

#### 二阶齐次线性方程

$$y'' + P(x) y' + Q(x) y = 0$$

**定理**：若 $y_1, y_2$ 是两个**线性无关**的解（即 $\frac{y_1}{y_2} \ne \text{常数}$），则通解为

$$\boxed{y = C_1 y_1 + C_2 y_2}$$

#### 二阶非齐次线性方程

$$y'' + P(x) y' + Q(x) y = f(x)$$

**定理**：通解 = 齐次方程通解 + 非齐次方程的一个特解

$$\boxed{y = C_1 y_1 + C_2 y_2 + y^*}$$

其中 $C_1 y_1 + C_2 y_2$ 为对应齐次方程的通解，$y^*$ 为非齐次方程的某个特解。

> **叠加原理**：若 $y_1^*$ 是 $y'' + Py' + Qy = f_1$ 的特解，$y_2^*$ 是 $y'' + Py' + Qy = f_2$ 的特解，则 $y_1^* + y_2^*$ 是 $y'' + Py' + Qy = f_1 + f_2$ 的特解。

### 2.4 二阶常系数齐次方程

$$\boxed{y'' + py' + qy = 0}$$

**特征方程**：$r^2 + pr + q = 0$，判别式 $\Delta = p^2 - 4q$。

| 情况 | 特征根 | 通解 |
|------|--------|------|
| $\Delta > 0$ | 两不等实根 $r_1 \ne r_2$ | $y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$ |
| $\Delta = 0$ | 重根 $r_1 = r_2 = r$ | $y = (C_1 + C_2 x) e^{rx}$ |
| $\Delta < 0$ | 共轭复根 $r = \alpha \pm \beta i$ | $y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$ |

### 2.5 二阶常系数非齐次方程

$$y'' + py' + qy = f(x)$$

方法：先求齐次通解 $Y$，再求非齐次特解 $y^*$，通解 $y = Y + y^*$。

#### 类型一：$f(x) = e^{\lambda x} P_m(x)$

设 $P_m(x)$ 为 $m$ 次多项式，特解设为

$$\boxed{y^* = x^k \cdot e^{\lambda x} \cdot Q_m(x)}$$

其中 $Q_m(x)$ 为 $m$ 次待定多项式，$k$ 的取值：

$$k = \begin{cases} 0, & \lambda \text{ 不是特征根} \\ 1, & \lambda \text{ 是单特征根} \\ 2, & \lambda \text{ 是重特征根} \end{cases}$$

#### 类型二：$f(x) = e^{\lambda x}[P_l(x)\cos\omega x + P_n(x)\sin\omega x]$

设 $m = \max(l, n)$，特解设为

$$\boxed{y^* = x^k \cdot e^{\lambda x} [Q_m(x)\cos\omega x + R_m(x)\sin\omega x]}$$

其中 $Q_m(x), R_m(x)$ 为 $m$ 次待定多项式，$k$ 的取值：

$$k = \begin{cases} 0, & \lambda + \omega i \text{ 不是特征根} \\ 1, & \lambda + \omega i \text{ 是特征根} \end{cases}$$

### 2.6 欧拉方程

$$\boxed{x^2 y'' + pxy' + qy = f(x)}$$

**解法**：令 $x = e^t$（即 $t = \ln x$），则

$$xy' = \frac{dy}{dt}, \quad x^2 y'' = \frac{d^2 y}{dt^2} - \frac{dy}{dt}$$

代入后化为关于 $t$ 的**常系数**线性方程，求解后再将 $t = \ln x$ 代回。

## 三、深入理解

### 3.1 一阶方程求解的判断流程

```
y' + P(x)y = Q(x)         → 一阶线性公式
y' + P(x)y = Q(x)y^n      → 伯努利方程（令 z = y^{1-n}）
M(x,y)dx + N(x,y)dy = 0
  ├─ ∂M/∂y = ∂N/∂x        → 全微分方程
  ├─ 可分离               → 分离变量
  └─ y' = φ(y/x)          → 齐次方程（令 u = y/x）
```

### 3.2 特征方程的三种情况的直觉

- **两不等实根**：两个指数函数叠加，可描述过阻尼振动
- **重根**：乘以 $x$ 保持线性无关，描述临界阻尼
- **共轭复根**：产生振荡项 $\cos\beta x$ 和 $\sin\beta x$，描述欠阻尼振动

### 3.3 常数变易法的核心思想

齐次方程 $y' + P(x)y = 0$ 的通解为 $y = Ce^{-\int P \, dx}$。

将常数 $C$ "变易"为函数 $C(x)$，得到 $y = C(x)e^{-\int P \, dx}$。

代入非齐次方程，自然消去齐次部分，只剩关于 $C'(x)$ 的简单方程。

## 五、典型例题

### 例题 1：一阶线性方程

**题目**：求微分方程 $y' + \frac{y}{x} = x^2$ 的通解。

**思路**：标准一阶线性方程，直接用通解公式。

**解答**：

$P(x) = \frac{1}{x}$，$Q(x) = x^2$。

$$\int P \, dx = \int \frac{1}{x} \, dx = \ln|x|$$

$$e^{\int P \, dx} = e^{\ln|x|} = |x| \quad \text{（取 } x > 0 \text{ 时为 } x\text{）}$$

$$y = e^{-\ln x} \left[\int x^2 \cdot e^{\ln x} \, dx + C\right] = \frac{1}{x}\left[\int x^3 \, dx + C\right]$$

$$\boxed{y = \frac{x^3}{4} + \frac{C}{x}}$$

**总结**：一阶线性方程的通解公式是最基本的公式，必须熟练掌握。关键是正确识别 $P(x)$ 和 $Q(x)$。

---

### 例题 2：二阶常系数非齐次方程

**题目**：求 $y'' - 3y' + 2y = xe^{2x}$ 的通解。

**思路**：先求齐次通解，再用待定系数法求特解。

**解答**：

**齐次方程**：$r^2 - 3r + 2 = 0$，$(r-1)(r-2) = 0$，$r_1 = 1$，$r_2 = 2$。

$$Y = C_1 e^x + C_2 e^{2x}$$

**特解**：$f(x) = xe^{2x}$，$\lambda = 2$ 是**单特征根**（$r_2 = 2$），故 $k = 1$。

$$y^* = x \cdot e^{2x}(ax + b) = e^{2x}(ax^2 + bx)$$

代入方程：

$y^* = e^{2x}(ax^2 + bx)$

$(y^*)' = e^{2x}(2ax^2 + 2bx + 2ax + b) = e^{2x}[2ax^2 + (2a+2b)x + b]$

$(y^*)'' = e^{2x}[4ax^2 + (8a+4b)x + (2a+4b)]$

代入 $y'' - 3y' + 2y$：

$e^{2x}\{[4ax^2 + (8a+4b)x + (2a+4b)] - 3[2ax^2 + (2a+2b)x + b] + 2[ax^2 + bx]\}$

$x^2$ 系数：$4a - 6a + 2a = 0$ ✓

$x^1$ 系数：$(8a+4b) - (6a+6b) + 2b = 2a = 1$，故 $a = \frac{1}{2}$

$x^0$ 系数：$(2a+4b) - 3b = 2a + b = 0$，故 $b = -2a = -1$

$$y^* = e^{2x}\left(\frac{x^2}{2} - x\right)$$

**通解**：

$$\boxed{y = C_1 e^x + C_2 e^{2x} + e^{2x}\left(\frac{x^2}{2} - x\right)}$$

**总结**：二阶常系数非齐次方程的核心是确定 $k$ 值（$\lambda$ 与特征根的关系）和待定多项式的系数。代入比较系数时要仔细。

---

### 例题 3：伯努利方程

**题目**：求 $y' + \frac{y}{x} = y^2 \ln x$ 的通解。

**思路**：$n = 2$ 的伯努利方程，令 $z = y^{1-2} = y^{-1}$。

**解答**：

方程两边除以 $y^2$：$\frac{y'}{y^2} + \frac{1}{xy} = \ln x$

令 $z = \frac{1}{y}$，$z' = -\frac{y'}{y^2}$，代入得

$$-z' + \frac{z}{x} = \ln x \quad \Rightarrow \quad z' - \frac{z}{x} = -\ln x$$

这是一阶线性方程，$P(x) = -\frac{1}{x}$，$Q(x) = -\ln x$。

$$e^{\int P \, dx} = e^{-\ln x} = \frac{1}{x}$$

$$z = x \left[\int (-\ln x) \cdot \frac{1}{x} \, dx + C\right] = x\left[-\frac{(\ln x)^2}{2} + C\right]$$

$$\frac{1}{y} = Cx - \frac{x(\ln x)^2}{2}$$

$$\boxed{y = \frac{1}{Cx - \frac{x(\ln x)^2}{2}}}$$

**总结**：伯努利方程的关键变换是 $z = y^{1-n}$。变换后变为一阶线性方程，直接用公式。

---

### 例题 4：解的结构与线性无关

**题目**：已知 $y_1 = e^x$，$y_2 = e^{-x}$ 是某二阶齐次线性方程的两个解，且该方程对应的非齐次方程有一个特解 $y^* = x$，求非齐次方程的通解。

**思路**：利用线性微分方程解的结构定理。

**解答**：

$y_1 = e^x$ 和 $y_2 = e^{-x}$ 线性无关（$\frac{y_1}{y_2} = e^{2x} \ne$ 常数）。

齐次方程通解：$Y = C_1 e^x + C_2 e^{-x}$

非齐次方程通解：

$$\boxed{y = C_1 e^x + C_2 e^{-x} + x}$$

**总结**：解的结构是选择题高频考点。记住：
- 齐次通解 = 独立解的线性组合（个数 = 阶数）
- 非齐次通解 = 齐次通解 + 非齐次特解
- 验证线性无关用 Wronsky 行列式 $W = \begin{vmatrix} y_1 & y_2 \\ y_1' & y_2' \end{vmatrix} \ne 0$

## 六、易错点

1. **一阶线性通解公式中的 $\int P \, dx$**：不要忘掉积分常数——公式中已有 $C$，所以 $\int P \, dx$ 取一个原函数即可（不加常数）。

2. **齐次方程中的"齐次"含义不同**：微分方程中的"齐次"指 $y' = \varphi(y/x)$ 的形式，不是 $f(\lambda x, \lambda y) = \lambda^n f(x,y)$ 的齐次函数。

3. **伯努利方程 $n = 0, 1$ 排除**：$n = 0$ 时为一阶线性，$n = 1$ 时可直接分离变量，不需要伯努利变换。

4. **特征方程重根时通解乘 $x$**：$r_1 = r_2 = r$ 时通解为 $(C_1 + C_2 x)e^{rx}$，不要漏掉 $x$。

5. **待定系数法中 $k$ 的取值**：$\lambda$ 是特征方程的几重根，$k$ 就取几。单根 $k = 1$，重根 $k = 2$，不是根 $k = 0$。

6. **可降阶方程类型二和三的区别**：不显含 $y$ 用 $p = y'$ 直接代换；不显含 $x$ 用 $y'' = p\frac{dp}{dy}$，不要混淆。

7. **全微分方程的判断**：必须在**单连通区域**内验证 $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$，多连通区域中此条件不充分。

8. **通解中独立常数的个数**：$n$ 阶方程的通解必须恰好含 $n$ 个独立常数。若化简后少于 $n$ 个，说明有解被遗漏。
