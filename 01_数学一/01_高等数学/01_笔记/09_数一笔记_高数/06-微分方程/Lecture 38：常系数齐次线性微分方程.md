# Lecture 38：常系数齐次线性微分方程

## 一、二阶常系数齐次线性方程

> [!important|purple] 定义
> 二阶常系数齐次线性微分方程的标准形式：
> $$y'' + py' + qy = 0$$
> 其中 $p, q$ 为常数。

---

## 二、特征方程法

> [!important|blue] 定理：特征方程法
> 设 $y = e^{rx}$（$r$ 为待定常数），代入 $y'' + py' + qy = 0$：
> $$r^2 e^{rx} + pr e^{rx} + q e^{rx} = 0 \implies e^{rx}(r^2 + pr + q) = 0$$
>
> 因为 $e^{rx} \neq 0$，故：
> $$r^2 + pr + q = 0$$
>
> 这就是**特征方程**。解出特征根 $r_1, r_2$，即可得通解。

---

## 三、三种情况的通解

### 3.1 两个不等实根（$\Delta = p^2 - 4q > 0$）

> [!important|blue] 情况一：$r_1 \neq r_2$（实数）
> 两个线性无关的解：$y_1 = e^{r_1 x}$，$y_2 = e^{r_2 x}$。
>
> **通解：**
> $$y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$$

#### 例题 1

求 $y'' - 5y' + 6y = 0$ 的通解。

**解：** 特征方程 $r^2 - 5r + 6 = 0$，$(r-2)(r-3) = 0$，$r_1 = 2$，$r_2 = 3$。

通解：$y = C_1 e^{2x} + C_2 e^{3x}$。

#### 例题 2

求 $y'' + y' - 2y = 0$ 满足 $y(0) = 1$，$y'(0) = 0$ 的特解。

**解：** 特征方程 $r^2 + r - 2 = 0$，$(r+2)(r-1) = 0$，$r_1 = 1$，$r_2 = -2$。

通解：$y = C_1 e^x + C_2 e^{-2x}$。

$y' = C_1 e^x - 2C_2 e^{-2x}$。

由初始条件：
- $y(0) = C_1 + C_2 = 1$
- $y'(0) = C_1 - 2C_2 = 0$

解得 $C_1 = \dfrac{2}{3}$，$C_2 = \dfrac{1}{3}$。

特解：$y = \dfrac{2}{3}e^x + \dfrac{1}{3}e^{-2x}$。

---

### 3.2 两个相等实根（$\Delta = p^2 - 4q = 0$）

> [!important|blue] 情况二：$r_1 = r_2 = r$（重根）
> 一个解 $y_1 = e^{rx}$，另一个线性无关解由**降阶法**得 $y_2 = x e^{rx}$。
>
> **通解：**
> $$y = (C_1 + C_2 x) e^{rx}$$

**为什么 $y_2 = xe^{rx}$ 是解？**

设 $y_2 = v(x)e^{rx}$，代入方程：
$$y_2' = (v' + rv)e^{rx}, \quad y_2'' = (v'' + 2rv' + r^2 v)e^{rx}$$

代入 $y'' + py' + qy = 0$，利用 $r^2 + pr + q = 0$ 和 $2r + p = 0$（重根条件）：
$$e^{rx}[v'' + (2r+p)v' + (r^2+pr+q)v] = e^{rx} v'' = 0$$

故 $v'' = 0$，$v = C_1 + C_2 x$。取 $v = x$ 得 $y_2 = xe^{rx}$。

#### 例题 3

求 $y'' - 4y' + 4y = 0$ 的通解。

**解：** 特征方程 $r^2 - 4r + 4 = 0$，$(r-2)^2 = 0$，$r = 2$（重根）。

通解：$y = (C_1 + C_2 x)e^{2x}$。

#### 例题 4

求 $y'' + 6y' + 9y = 0$ 满足 $y(0) = 0$，$y'(0) = 1$ 的特解。

**解：** 特征方程 $r^2 + 6r + 9 = 0$，$(r+3)^2 = 0$，$r = -3$（重根）。

通解：$y = (C_1 + C_2 x)e^{-3x}$。

$y' = C_2 e^{-3x} - 3(C_1 + C_2 x)e^{-3x} = (C_2 - 3C_1 - 3C_2 x)e^{-3x}$。

由 $y(0) = C_1 = 0$，$y'(0) = C_2 = 1$。

特解：$y = xe^{-3x}$。

---

### 3.3 一对共轭复根（$\Delta = p^2 - 4q < 0$）

> [!important|blue] 情况三：$r = \alpha \pm \beta i$（共轭复根）
> 其中 $\alpha = -\dfrac{p}{2}$，$\beta = \dfrac{\sqrt{4q - p^2}}{2}$。
>
> 两个复值解：$e^{(\alpha + \beta i)x}$ 和 $e^{(\alpha - \beta i)x}$。
>
> 取实部和虚部得到两个实值线性无关解：
> $$y_1 = e^{\alpha x}\cos\beta x, \quad y_2 = e^{\alpha x}\sin\beta x$$
>
> **通解：**
> $$y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$$

**推导：** 由 Euler 公式 $e^{(\alpha+\beta i)x} = e^{\alpha x}(\cos\beta x + i\sin\beta x)$，

实部 $= e^{\alpha x}\cos\beta x$ 和虚部 $= e^{\alpha x}\sin\beta x$ 都是方程的解（因为方程系数为实数，复值解的实部和虚部分别是解）。

#### 例题 5

求 $y'' + 2y' + 5y = 0$ 的通解。

**解：** 特征方程 $r^2 + 2r + 5 = 0$，$r = \dfrac{-2 \pm \sqrt{4-20}}{2} = -1 \pm 2i$。

$\alpha = -1$，$\beta = 2$。

通解：$y = e^{-x}(C_1 \cos 2x + C_2 \sin 2x)$。

#### 例题 6

求 $y'' + y = 0$ 满足 $y(0) = 2$，$y'(0) = -1$ 的特解。

**解：** 特征方程 $r^2 + 1 = 0$，$r = \pm i$。$\alpha = 0$，$\beta = 1$。

通解：$y = C_1 \cos x + C_2 \sin x$。

$y' = -C_1 \sin x + C_2 \cos x$。

$y(0) = C_1 = 2$，$y'(0) = C_2 = -1$。

特解：$y = 2\cos x - \sin x$。

#### 例题 7

求 $y'' - 2y' + 2y = 0$ 的通解。

**解：** 特征方程 $r^2 - 2r + 2 = 0$，$r = \dfrac{2 \pm \sqrt{4-8}}{2} = 1 \pm i$。

$\alpha = 1$，$\beta = 1$。

通解：$y = e^x(C_1 \cos x + C_2 \sin x)$。

---

## 四、总结表

| 特征根情况 | 条件 | 通解形式 |
|:---:|:---:|:---:|
| 两个不等实根 $r_1 \neq r_2$ | $\Delta > 0$ | $y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$ |
| 重根 $r_1 = r_2 = r$ | $\Delta = 0$ | $y = (C_1 + C_2 x)e^{rx}$ |
| 共轭复根 $\alpha \pm \beta i$ | $\Delta < 0$ | $y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$ |

---

## 五、高阶常系数齐次线性方程

> [!important|blue] 推广：$n$ 阶常系数齐次方程
> $$y^{(n)} + p_1 y^{(n-1)} + \cdots + p_n y = 0$$
> 特征方程：$r^n + p_1 r^{n-1} + \cdots + p_n = 0$
>
> - 每个**单实根** $r$：对应解 $e^{rx}$；
> - 每个 $k$ **重实根** $r$：对应 $k$ 个解 $e^{rx}, xe^{rx}, \ldots, x^{k-1}e^{rx}$；
> - 每对**单共轭复根** $\alpha \pm \beta i$：对应 $e^{\alpha x}\cos\beta x$，$e^{\alpha x}\sin\beta x$；
> - 每对 $k$ **重复根** $\alpha \pm \beta i$：对应 $e^{\alpha x}\cos\beta x, xe^{\alpha x}\cos\beta x, \ldots, x^{k-1}e^{\alpha x}\cos\beta x$ 和对应的 $\sin$ 项。

#### 例题 8

求 $y''' - 3y'' + 3y' - y = 0$ 的通解。

**解：** 特征方程 $r^3 - 3r^2 + 3r - 1 = 0$，即 $(r-1)^3 = 0$，$r = 1$（三重根）。

通解：$y = (C_1 + C_2 x + C_3 x^2)e^x$。

#### 例题 9

求 $y^{(4)} + 2y'' + y = 0$ 的通解。

**解：** 特征方程 $r^4 + 2r^2 + 1 = 0$，即 $(r^2 + 1)^2 = 0$，$r = \pm i$（各二重）。

通解：$y = (C_1 + C_2 x)\cos x + (C_3 + C_4 x)\sin x$。

---

## 六、易错点总结

> 1. **重根情况不要写成 $y = C_1 e^{rx} + C_2 e^{rx}$**：两个解相同，线性相关。正确为 $y = (C_1 + C_2 x)e^{rx}$。
> 2. **复根情况的 $\alpha$ 和 $\beta$ 不要搞混**：$\alpha = -\dfrac{p}{2}$（实部），$\beta = \dfrac{\sqrt{4q-p^2}}{2}$（虚部的绝对值）。
> 3. **特征方程是从 $y'' + py' + qy = 0$ 中 $y \to r^2$, $y' \to r$, $y \to 1$ 得到的**：注意符号，$p, q$ 可正可负。
> 4. **初值条件要在通解中代入**：先求通解（含两个任意常数），再代入两个条件定常数。
> 5. **高阶方程的特征根要因式分解彻底**：可能需要试根（$r = \pm 1, \pm 2$ 等）。
> 6. **通解中 $C_1, C_2$ 是任意常数**，不同题中取不同值，不要混淆。
