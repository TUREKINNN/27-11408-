# Lecture 40：欧拉方程

## 一、欧拉方程的定义

> [!important|purple] 定义：欧拉方程（Euler Equation）
> 形如
> $$x^2 y'' + p x y' + q y = f(x)$$
> 的方程称为**欧拉方程**（或 Euler-Cauchy 方程），其中 $p, q$ 为常数。
>
> 更一般的 $n$ 阶形式：
> $$x^n y^{(n)} + p_1 x^{n-1} y^{(n-1)} + \cdots + p_{n-1} x y' + p_n y = f(x)$$
>
> **特征：** 各项中 $x$ 的幂次与导数的阶数相同——$k$ 阶导数前乘 $x^k$。

**与常系数方程的区别：** 常系数方程的系数是常数，欧拉方程的系数是 $x$ 的幂函数。但通过变量替换，欧拉方程可以转化为常系数方程。

---

## 二、核心方法：令 $x = e^t$ 换元

> [!important|blue] 定理：欧拉方程的求解方法
> 作变换 $x = e^t$（即 $t = \ln x$，要求 $x > 0$），将欧拉方程转化为关于 $t$ 的**常系数线性方程**。
>
> **关键导数关系：**
>
> 设 $D = \dfrac{d}{dt}$，则：
> $$xy' = Dy$$
> $$x^2 y'' = D(D-1)y = D^2 y - Dy$$
> $$x^3 y''' = D(D-1)(D-2)y$$
>
> 一般地：$x^k y^{(k)} = D(D-1)\cdots(D-k+1)y$

### 推导 $xy' = Dy$

设 $t = \ln x$，$\dfrac{dt}{dx} = \dfrac{1}{x}$。

$$y' = \frac{dy}{dx} = \frac{dy}{dt} \cdot \frac{dt}{dx} = \frac{1}{x}\frac{dy}{dt}$$

故 $xy' = \dfrac{dy}{dt} = Dy$。

### 推导 $x^2 y'' = D(D-1)y$

$$y'' = \frac{d}{dx}\left(\frac{1}{x}\frac{dy}{dt}\right) = -\frac{1}{x^2}\frac{dy}{dt} + \frac{1}{x}\frac{d}{dx}\frac{dy}{dt}$$

$$= -\frac{1}{x^2}\frac{dy}{dt} + \frac{1}{x} \cdot \frac{1}{x}\frac{d^2y}{dt^2} = \frac{1}{x^2}\left(\frac{d^2y}{dt^2} - \frac{dy}{dt}\right)$$

故 $x^2 y'' = \dfrac{d^2y}{dt^2} - \dfrac{dy}{dt} = D(D-1)y$。

### 推导 $x^3 y''' = D(D-1)(D-2)y$

继续微分 $x^2 y'' = D(D-1)y$ 并利用 $xy' = Dy$：

$$\frac{d}{dx}(x^2 y'') = 2xy'' + x^2 y''' = \frac{1}{x}\frac{d}{dt}[D(D-1)y] = \frac{1}{x}D^2(D-1)y$$

$$x^2 y''' = \frac{1}{x}D^2(D-1)y - 2xy''$$

利用 $xy'' = \dfrac{1}{x}[D(D-1)y - Dy \cdot 0] = \dfrac{1}{x}D(D-1)y - \ldots$（过程较长）

最终：$x^3 y''' = D(D-1)(D-2)y$。$\square$

---

## 三、二阶欧拉方程的详细求解

将 $x^2 y'' + pxy' + qy = f(x)$ 代入换元关系：

$$D(D-1)y + pDy + qy = f(e^t)$$

$$(D^2 - D + pD + q)y = f(e^t)$$

$$\left(D^2 + (p-1)D + q\right)y = f(e^t)$$

这是关于 $t$ 的**二阶常系数线性方程**！

> | 欧拉方程 | 常系数方程（$t$ 的方程） |
> |:---:|:---:|
> | $x^2 y'' + pxy' + qy = 0$ | $y''_t + (p-1)y'_t + qy = 0$ |
> | 特征方程（$r$） | $r^2 + (p-1)r + q = 0$ |

### 齐次欧拉方程的解

齐次欧拉方程 $x^2 y'' + pxy' + qy = 0$ 的特征方程为：
$$r^2 + (p-1)r + q = 0$$

根据特征根的情况：

| 特征根 | 关于 $t$ 的通解 | 关于 $x$ 的通解（$x = e^t$） |
|:---:|:---:|:---:|
| $r_1 \neq r_2$（实数） | $y = C_1 e^{r_1 t} + C_2 e^{r_2 t}$ | $y = C_1 x^{r_1} + C_2 x^{r_2}$ |
| $r_1 = r_2 = r$（重根） | $y = (C_1 + C_2 t)e^{rt}$ | $y = (C_1 + C_2 \ln x)x^r$ |
| $r = \alpha \pm \beta i$ | $y = e^{\alpha t}(C_1\cos\beta t + C_2\sin\beta t)$ | $y = x^\alpha(C_1\cos(\beta\ln x) + C_2\sin(\beta\ln x))$ |

**注意：** $e^{rt} = (e^t)^r = x^r$，$t = \ln x$。这是替换回 $x$ 的关键。

#### 例题 1

求 $x^2 y'' - xy' + y = 0$ 的通解（$x > 0$）。

**解：**
令 $x = e^t$，方程变为：
$$D(D-1)y - Dy + y = 0 \implies (D^2 - 2D + 1)y = 0$$

特征方程 $r^2 - 2r + 1 = 0$，$(r-1)^2 = 0$，$r = 1$（重根）。

关于 $t$ 的通解：$y = (C_1 + C_2 t)e^t$。

换回 $x$：$y = (C_1 + C_2 \ln x) \cdot x$。

#### 例题 2

求 $x^2 y'' + xy' + y = 0$ 的通解（$x > 0$）。

**解：** 换元后：
$$D(D-1)y + Dy + y = 0 \implies (D^2 + 1)y = 0$$

特征方程 $r^2 + 1 = 0$，$r = \pm i$。$\alpha = 0$，$\beta = 1$。

关于 $t$ 的通解：$y = C_1\cos t + C_2\sin t$。

换回 $x$：$y = C_1\cos(\ln x) + C_2\sin(\ln x)$。

#### 例题 3

求 $4x^2 y'' + 8xy' + y = 0$ 的通解（$x > 0$）。

**解：** 先化为标准形式：$x^2 y'' + 2xy' + \dfrac{1}{4}y = 0$（各项除以 4）。

换元后：$D(D-1)y + 2Dy + \dfrac{1}{4}y = 0$ → $\left(D^2 + D + \dfrac{1}{4}\right)y = 0$。

特征方程 $r^2 + r + \dfrac{1}{4} = 0$，$\left(r + \dfrac{1}{2}\right)^2 = 0$，$r = -\dfrac{1}{2}$（重根）。

通解：$y = (C_1 + C_2 t)e^{-t/2} = (C_1 + C_2\ln x) \cdot x^{-1/2} = \dfrac{C_1 + C_2\ln x}{\sqrt{x}}$。

#### 例题 4

求 $x^2 y'' - 2xy' + 2y = x^3$ 的通解（$x > 0$）。

**解：**

1. **齐次方程** $x^2 y'' - 2xy' + 2y = 0$：
   换元：$D(D-1)y - 2Dy + 2y = 0$ → $(D^2 - 3D + 2)y = 0$。
   特征方程 $r^2 - 3r + 2 = 0$，$(r-1)(r-2) = 0$，$r = 1, 2$。
   齐次通解：$Y = C_1 x + C_2 x^2$。

2. **非齐次特解**：$f(x) = x^3$，换元后 $f(e^t) = e^{3t}$。
   $(D^2 - 3D + 2)y = e^{3t}$。
   $\lambda = 3$ 不是特征根，$k = 0$。设 $y^* = Ae^{3t}$。
   $(9 - 9 + 2)A = 1$ → $A = \dfrac{1}{2}$。
   $y^* = \dfrac{1}{2}e^{3t} = \dfrac{x^3}{2}$。

3. **通解**：$y = C_1 x + C_2 x^2 + \dfrac{x^3}{2}$。

#### 例题 5

求 $x^2 y'' - xy' + 4y = x\sin(\ln x)$ 的通解（$x > 0$）。

**解：**

1. **齐次**：$D(D-1)y - Dy + 4y = 0$ → $(D^2 - 2D + 4)y = 0$。
   $r^2 - 2r + 4 = 0$，$r = 1 \pm \sqrt{3}i$。
   $Y = x(C_1\cos(\sqrt{3}\ln x) + C_2\sin(\sqrt{3}\ln x))$。

2. **特解**：$f(x) = x\sin(\ln x)$，换元后 $= e^t\sin t$。
   $(D^2 - 2D + 4)y = e^t\sin t$。
   $\lambda + \omega i = 1 + i$，不是特征根（特征根是 $1 \pm \sqrt{3}i$），$k = 0$。
   设 $y^* = e^t(A\cos t + B\sin t)$。
   代入计算得 $A = \dfrac{1}{6}$，$B = \dfrac{1}{3}$。
   $y^* = e^t\left(\dfrac{1}{6}\cos t + \dfrac{1}{3}\sin t\right) = x\left(\dfrac{1}{6}\cos(\ln x) + \dfrac{1}{3}\sin(\ln x)\right)$。

3. **通解**：$y = x\left(C_1\cos(\sqrt{3}\ln x) + C_2\sin(\sqrt{3}\ln x) + \dfrac{1}{6}\cos(\ln x) + \dfrac{1}{3}\sin(\ln x)\right)$。

---

## 四、$x < 0$ 的情形

当 $x < 0$ 时，令 $x = -e^t$（$t = \ln(-x)$），类似的推导仍然成立。最终结果中 $|x|$ 替代 $x$，$|\ln|x||$ 替代 $\ln x$。

实际考试中通常给出 $x > 0$ 的条件，或直接写 $|x|$ 形式的通解。

---

## 五、高阶欧拉方程

#### 例题 6

求 $x^3 y''' + x^2 y'' - 2xy' + 2y = 0$（$x > 0$）。

**解：** 换元 $x = e^t$：
- $xy' = Dy$
- $x^2 y'' = D(D-1)y$
- $x^3 y''' = D(D-1)(D-2)y$

代入：$D(D-1)(D-2)y + D(D-1)y - 2Dy + 2y = 0$

展开：$[D^3 - 3D^2 + 2D + D^2 - D - 2D + 2]y = 0$

$[D^3 - 2D^2 - D + 2]y = 0$

特征方程：$r^3 - 2r^2 - r + 2 = 0$。

试根：$r = 1$：$1 - 2 - 1 + 2 = 0$ ✓

因式分解：$(r-1)(r^2 - r - 2) = (r-1)(r-2)(r+1) = 0$

$r = 1, 2, -1$。

通解：$y = C_1 x + C_2 x^2 + C_3 x^{-1} = C_1 x + C_2 x^2 + \dfrac{C_3}{x}$。

## 六、求解步骤总结

> 1. **确认是欧拉方程**：$k$ 阶导数前乘 $x^k$。
> 2. **令 $x = e^t$**（$x > 0$），将方程转化为常系数方程。
> 3. **用算子 $D$ 替换**：$x^k y^{(k)} \to D(D-1)\cdots(D-k+1)y$。
> 4. **解常系数方程**：特征方程法求齐次通解 + 待定系数法求特解。
> 5. **换回 $x$**：$e^{rt} = x^r$，$t = \ln x$。

---

## 七、易错点总结

> 1. **换元后特征方程不是直接用原方程系数**：$x^2 y'' + pxy' + qy = 0$ 的特征方程是 $r^2 + (p-1)r + q = 0$，不是 $r^2 + pr + q = 0$。$p$ 要减 1！
> 2. **$x = e^t$ 要求 $x > 0$**：若题目没有限制 $x > 0$，需考虑 $x < 0$ 的情形。
> 3. **通解换回 $x$ 时不要遗漏**：$e^{rt} = x^r$，$t = \ln x$，不要把 $t$ 留在答案中。
> 4. **重根情形中 $te^{rt} = x^r \ln x$**，不要忘记 $\ln x$。
> 5. **复根情形中 $\alpha \pm \beta i$**：通解为 $x^\alpha[C_1\cos(\beta\ln x) + C_2\sin(\beta\ln x)]$，注意是 $\ln x$ 而不是 $x$ 在三角函数内部。
> 6. **标准形式**：先化为 $x^2 y'' + pxy' + qy = f(x)$ 的形式（首项系数为 1 的 $x^2 y''$），如果原方程首项系数不是 1，先除以该系数。
> 7. **非齐次方程的特解中 $f(e^t)$ 要正确替换**：如 $f(x) = x^3$ 变为 $f(e^t) = e^{3t}$。
