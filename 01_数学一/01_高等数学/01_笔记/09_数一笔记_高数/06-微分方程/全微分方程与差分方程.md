# 全微分方程与差分方程

> **06-微分方程** | **06-微分方程**

## 全微分方程

### 形式

$$P(x, y)\,dx + Q(x, y)\,dy = 0$$

若存在 $u(x, y)$ 使得 $du = P\,dx + Q\,dy$，则称该方程为**全微分方程**。

### 判定条件

**定理**：$P, Q$ 在单连通域 $D$ 上有连续一阶偏导数，则方程为全微分方程的充要条件是
$$\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$$

### 求解方法

#### 方法 1：凑微分

找到原函数 $u(x, y)$，通解为 $u(x, y) = C$。

**例**：$(3x^2 + 6xy^2)\,dx + (4y^3 + 6x^2y)\,dy = 0$

**解**：
```
P = 3x² + 6xy², Q = 4y³ + 6x²y
∂P/∂y = 12xy = ∂Q/∂x ✓

u = ∫P dx = x³ + 3x²y² + φ(y)
∂u/∂y = 6x²y + φ'(y) = Q
φ'(y) = 4y³, φ(y) = y⁴

通解：x³ + 3x²y² + y⁴ = C
```

#### 方法 2：积分因子法

当 $\frac{\partial P}{\partial y} \neq \frac{\partial Q}{\partial x}$ 时，找 $\mu(x)$ 或 $\mu(y)$ 使方程变为全微分。

**常见积分因子**：

| 条件 | 积分因子 |
|------|----------|
| $\frac{1}{Q}\left(\frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x}\right)$ 仅含 $x$ | $\mu(x)$ |
| $\frac{1}{P}\left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)$ 仅含 $y$ | $\mu(y)$ |
| $P, Q$ 为齐次 | $\frac{1}{xP + yQ}$ |

## 差分方程

### 概念

**差分方程**：包含未知函数差分的方程。

- **一阶前向差分**：$\Delta y_t = y_{t+1} - y_t$
- **二阶前向差分**：$\Delta^2 y_t = \Delta y_{t+1} - \Delta y_t$

### 一阶常系数线性差分方程

$$y_{t+1} + ay_t = f(t)$$

#### 齐次方程 $y_{t+1} + ay_t = 0$

通解：$y_t = C \cdot (-a)^t$

#### 非齐次方程

通解 = 齐次通解 + 特解

**特解形式**（按 $f(t)$ 类型）：

| $f(t)$ | 特解形式 |
|--------|----------|
| $C$（常数） | $y^*_t = A$（常数） |
| $C\lambda^t$ | $y^*_t = A\lambda^t$（若 $\lambda \neq -a$） |
| $C t^n$ | $y^*_t = t^n(B_n t^n + ... + B_0)$ |

### 例题

**例**：求 $y_{t+1} - 2y_t = 3 \cdot 2^t$ 的通解

**解**：
```
齐次：y_{t+1} - 2y_t = 0
特征根：λ = 2
齐次通解：y^(h)_t = C · 2^t

非齐次：f(t) = 3 · 2^t
λ = 2 是特征根，特解形式：y*_t = At · 2^t
代入：At · 2^(t+1) - 2 · At · 2^t = 3 · 2^t
2At · 2^t - 2At · 2^t = 3 · 2^t  ???

重设：y*_t = At · 2^t
y*_{t+1} = A(t+1) · 2^(t+1)
代入：A(t+1) · 2^(t+1) - 2At · 2^t = 3 · 2^t
2A(t+1) · 2^t - 2At · 2^t = 3 · 2^t
2A · 2^t = 3 · 2^t
A = 3/2

特解：y*_t = (3/2)t · 2^t

通解：y_t = C · 2^t + (3/2)t · 2^t
```

### 经济学应用

- **储蓄模型**：$S_t = \alpha Y_t$
- **收入-消费模型**：$Y_t = C_t + I_t$
- **Harrod-Domar 增长模型**
