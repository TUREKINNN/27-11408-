# Lecture 39：常系数非齐次线性微分方程

## 一、基本理论回顾

> [!important|blue] 定理：非齐次方程通解结构
> 非齐次方程 $y'' + py' + qy = f(x)$ 的通解为：
> $$y = Y + y^*$$
> 其中 $Y$ 是对应齐次方程 $y'' + py' + qy = 0$ 的通解，$y^*$ 是非齐次方程的一个**特解**。

**核心问题：如何求特解 $y^*$？**

本讲介绍**待定系数法**（也称"猜测法"），适用于 $f(x)$ 的特殊形式。

---

## 二、$f(x) = e^{\lambda x} P_m(x)$ 型

> [!important|blue] 定理：特解形式
> 设 $f(x) = e^{\lambda x} P_m(x)$，其中 $P_m(x)$ 是 $m$ 次多项式。
>
> 特解形式为：
> $$y^* = x^k e^{\lambda x} Q_m(x)$$
> 其中 $Q_m(x)$ 是**待定的 $m$ 次多项式**，$k$ 的取值为：
>
> | $\lambda$ 与特征根的关系 | $k$ 的值 |
> |:---:|:---:|
> | $\lambda$ 不是特征根 | $k = 0$ |
> | $\lambda$ 是单特征根 | $k = 1$ |
> | $\lambda$ 是二重特征根 | $k = 2$ |

**记忆口诀：** "$\lambda$ 不是根 $x^0$，单根 $x^1$，重根 $x^2$"——$k$ 等于 $\lambda$ 作为特征根的重数。

**直觉理解：** $e^{\lambda x}$ 本身已经是齐次方程的解时（$\lambda$ 是特征根），直接猜 $e^{\lambda x} Q_m(x)$ 代入方程会被"消掉"，所以需要乘以 $x^k$ 来调整。

#### 例题 1

求 $y'' - 3y' + 2y = e^{3x}$ 的通解。

**解：**

1. **齐次通解**：$r^2 - 3r + 2 = 0$，$(r-1)(r-2) = 0$，$Y = C_1 e^x + C_2 e^{2x}$。

2. **特解**：$f(x) = e^{3x}$，即 $P_0(x) = 1$（零次多项式），$\lambda = 3$。
   - 特征根 $r_1 = 1, r_2 = 2$，$\lambda = 3$ 不是特征根，$k = 0$。
   - 设 $y^* = Ae^{3x}$。
   - $y^{*\prime} = 3Ae^{3x}$，$y^{*\prime\prime} = 9Ae^{3x}$。
   - 代入：$9A - 9A + 2A = 1$ → $2A = 1$ → $A = \dfrac{1}{2}$。

3. **通解**：$y = C_1 e^x + C_2 e^{2x} + \dfrac{1}{2}e^{3x}$。

#### 例题 2

求 $y'' - 3y' + 2y = e^x$ 的通解。

**解：**

1. 齐次通解同上：$Y = C_1 e^x + C_2 e^{2x}$。

2. 特解：$\lambda = 1$ 是单特征根，$k = 1$。
   - 设 $y^* = Axe^x$。
   - $y^{*\prime} = A(x+1)e^x$，$y^{*\prime\prime} = A(x+2)e^x$。
   - 代入：$A(x+2) - 3A(x+1) + 2Ax = 1$ → $A(2-3) = 1$ → $-A = 1$ → $A = -1$。

3. 通解：$y = C_1 e^x + C_2 e^{2x} - xe^x$。

#### 例题 3

求 $y'' - 2y' + y = (x+1)e^x$ 的通解。

**解：**

1. 齐次：$r^2 - 2r + 1 = 0$，$(r-1)^2 = 0$，$r = 1$（重根），$Y = (C_1 + C_2 x)e^x$。

2. 特解：$\lambda = 1$ 是二重根，$k = 2$。$P_1(x) = x + 1$（一次多项式）。
   - 设 $y^* = x^2(Ax + B)e^x$。
   - 计算 $y^{*\prime}$ 和 $y^{*\prime\prime}$ 并代入（过程略）：
   - 最终得 $A = \dfrac{1}{3}$，$B = 1$。

3. 通解：$y = (C_1 + C_2 x)e^x + x^2\left(\dfrac{x}{3} + 1\right)e^x$。

#### 例题 4

求 $y'' + y = 2x^2 - 3$ 的通解。

**解：**

1. 齐次：$r^2 + 1 = 0$，$r = \pm i$，$Y = C_1\cos x + C_2\sin x$。

2. 特解：$f(x) = 2x^2 - 3$，$\lambda = 0$。特征根为 $\pm i$，$\lambda = 0$ 不是特征根，$k = 0$。
   - 设 $y^* = Ax^2 + Bx + C$。
   - $y^{*\prime\prime} = 2A$。
   - 代入：$2A + Ax^2 + Bx + C = 2x^2 - 3$。
   - 比较系数：$A = 2$，$B = 0$，$2A + C = -3$ → $C = -7$。

3. 通解：$y = C_1\cos x + C_2\sin x + 2x^2 - 7$。

## 三、$f(x) = e^{\lambda x}[P_m(x)\cos\omega x + P_n(x)\sin\omega x]$ 型

> [!important|blue] 定理：三角函数型特解
> 设 $f(x) = e^{\lambda x}[P_l(x)\cos\omega x + P_n(x)\sin\omega x]$，其中 $P_l, P_n$ 分别是 $l$ 次和 $n$ 次多项式。
>
> 令 $m = \max(l, n)$，特解形式为：
> $$y^* = x^k e^{\lambda x}[Q_m(x)\cos\omega x + R_m(x)\sin\omega x]$$
> 其中 $Q_m(x), R_m(x)$ 是待定的 $m$ 次多项式，$k$ 的取值为：
>
> | $\lambda + \omega i$ 与特征根的关系 | $k$ 的值 |
> |:---:|:---:|
> | $\lambda + \omega i$ 不是特征根 | $k = 0$ |
> | $\lambda + \omega i$ 是特征根 | $k = 1$ |

**关键：** 判断 $\lambda + \omega i$ 是否为特征根（注意是复数形式）。

#### 例题 5

求 $y'' + y = \cos x$ 的通解。

**解：**

1. 齐次：$Y = C_1\cos x + C_2\sin x$。

2. 特解：$f(x) = \cos x$，即 $\lambda = 0$，$\omega = 1$，$\lambda + \omega i = i$。
   - 特征根 $r = \pm i$，$i$ 是特征根，$k = 1$。
   - 设 $y^* = x(A\cos x + B\sin x)$。
   - $y^{*\prime} = (A\cos x + B\sin x) + x(-A\sin x + B\cos x)$
   - $y^{*\prime\prime} = 2(-A\sin x + B\cos x) + x(-A\cos x - B\sin x)$
   - 代入 $y'' + y = \cos x$：
     $2(-A\sin x + B\cos x) = \cos x$
   - $2B = 1$，$-2A = 0$ → $A = 0$，$B = \dfrac{1}{2}$。

3. 通解：$y = C_1\cos x + C_2\sin x + \dfrac{x}{2}\sin x$。

#### 例题 6

求 $y'' - 2y' + 5y = e^x\sin 2x$ 的通解。

**解：**

1. 齐次：$r^2 - 2r + 5 = 0$，$r = 1 \pm 2i$，$Y = e^x(C_1\cos 2x + C_2\sin 2x)$。

2. 特解：$\lambda = 1$，$\omega = 2$，$\lambda + \omega i = 1 + 2i$。
   - $1 + 2i$ 是特征根，$k = 1$。
   - 设 $y^* = xe^x(A\cos 2x + B\sin 2x)$。
   - 代入计算（过程较长）：
     最终得 $A = -\dfrac{1}{4}$，$B = 0$。

3. 通解：$y = e^x(C_1\cos 2x + C_2\sin 2x) - \dfrac{x}{4}e^x\cos 2x$。

#### 例题 7

求 $y'' + 4y = \sin 2x$ 的特解。

**解：**

齐次：$r^2 + 4 = 0$，$r = \pm 2i$。

$\lambda = 0$，$\omega = 2$，$\lambda + \omega i = 2i$ 是特征根，$k = 1$。

设 $y^* = x(A\cos 2x + B\sin 2x)$。

$y^{*\prime\prime} + 4y^* = -4A\sin 2x + 4B\cos 2x = \sin 2x$。

$-4A = 1$，$4B = 0$ → $A = -\dfrac{1}{4}$，$B = 0$。

$y^* = -\dfrac{x}{4}\cos 2x$。

#### 例题 8

求 $y'' + y = 2\cos x + 3\sin x$ 的特解。

**解：** $\lambda = 0$，$\omega = 1$，$i$ 是特征根，$k = 1$。

设 $y^* = x(A\cos x + B\sin x)$。

代入得 $-2A\sin x + 2B\cos x = 2\cos x + 3\sin x$。

$2B = 2$，$-2A = 3$ → $A = -\dfrac{3}{2}$，$B = 1$。

$y^* = x\left(-\dfrac{3}{2}\cos x + \sin x\right)$。

## 四、两种类型的对比总结

| 右端 $f(x)$ | 特解形式 | $k$ 的判断 |
|:---:|:---:|:---:|
| $e^{\lambda x}P_m(x)$ | $x^k e^{\lambda x}Q_m(x)$ | $\lambda$ 作为特征根的重数 |
| $e^{\lambda x}[P_l\cos\omega x + P_n\sin\omega x]$ | $x^k e^{\lambda x}[Q_m\cos\omega x + R_m\sin\omega x]$ | $\lambda + \omega i$ 是特征根则 $k=1$，否则 $k=0$ |

> 若 $f(x) = f_1(x) + f_2(x)$，可分别求 $f_1$ 和 $f_2$ 对应的特解 $y_1^*$ 和 $y_2^*$，则 $y^* = y_1^* + y_2^*$（叠加原理）。

#### 例题 9

求 $y'' + y = x + \cos x$ 的特解。

**解：** 分成两部分：
- $y'' + y = x$：设 $y_1^* = Ax + B$，代入得 $Ax + B = x$，$A = 1$，$B = 0$。$y_1^* = x$。
- $y'' + y = \cos x$：由例题 7 类似方法，$y_2^* = \dfrac{x}{2}\sin x$。

$y^* = x + \dfrac{x}{2}\sin x$。

## 五、求解步骤总结

> 1. **求齐次通解 $Y$**：解特征方程 $r^2 + pr + q = 0$。
> 2. **判断 $f(x)$ 的类型**：是 $e^{\lambda x}P_m(x)$ 型还是三角函数型。
> 3. **确定特解形式**：确定 $k$ 和待定多项式的次数。
> 4. **代入原方程**：比较系数，确定待定常数。
> 5. **写出通解**：$y = Y + y^*$。
> 6. **如有初始条件**：代入通解确定 $C_1, C_2$。

---

## 六、易错点总结

> 1. **$k$ 的取值是核心**：$\lambda$ 不是特征根时 $k=0$，是单根时 $k=1$，是重根时 $k=2$。忘记乘 $x^k$ 是最常见错误。
> 2. **三角函数型中 $Q_m$ 和 $R_m$ 都要设**：即使原题只有 $\cos$ 或只有 $\sin$，特解中 $\cos$ 和 $\sin$ 项都要保留（除非通过计算发现某系数为零）。
> 3. **$m = \max(l, n)$**：$P_l$ 和 $P_n$ 次数不同时，$Q_m$ 和 $R_m$ 的次数取较大者。
> 4. **$e^{\lambda x}$ 的 $\lambda$ 是指数中的系数**：如 $e^{2x}$ 的 $\lambda = 2$，不要和 $2e^x$（$\lambda = 1$, 系数为 2）混淆。
> 5. **叠加原理**：$f(x) = f_1(x) + f_2(x)$ 时可分别求特解再相加，但注意每部分的 $k$ 要分别判断。
> 6. **代入求系数时的计算**：$y^*$ 代入后要仔细整理同类项，特别是三角函数型。
