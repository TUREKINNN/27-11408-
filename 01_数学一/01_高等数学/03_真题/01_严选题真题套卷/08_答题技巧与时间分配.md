## 🎯 三、选填题 7 大秒杀技巧

### 3.1 特例法

**原理**：将抽象问题用具体值代入

**适用**：判别题、存在性题、抽象函数题

**例**：判断 f(x)=|x|sin(1/x) 在 x=0 处是否可导
- 特例 x>0：f(x) = x·sin(1/x)
- lim h→0 [f(h)-f(0)]/h = lim sin(1/h) 不存在
- 结论：不可导

### 3.2 反例法

**原理**：举反例排除错误选项

**适用**：恒成立判断、充分必要条件

**例**：判断"若 f(x) 在 [0,1] 连续，则必有最大值"
- 反例：f(x) = 1/(x-0.5)（除去 0.5 点）
- 该函数在 [0,1] 不连续（除 0.5 外）→ 排除

### 3.3 几何意义

**原理**：将代数问题转为几何

**适用**：积分/导数/向量题

**例**：∫_0^1 √(1-x²) dx = π/4
- 几何意义：单位圆在第一象限面积
- 不用计算即可得答案

### 3.4 数形结合

**原理**：画图辅助判断

**适用**：多元函数、方程根、极值

**例**：方程 ln x = -x² + 1 的根
- 画图：ln x 与抛物线 -x²+1 交点
- 1 个根

### 3.5 量纲分析

**原理**：利用量纲（单位）检查答案

**适用**：物理应用题、定积分

**例**：求 d/dt ∫_0^t f(x) dx
- 量纲：被积函数 f(x) 的量纲 = dx
- = f(t) 量纲正确

### 3.6 估值法

**原理**：用夹逼估计数值范围

**适用**：证明题、不等式题

**例**：证明 ln(1+x) ≤ x
- 0 ≤ ∫_0^x (1/(1+t)) dt ≤ ∫_0^x 1 dt = x

### 3.7 对称性

**原理**：利用函数对称性简化

**适用**：积分、行列式、矩阵

**例**：∫_{-a}^a f(x) dx
- 若 f 偶：= 2∫_0^a f(x) dx
- 若 f 奇：= 0

## ✍️ 四、解答题 7 大答题规范

### 4.1 解题 7 步法

```
第 1 步：审题（30 秒）
  - 标出已知条件
  - 标出所求
  - 联想相关公式

第 2 步：写方案（30 秒）
  - 列出本题关键步骤
  - 写出所用的核心公式

第 3 步：详细计算（10 分钟）
  - 分步标号
  - 写出关键公式
  - 化简到底

第 4 步：检查（2 分钟）
  - 特殊值验证
  - 量纲检查
  - 极限情况

第 5 步：誊写（备用时间）
  - 字体清晰
  - 标点规范

第 6 步：检查答案（1 分钟）
  - 是否化简
  - 是否漏负号
  - 是否漏条件

第 7 步：检查卷面（1 分钟）
  - 是否漏题
  - 是否漏空
```

### 4.2 解答题 5 大扣分雷区

| 雷区 | 扣分 | 防范 |
|------|------|------|
| **不写"解"** | -1 | 写"解："或"证明：" |
| **不写公式** | -2 | 关键公式必写 |
| **计算跳步** | -2 | 重要步骤必写 |
| **结果未化简** | -2 | 化简为最简 |
| **缺单位/定义域** | -1 | 注意上下文 |

### 4.3 解答题 5 大抢分技巧

| 技巧 | 适用 | 抢分点 |
|------|------|--------|
| **写出公式** | 不会做 | 公式分 |
| **写思路** | 计算卡 | 步骤分 |
| **特殊值代入** | 求值 | 验证分 |
| **写出关键结论** | 时间紧 | 结论分 |
| **反向验算** | 任何题 | 减少失误 |

## 🎲 五、不会做的题怎么办？

### 5.1 选填题不会做（4 大策略）

| 策略 | 操作 | 得分概率 |
|------|------|---------|
| **猜 1 个** | 选 B 或 C | 25% |
| **排除 2 个** | 在剩下选 | 50% |
| **特例反例** | 用特例排除 | 50-75% |
| **蒙** | 凭感觉 | 25% |

### 5.2 解答题不会做（5 大策略）

```
1. 抄已知条件（1 分）
   - 列出题目所有条件
   - 写"由已知..."
   
2. 抄相关公式（2-3 分）
   - 写出本题可能用到的公式
   - 写"由 ... 公式得..."

3. 写思路（2-3 分）
   - 写出"设 ... 求 ..."
   - 写"要证 ... 只需证 ..."

4. 计算关键步骤（4-6 分）
   - 即便不会，也能写出 1-2 步

5. 写部分结论（1-2 分）
   - 写出"由题意得..." 之类的废话
```

### 5.3 压轴题（最后一题）不会做

```
策略：拿步骤分
- 写出题目条件（1-2 分）
- 写相关公式（1-2 分）
- 写出关键思路（1-2 分）
- 计算前 2 步（2-3 分）
- 写部分结论（1-2 分）

预期：5-10 分（不会做也能拿）
```

## 🛡️ 六、6 大应急策略

### 6.1 时间不够

```
1. 选择题：直接蒙（每题 25%）
2. 填空题：写公式
3. 解答题：写关键步骤
4. 放弃最后一题或最后 2 题
```

### 6.2 计算卡住

```
1. 跳过，做下一题
2. 用特例验算
3. 用几何意义判断
```

### 6.3 思路中断

```
1. 回到定义
2. 用特例法试
3. 类比类似题
```

### 6.4 出现粗心错

```
1. 标出关键步骤
2. 留 5 分钟检查
3. 用特例法验算
```

### 6.5 完全不会

```
1. 选填：25% 蒙
2. 解答：写公式 + 思路
3. 至少拿 2-3 分
```

### 6.6 紧张

```
1. 深呼吸 3 次
2. 先做最易题
3. 找回节奏
```
