## 📝 四、答题策略

### 4.1 选择题（8 道 × 4 分 = 32 分）

**3 大核心策略**：

#### 策略 1：直接法（首选）

```
对简单题直接计算
耗时：1-2 分钟/题
适合：极限、积分、矩阵运算
```

#### 策略 2：排除法（次选）

```
对难计算题用反例排除
耗时：1-2 分钟/题
适合：抽象题、判别题
```

#### 策略 3：特殊值法（应急）

```
代入特殊值快速判断
耗时：30 秒-1 分钟
适合：选择题任何题型
```

#### 选择题 4 大技巧

| 技巧 | 适用 | 示例 |
|------|------|------|
| **特例法** | 抽象判别 | 令 n=1, x=0 |
| **反例法** | "恒成立"判断 | 找反例排除 |
| **几何意义** | 积分/导数 | 看图判断 |
| **数形结合** | 多元函数 | 画等高线 |

### 4.2 填空题（6 道 × 4 分 = 24 分）

**3 大注意事项**：
1. **不需要过程**：只填答案
2. **结果必须化简**：如 2/3 不写 0.667
3. **看清定义域**：如 x>0

### 4.3 解答题（9 道 = 94 分）

**7 大答题步骤**：

```
1. 抄题：先抄关键条件
2. 画图：几何题必画图
3. 标号：分步标号，方便检查
4. 写公式：每步写明公式依据
5. 计算：化简到底，结果必须化简
6. 检验：特殊值验证
7. 总结：难度题写出关键思路
```

#### 解答题 5 大拿分原则

| 原则 | 说明 |
|------|------|
| **过程清晰** | 每步标号 |
| **公式正确** | 关键公式必须写出 |
| **计算准确** | 化简到底 |
| **结果规范** | 单位/最简 |
| **检查** | 特殊值法验算 |

## ⚠️ 六、2023 年易错点预警

### 6.1 高数 8 大易错

1. **极限**：忽视定义域（如 x→0 但 x≠0）
2. **导数**：复合求导漏项
3. **积分**：分段函数漏分段
4. **微分方程**：齐次/非齐次判断错
5. **多元函数**：偏导存在 ≠ 连续
6. **二重积分**：极坐标变换漏雅可比
7. **级数**：交错级数审敛漏莱布尼茨
8. **中值定理**：辅助函数构造错

### 6.2 线代 5 大易错

1. **行列式**：|kA| = k^n|A|（非 k|A|）
2. **矩阵**：AB=O 不能推出 A=O 或 B=O
3. **特征值**：忘记求特征向量
4. **对角化**：判断条件（n 个无关特征向量）
5. **二次型**：正定判别（顺序主子式 > 0）

### 6.3 概率 5 大易错

1. **分布**：连续型忘写密度函数
2. **独立性**：互斥 ≠ 独立
3. **条件概率**：P(A|B) = P(AB)/P(B)
4. **期望**：E(XY) = E(X)E(Y) 仅独立时
5. **估计**：最大似然估计求解步骤
