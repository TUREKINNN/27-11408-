### 行列式公式

| 公式 | 位置 |
|------|------|
| $|A| = \sum_{j=1}^{n} a_{1j}A_{1j}$（按行展开） | Lecture 01 |

| $|AB| = |A||B|$（行列式乘法） | Lecture 01 |
| $|kA| = k^n|A|$（数乘行列式） | Lecture 01 |
| $|A^{-1}| = \frac{1}{|A|}$（逆矩阵行列式） | Lecture 01 |

### 矩阵运算公式

| 公式 | 位置 |
|------|------|
| $(\mathbf{A}\mathbf{B})\mathbf{C} = \mathbf{A}(\mathbf{B}\mathbf{C})$（结合律） | Lecture 02 |
| $(\mathbf{A}\mathbf{B})^T = \mathbf{B}^T\mathbf{A}^T$（转置） | Lecture 02 |
| $(\mathbf{A}^{-1})^{-1} = \mathbf{A}$（逆矩阵性质） | Lecture 02 |
| $\mathbf{A}\mathbf{A}^* = \mathbf{A}^*\mathbf{A} = |A|\mathbf{E}$（伴随矩阵） | Lecture 02 |
| $r(\mathbf{A}) = r(\mathbf{A}^T) = r(\mathbf{A}^T\mathbf{A})$（秩的性质） | Lecture 02 |

### 线性方程组公式

| 公式 | 位置 |
|------|------|
| $\mathbf{A}\mathbf{x} = \mathbf{b}$（矩阵形式） | Lecture 04 |

| $r(\mathbf{A}) < n$（齐次方程组有非零解） | Lecture 04 |
| $r(\mathbf{A}) = r(\bar{\mathbf{A}})$（非齐次方程组有解） | Lecture 04 |

| $\sum \lambda_i = \text{tr}(\mathbf{A})$（迹） | Lecture 05 |
| $\prod \lambda_i = |\mathbf{A}|$（行列式） | Lecture 05 |
