## 知识脉络图

```
行列式 ──→ 矩阵运算 ──→ 初等变换/秩 ──→ 分块矩阵
  │              │
  ▼              ▼
向量组相关性 ←── 线性方程组 ──→ 解的结构（齐次/非齐次）
                    │
                    ▼
           特征值与特征向量
                    │
                    ▼
              相似对角化
                    │
                    ▼
          二次型 ← 正交变换 → 正定性判定
```

**核心主线**：行列式 → 矩阵 → 向量 → 方程组 → 特征值 → 二次型，六章内容环环相扣。

## 复习建议

1. **行列式计算**必须熟练（展开法、逐行化简、递推法）
2. **矩阵可逆 ↔ 满秩 ↔ 行列式非零 ↔ 齐次方程只有零解 ↔ 特征值全非零** — 这条等价链贯穿全局
3. **实对称矩阵**必可正交对角化，是二次型的基石
4. 每个**二次型**题都要能还原到**矩阵**运算
