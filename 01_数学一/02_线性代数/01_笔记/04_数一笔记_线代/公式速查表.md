# 线性代数公式速查表

---

## 一、行列式

### 基本性质
$$|A^T| = |A|, \quad |kA| = k^n|A| \text{（n 为阶数）}$$
$$|AB| = |A||B|, \quad |A^{-1}| = \frac{1}{|A|}, \quad |A^*| = |A|^{n-1}$$

### 展开定理
$$|A| = \sum_{j=1}^{n} a_{ij} A_{ij} \text{（按第 i 行展开）}$$
$$\sum_{j=1}^{n} a_{ij} A_{kj} = 0 \quad (i \neq k)$$

### 常用行列式
- **上/下三角行列式**：$|A| = a_{11}a_{22}\cdots a_{nn}$
- **对角行列式**：$|A| = \prod_{i=1}^n a_{ii}$
- **范德蒙德行列式**：$\prod_{1 \le j < i \le n}(x_i - x_j)$

---

## 二、矩阵

### 逆矩阵
$$A^{-1} = \frac{1}{|A|} A^* \text{（伴随矩阵法）}$$
$$\text{求法：} [A|E] \xrightarrow{\text{初等行变换}} [E|A^{-1}]$$

### 伴随矩阵性质
$$AA^* = A^*A = |A|E$$
$$(A^*)^{-1} = (A^{-1})^* = \frac{1}{|A|}A$$
$$|A^*| = |A|^{n-1}, \quad (A^*)^* = |A|^{n-2}A$$

### 秩的公式
$$r(A) = r(A^T)$$
$$r(AB) \le \min\{r(A), r(B)\}$$
$$r(A+B) \le r(A) + r(B)$$
$$r(A) + r(B) - n \le r(AB) \quad \text{（Sylvester 不等式）}$$

---

## 三、线性方程组

### 有解判定
- $r(A) = r(\bar{A}) = n$：唯一解
- $r(A) = r(\bar{A}) < n$：无穷多解
- $r(A) \neq r(\bar{A})$：无解

### 解的结构
- **齐次**：$\alpha = k_1\xi_1 + k_2\xi_2 + \cdots + k_{n-r}\xi_{n-r}$（基础解系线性组合）
- **非齐次**：$x = \alpha^* + k_1\xi_1 + \cdots + k_{n-r}\xi_{n-r}$（特解 + 齐次通解）

---

## 四、特征值与特征向量

### 基本公式
$$Ax = \lambda x, \quad |A - \lambda E| = 0$$

### 特征值性质
$$\sum_{i=1}^n \lambda_i = \text{tr}(A) = \sum_{i=1}^n a_{ii}$$
$$\prod_{i=1}^n \lambda_i = |A|$$

### 常见矩阵的特征值
| 矩阵 | 特征值 |
|------|--------|
| $A$ | $\lambda$ |
| $A^k$ | $\lambda^k$ |
| $aA+bE$ | $a\lambda+b$ |
| $A^{-1}$ | $1/\lambda$ |
| $A^*$ | $\|A\|/\lambda$ |
| $A^T$ | $\lambda$ |

### 对角化
$$A \sim \Lambda \iff A \text{ 有 n 个线性无关的特征向量}$$
$$P^{-1}AP = \Lambda \implies A = P\Lambda P^{-1}$$

### 实对称矩阵
- 特征值均为实数
- 不同特征值的特征向量正交
- 必可正交对角化：$Q^TAQ = \Lambda$（$Q$ 为正交矩阵）

---

## 五、二次型

### 标准形
$$f(x) = x^TAx \xrightarrow{x=Cy} d_1y_1^2 + d_2y_2^2 + \cdots + d_ny_n^2$$

### 正定判定（充要条件）
1. 所有特征值 $> 0$
2. 各阶顺序主子式 $> 0$
3. 正惯性指数 $= n$
4. 存在可逆矩阵 $P$ 使 $A = P^TP$

### 惯性定理
$$p + q = r(A), \quad p - q = \text{符号差}$$
其中 $p$ 为正惯性指数，$q$ 为负惯性指数。

---

## 六、向量空间

### 过渡矩阵
$$(\beta_1, \beta_2, \beta_3) = (\alpha_1, \alpha_2, \alpha_3)P$$
$$P = (\alpha_1, \alpha_2, \alpha_3)^{-1}(\beta_1, \beta_2, \beta_3)$$

### 正交化（施密特）
$$\beta_1 = \alpha_1$$
$$\beta_2 = \alpha_2 - \frac{(\alpha_2, \beta_1)}{(\beta_1, \beta_1)}\beta_1$$
$$\beta_3 = \alpha_3 - \frac{(\alpha_3, \beta_1)}{(\beta_1, \beta_1)}\beta_1 - \frac{(\alpha_3, \beta_2)}{(\beta_2, \beta_2)}\beta_2$$
