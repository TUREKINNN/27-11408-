# 线性代数学习路径

## 📅 总体规划（约 6-8 周）

---

## 第一阶段：基础夯实（2 周）

### 第 1 周：行列式
- [ ] 行列式的定义（逆序数法、递推法）
- [ ] 行列式的性质（转置、换行、提公因子、拆分）
- [ ] 行列式按行/列展开定理
- [ ] 计算方法：三角化法、加边法、递推法、范德蒙德行列式
- [ ] 克拉默法则

### 第 2 周：矩阵
- [ ] 矩阵的定义与运算（加、减、乘、转置）
- [ ] 逆矩阵的定义、性质与求法（伴随矩阵法、初等变换法）
- [ ] 初等变换与初等矩阵
- [ ] 矩阵的秩
- [ ] 分块矩阵的运算

---

## 第二阶段：核心突破（2 周）

### 第 3 周：向量
- [ ] 向量的线性运算
- [ ] 线性相关与线性无关
- [ ] 向量组的秩与极大线性无关组
- [ ] 向量空间、基、维数、坐标

### 第 4 周：线性方程组
- [ ] 齐次线性方程组（基础解系、解的结构）
- [ ] 非齐次线性方程组（特解、通解）
- [ ] 有解判定定理
- [ ] 方程组与矩阵秩的关系

---

## 第三阶段：进阶提升（2 周）

### 第 5 周：特征值与特征向量
- [ ] 特征值与特征向量的定义与求法
- [ ] 特征多项式、特征方程
- [ ] 相似矩阵与对角化
- [ ] 实对称矩阵的正交对角化

### 第 6 周：二次型
- [ ] 二次型的定义与矩阵表示
- [ ] 合同变换与标准形
- [ ] 正定二次型与正定矩阵的判定
- [ ] 惯性定理

---

## 第四阶段：冲刺综合（1-2 周）

- [ ] 章节间联系梳理（矩阵 ↔ 方程组 ↔ 向量组）
- [ ] 历年真题分类训练
- [ ] 错题回顾与薄弱环节强化
- [ ] 模拟套卷限时训练

---

## 💡 学习建议

1. **矩阵是核心工具**：行列式、方程组、特征值都离不开矩阵运算
2. **理解线性相关性的几何意义**：这是贯穿线代的灵魂概念
3. **重视计算能力**：考研线代题计算量大，务必多练手
4. **注意各章联系**：方程组的解 = 向量组的线性表示问题
