## 📐 核心公式索引 (一站式查表)

>
> 复习时遇到具体公式, 直接查本表定位章节 + 速记口诀

### 第一章 行列式

| 公式 | 名称 | 应用场景 |
|------|------|---------|
| $\det A = \det A^T$ | 转置不变 | 行列互换 |
| $\|kA\| = k^n \|A\|$ | 数乘提公因子 | 缩放 |
| $\|AB\| = \|A\| \\|B\|$ | 乘积公式 | 分块矩阵 |
| $\|A^{-1}\| = 1/\|A\|$ | 逆矩阵 | 可逆判定 |
| $\|A^*\| = \|A\|^{n-1}$ | 伴随矩阵 | $A^*$ 计算 |
| 范德蒙德 $\|V_n\| = \prod_{1 \le j < i \le n} (x_i - x_j)$ | 范德蒙德 | Vandermonde |

### 第二章 矩阵

| 公式 | 名称 | 应用 |
|------|------|-----|
| $(AB)^{-1} = B^{-1} A^{-1}$ | 逆矩阵反转 | 矩阵方程 |
| $(AB)^T = B^T A^T$ | 转置反转 | 转置计算 |
| $A^* = \|A\| A^{-1}$ | 伴随矩阵 | $A^*$ 公式 |
| $(kA)^* = k^{n-1} A^*$ | 缩放伴随 | 缩放后伴随 |
| $r(AB) \le \min\{r(A), r(B)\}$ | 秩不等式 | 秩估计 |
| $r(A^T A) = r(A)$ | 重要等式 | 最小二乘 |
| $r(A) + r(B) - n \le r(AB) \le \min\{r(A), r(B)\}$ | Frobenius | 秩精确范围 |
| $A$ 可逆 $\Leftrightarrow$ $\|A\| \ne 0 \Leftrightarrow r(A) = n \Leftrightarrow$ 满秩 | **可逆 10 等价** | 可逆判定 |

### 第三章 向量组

| 公式/定理 | 名称 | 应用 |
|----------|------|-----|
| $\alpha_1, \alpha_2, \ldots, \alpha_m$ 线性相关 $\Leftrightarrow$ 存在不全为 0 的 $k_i$ 使 $\sum k_i \alpha_i = 0$ | **相关性定义** | 相关性判定 |
| $n+1$ 个 $n$ 维向量必线性相关 | **多则相关** | 维数约束 |
| 线性无关组加长仍无关, 缩短不一定 | 增减性质 | 无关性传递 |
| $r(\alpha_1, \alpha_2, \ldots, \alpha_m) = r(A)$ | 矩阵秩 = 向量组秩 | 秩计算 |
| $r(A) + r(B) - n \le r(AB) \le \min\{r(A), r(B)\}$ | 同上 | 矩阵乘积的秩 |
| $\alpha_1, \ldots, \alpha_s$ 线性无关 $\Leftrightarrow r(\alpha_1, \ldots, \alpha_s) = s$ | **秩 = 个数 ⇔ 无关** | 无关判定 |
| $A$ 列满秩 $\Leftrightarrow Ax = 0$ 只有零解 | 列满秩 | 解结构 |

### 第四章 线性方程组

| 公式/定理 | 名称 | 应用 |
|----------|------|-----|
| $Ax = b$ 有解 $\Leftrightarrow r(A) = r(A \\| b)$ | **有解判定** | 求解可行性 |
| $Ax = b$ 唯一解 $\Leftrightarrow r(A) = n$ | 唯一解 | 满秩 |
| $Ax = b$ 无穷多解 $\Leftrightarrow r(A) = r(A \\| b) < n$ | 无穷多解 | 基础解系 |
| $Ax = 0$ 基础解系含 $n - r(A)$ 个解 | **基础解系个数** | 自由度 |
| $Ax = b$ 通解 = 特解 + 齐次通解 | **非齐次通解结构** | 通解构造 |
| $A$ 的行向量组与 $A^T x = 0$ 解正交 | 行向量与转置解 | 正交补 |
| $AB = O \Rightarrow B$ 的列是 $Ax = 0$ 的解 | **$AB = O$ 推论** | 解结构 |

### 第五章 特征值与特征向量

| 公式/定理 | 名称 | 应用 |
|----------|------|-----|
| $A \xi = \lambda \xi$ ($\xi \ne 0$) | **定义** | 特征值 |
| $\|A - \lambda E\| = 0$ | **特征方程** | 求解 $\lambda$ |
| $\sum \lambda_i = \text{tr}(A) = \sum a_{ii}$ | **迹 = 特征值和** | 迹公式 |
| $\prod \lambda_i = \|A\|$ | 特征值积 = 行列式 | 行列式 |
| $\lambda$ 单根 $\Rightarrow$ 对应 1 个线性无关特征向量 | 单根 1 维 | 几何重数 |
| $k$ 重根 $\Rightarrow$ 至少 1 个, 至多 $k$ 个 | 重根范围 | 几何重数 |
| $A \sim \Lambda \Leftrightarrow A$ 有 $n$ 个线性无关特征向量 | **可对角化充要** | 对角化 |
| 实对称 $\Rightarrow$ 必可正交对角化 | **实对称特殊** | 正交相似 |
| $A$ 的不同 $\lambda$ 对应 $\xi$ 必线性无关 | 不同 $\lambda$ | 无关性 |

### 第六章 二次型

| 公式/定理 | 名称 | 应用 |
|----------|------|-----|
| $f(x_1, \ldots, x_n) = x^T A x$ ($A$ 对称) | **矩阵表示** | 二次型 |
| $A$ 与 $B$ 合同 $\Leftrightarrow \exists P$ 可逆使 $B = P^T A P$ | 合同 | 等价关系 |
| 任何二次型可用正交变换化为标准形 $\sum \lambda_i y_i^2$ | **主轴定理** | 标准形 |
| 任何二次型可用配方法化为标准形 | 配方 | 配方法 |
| 惯性定理: 标准形正项个数 $p$ 负项个数 $q$ 不变 | **惯性定理** | 规范形 |
| $A$ 正定 $\Leftrightarrow$ 所有 $\lambda_i > 0$ | 特征值判别 | 正定 |
| $A$ 正定 $\Leftrightarrow$ 所有顺序主子式 > 0 | **顺序主子式** | 正定 |
| $A$ 正定 $\Leftrightarrow$ 合同于单位阵 | 合同单位 | 正定 |

### 🟡 易错 10 大陷阱 (考前必看)

| # | 陷阱 | 错误率 | 正确理解 |
|---|------|--------|----------|
| 1 | $\|kA\| = k \|A\|$ | ❌ 90% | $\|kA\| = k^n \|A\|$ ($n$ 阶) |
| 2 | $A^T = A^{-1}$ | ❌ 70% | 仅正交阵成立 |
| 3 | $AB = O \Rightarrow A = O$ 或 $B = O$ | ❌ 80% | $A, B$ 均可能非零 |
| 4 | $r(A + B) = r(A) + r(B)$ | ❌ 60% | $r(A + B) \le r(A) + r(B)$ |
| 5 | $A \sim B \Rightarrow A = B$ | ❌ 50% | 相似只是特征值相同, 矩阵可不同 |
| 6 | $A$ 可对角化 $\Rightarrow A$ 对称 | ❌ 40% | 反过来: 对称必可对角化 |
| 7 | $A, B$ 正定 $\Rightarrow AB$ 正定 | ❌ 30% | 不一定, 但 $A + B$ 正定 |
| 8 | $n$ 阶矩阵有 $n$ 个特征值 $\Rightarrow$ 可对角化 | ❌ 80% | 还需线性无关 |
| 9 | $A^T A = O \Rightarrow A = O$ | ❌ 20% | 正确 (实矩阵) |
| 10 | $A^2 = A \Rightarrow A$ 可逆 | ❌ 50% | 幂等矩阵不一定可逆 |

## ⚡ 考前 24 小时速查表


### ✅ 必记 30 个核心结论

1. $\|kA\| = k^n \|A\|$ (数乘)
2. $\|A^*\| = \|A\|^{n-1}$ (伴随)
3. $A^* A = AA^* = \|A\| E$ (伴随核心)
4. $(AB)^{-1} = B^{-1} A^{-1}$ (逆反转)
5. $(AB)^T = B^T A^T$ (转置反转)
6. $r(AB) \le \min\{r(A), r(B)\}$ (秩不等式)
7. $r(A) + r(B) - n \le r(AB)$ (Frobenius)
8. $r(A^T A) = r(A)$ (重要等式)
9. $A$ 可逆 $\Leftrightarrow \|A\| \ne 0 \Leftrightarrow r(A) = n$
10. 线性无关组 + 一个向量 = 仍无关; + 相关组 = 必相关
11. $n+1$ 个 $n$ 维向量必相关
12. $r(\alpha_1, \ldots, \alpha_s) = s \Leftrightarrow$ 线性无关
13. $Ax = 0$ 基础解系含 $n - r(A)$ 个解
14. $Ax = b$ 有解 $\Leftrightarrow r(A) = r(A \| b)$
15. $AB = O \Rightarrow B$ 的列 $\subset$ $Ax = 0$ 的解空间
16. $\sum \lambda_i = \text{tr}(A)$ (迹)
17. $\prod \lambda_i = \|A\|$ (积)
18. $A$ 的不同 $\lambda$ 对应 $\xi$ 必无关
19. $A$ 可对角化 $\Leftrightarrow$ 有 $n$ 个无关 $\xi$
20. 实对称 $\Rightarrow$ 必可正交对角化
21. $A \sim B \Rightarrow$ 相同特征值 + 相同秩 + 相同迹
22. $A$ 与 $A^T$ 特征值相同
23. 任何二次型 $\exists$ 正交变换 $\to$ 标准形
24. 惯性定理: 正负项个数不变
25. $A$ 正定 $\Leftrightarrow$ 所有 $\lambda_i > 0 \Leftrightarrow$ 所有顺序主子式 > 0
26. $A$ 正定 $\Rightarrow$ $A^T, A^{-1}, A^*$ 也正定
27. $A, B$ 正定 $\Rightarrow$ $A + B$ 正定
28. $A^T A$ 半正定
29. $|A| = \prod \lambda_i$ (复数域)
30. 配方法: **先平方法后交叉法**
